让我们来了解2022年度工博会的参展商,第一章

中国意大利商会 2022-09-02




中国意大利商会

China-Italy Chamber of Commerce


通过这些每周发布的新闻稿，我们将更深入地了解2022年度中国国际工业博览会的52家意大利参展商



自2020年意大利国家馆在中国国际工业博览会成功举办之后，2021年由于新冠疫情中断，今年意大利国家馆将重返2022年中国国际工业博览会，意大利国家馆面积达1300平方米，近50家意大利企业参展，向中国公众集中展示意大利制造业的创新和先进技术及其新产品。


意大利国家馆由意大利对外贸易委员会和中国意大利商会主办，意大利驻华大使馆和意大利驻上海总领事馆提供支持。


随着2022年中国国际工业博览会的举行，第二届中国意大利机械工程周（WIMEC）也将再次拉开帷幕。其主要目标是彰显意大利在该领域的技术形象和领先地位。


意大利是机械工程领域的世界领导者，也是世界主要工业强国的技术和机器供应商。意大利对华出口总额的50%来自机械和工程技术。

通过周新闻，我们可以更加深入了解参加2022年度中国国际工业博览会的52家意大利参展商。


以下是第一组10家参展商的详细信息：



阿里斯顿集团



展台位置: 1.1 H - D002 - G9H8




阿里斯顿集团是国际供暖热水系统及服务供应商，总部设在意大利北部城市法布里亚诺。自1904年阿里斯顿首台燃气热水器在意大利问世以来，阿里斯顿热能集团以品质和安全为使命，将创新技术、节能、时尚设计完美结合，为国际用户提供从家用到商用的全系列供暖和热水产品，致力于为消费者提供舒适享受，并实现较低的能源消耗。

 

阿里斯顿集团在国际16个国家拥有25个自动化制造基地，在15个国家建有26个研发中心，在42个国家设有70个分支机构及6个代表处，销售遍及150多个国家，每天有上亿用户使用和体验阿里斯顿产品。集团三大事业部主要涵盖热能产品、燃烧器、系统零部件等。其中热能产品包括家用和商用燃气采暖热水炉，电热水器，燃气热水器，空气能热水器，太阳能系统，供暖系统等全系列热暖产品及服务。


网站: www.ariston.com.cn



凯斯帕液压（上海）有限公司



展台位置: 1.1 H - D001 - C4




凯斯帕成立于1952年，是一家意大利家族企业，专注于设计、制造和销售液压系统主要零部件，广泛应用于建筑工程，物料搬运，农业与草坪护理，工程车辆等行业领域。2007年，凯斯帕液压（上海）有限公司在上海浦东康桥工业园区成立并投产，工厂面积为9000平方米。我们为客户提供各种铝和铸铁齿轮泵，马达，分流器，变量柱塞泵，电子控制装置，过滤器。此外，我们还与客户密切合作共同开发定制化解决方案。如今，凯斯帕上海已经连续五年荣获卡特彼勒公司颁发的铂金供应商奖，并且是三一卡车起重机的顶级供应商之一。

 

“凯斯帕节能”系列的代表产品-电控液压风扇驱动是一款高性能的散热系统。相比于传统的风扇驱动系统，结合了液压和电控装置的风扇驱动系统具有优化格局，提高效益，改良冷却策略的优越性能，更具有节能减排的显著优势。即使在中国“环保风暴”的大背景下，凯斯帕电控液压风扇系统也能应对这些挑战，并助力中国工程机械行业进步发展。


网站: www.casappa.cn /  www.casappa.com



易器杰商务咨询（上海）有限公司



展台位置: 1.1 H - D002 - F2




ECM国际认证集团（Ente Certificazione Macchine）于1998年成立于意大利萨维亚诺素帕拉诺（Savignano sul Panaro）(MO)。ECM重视国际环境，通过在中国、韩国、日本、美国和南美开展业务，建立了紧密的商业合作网络，将业务推向了全球。2017年2月，ECM获得了卫生部认证，可根据医疗器械指令93/42/EC颁发CE认证。此外，它还有资格依据ISO 13485标准颁发质量管理体系认证。在2020年7月揭幕的ECM医疗设备新总部中，ECM作为欧洲医疗器械法规认证公告机构一员，来实施即将生效的《医疗器械法规EU 2017/725》。


网站: www.entecerma.it



法阿姆中国



展台位置: 1.1 H - D002 - E2




FAAM集团成立于1974年，总部位于意大利的一个风景优美的小镇，在意大利和中国都建有厂房，其生产的工业电池广泛应用于世界各地的电动叉车、电动汽车和 UPS系统、警报系统等。2006年FAAM集团总投资700万欧元，在江苏宜兴成立了法阿姆工业电池有限公司，公司使用欧洲先进的设备，使用的原材料也沿用欧洲的生产标准，部分关键零部件从欧洲进口。FAAM集团为保证其产品在中国的生产质量，一切都从严把关。


网站: www.faam.com.cn



法麦凯尼柯机械(上海)有限公司



展台位置: 1.1 H - D002 - E3




法麦凯尼柯机械有限公司成立于1975年，主要生产婴儿纸尿裤、妇女卫生巾生产设备等，至今不仅在人员和规模方面经历了持续性的增长（目前为止在全世界拥有三个生产厂分别位于意大利、美国和中国），而且在本行业中也不断开发数字化应用和机器人技术，从涉足第一台纸尿裤设备到拥有女性卫生巾线及失禁产品线等，这些都证明了我们在全球技术的领先地位。


作为意大利制造，法麦灵活运用技术手段解决创新方案并在世界上独领风骚。法麦现拥有200多名工程师、600多项专利并已完成交付多达1200台设备。法麦拥有一支高度专业化的技术队伍，能在传统行业保持高度竞争力的同时，还能不断接受市场变化带来的技术挑战，让法麦不仅成为大型跨国公司考虑合作的对象并且也是开发创新项目时的首选。


如今法麦作为一家面向快消行业的国际集团，公司秉承可持续性发展的目标，不断开发高效设备和技术、以及高附加值的数字化服务，以满足全球客户的竞争和创新需求。


网站: www.fameccanica.com



意世必(上海)轴承有限公司



展台位置: 1.1 H - D002 - G8H7




ISB是一家有着40多年丰富经验的机械领域的服务商。公司三个事业部致力于为客户提供高品质的轴承配套解决方案，主要产品涵盖了转动件（轴承、带座轴承、衬套及其它零部件等）,直线运动系列零部件(导轨及滚珠丝杠),以及动力传动件(工业皮带，连轴器等)。


网站: www.isb-industries.com



马波斯（上海）测量设备科技有限公司



展台位置: 1.1 H - D002 - G6H5




1952年Mario Possati先生在意大利博洛尼亚创立了马波斯。马波斯是世界领先的测量设备和技术供应商。马波斯致力于为工业领域提供高精度的测量及监控设备，产品从单个测量元件到“交钥匙”测量设备，可根据用户需求定制自动生产线，无损检测设备，泄漏检测系统，各种传感器，机床监控系统等。

 

目前马波斯在全球30多个国家地区建立了销售、服务网络，集团直属雇员超过3600名。 


网站: www.marposs.com



德建建筑设计咨询（上海）有限公司



展台位置: 1.1 H - D002 - E4




VIRTUARCH德建起源于瑞士，在建筑设计、室内设计、项目管理等领域为客户提供高质量的服务。目前，德建拥有一支多元文化团队，大约有100名经验丰富的建筑师、设计师、工程师和项目经理分布在瑞士,中国和泰国办公室。德建积累了近20年的项目交付经验，500多个项目均已成功在中国和其它亚洲国家建成，专业涉及领域包括：工业项目、研发项目、办公项目、教育类项目以及企业/商业项目等。


网站: www.virtuarch.net



伍尔特（上海）工业紧固件有限公司



展台位置: 1.1 H - D002 - G7H6




伍尔特在紧固件、化工品、工具和库存管理方面拥有75年的丰富经验。伍尔特集团的工业部门负责为工业客户提供C类零件和物流服务。伍尔特工业拥有超过100万种C类物料，凭借其独特的物流供应理念，成为该行业中强有力的合作伙伴。

 

伍尔特工业中国秉承德国工业服务模式，为中国的工业客户提供持续稳定的服务和产品质量，并提供工业4.0下定制化的物流解决方案，助力企业提高生产效率，让企业工厂更加精益。


网站: www.wuerth-industry.cn



杭州佐帕斯工业有限公司



展台位置: 1.1 H - D002 - G5H4




佐帕斯工业是全球知名的电热产品供应商，产品应用范围涵盖家用、商用、工业用市场。在电热元件和电加热系统的设计与制造领域，佐帕斯工业已拥有超过50年的丰富经验。佐帕斯工业高度全球化，总部设在意大利，在欧洲、北美、南美和亚洲均有研发和生产基地，以便为世界各地客户提供优质快捷的服务。


杭州佐帕斯工业有限公司成立于2000年，工厂占地面积54,000平方米，配有欧洲最先进的设备和技术，在管状加热器、单头加热棒、加热圈、加热线、裸丝加热器等产品的研发和制造方面具有雄厚的实力，为亚洲市场提供专业的加热解决方案。


网站: www.zoppasindustries.com
