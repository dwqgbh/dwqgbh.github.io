2026年度策略:确定性为王

南宫云遥录 2026-01-05

本文的构思始于2025年11月。过去一个多月，我几乎每天工作15小时以上，看过所有的内资与外资年度展望报告，策略会、路演，以及各路大V的观点。

我相信，等到2026年结束，回看此文时，它会是这一年最具实操价值、准确率、且免费公开的文章之一。

至于明年……或许不会再写，或许不能再写，又或许不再免费分享。所以，请珍惜此刻的坦诚。


2026年，我依然看好A股市场的中期走势，尤其是上半年——确定性相对较高。但进入下半年后，随着宏观节奏切换与政策边际效应减弱，不确定性增加，市场波动可能加大。

上证指数目标位：

  • 第一目标：4350–4450点
  • 第二目标：5000点（途中或有一次300点以上的深度调整）


基于“确定性”原则，重点看好以下六大方向：

一、新能源（储能、固态电池）


二、科技（国产替代、半导体设备）


三、金融（券商、保险）


四、军工（商业航天、军贸出海）


五、消费（旅游、食品饮料、医疗服务）


六、有色金属（金银铜）


一、新能源（储能、固态电池）


核心逻辑：需求强劲 + 供给紧缺 + 价格传导顺畅


当前储能行业呈现“供需两旺”格局：

  • 国内外需求共振，新型储能“一芯难求”；
  • 头部电池企业持续满产，电池价格延续上涨趋势；
  • 供应链具备强定价能力，景气度有望贯穿全年。

业内机构普遍预计，2026年全球储能市场增速接近50%，是少有的高增长+高确定性赛道。


固态电池方向与储能高度重合，可一并布局。


特别期待：宁德时代有望成为中国历史上首家净利润突破1000亿元的制造业公司——新能源时代的“英伟达”，未来A股市值最高的企业。


二、科技（国产替代、半导体设备）

核心逻辑：全面自主可控 + 产业资本加速投入


2025年科技板块已强势启动，2026年一季度仍可能领涨。但需警惕：随着估值抬升，泡沫成分或将显现，风险可能在下半年释放。


关键催化：

  • 长鑫存储、长江存储
    预计于2026年Q1–Q2上市，融资后将大规模扩产；
  • 存储产业链（设备、材料、封测）迎来历史性机遇。

2026，是真正的“国产元年”——从芯片到操作系统，从设备到EDA，国产化不再只是口号，而是订单与财报。


三、金融（券商+保险） 

核心逻辑：估值低位 + 业绩稳定 + 政策托底


当前券商和保险板块均处于历史估值底部，且业绩高增，看好中长期价值修复。

  • 保险有望复制银行走势；

  • 券商并非全年持有型板块，只在市场情绪高潮阶段爆发，更适合作为春季躁动或关键时点的进攻工具。


四、军工（商业航天、军贸出海）



核心逻辑：“十五五”规划启动 + 玄学逻辑（赤马红羊劫-火旺-必有大事发生）+ 建军百年冲刺（2027）


  • 上半年聚焦商业航天
    火箭发射商业化、卫星互联网组网加速、地面终端落地；
  • 下半年聚焦军贸出海
    我国军贸正从“性价比替代”迈向“技术标杆”，出口创历史新高。

重要节点：

  • 2026年8月1日：建军99周年
  • 2026年11月10日：第十六届珠海航展
  • 2027年8月1日：建军100周年

新质战斗力方向：先进战机、军工AI、军用机器人、精确打击、反无人机系统、水下攻防等。


五、消费（旅游、食品饮料、医疗服务）

核心逻辑：跌得够久、估值够低、预期够差，但基本面正在筑底


消费板块最大的优势不是弹性，而是安全边际。

  • 旅游率先修复；
  • 整体反转或在26年Q2之后，主升浪可能出现在下半年——作为牛市的尾巴结束。


六、有色金属 


核心逻辑：美联储降息 + 通胀预期 + AI/新能源需求 + 地缘风险


我依然相信《六金纪元》提出的大周期逻辑。但需注意：

  • 有色板块在2025年涨幅较大，2026年大概率转入结构性行情，难现全面牛市；
  • 重点关注铜（工业+AI）、银（光伏+电子）、金（美元信用+避险）、钨（军工+硬科技）。


七、择时畅想


提升收益率的关键：

择时 + 板块切换 + 高集中度

结语

以上策略完稿于2026年第一个交易日前，是对新一年市场的展望与推演。未来必然会有黑天鹅、灰犀牛、突发事件……所以，所有判断仅供参考。


但正因当下世界充满变数，我们才更需聚焦确定性。

2026年。想对自己说，也愿与各位共勉：


多做确定性的事情去对冲不确定性。

模糊的正确，胜过精确的错误。


后续，我会持续跟踪市场变化，尽量及时更新各板块的细分逻辑与择时信号，欢迎一路同行。


新年快乐，愿你我2026一路长红！
