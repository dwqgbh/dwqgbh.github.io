【2025年大湾区未来科技营】探秘湾期,洞见未来,感受中国最强的科技与经济圈!

熊宝亲旅 2025-03-16


RATIONALE

研学缘由


湾区，是带动全球经济发展的重要增长极和引领技术变革的领头羊，放眼全球，国际一流湾区主要包括：纽约湾区、旧金山湾区和东京湾区。目前，粤港澳大湾区已经成为世界的第二大湾区，她不仅是世界的经济中心，同时也是创新中心、金融中心、文化枢纽。



到粤港澳大湾区研学，有助于塑造孩子的未来视野、激发他们的创新思维、创业精神、开放进取的人生态度，强化他们“时间就是金钱，效率就是生命”的生活与学习观念。



 RESEARCH HIGHLIGHTS

研学亮点


RESEARCH  CONTENT

研学内容


 # 探秘国家基因库 #


永存即永生，深圳国家基因库，是全球领先的基因资源中心，以其超大规模的数据存储与样本保藏能力为特色，融合“存、读、写”遗传资源功能于一体。同时，它也是全球生物资源保护与利用的关键平台，展现了我国在基因科技领域的卓越实力与国际影响力。



认真的小科学家们在做奇妙的基因实验！



荣获由国家基因库颁发的研学证书！



 # 世界最大水陆两栖飞机制造厂 #


AG600是在研的世界最大水陆两栖飞机，中航体系内仅有的开放参观的大飞机生产线，厂房里更有众多在研及已量产的尖端机型。



给你一个机会走光荣的“主席路”!零距离和飞机接触,参观两种飞机的组装现场:全世界在研最大的水陆两栖飞机——AG600,西锐公司设计制造的小型私人飞机—一西锐SR20。




独立操作飞行模拟器，了解飞机各个操作面的控制原理，包括升降舵，副翼，襟翼，方向舵。识读传统仪表和综合航电系统，驾驶初级教练机进行五边飞行训练。





 # 亚洲领先的国际名校——香港大学 #




香港大学现位居QS亚洲大学排名第2位，全球大学排名第21位，被称为“亚洲常青藤”，孙中山、朱光潜和张爱玲等名人都是从港大毕业。


对于有留学规划的孩子来说，可以提前感受先进的国际教育氛围，明白在名校如何享受获得知识的乐趣，从而培养学习兴趣。



大家由港大学霸带领，一起走进香港大学，了解港大的文化底蕴，感受学术氛围和教学风格；观览校园，知晓校内历史建筑、名人典故。



本部大楼以英国“红砖大学”做参考，使用清水红砖这样朴素的建筑材料，反映港大是踏实、开放民主的教育机构，也是校园历史最悠久的建筑。



墙面的学术活动海报，这样的设计与张贴在内地并不多见，更有活力、鼓励平等讨论、学生自发的感受。




 # 香港金管局 #


香港金融管理局（金管局）以高效监管、稳健政策与国际化视野著称，推动香港作为国际金融中心的地位。



营员们了解香港货币及其发行管理的相关知识。




 # 金巴穿越港珠澳大桥 #


港珠澳大桥是连接香港、珠海、澳门的桥隧工程、因其超大的建筑规模、空前的施工难度和顶尖的建造技术而闻名世界，是世界上最大的跨海大桥，被称为“新世界七大人工奇迹之一”。


大巴穿越港珠澳大桥，近距离看看历时9年，花费1269个亿打造的世界最长跨海大桥，从上桥到海底隧道，全程震撼！



 # 澳门大学 #


澳门大学，是澳门唯一一所综合性公立大学，在亚洲乃至全世界的高等学府中享有极高的声誉，是澳门教育现代化的重要标志之一。



澳大学霸带队参访，孩子们除了欣赏优美的校园风光之余，更重要的是，了解不同文化熏陶下的校园文化，深刻感受高等学府的魅力所在。




住宿式书院是澳大特色，和牛津大学的书院管理如出一辙，澳门大学也是亚洲第一所书院制大学。孩子们步行参观书院区域及教学区，感受澳门大学的课堂氛围和学习方式，打开对大学课程的好奇大门。





 # 铭记历史-澳门回归贺礼陈列馆 #



“你可知Macau不是我真姓？”澳门经历了400多年的颠沛流离，1999年终于回归祖国母亲的怀抱，如今这朵莲花正骄傲的绽放着。



人们设立专门的回归陈列馆作为青少年教育基地，这里有每个省赠送的、颇有特色的欢迎礼，编钟、玉雕、瓷器......琳琅满目。




 # 游轮出海—珠澳海湾夜游 #


乘游轮夜游，领略港珠澳大桥无敌夜景、珠澳城市海岸线的璀璨灯光。


港珠澳大桥宛如一条发着光的海上巨龙，蜿蜒游向大海深处。游轮从大桥底下经过时，瞬间便能感受到它的宏伟和壮观。


美好的水上观光之旅，丰富的表演必不可少，珠澳海湾夜游航线会在一楼船舱带来精彩的节目表演。



 # 直升机飞行体验 #


素有百岛之城美誉的珠海，你有在空中看过吗？孩子们零距离接触直升机，跟飞行员一起飞上蓝天，解锁珠海新玩法。



掌握了飞行安全与基础航空知识培训后，专业飞行员将在停机坪起降台发动引擎，由机场运营管理团队的安全保障下，学员们坐上直升机开始天空之旅！


一名飞行员带着三名学员翱翔起飞，保证每个孩子都享有广阔的高空视野（飞行时间大约5分钟，机上无老师跟随，无法拍摄飞行时照片）



换个角度看世界，看百岛之城的海岸线，俯瞰蜿蜒如长龙的港珠澳大桥。深刻体会国家逢山开路，遇水架桥的奋斗精神，感受大国工匠精神，感受永创世界一流的民族精神。



 # 赶海，探秘有趣的潮间带生物 #


每一个孩子都应该有快乐的赶海童年 

去感受海的广袤与博大

去发现海岸线上的生命奇迹




在海岸潮间带上，你总能发现一些奇特的生物，蛤、蚶、螺、贝、珊瑚、寄居蟹、小鱼、小虾、小螃蟹，还有时不时漂来的奇形怪状的藻类，总能让人惊喜连连、眼界大开。








喜欢在沙滩上打洞的沙蟹，有着竖起来的眼睛，是行走速度最快的一类蟹啦~



几乎和环境融为一体的野生短桨蟹。



我在哪？

形状各异、花纹各异的各种贝类。



可以看到被称为海葡萄的长茎葡萄厥藻。



不止是“葡萄”你甚至还能在海里赶到“香蕉”。




 # 大潮起珠江—改革开放40周年历程 #


回顾和展示广东改革开放历程，了解中国改革开放史。



展览以时间节点为线索，通过“伟大转折”、“春天的故事”、“小康路上”等主题，用经典力作来展现自1978年改革开放以来中国发展的历史故事。



“时间就是金钱，效率就是生命”提升营员的时间效率观念！
