大湾区（珠海）国际汽车文化节
0
1


举办时间
2025年3月15日—16日
举办地点
珠海国际航展中心
展会介绍

大湾区（珠海）国际汽车文化节是粤港澳大湾区知名汽车文化赛事活动，已连续在航展中心举办了7年，是集汽车动态表演、改装静态展示及沉浸式互动体验于一体的汽车文娱活动新场景。本届将包含WOFA直线加速赛（约100—150台车）、越野大会（超100位专业选手与专业改装车辆展示竞速、漂移、冲坡、涉水等）、WOFA SHOW潮流车展（超500台车参与姿态、性能、内饰等五大主题展示）、CA Classic银石经典车展（经典车、老爷车等）、数字星球挑战赛（数字孪生山路漂移赛道）等。

2025珠海国际海洋智能科技展


举办时间
2025年4月25日—27日
举办地点
珠海国际航展中心
展会介绍

展会以“智汇海洋，科创未来”为主题，搭建开放合作、共赢共享的国际海洋交流平台，打造具有上下游产业链、智能科技应用、产学联动的海洋特色综合性展会，汇聚海洋智能制造、无人装备、海洋新能源、智慧船舶、海洋防腐及新材料、海洋牧场、游艇等海洋领域企业参展，同期将举办会议论坛活动约20场。

珠海AS动漫展
0
3


举办时间
2025年5月3日、8月2日、10月4日
举办地点
珠海国际会展中心
展会介绍

珠海AS动漫展作为珠海二次元盛典，是“动漫游戏”爱好者的聚集地。新潮的文化活动，化身聚集人气的新火炬，点燃着年轻人的热情，以动漫游戏展创造新体验。举办期间有盛装自由行、摄影自由行、宅舞比赛、随机舞蹈、嘉宾签售、签绘墙长廊、闯关游戏、无料专区、动漫周边大汇聚等一系列活动。活动期间，将承载每一位cosplayer的高光时刻。

2025粤港澳学生信息科技创新大赛


举办时间
2025年5月下旬（待定）
举办地点
珠海国际航展中心
展会介绍

粤港澳学生信息科技创新大赛已连续在珠海成功举办两届，省教育厅决定至2030年之前的大赛活动均选址航展中心举办。该赛事的活动对象为广东、香港和澳门的在读全日制中小学（含中职）学生、广东省内的高校教育类专业师范生，活动共设置五大类27个项目，含创意智造和人工智能应用专项等科创智造类项目6个、智能机器人类项目8个、数字创作类项目7个、计算思维类项目3个，科创教育成果展示类项目3个。

第八届华南CIO大会暨信息技术交易会
0
5


举办时间
2025年6月（待定）
举办地点
珠海国际会展中心
展会介绍

华南CIO大会暨信息技术交易会（S-CIO）是一个专注于信息技术领域的重要会议，旨在推动数字化转型和信息技术的广泛应用。自2017年以来，该大会已连续成功举办多届，成为华南地区CIO及相关专业人士的重要聚会平台。华南CIO大会的主要目的是集结产业最新信息动能，吸引国内外企业信息化管理者、专家学者、企业主、软硬件服务商等，共同探讨和分享数字化转型的最佳实践和经验。通过这一平台，大会致力于帮助企业实现数智化变革，深入打通经济发展的信息大动脉。

2025珠海国际工业博览会


举办时间
2025年9月11日—13日
举办地点
珠海国际会展中心
展会介绍

为发展新质生产力、构建现代化产业体系、推动高质量发展、提高产业创新能力和竞争力、助推工业经济高质量发展，由广东亚联展览股份有限公司主办的“2025 珠海国际工业博览会”将于2025年9月11日—13日在珠海国际会展中心举办。本届博览会将集中展示智能制造全产业链最先进的产品、技术及解决方案，同期还将举办大型主题论坛、讲座、供需对接会等活动，吸引装备制造业、新能源行业学术专家，企业家，工程师等高端人才参与，将打造一场集展示、交流、采购于一体的大型工业盛会。

第十九届珠海国际办公设备及耗材展览会
0
7


举办时间
2025年10月16日—18日
举办地点
珠海国际会展中心
展会介绍

珠海国际办公设备及耗材展览会（RemaxWorld Expo）自2007年首次在世界“打印耗材之都”——珠海面世后，至今已成功举办18届。得益于展会举办地珠海的产业地理优势，以及主办方再生时代的不断努力，该展会已成为全球办公设备及耗材产业极具权威性和影响力的展览会之一。截至2023年，展会已累计接待服务超过21万名办公行业人士，为推动全球办公设备及耗材产业交流合作，以及行业的繁荣发展做出了重要贡献。

第四届粤港澳大湾区服务贸易大会


举办时间
2025年10月或11月（待定）
举办地点
珠海国际会展中心
展会介绍

粤港澳大湾区服贸会是粤港澳大湾区首个以服务贸易为主题，融展览展示、行业对话、商务交流于一体的高规格、综合性展会。往届展会开创了“一会三地”新模式，在珠海设主会场，在香港、澳门设分会场。第四届大会将深入贯彻落实广东省委十三届五次全会精神和省委“1310”具体部署，打好外贸、外资、外包、外经、外智“五外联动”组合拳，以“促成交、促融合、促跨境服务提升”为目标，通过专业展览展示、论坛、B2B商务洽谈、项目路演等形式展示大湾区服务贸易发展优势，全方位搭建高效精准的对接交流和行业交易平台，推动大湾区服务贸易高质量发展。

2025第八届全球无人系统大会、
2025珠海国际低空经济博览会/
2025国际无人系统新技术成果展





0
9


举办时间
2025年11月1日—3日
举办地点
珠海国际会展中心
展会介绍

为贯彻落实中央经济工作会议和党的二十届三中全会精神，加速推进低空经济快速发展，“2025年第八届全球无人系统大会(GUSC2025)”暨“2025珠海国际低空经济博览会、国际无人系统新技术成果展”将于2025年11月1日—3日在珠海国际会展中心举行。全球无人系统大会自2017年创立以来，已成功举办七届。作为专注低空经济与无人系统创新发展的高端展会，大会汇聚了来自全球30多个地区的2000多位行业专家、学者和企业代表，吸引了500多家优质企业参展，覆盖低空交通、无人系统、智能制造、技术研发与应用等多个板块。大会打造了集产业交流、技术展示、商务合作于一体的专业平台，推动跨区域协同创新，为低空经济与无人系统产业的未来发展奠定了坚实基础。

2025亚洲通用航空展


举办时间
2025年11月6日—9日
举办地点
珠海国际航展中心
展会介绍

展会以“通航新纪元、探索低空无限可能”为主题，突出低空经济领域展示内容，尤其是电动航空、eVTOL等方面优势科技，聚焦“无人化”“智能化”“低碳化”等新兴主题，覆盖全通航产业链，打造市场化的通航展。展会划分五大展区（固定翼航空器展区、旋翼类航空器展区、无人机及低空经济应用场景展区、航空动力和绿色新能源展区、航电/零部件/数字化低空管理/运营维修和人工智能展区），并规划需求对接区、航空金融主题展区、初创展区及省市展区，举办各类投融资对接、企业路演、职业需求对接、飞行表演、飞机性能展示等，会议论坛活动共约40场。

第七届中国教育创新成果公益博览会
1
1


举办时间
2025年11月12日—18日
举办地点
珠海国际会展中心
展会介绍

中国教育创新成果公益博览会由北京师范大学于2015年首倡发起，至今已成功举办六届。该展会以“共建共享教育创新成果，汇聚中国教育创新智慧，服务中国教育创新实践，发出中国教育创新声音，提供中国教育创新方案，影响世界教育创新趋势”为宗旨，引导社会各界共同参与教育创新，在全社会形成了广泛而积极的影响。教博会已成为国内唯一一个由高校主办、指向教育内涵发展、服务中小学实践、最具影响力的专业盛会。
