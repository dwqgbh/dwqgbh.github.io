官方通知核心信息



（1）赛事全称：第19届全国三维数字化创新设计大赛（Digital Design Dimensions Show, 3DDS）

（2）主题：Ai3D 数实融合 全民共创

（3）性质：教育部《全国普通高校大学生竞赛排行榜》入库赛事

（4）官网：https://3dds.3ddl.net

（5）作品展示平台：3DShow https://3dshow.3ddl.net

（6）参赛对象：研究生组/本科生组/高职高专组等，每队学生2-5人 + 指导教师1-2人

（7）报名方式：官网注册→队长建队→邀请队员→审核通过→提交作品

（8）2026年关键时间节点（官方参考）

⚠️ 各高校校赛截止日、专项赛时间略有不同，以本校通知和官网赛项细则为准。



赛项分类



（1）开放赛项（年度赛，自主命题）

（2）产业应用/命题专项赛（动态发布）

①"AMD杯"AI+3D创新设计大赛：须使用AI工具（ChatGPT/DeepSeek/MJ/SD等），体现AI创作过程

②CURC机器人+创新设计大赛（格斗/灌篮机器人方案设计）

③增减材复合制造及数字孪生技术创新应用大赛（数马&华教杯）

④昆仑谣·数字文旅创作大赛 / AIGC多模态大赛 / 数智空天 / 低空无人机等



作品提交要求（官方规范）



（1）作品须使用 3DShow组织并提交，一般包括：

（2）3D数字模型：结构合理、PBR材质，格式建议 .fbx/.step/.gltf，≤50MB

（3）设计说明文档（PDF）：创意来源→设计思路→技术应用路线→创新点→技术难点→仿真/验证→应用前景

（4）展示图片：分辨率 ≥1920×1080，含效果图、三视图、爆炸图

（5）演示视频：≥60秒，1080P，含设计过程/花絮（AI专项赛强调AI工具使用过程展示）

（6）匿名要求：3DShow作品及说明书中不得出现校名、校徽、队名、成员姓名，违者初审不通过

AMD杯额外要求：须附AI工具使用痕迹（提示词截图/AI生成过程说明）



评审标准（100分制参考）





备赛全流程攻略



（1）组队与分工（推荐3人黄金配置）

①建模手（核心）：机械→SolidWorks/UGNX/CREO；设计→Blender/Rhino；负责主体建模、装配、工程校验

②渲染/美工：KeyShot/Enscape/V-Ray出图，剪辑演示视频，排版3DShow

③文案/答辩：撰写说明书、提炼创新点、准备PPT、负责答辩演练

（2）选题策略——"小而美，重落地"

❌ 避开"智慧城市""工业4.0"等宏大空泛题

✅ 选生活化/校园痛点/非遗文创/小型机械装置，功能合理+可制造+有文化/社会价值

开放赛项鼓励结合地方特色（地方非遗、校园设施改良）

（3）建模与工程细节

①零件无干涉、装配约束正确、重要尺寸标注

②工业类建议做简单受力/运动仿真或有限元分析（ANSYS/SolidWorks Simulation）——省赛→国赛加分项

③文化/人居类重视UV展开、PBR贴图、场景氛围

（4）文档与展示打磨

①说明书结构：背景→设计思路→结构原理→创新点→仿真/验证→应用前景

②3DShow排版：首屏放精华渲染图+slogan，中间放爆炸图/原理说明，末页放AI使用说明（专项赛必填）

③演示视频分镜：问题引入→方案展示→模型旋转→细节特写→总结

（5）答辩准备（省/国赛）

①3分钟讲清：做什么→为什么做→怎么做的（创新点）→效果如何

②预判问题：结构合理性？加工工艺？成本？为何选此方案？

携带源文件、关键截图备用



常见失分雷区





现阶段行动清单（5月下旬）



✅确定赛项方向（开放or专项）+ 官网注册组队报名（6月30日前完成报名）

✅敲定选题，做草图/概念验证（一周内）

✅ 开始主体建模+初步渲染，6月中下旬完成初版

✅ 同步准备设计说明书框架和AI工具使用记录（报专项赛必做）

✅7月10日前按提交要求上传3DShow自审，参加校赛选拔（7月20日截止提交）



按获奖友好度排序

（省赛/国赛综合）



💡结论：普通本科/高职想稳拿省奖→专项赛或数字文化设计；有扎实工科背景（能做CAE仿真、运动仿真）→可冲数字工业。


哪些作品特征更容易拿高分

（评委偏好）



根据评审标准和近年国奖作品复盘，高胜率作品共性：

（1）紧扣"数实融合/新质生产力"

①工业类：带ANSYS受力分析/运动仿真/拓扑优化+3D打印实物验证，而不只是外观建模

②文化类：非遗元素提取→参数化重构→可3D打印/可交互展示，而非简单贴图

③专项赛（AMD杯）：AI工具贯穿全流程（提示词记录、AI辅助生成→手动精修→对比说明），缺一不可

（2）"小而美+解决真问题"

①省赛国奖常出自：助老/助残辅助器械、校园设施改良、地方非遗数字化复原、小型特种机器人——具体场景+完整验证

②避开空洞的"智慧城市""工业4.0"宏大叙事

（3）技术深度+视觉表现双达标

①PBR材质、UV展开正确、无破面穿模

②工业组做爆炸图+装配动画+仿真截图；文化组做XR交互或360°展示

③3DShow云展板排版清晰，首屏一句Slogan点明创新点

（4）文档逻辑闭环

背景痛点→设计方案→技术路线（附仿真/AI使用说明）→创新点提炼→应用前景，数据/截图佐证

（5） 容易被刷的特征

①纯网上下载模型改色/换贴图 → 创意分为零

②只有漂亮渲染图但装配有干涉、无尺寸标注（工业组致命）

③AI专项赛未展示AI使用过程/提示词

④3DShow出现校名队名 → 初审打回



结合你现在的时间建议

（5月下旬）



（1）跨专业/艺术类背景 → 报数字文化设计（非遗+AIGC）或AMD杯AI+3D专项，重点打磨渲染+XR交互+AI过程记录

（2）机械/车辆/自动化背景→ 若学校往年有国奖传统可冲数字工业（务必加仿真+3D打印样机）；否则也可走CURC机器人+专项赛降维竞争

（3）建筑/环艺背景 → 数字人居+BIM数据或转数字文化/元宇宙虚拟展馆

（4）计算机/数媒背景 → 元宇宙创建（Unity/UE5数字人/虚拟展馆）或AI+3D专项
