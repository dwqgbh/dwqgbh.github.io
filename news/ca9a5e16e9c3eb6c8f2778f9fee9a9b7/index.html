2025年库卡工业机器人培训报名开始啦!

天天智能制造培训中心 2025-02-20

天天智能制造培训中心

库卡工业机器人培训

培训目标

会使用示教器、操作面板等人机交互设备及相关机械工具，对工业机器人、工业机器人工作站或系统进行装配、编程、调试、工艺参数更改、工装夹具更换及其他辅助作业的人员。


课程时间

上课日期：3月2、9、16、23日

上课时间：9：00-17:30

培训费：1500元（4天）

上课地址：

江苏省苏州市虎丘区金枫路151号

苏州天天智培培训有限公司

课程内容

一、机械系统装调部分


1. 工具和工装夹具机械零部件装配图和装配工艺文件的识读方法

2. 常用机械装配工具的使用方法

3. 末端执行器自动更换系统装配方法

4. 传动系统元件使用方法及原理图识图方法 

二、电气系统装调部分


1. 电气图识图方法

2. 电气装配工具的使用方法

3. 电气线路连接规范及要求

4. 工业机器人电气系统调试方法

三、工业机器人系统操作与编程调试部分


1、工业机器人安全

1. 工业机器人安全标识

2. 工业机器人安全操作注意事项

3. 工业机器人及周边配套设备安全操作方法

4. 工业机器人安全运行机制

2、工业机器人系统操作与设定

1. 工业机器人组成及连接操作方法

2. 工业机器人示教器的操作方法

3. 工业机器人运行模式设定

4. 工业机器人坐标系设定方法及其使用技巧

5. 工业机器人零位校准方法

6. 工业机器人输入/输出信号设定方法

3、示教编程与调试

1. 工业机器人程序点示教方法

2. 工业机器人运动指令的编辑及使用方法

3. 工业机器人逻辑指令的编辑及使用方法

4. 工业机器人轨迹编程方法及技巧

5. 工业机器人位置数据的读取与设置方法

6. 工业机器人系统文件、程序文件等的备份和恢复/加载方法

智能制造职场路，天天助你展宏图


   

    天天智能制造培训中心 (简称: 天天智培) 成立于2006年立足苏州，服务全国，专业为电气自动化领域培养技术人才目前学员数量累计超过2万人。从2022年起，天天智培以“毕业即上岗”“高质量对口就业”作为培养目标，打破固有的学习模式，集中力量，开发了一系列组合课程，以电工基础、三菱/西门子PLC、工业机器人、项目实训为基础，在教学形式和内容上做了全面升级，除了学习行业知识以外，更注重培养学员自学的方法和举一反三的能力，使学员实现自动化领域中“从0到1”的华丽转身!

    此外，天天智培为学员提供更优质的企业招聘对口岗位，由专业老师为学员从入学开始进行跟踪指导，保证学员在学习过程中管理能力、总结能力、表达能力都得到全方位的提升，也确保了学员在毕业后能够顺利从事电气自动化编程调试或设备维修维护岗位。目前，项目制包就业培训已毕业的学员主要服务于新能源、汽车、光伏、电子等领域的自动化或制造型企业。

    同时，天天智培也致力于为智能制造行业提供专业的人才输出平台，目前已与国内多家智能制造自动化企业达成战略合作，其中有外资企业、上市公司、国有企业、民营企业等，天天智培已经成为企业首选的电气工程师招聘和培养的平台!
