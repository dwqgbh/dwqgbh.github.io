展商推介 | 海克斯康整装待发,闪耀2025工博会:以精度筑基,用数据赋能,共创智造新生态!

中国工博会 2025-09-14







6.1H-B001


海克斯康制造智能技术（青岛）有限公司

企业简介

备受全球工业界瞩目的2025年中国国际工业博览会即将拉开帷幕。作为数字化信息技术解决方案的全球领导者，海克斯康已蓄势待发，将以强大阵容深度参与这一年度盛会。我们不仅积极响应国家“智能制造主攻方向”与“数据要素×新型工业化”的战略号召，更将携贯穿“精度铸基、数据协同、生态互联”的完整技术演进故事线与最新成果，向业界展示赋能重点产业“智改数转”的坚实路径与创新范式。

海克斯康此次参展紧扣国家“智能制造”与“数据要素×工业化”战略，清晰演绎技术赋能产业升级的路径：以海克斯康经典超高精度计量设备为精度铸基；通过AI+半导体制造、核心工业软件、逆向工程 等方案实现“数据协同”，深度激活工业数据价值；并呈现人形机器人精密零部件测量解决方案、PRESTO自动检测系统、无线3D扫描解决方案等“生态互联” 创新应用，展示自动化、智能化互联的未来工厂图景，系统性呈现从根基夯实到生态构建的智造范式。

工博会不仅是展示实力的舞台，更是连接产业、共谋发展的枢纽。海克斯康已组建强大的技术专家与最新产品，满怀诚意期待您莅临6.1H-B001海克斯康展台，与海克斯康一起，以精度筑基，用数据赋能，携手迈向智能制造与新型工业化的璀璨未来！


💫企业成就：

海克斯康是全球领先的数字现实技术解决方案供应商，是胡润世界500强、财富中国科技50强、全球科技公司市值百强、工业互联网500强，致力于在“智能制造”和“智慧城市”领域推动技术变革和数字化转型。海克斯康的技术广泛应用于飞机、汽车、电子设备等28个行业，持续以全球本地化技术助力爱达魔都号、海上巨型风机、新能源汽车、国产大飞机、人形机器人等中国超级工程与新兴产业发展。未来，海克斯康将聚焦Glocal战略、生态共建平台战略、以质量为核心的智能制造、可持续发展战略，把全球突破性创新技术融入中国的发展脉搏之中，持续赋能中国经济实现高质量发展。


产品介绍

人形机器人精密零部件检测方案


基于 Reference 15.9.7超高精度坐标测量平台、Senmation多传感器全自动切换系统及HP-O光纤非接触测头三大核心模块，构建面向微型传动部件的全闭环质量控制体系。该方案攻克了人形机器人关节模组中谐波减速器柔轮、RV摆线轮、行星滚柱丝杠等超精密零件的检测瓶颈：


✨主要创新点1：Senmation全自动测头更换系统是行业首创，目前为止，也是唯一一个可以全自动自动更换测头传感器的更换系统，可解决旋转测头、固定测头、白光测头、以及粗糙度等多传感器自动更换，可以兼容多个应用场景，并提升测量效率。


✨创新点2：HP-O光纤测头，有多种发光角度，光学测头最小测针直径3mm，可测量最小直径5mm的内孔尺寸，目前为行业首创。可以很好的应用在人形机器人的微小齿轮测量应用上。



无线3D扫描系统


CereScan SE 智能灵动全无线手持3D扫描仪，以全无线轻量化设计重塑手持扫描体验，凭借更自由、更精准、更全能的革新架构，再拓便携式3D扫描效能边界，以更强的场景适应性为用户提供专业级高性价比解决方案。


🛠关键参数：

净重540g / 尺寸215×95×45mm / 精度0.02mm / 速率4,000,000次测量/秒 / 孔位闪测 


🛠产品亮点：

  • 全无线集成式设计赋予设备灵锐无限，机身净重仅540g，可轻畅无阻地突破各类深窄特征和内部盲区；

  • 可拆卸式无线底座实现无线/有线自由切换，110帧高速数据传输，无线/有线同性能；

  • 升级激光配置，激光线增至52束，更密集的激光矩阵，大幅提升细节捕获能力；

  • 除集成摄影测量外，另创新搭载孔位闪测功能，显著提升设备实用性与性价比。



HyperScan SE 智能灵动全无线光学追踪3D扫描仪，基于光学追踪技术无需贴点、高精度动态测量的优势，以全域无线化、轻量化革新为核心，重新定义3D扫描的灵活性与实用性。搭配全新Innov3DStudio计量软件，打造智能、灵动、简便的“零门槛”计量生态。


🛠关键参数：

扫描仪净重1.3kg / 精度0.03mm / 速率4,000,000次测量/秒 / 孔位闪测 / 边界扫描


🛠产品亮点：

  • 扫描仪、跟踪器双端全无线设计，轻松应对无电源环境/狭小空间，场景适应力倍增；

  • 扫描仪、跟踪器均搭载高性能计算模组，「双」算力赋能，效能大跃升；

  • 通过升级激光配置、优化靶球设计，实现多维度精度提升，筑牢计量根基；

  • 全新设计迭代，融合「航空级」人机工学，轻巧省力，盈握之间掌控自如；

  • 承袭HyperScan系列语音控制功能，简化操作流程，复杂高难度场景中也可轻松作业。



PRESTO自动化测量系统


专为实现大中型零部件的自动化测量而开发，可广泛应用于汽车大型结构件的快速检测。它将先进的光学测量技术与高效率的机器人测量技术相结合，能快速而精确地完成多种零部件的三维数字化及尺寸测量任务。

PRESTO还是一套面向未来的测量系统，具有扩展性高、无缝集成、适应性强等特点，可满足现代制造业不断变化的需求，帮助客户有效减少测量时间、简化工作流程。
