【会展业日报】2025.04.03星期四

行业风向标Daily 2025-04-03

今日要闻

  1. 中国会展经济研究会发布《2024年度中国展览数据统计报告1.0》，全面展示会展行业发展现状和趋势。报告显示，2024年全国共举办线下展览8916场，展览总面积为14678.66万平方米，较上年展览总数增加1113个、展览总面积增加164.66万平方米，增幅分别为14.26%和1.13%。境外自主办展数量攀升，增幅达13.68%，为行业高质量发展注入新动能。

  2. 第三届链博会广东推介会在广州成功举办，中国贸促会会长任鸿斌、广东省副省长王胜共同出席。会上一批广东企业与中展集团签约参展第三届链博会，并公布了广东第一批专业观众和采购商名单。企业表示，链博会是参与国际产业分工、整合资源的宝贵机会，将充分展示自身优势，与产业链上下游伙伴积极开展合作对话。

  3. 美国总统特朗普宣布了全球瞩目的"对等关税"计划，对中国加征34%关税，对欧盟、越南等贸易伙伴征收20%到49%不等的关税，对任何贸易伙伴的最低对等税率为10%。这项近百年来对全球经贸格局冲击最强烈、覆盖面最广泛的关税举措，预计将重塑美国和全球经济，推高通胀并拖累经济增长。特朗普公布的关税方案整体高于投资者预期，全球市场震荡加剧，将对国际贸易秩序产生深远影响。

会展活动与项目

1. 多玛凯拔亮相2025年上海国际酒店展

多玛凯拔携客制化安防及门控解决方案亮相上海国际酒店工程设计与用品展览会，展示包括Quantum Pixel+酒店锁、定制五金及客房用品、空间优化解决方案和酒店特殊客房设施等产品。凭借卓越产品品质和行业贡献**荣膺"行业特别贡献奖"**，其创新产品获得众多酒店管理集团关注。

2. 瑞联电气亮相2025年汉诺威工业博览会

瑞联电气跟随百利装备集团亮相德国汉诺威工业博览会，展示新一代智能配电解决方案及绿色能源技术。作为"有电的地方就有瑞联"的企业代表，积极响应集团"走出去"国际化战略，对标国际先进水平，聚焦自身优势，打造新质生产力，为集团建设世界一流装备制造企业贡献力量。

3. 希森美克亮相2025中国国际清洁能源博览会

广州希森美克新材料科技股份有限公司以新材料创新解决方案领军者之姿，携前沿纳米科技亮相北京国家会议中心，展出覆盖光伏、风电、储能等领域的特种功能涂层材料及系统解决方案。其产品已在16个省份及全球多个城市应用，发电效率提升均超预期，成为支撑"双碳"战略的新质生产力标杆。

4. 荣程新能200kW氢能重卡亮相2025年HEIE国际氢能展

荣程新能与吉利远程联合研发的200kW氢能重卡精彩亮相国际氢能展，是国内首批实现大批量上装的200kW氢燃料电池重卡。该车型基于远程星瀚G新能源重卡平台打造，搭载荣程氢扬动力燃料电池系统，具备高效能、长续航与轻量化优势，最大续航里程突破700公里，为氢燃料电池商用车规模化推广提供标杆案例。

5. 台湾展团首次亮相2025大湾区国际橡塑技术展

台湾橡胶暨弹性体工业同业公会首次携台湾企业参与大湾区橡塑展，展出橡硅胶加硫机、射出成型机、密炼机等机械设备及轮胎垫布、氧化锌、促进剂等橡胶原物料。参展企业包括威福兴业、玮祥机械、东毓油压等台湾知名企业，期待创造两岸橡塑行业合作新契机，共同探索低碳环保发展路径。

高层会议与论坛

1. 无锡经开区会展支持政策引发关注

无锡经开区推出会展支持政策，引发业内广泛讨论，关注焦点集中在政策兑现的可靠性和实施效果上。业内人士就"政策是否能如期兑现"、"兑现过程是否会打折扣"等问题展开热议。该政策旨在吸引更多高质量会展项目落地，促进当地会展经济发展，但政策落实的实际效果仍有待观察。

2. 2025国际电子电路（上海）展览会数据简报出炉

2025国际电子电路（上海）展览会发布数据简报，多项指标刷新纪录，再次体现行业领先地位。展会通过"会+展+X"的持续创新模式，聚焦印制电路板、半导体、封装基板等领域最新产品及科技成果，深度探索电子电路产业在新能源、汽车电子等领域的应用。展会激活产业创新力，推动整个产业链升级发展，为全球电子电路及半导体行业发展"精准把脉"。

3. 中国馆将成为大阪世博会靓丽的"中国名片"

中国贸促会举办专题新闻发布会介绍大阪世博会中国馆筹备情况。中国馆是大阪世博园第一个开工建设的大型外国自建馆，也是目前唯一通过各项许可验收的外国大型自建馆。占地约3500平方米的中国馆将围绕"共同构建人与自然生命共同体"主题，以"天人合一""绿水青山""生生不息"三大篇章为叙事主线，展示中华文明孕育的传统生态智慧和新时代绿色发展成就。

4. "投资中国"连续八年亮相汉诺威工博会

"投资中国"展区连续八年亮相汉诺威工业博览会，持续向全球企业展示中国优化的营商环境和广阔投资机遇。本次展区由中国商务部投资促进事务局主办、中国国际投资促进中心（德国）承办，期间举办"投资中国"系列专场活动，包括主题日展示、重点产业集群交流、企业路演等。商务部投资促进事务局副局长于广生强调，中国开放的大门不会关闭，只会越开越大，为全球企业提供更多确定性。

5. TSCI杭州数码印花工业应用展海外预登记大幅增长

TSCI杭州数码印花工业应用展海外预登记同比绍兴展暴增100%，多国免签政策助推国际观众数量创历史新高。截至3月25日，海外预登记人数已突破3500人，整体曝光超过10万。展会吸引了来自全球93个国家和地区的专业观众，预登记观众主要集中在高采购力市场。展会同期将举办国际纺织数码印花峰会等多场高端论坛，汇聚国际纺织大咖，共同探讨全球数码印花趋势。

科技创新与数字化

1. 小米回应网传SU7事故公开信：官方未发布

针对网传"雷军就SU7事故发布致用户和公众的一封信"，小米公司相关负责人明确表示："如果我们要披露什么，一定是通过官方账号发"。该网传"公开信"包含"此刻的承诺"、"反思与行动"等内容，但经查证并非官方发布。此前，小米已就"安徽小米SU7高速碰撞爆燃"事故做出回应，并表示将配合警方调查，回应家属和社会关心的问题，体现了企业在危机事件中的信息发布管理。

2. DeepSeek招聘财务高管引发融资猜测

DeepSeek近期放出多个财务相关岗位招聘信息，包括CFO、COO、财务副总裁等，引发市场对其可能开放融资的猜测。此前DeepSeek一直未进行外部融资，资金主要来自幻方量化。据报道，DeepSeek创始人梁文锋持有超80%股份，个人财富达330亿元人民币。业内预计，如果DeepSeek开放融资，估值可能超过100亿，而投资者对这一AI领域明星企业的热情高涨，将对整个AI投资生态产生重大影响。

3. Hugging Face新增一键查看电脑可运行模型功能

人工智能开源社区平台Hugging Face推出实用新功能，允许用户快速查看自己电脑硬件能运行哪些机器学习模型。用户只需在个人设置页面添加硬件信息，如GPU型号、内存容量等，系统便会智能分析并显示兼容模型及其量化版本。这一功能极大提升了开发者和AI爱好者的使用体验，降低了模型选择的技术门槛，为AI技术的普及和应用推广提供了便利。

4. 字节跳动发布轻量化语音合成模型MegaTTS3

字节跳动在Hugging Face上发布最新文本转语音模型MegaTTS3，以仅0.45亿参数的轻量化设计实现高质量语音合成。该模型由字节跳动与浙江大学合作开发，在降低计算资源需求的同时保持出色的语音合成效果，支持多语言处理能力。MegaTTS3的发布标志着语音合成技术的重要进步，为移动设备和边缘计算场景下的语音应用提供了新选择。

5. 武汉体育中考引入AI智能系统，近10万学生受益

武汉市体育中考首次引入AI智能判定系统，实现了考试评判从人工到智能化的重大转变。系统采用人脸识别、电子测距和无线传感等多项AI技术，涵盖长跑、跳绳、足球绕杆等14个项目，几乎实现零人工干预的考试过程。这一技术创新不仅提高了考试的公正性和效率，也为教育评价体系的数字化转型提供了有益探索，对推动智慧教育发展具有示范意义。

国际交流与合作

1. 习近平同印度总统穆尔穆就中印建交75周年互致贺电

国家主席习近平与印度总统穆尔穆互致贺电庆祝两国建交75周年，强调"龙象共舞"是双方的正确选择，符合两国和两国人民根本利益。习近平表示愿与穆尔穆一道增进双方战略互信，加强各领域交流合作，深化在重大国际事务中的沟通协调，推动中印关系沿着健康稳定轨道向前发展，为促进世界和平繁荣作出贡献。

2. 外交部：将继续携手各方以绿色赋能发展、以蓝色助力合作

外交部发言人郭嘉昆表示，中国将继续携手各方，以绿色赋能发展、以蓝色助力合作，呼吁各方共同为国际绿色合作创造有利条件。他指出，中国每10度发电中有4度来自清洁能源，作为全球最大清洁技术出口国，创造了全球可再生能源领域46%的工作岗位。中国还与越南、泰国等开展清洁能源合作，与印尼等国签署蓝色经济合作文件，推进相关领域合作。

3. 普京会见王毅，俄方期盼中方来俄出席战胜80周年纪念活动

俄罗斯总统普京在莫斯科会见中共中央政治局委员、外交部长王毅时表示，期盼中方来俄出席苏联伟大卫国战争胜利80周年纪念活动，共同庆祝抗击纳粹法西斯和日本军国主义的胜利。普京强调，俄方正全力推进筹备工作，愿以此为契机推动俄中全面战略协作伙伴关系迈上新台阶，不断深化双方在联合国、上合组织、金砖国家等多边框架下的合作。

4. 印度越南泰国面临26%-46%对等关税，均表示将谋求贸易谈判

美国总统特朗普宣布的"对等关税"计划中，对印度征收26%、越南46%、泰国36%的关税，这些亚洲国家均表示将谋求贸易谈判以应对挑战。特朗普在宣布时特别点名这些国家，称他们对美国征收了高额关税，使美国工人遭到"恶毒的攻击"。这一关税政策将对全球贸易格局产生深远影响，相关国家正积极寻求通过外交和贸易谈判来缓解紧张局势。

5. 国际观察："伤人八百，自损一千"——"对等关税"将反噬美国经济

新华社评论指出，美国总统特朗普的"对等关税"计划将对美国家庭、产业和宏观经济造成多重反噬，被形容为"伤人八百，自损一千"。美国竞争企业协会高级经济学家认为，在通胀仍未得到控制之际，特朗普的关税已在推高消费者价格。耶鲁大学预测，如果其他国家采取报复措施，美国个人消费支出将上涨2.1%，实际GDP增长率将下降1个百分点。高盛集团预计，由于关税飙升可能阻碍经济增长、重燃通胀并推高失业率，未来12个月美国经济出现衰退的概率升至35%。
