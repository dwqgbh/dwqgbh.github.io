2022年全球数控机床行业技术竞争格局(附区域申请分布、申请人排名、专利申请集中度等)

行讯制造业资源 2022-11-01

全球数控机床技术区域竞争格局

1、目标市场国 / 地区分布：中国占比最高

目前，全球数控机床第一大目标市场国 / 地区为中国，中国数控机床专利申请量占全球数控机床专利总申请量的 80% 以上 ; 其次是美国、日本等国家和地区。

统计说明：①按每件申请显示一个公开文本的去重规则进行统计，并选择公开日最新的文本计算。②按照专利优先权国家进行统计，若无优先权，则按照受理局国家计算。如果有多个优先权国家，则按照最早优先权国家计算。

2、专利申请趋势：2014 年以来中国大幅领先，美国位居次席

从趋势上看，2010-2021 年，中国数控机床专利申请数量处于领先。2021 年，中国数控机床专利申请量为 18934 项，热度维持在高位。美国数控机床专利申请量总体呈现波动态势，2021 年，美国数控机床专利申请量为 177 项，同比有所下降。

3、中国区域专利申请分布：江苏位居榜首，广东紧随其后

中国方面，江苏为中国当前申请数控机床专利数量最多的省市，累计当前数控机床专利申请数量高达 17073 项。广东当前申请数控机床专利数量 15179 项，位居第二。中国当前申请省 ( 市、自治区 ) 数控机床专利数量排名前十的省份还有浙江、山东、辽宁、上海、安徽、湖北、福建和四川。

统计口径说明：按照专利申请人提交的地址统计。

趋势方面，2010-2021 年期间，江苏、广东数控机床专利申请量交替领先，2019 年后江苏数控机床专利申请量超过广东，位居榜首，广东紧随其后。除江苏、广东之外，2021 年，山东、浙江、辽宁等省份和地区数控机床专利申请量也相对较多。

全球数控机床技术申请人竞争格局

1、专利申请人集中度：市场集中度低，CR10 波动下降

2011-2021 年，全球数控机床专利申请人 CR10 总体呈现波动下降趋势，由 2011 年的 8.26% 波动下降至 2021 年的 1.48%，2022 年集中度有小幅回升的态势。整体来看，全球数控机床专利申请人集中度不高，且集中度总体呈现下降趋势。

注：①市场集中度 CR10 为申请总量排名前 10 位的申请人的专利申请量占该领域专利申请总量的比例 ( 其中，有联合申请时，专利数量不会被去重计算 ) 。

② 2022 年数据截至 2022 年 10 月 18 日。

2、TOP10 专利申请人

——总量及趋势：大连理工大学夺得桂冠

全球数控机床行业专利申请数量 TOP10 申请人分别是大连理工大学、成都飞机工业 ( 集团 ) 有限责任公司、深圳市创世纪机械有限公司、华中科技大学、广东普拉迪科技股份有限公司、发那科株式会社、珠海格力电器股份有限公司、科德数控股份有限公司、乔锋智能装备股份有限公司、沈阳飞机工业 ( 集团 ) 有限公司。

其中，大连理工大学数控机床专利申请数量最多，为 181 项。成都飞机工业 ( 集团 ) 有限责任公司排名第二，其数控机床专利申请数量达 167 项。

注：未剔除联合申请数量。

趋势方面，2011-2021 年，全球前十大数控机床专利申请人申请的数控机床专利申请数波动较大，十大数控机床专利申请人交替领先，竞争十分激烈。

——专利技术分布：B23011、B2303 及相关领域布局较多

目前，全球数控机床行业专利申请主要布局在 B23011、B2303 等细分领域，其中发那科株式会社在 G05B19 领域专利申请量达到 107 项 ; 大连理工大学在 G05B19 领域专利申请量达到 70 项。

3、市场价值最高 TOP10 专利的申请人：太陽專利信託公司处于领先

全球数控机床市场价值最高 TOP10 专利中，价值最高的专利由太陽專利信託公司掌握，价值为 1341 万美元。其余数控机床市场价值最高 TOP10 持有企业有苹果公司、BIS 公司、NNT 都科摩股份有限公司、安赛乐米塔尔公司、艾伯维爱尔兰无限公司、井博士有限责任公司、弗哈夫曼拉罗切有限公司和艾丽奥斯生物制药有限公司。

注：最有价值的专利是指该技术领域内具有最高专利价值的简单同族。当前统计口径按每组简单同族一个专利代表的去重规则进行统计，并选择同族中有专利价值的任意一件专利进行显示。

4、专利申请新进入者：四大新进入者

全球主要新进入者有四位，分别是浙江意控机床有限公司、意特利 ( 滁州 ) 智能数控科技有限公司、江苏贵钰航空工业有限公司以及南京林业大学。2021 年，意特利 ( 滁州 ) 智能数控科技有限公司数控机床专利申请量最多，达 16 项 ; 江苏贵钰航空工业有限公司申请 14 项，数量紧随其后。
