2025 年 11月 14日新闻早班车

江湖如闻 2025-11-14

工作路上皆寂寞，唯有关注不迷路。

1. 头条
1. 习近平复信51国61名青年汉学家，勉励其当好融通中外文明的使者，助力构建人类命运共同体。
2. 习近平同科摩罗总统阿扎利就中科建交50周年互致贺电。
3. 第二十七届中国国际高新技术成果交易会在深圳开幕，鸿蒙、麒麟、昇腾等国产核心技术集中亮相。
4. 宁波首次举办健康养老与银发经济产业博览会，12000平方米展区集结200余家展商，聚焦养老黑科技与品质服务。
5. 贵州黔南“村GT”汽车极限漂移大会启幕，上演“车马双驰”文体旅融合盛宴，抖音相关曝光量达1.7亿。
6. 美国联邦政府结束持续43天的历史最长停摆，总统签署众议院通过的临时拨款法案。
2. 政治
1. 中央纪委国家监委通报5起违纪违法案件，南昌市政协主席肖玉文等多名地方及国企官员接受审查调查或被开除党籍。
2. 中国纪检监察报发文强调，推进“十五五”发展需以党的自我革命引领社会革命，持之以恒推进全面从严治党。
3. 江苏纪委监委推进“高质量办案”，将案件审理室作为统筹牵头部门提升监督办案质效。
4. 英国新当选领导团队推进移民、教育和环境领域规则改革，多国2025年大选结果引发全球对国际关系格局的关注。
5. 安徽宿州埇桥区医保局原副局长贾闽波因严重违纪违法被开除党籍和公职。
6. 高标准高质量推进雄安新区建设，当地民间投资活力持续释放，助力现代化城市建设。
3. 经济金融
1. 国际白银现货价格突破54美元/盎司逼近历史高点，沪银主力合约同步创历史新高，避险情绪等多重因素推动涨势。
2. 中国央行公布10月金融数据，社融存量同比增8.5%，前十月增量超30万亿元，M1 - M2剪刀差收窄释放经济回暖信号。
3. 美国12万亿美元短期融资市场压力升温，银行警告美联储需出手缓解流动性紧张，纽约联储暗示或购买资产。
4. 欧洲金融官员探讨创建美联储资金支持替代机制，拟汇集美元储备减少对美国金融体系的依赖。
5. 多家中小银行下架5年期甚至3年期存款产品，以此应对净息差收窄带来的经营压力。
6. 软件巨头甲骨文股价持续下跌，五个交易日抹去9月涨幅，投资者质疑其人工智能和云业务盈利能力。
4. 军事
1. 俄罗斯武装力量正式组建无人系统部队，强化新型作战力量建设。
2. 美国“福特”号航母进入加勒比海，委内瑞拉宣布全面进入战备状态。
3. 法国大幅增加太空国防预算，应对太空领域安全新挑战。
4. 土耳其停飞所有现役C - 130“大力神”军用运输机，具体原因未公开披露。
5. 美军在加勒比海实施第20次反贩毒军事打击，击毙4名涉嫌贩毒的纳粹恐怖分子。
6. 巴基斯坦首都发生自杀式爆炸致12人死亡，巴方指认袭击获印度支持，印方予以否认。
5. 社会民生
1. 今年“双11”全国快递包裹揽收量达139.38亿件，单日峰值7.77亿件，两项数据均创历史新高。
2. 北京放宽公务员考试年龄上限至38岁，发布2026年度考试录用公务员公告。
3. 中国支付清算协会倡议“免密支付”不得默认开通，需为用户提供一键取消功能。
4. 7省份实现生娃基本医疗费用“零负担”，北京计划将胚胎冷冻与保存费用纳入医保支付范围。
5. 缅甸妙瓦底赌诈头目佘智江被从泰国引渡回国，其涉案金额超27亿元。
6. 黑龙江多地因地磁暴观测到绚丽极光，为国内罕见的极光观测景象。
6. 科教
1. 复旦科研团队揭示“饿一饿更健康”相关原理，在肿瘤治疗领域取得新突破。
2. 加州大学圣地亚哥分校获2100万美元科研经费，用于研究感官输入与头部运动的协调机制。
3. 该校另一团队获1000万美元资助，成立研究中心探索气候变化对海洋动物神经系统的影响。
4. 第五届国际新材料产业大会将于11月20日在蚌埠举办，聚焦新科技新材料产业发展。
5. 俄克拉荷马大学举办细胞生物学研讨会，同步开设冥想课程助力师生缓解压力、提升专注力。
6. 多国推进太空探索，中美均派航天员绕月飞行并计划建月球基地，日本企业测试太空酒店雏形。
7. 文体
1. 欧洲预选多场赛事开赛，挪威对阵爱沙尼亚、英格兰迎战塞尔维亚等焦点比赛引发球迷关注。
2. 立陶宛考纳斯一场以以足球友谊赛前，数十人聚集抗议反对与以色列进行体育合作。
3. 贵州三都将水族端节赛马与汽车漂移结合，举办体育旅游欢乐夜，展现传统与现代体育融合魅力。
4. 国际奥委会拟出台新政，考虑禁止跨性别女性参加奥运会女子项目。
5. 韩国打捞600年前沉船，发现63块写有“罗州广兴仓”的汉字木简及150余件青瓷。
6. 黔南“村GT”赛事举办超跑巡游活动，沿都柳江畔穿行，打造水族文化与汽车文化的流动展演。
8. 证券
1. 11月13日A股三大指数集体走强，上证指数创新高，锂电与储能板块爆发，多只个股涨停。
2. 22只个股获机构买入评级，其中8只上涨空间超20%，古井贡酒预期上涨空间达92.55%。
3. 领益智造重大资产重组审核终止，公司调整交易作价和支付方式，不再涉及可转债发行。
4. ST中迪因19连板股价涨幅过大停牌核查，合富中国等多只大牛股密集发布交易风险提示。
5. 比特币反弹遇阻，过去一个月现货比特币ETF净流出约28亿美元，机构大买家纷纷离场。
6. 美股道指续创收盘新高，而甲骨文股价连续下跌，市场对其云业务盈利前景存疑。
9. 气象灾害
1. 新疆富蕴县发布暴雪红色预警，吐尔洪乡等区域6小时降雪量达21.1毫米，后续仍有大暴雪。
2. 中央气象台预报14 - 17日北方和中东部气温普降6 - 10℃，0℃线将推进至长江沿线，或影响河南山东冬小麦生长。
3. 美国俄克拉荷马州发生氨气泄漏事件，导致当地数百人被紧急疏散。
4. 2025年全球极端天气频发，欧洲遭遇创纪录高温，南亚爆发洪灾，北美和澳大利亚多地遭野火侵袭。
5. 阿勒泰因寒潮遭遇暴雪，部分道路通行受阻，当地启动防雪灾应急抢险准备工作。
6. 秘鲁发生长途客车坠谷事故致37人遇难，印度新德里汽车爆炸造成13死20余伤。
10. 养生
规律作息可延缓衰老，研究显示入睡时间不规律、工作日与周末睡眠差距大的人生理年龄可能比实际老9个月，建议每天23点前固定入睡，保证7 - 8小时睡眠；同时11月14日为世界糖尿病日，日常需通过健康饮食和适度运动预防糖尿病，关注血糖监测，降低并发症风险。
