边缘AI领航,国产算力崛起|凌华智能2026华南工博会精彩回顾

凌华智能 2026-06-16

展会回顾


2026年6月10-12日，2026华南国际工业博览会在深圳国际会展中心圆满落幕。凌华智能以“共筑边缘AI未来，赋能工业物理场景创新”为主题，携全栈边缘计算、国产化平台及行业解决方案重磅亮相10号馆B043展位。从微秒级实时控制到千亿级参数大模型本地部署，凌华用一场“可触摸的算力盛宴”，诠释了边缘AI在工业现场的真实力量。



国产算力平台全线出击

海光系列新品集中首发


SCIIF

COMEx-6HG4 

本次展会的核心亮点，当属凌华首款国产自主嵌入式模块——COMEx-6HG4的首次公开亮相。该计算模块严格遵循PICMG COM Express® 3.1 Type 6标准，搭载海光新一代C86-4G 3430和3450处理器，提供4核或8核的灵活配置，在性能与功耗之间实现精准平衡。最高支持64GB DDR5 SO-DIMM高速内存，频率达到5600MHz，充分满足工控领域高负载运算与大带宽数据吞吐需求。模块集成丰富的显示与扩展接口，支持宽温加固及国产化安全机制，兼容麒麟、Windows、统信UOS等主流操作系统。其搭载的海光C86-4G系列处理器原生支持x86指令集，用户无需大规模重构现有应用软件即可平滑迁移至国产硬件平台，是国产化替代场景中高性能、高可靠嵌入式计算的理想选择。




SCIIF

PXIe-3943/3945 

基于COMEx-6HG4模块，凌华同步推出多款重磅整机产品。PXIe-3943/3945控制器动态搭配PXES-2596机箱现场演示，成为国内首款搭载国产海光3430/3450芯片的3U PXIe控制器，100%符合信创要求，具备高安全、自主可控特性。采用业界领先的5600MHz DDR5内存，支持PCIe 3.0标准，提供高达16GB/s的系统传输带宽，并可自动识别适配PXI机箱内部复杂拓扑结构。前面板配备2个千兆以太网口、2个USB 3.0、4个USB 2.0、2个串口及2个DP显示接口，为核心测量任务释放更多通道资源。


SCIIF

cPCI-C3700 

cPCI-C3700 3U CompactPCI主控模块同步登场。该产品采用海光4核3430G处理器，支持TCM 2.0安全芯片，全面符合国产化信创安全标准。面板I/O包括2个USB 2.0、1个RJ-45 LAN、1个VGA端口及1个支持RS-232的DB-9串口，平台成熟稳定易扩展，生态丰富，是工厂自动化等工业应用及自主可控加固计算领域的优质选择。


SCIIF

AVA-7200

轨交领域的 AVA-7200 6HG4 无风扇可扩展边缘AI平台同样吸引众多目光。该平台搭载国产海光3430G芯片，专为响应轨交行业国产化趋势而全新打造，可实现复杂多源视频实时分析，已在全球头部轨交现场落地验证。平台兼顾高性能边缘分析与实时传感器处理能力，广泛适用于车载列车控制系统、车用AIoT视频分析等场景，在严苛环境与自主可控需求之间实现完美兼顾。


SCIIF

SuperCAT 

在应用方案上，SuperCAT运动控制系统展区人气持续高涨。这是目前国内唯一适配海光平台的EtherCAT主控方案，市面尚无同类国产替代产品，从底层彻底打破海外技术垄断。海光3400系列高频多核架构带来强劲并行处理能力，原生适配工业实时操作系统，具备优异的低延迟与高稳定特性，有效抑制工控环境中的抖动。芯片内置硬件级可信计算模块，充分满足工业信创合规要求。SuperCAT主控软件支持双主站运行，同时兼容Hypervisor虚拟化架构，使非实时Windows系统与实时控制核高效并行运转，兼顾友好的操作交互体验与高精度运动控制的稳定性。整套方案可同时支持多达128个同步轴和超过10,000个控制点，实时响应速度与运行稳定性均居行业高端水准，尤其适用于半导体、电子制造等自动化应用领域，整体性能与当前国际主流处理器方案相比毫不逊色






边缘AI算力矩阵全覆盖，

从千亿大模型到紧凑型推理无所不及



SCIIF

DLAP-701 

面向AI算力需求的全面爆发，凌华交出了一份“算力无所不及”的答卷。旗舰级边缘推理平台 DLAP-701 迎来重磅首秀，内建NVIDIA Jetson Thor™ T5000，提供最高2070 FP4 TFLOPS的极致AI运算效能，支持5GbE、USB 3.0、M.2及QSFP28高速接口，可在完全脱机环境中低延迟执行大语言模型（LLM），确保数据隐私与安全，让生成式AI真正落地于工厂产线、高端医疗等数据敏感型现场。




SCIIF

DLAP-411-Orin

DLAP-411-Orin 无风扇紧凑型推理平台，采用NVIDIA Jetson AGX Orin处理引擎，拥有8核Carmel Arm 64位CPU、1792核Ampere GPU、32GB LPDDR5及64GB eMMC，紧凑尺寸下提供千兆以太网、双USB及M.2接口，易于集成到各类边缘设备和系统中。DLAP-211-Orin 系列则以更灵活的身形覆盖广泛场景，从前代产品8倍AI性能跃升，结合多实例GPU与专用深度学习加速器，支撑智能城市、机器视觉检测、自动驾驶汽车、机器人等多元应用，在紧凑尺寸、可扩展性与高可靠性之间取得精妙平衡。


SCIIF

Express-PTL

模块化电脑产品线同样全面进化为AI尖兵。全新 Express-PTL COM Express Type 6计算模块基于Intel第三代酷睿Ultra处理器，最多支持16核及12个Xe3 GPU引擎，全处理器最高可提供180TOPS AI算力，专为高强度、高复杂度边缘AI工作负载而生。热设计功耗15-65瓦，支持USB4、DP、HDMI等多类型I/O，最高128GB DDR5 SO-DIMM内存，工业级SKU支持-40°C至+85°C宽温，确保10年以上长生命周期供货。


SCIIF

COM 模块化电脑

COM-HPC-mMTL Mini型COM-HPC模块以95x70mm的紧凑尺寸装入Intel第一代酷睿Ultra处理器，最多14核，集成高达64GB LPDDR5内存并板贴内存及SSD颗粒，在极小空间内实现高性能与高可靠性，是电池供电、空间受限边缘应用的理想选择，覆盖医疗超声、工业自动化、数据记录仪等场景。

cExpress-R8 Type 6紧凑型模块搭载AMD Ryzen™嵌入式8000系列处理器，集成最高16 TOPS算力的AMD XDNA™ NPU与RDNA™ 3显卡，支持最高96GB双通道DDR5内存及16条PCIe Gen4通道，在同等性能的x86平台产品中提供更高性价比和更快速稳定的交付。

另有 Express-VR7 COM Express Type 7基本尺寸模块，基于AMD嵌入式Ryzen™ V3000系列，支持双10G光通信口及2.5G以太网口，专攻边缘通信与网络场景，在同类产品中具备最佳单位能耗比；

NanoX-BT COM Express Mini Size Type 10模块作为尺寸最小的选项，支持英特尔凌动或赛扬处理器，宽温-40°C至+85°C，适配极端空间约束场景；LEC-IMX95 SMARC计算模块基于NXP i.MX95处理器，集成6核Cortex-A55与NPU最高2 TOPS，兼具高效能计算与图像信号处理能力，面向智能家居、智慧城市、工业4.0等下一代物联网应用。






深耕行业场景，视觉与测试方案合作成果



SCIIF

凌华智能 × 泉州华创

合作方案区汇聚多套行业验证的成熟系统。凌华智能 × 泉州华创 基于Camera Link接口的高精度印刷缺陷检测方案，主控单元采用加固级无风扇嵌入式电脑MVP-6230，支持第12/13代Intel® Alder Lake-S处理器，适配Windows或Linux Ubuntu操作系统，支持全天候24/7运行。图像采集卡PCIe-CPL64V具备PCIe Gen3 x4高带宽连接能力，支持Camera Link供电及多卡同步运行。配合8K线阵相机，特别适合高速印刷品在线缺陷检测、OLED面板与PCB自动光学检测等连续运动或超高分辨率场景。




SCIIF

凌华智能 × 华汉伟业

凌华智能 × 华汉伟业 电机定子绕线高度检测方案，以凌华W890工作站为核心，搭载Intel至强600系列工作站处理器，支持高达1TB DDR5-6400 ECC内存及灵活7槽PCIe扩展布局，为复杂机器视觉和多GPU AI推理奠定强大基础。系统联合华汉伟业超高精度VS2040型号3D结构光相机检测绕线定子高度，对定子单片的尺寸进行把控，如同心度，轮廓度等，以保证线圈顺利缠绕。此检测方案解决了新能源电机至关重要的质量问题，为电机安全使用提供有力的保障。


SCIIF

凌华智能 × 鼎阳科技

测试测量领域，凌华智能 × 鼎阳科技 合力打造的PXIe多通道拓展方案同样备受关注。PXES-2590 3U 9槽全混合机箱提供最高8GB/s系统带宽，1个系统插槽、1个系统定时插槽及7个混合外设插槽，可灵活兼容CompactPCI、PXI、CompactPCI Express和PXI Express模块，背部2个10MHz时钟BNC接口进一步提升同步能力。PXIe-3987控制器基于Intel® Core™ i7-7820EQ四核处理器，配备双DisplayPort、双千兆以太网、双USB 3.0及GPIB接口，构建混合测试系统的理想控制核心。鼎阳MS高分辨率模块化示波器，最高1GHz带宽、5GSa/s采样率、1.25Gpts/通道存储深度，12-bit垂直分辨率配合丰富的高级分析模式。整套系统通过SigScopeLab软件实现PC端实时与离线波形分析，多窗口分窗显示、各Memory波形水平时基独立调节，配合Zoom功能精准捕捉波形细节，为多元测量需求提供专业支撑。





工业显示与数据采集，让边缘更易用、更智能


SCIIF

AID-1560A

算力落地离不开可靠的人机交互与感知层支撑。工业一体机 AID-1560A 为凌华2026年全新推出的高性能触控一体机，搭载英特尔酷睿Ultra 1系列处理器，算力最高达34 TOPS，支持I/O板定制，内置RS485匹配电阻，兼容UPS供电、远程开关机及一键备份功能，全面匹配工业控制现场的严苛使用需求。




SCIIF

Titan 2

Titan 2 IP69K不锈钢工业平板电脑，采用第11代Intel Core i5/i3处理器，真平板设计搭配304不锈钢外壳（可选316），耐受高压、热水及酒精擦拭清洁。光学键合与高亮度增强技术确保阳光与暗光环境皆清晰可视，M12连接器提供即装即用的安全接线方案，以边缘数据可视化驱动AI辅助生产优化，是食品饮料、制药与化工行业的坚实HMI之选。


SCIIF

MLC-M

MLC-M多功能一体式医疗平板电脑专为卓越卫生设计，无螺丝全密封铝制外壳与无风扇冷却共同防止污垢灰尘积聚，IP54防护等级配合屏幕锁定键，使清洁消毒更彻底高效。PCAP多点触控屏支持手套操作，前玻璃四个发光软触功能键让用户无需鼠标键盘即可完成多项操作。


SCIIF

SDAQ-204

边缘数据采集层面，SDAQ-204 可编程紧凑型边缘数据采集器内置ARM Cortex-A9处理器与四通道24位高分辨率模拟输入，无需搭配主机即可独立运行，是旋转机械设备24小时振动监测的理想之选。24位Sigma-Delta ADC支持抗混叠滤波，双千兆以太网端口支持菊花链连接，可在边缘侧执行自定义Python iApp将原始数据过滤为FFT等可用信息后回传，显著降低网络负载与后端处理压力。内置Web控制台配合直观仪表板界面，无需编程即可快速部署机器状态监控环境。




从微秒级运动控制到千亿级大模型本地推理，从国产海光平台全线出击到全系列AI模块化电脑矩阵，从工业视觉方案现场展示到专业测试测量平台实战亮剑——凌华智能在本届华南工博会上展现的，正是一幅由边缘AI贯穿计算、连接、感知与行动的完整智能化图景。我们深信，真正的智能化，就发生在离机器、离产线、离现场最近的地方。
