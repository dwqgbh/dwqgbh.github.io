广西2026年“桂工 AI 动 健康同行”线上趣味运动会【职工群众答题】易错题

时光慢些叭 2026-05-28

一、《工会法》及广西实施办法


1. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，用人单位单方面解除职工劳动合同时，应当在作出解除合同的决定前（）将理由通知工会。
A 5日
B 10日
C 15日
D 30日
正确答案：B
解析：用人单位单方解除劳动合同，需在作出决定前10日将理由通知工会，接受工会监督。

2. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，工会收到用人单位单方面解除职工劳动合同的理由通知后，应当在（）内召开有关会议研究并提出意见。
A 5日
B 7日
C 10日
D 15日
正确答案：B
解析：工会收到通知后，须在7日内研究并出具意见。

3. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，对工会提出的不同解除劳动合同意见，用人单位应当及时研究并在（）内书面答复工会。
A 5日
B 7日
C 10日
D 15日
正确答案：A
解析：用人单位收到工会意见后，需在5日内书面回复。

4. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，工会经费审查委员会应当对本级工会经费的收支和财产管理情况每年至少进行（）次审查。
A 1次
B 2次
C 3次
D 4次
正确答案：A
解析：工会经费审查委员会每年至少开展1次经费与资产审查，并向会员代表大会报告。

5. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，有侵犯职工劳动权益情形的，企业、事业单位、社会组织应当在（）内向工会作出书面答复。
A 10日
B 15日
C 20日
D 30日
正确答案：B
解析：单位收到工会整改意见后，15日内必须书面答复。

6. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，企业、事业单位、社会组织发生伤亡事故或者出现严重危害职工安全和健康的情况时，所在单位和有关部门应当在收到工会处理意见之日起（）内书面答复。
A 10日
B 15日
C 20日
D 30日
正确答案：B
解析：发生安全事故、职业危害事件后，相关单位需在15日内回复工会处理意见。

7. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，因工作需要调动工会主席、副主席工作的，应当事先书面征得本级工会委员会和上一级工会的同意，有关工会组织应当在接到书面意见之日起（）内作出书面答复。
A 15日
B 30日
C 45日
D 60日
正确答案：B
解析：工会组织接到调动征询意见后，需在30日内作出书面答复。

8. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，建立工会组织的用人单位，应当于每月（）前按上月全部职工工资总额的百分之二向工会拨缴当月工会经费。
A 5日
B 10日
C 15日
D 20日
正确答案：C
解析：用人单位每月15日前，完成工会经费拨缴工作。

9. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，用人单位自上级工会批准筹建工会组织的次月起，由上级工会收取建会筹备金，待工会建立后按规定返还，建会筹备金的收取依据是（）。
A 工会法有关经费数额的规定
B 职工工资总额的百分之一
C 当地最低工资标准的百分之二
D 企业利润的千分之五
正确答案：A
解析：工会筹备金按照《工会法》规定的经费标准收取，工会正式成立后依规返还。

10. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，各级地方总工会，产业工会和具有社会团体法人资格的基层工会，可以依法登记设立（）。
A 盈利性企业
B 为职工服务的企业、事业单位
C 公益组织
D 行业协会
正确答案：B
解析：工会可依法设立服务职工的企事业单位，提升服务职工能力。

11. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，工会对其登记设立的企业、事业单位的财产享有（）。
A 使用权
B 所有权
C 管理权
D 处置权
正确答案：B
解析：工会对下属服务型企事业单位的财产依法享有所有权。

12. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，工会专职工作人员的人数由工会与企业、事业单位、社会组织参照职工总数的（）比例协商确定。
A 千分之一
B 千分之三
C 千分之五
D 百分之一
正确答案：B
解析：工会专职人员配置，参照职工总人数千分之三的比例协商确定。

13. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，工会工作人员截留、挪用、侵占、贪污工会经费的，情节严重的应当（）。
A 予以警告
B 罚款
C 依照《中国工会章程》予以罢免
D 行政拘留
正确答案：C
解析：侵占、挪用工会经费且情节严重的，按《中国工会章程》对相关人员予以罢免。

14. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，不经上级工会批准，擅自建立工会组织的，工会可以提请哪个部门依法处理。
A 纪检部门
B 政府有关行政主管部门
C 司法机关
D 行业主管部门
正确答案：B
解析：违规私自组建工会的，工会可提请政府相关行政主管部门处置。

15. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，本办法自（）起施行。
A 2005年7月29日
B 2005年9月1日
C 2025年11月27日
D 2026年1月1日
正确答案：B
解析：该办法2005年7月29日通过，自2005年9月1日正式施行。

16. 题目：依据《广西壮族自治区实施〈中华人民共和国工会法〉办法》，县级以上人民政府劳动行政部门应当会同同级工会和企业方面代表，建立健全（）。
A 劳动监察机制
B 劳动关系三方协商机制
C 劳动争议调解机制
D 用工监管机制
正确答案：B
解析：劳动部门、工会、企业代表三方建立协商机制，统筹协调劳动关系。

17. 题目：依据《中华人民共和国工会法》，工会经费的主要来源是（）。
A 政府财政拨款
B 工会会员缴纳的会费
C 用人单位缴纳的工会经费
D 社会捐赠
正确答案：C
解析：用人单位按职工工资总额2%拨缴的经费，是工会经费最主要、最稳定的来源。

18. 题目：依据《中华人民共和国工会法》，工会组织职工参与本单位的民主管理形式不包括（）。
A 民主选举
B 民主协商
C 民主决策
D 民主裁决
正确答案：D
解析：职工民主管理包含民主选举、协商、决策、监督，民主裁决不属于法定形式。

19. 题目：依据《中华人民共和国工会法》，工会发现企业违章指挥、强令工人冒险作业，或者生产过程中发现明显重大事故隐患和职业危害，有权提出（）。
A 批评
B 整改建议
C 处罚建议
D 停产要求
正确答案：B
解析：发现安全隐患、违规作业行为，工会有权提出整改建议，督促企业落实安全责任。

20. 题目：依据《中华人民共和国工会法》，基层工会委员会每届任期为（）。
A 2年
B 3年
C 5年
D 3年至5年
正确答案：D
解析：基层工会委员会每届任期为3年至5年，具体由会员大会确定。

21. 题目：依据《中华人民共和国工会法》，工会动员和组织职工积极参加经济建设，开展（）活动，提高劳动生产率和经济效益。
A 劳动竞赛、合理化建议
B 安全生产、技能竞赛
C 节能减排、技术革新
D 岗位练兵、业务培训
正确答案：A
解析：工会组织职工开展劳动竞赛、合理化建议等活动，助力企业发展。

22. 题目：依据《中华人民共和国工会法》，工会参加企业的劳动争议（）工作。
A 仲裁
B 诉讼
C 调解
D 裁决
正确答案：C
解析：工会主要参与企业内部劳动争议调解工作，化解劳资矛盾。

23. 题目：依据《中华人民共和国工会法》，工会应当根据（）原则，建立预算、决算和经费审查监督制度。
A 民主集中制
B 公开透明
C 统一领导、分级管理
D 勤俭节约
正确答案：C
解析：工会经费管理遵循统一领导、分级管理原则，建立完整的预决算与监督制度。

24. 题目：依据《中华人民共和国工会法》，县级以上地方各级人民政府可以根据实际情况，决定是否听取同级工会对涉及职工利益重大问题的意见。（）
A 正确
B 错误
正确答案：B
解析：涉及职工利益的重大问题，政府必须听取同级工会意见，并非可自主决定。

25. 题目：依据《中华人民共和国工会法》，工会对职工进行思想政治教育，提高职工的思想道德、科学文化和（）素质。
A 职业技能
B 法律意识
C 安全意识
D 综合素质
正确答案：A
解析：工会负责提升职工思想、文化以及职业技能综合素养。

26. 题目：依据《中华人民共和国工会法》，工会组织职工开展文娱、体育活动，丰富职工的（）生活。
A 物质
B 日常
C 业余
D 精神
正确答案：C
解析：文体活动主要用于丰富职工业余文化生活。

27. 题目：依据《中华人民共和国工会法》，县级以上地方各级人民政府制定国民经济和社会发展计划，对涉及职工利益的重大问题，应当听取（）的意见。
A 同级工会
B 上级工会
C 职工代表
D 行业工会
正确答案：A
解析：政府制定规划时，涉及职工权益事项必须听取同级工会意见。

28. 题目：依据《中华人民共和国工会法》，工会经费的经费审查委员会由（）选举产生。
A 工会会员代表大会
B 工会委员会
C 职工大会
D 管理层会议
正确答案：A
解析：工会经费审查委员会由会员代表大会选举产生，行使监督职权。

29. 题目：依据《中华人民共和国工会法》，职工（）以上的企业、事业单位、社会组织的工会，可以设专职工会主席。
A 100人
B 150人
C 200人
D 300人
正确答案：C
解析：职工人数达到200人及以上的单位，其工会可设置专职工会主席。

30. 题目：依据《中华人民共和国工会法》，工会帮助、指导职工与用人单位依法订立和履行劳动合同，并与用人单位建立（），维护职工的劳动权益。
A 集体协商机制
B 用工沟通机制
C 权益保障机制
D 纠纷化解机制
正确答案：A
解析：工会与用人单位建立集体协商机制，签订集体合同，整体维护职工权益。

31. 题目：依据《中华人民共和国工会法》，工会主席、副主席任期未满时，不得随意调动其工作。因工作需要调动时，应当征得（）的同意。
A 上一级工会
B 职工代表大会
C 用人单位管理层
D 劳动行政部门
正确答案：A
解析：任期内调动工会主席、副主席，必须征得本级工会委员会及上一级工会同意。

 

二、《新时期产业工人队伍建设改革方案》

1. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人社会保险权益保障的重点是（）。
A 养老保险
B 医疗保险
C 工伤保险
D 失业保险
正确答案：C
解析：产业工人多从事一线作业，工伤风险高，工伤保险是权益保障重点。

2. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人薪酬分配的公平性要求是（）。
A 同工同酬
B 按劳分配
C 按能分配
D 按绩分配
正确答案：A
解析：落实同工同酬，消除用工身份差异带来的薪酬不公。

3. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人队伍建设改革的重点领域不包括（）。
A 传统制造业
B 新兴产业
C 服务业
D 农业
正确答案：D
解析：改革聚焦制造业、新兴产业、服务业，农业从业者不属于产业工人范畴。

4. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人技能培训设计的首要出发点是（）。
A 针对产业需求
B 针对岗位需求
C 针对个人需求
D 针对考证需求
正确答案：A
解析：技能培训以产业发展需求为首要导向，匹配产业升级方向。

5. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人队伍建设改革中，要加强的企业文化建设重点是（）。
A 创新文化
B 安全文化
C 工匠文化
D 团队文化
正确答案：C
解析：重点弘扬工匠精神，打造工匠文化，营造崇尚技能的氛围。

6. 题目：依据《新时期产业工人队伍建设改革方案》，我国企业开展职业技能培训的主要责任主体是（）。
A 政府
B 企业
C 行业协会
D 培训机构
正确答案：B
解析：企业是职工职业技能培训的第一责任主体。

7. 题目：依据《新时期产业工人队伍建设改革方案》，我国在产业工人队伍建设改革中，要建立的职业技能培训模式是（）。
A 政府主导
B 企业主导
C 校企合作
D 社会参与
正确答案：C
解析：推行校企合作、工学结合的培训模式，实现教学与岗位无缝衔接。

8. 题目：依据《新时期产业工人队伍建设改革方案》，我国推动产业工人向技术工人转型，关键是（）。
A 提高工资待遇
B 加强技能培训
C 改善工作环境
D 拓宽晋升渠道
正确答案：B
解析：强化职业技能培训，是普通产业工人转型为技术工人的核心举措。

9. 题目：依据《新时期产业工人队伍建设改革方案》，我国企业内部产业工人民主管理监督的基础形式是（）。
A 职工代表大会监督
B 工会专项监督
C 职工群众自发监督
D 行业部门监督
正确答案：A
解析：职工代表大会是企业民主管理、民主监督的基础形式。

10. 题目：依据《新时期产业工人队伍建设改革方案》，产业工人队伍建设改革的核心是提升产业工人的技能水平。（）
A 正确
B 错误
正确答案：B
解析：改革主线是思想政治引领，提升技能是重要内容，题干表述错误。

11. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人队伍建设改革的主线是（）。
A 提升职业技能
B 加强思想政治引领
C 完善权益保障
D 推动产业工人全面发展
正确答案：B
解析：加强思想政治引领是产业工人队伍改革的主线。

12. 题目：依据《新时期产业工人队伍建设改革方案》，产业工人队伍建设改革的根本目的是提高产业工人的社会地位。（）
A 正确
B 错误
正确答案：B
解析：改革根本目的是打造知识型、技能型、创新型高素质产业工人队伍。

13. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人薪酬待遇的增长机制是（）。
A 定期增长
B 按需增长
C 按绩增长
D 随机增长
正确答案：A
解析略。

14. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人队伍建设改革的目标实现时间节点是（）。

15. 题目：依据《新时期产业工人队伍建设改革方案》，我国产业工人队伍建设改革的直接发展目标是（）。

三、《中国共产党统一战线工作条例》

2. 题目：依据《中国共产党统一战线工作条例》，修订后的《中国共产党统一战线工作条例》正式发布日期是（）。

B 2020年11月30日

D 2021年1月1日

3. 题目：依据《中国共产党统一战线工作条例》，中国共产党领导的多党合作和政治协商制度是我国的（）。

B 基本政治制度
C 重要政治制度
D 核心政治制度

4. 题目：依据《中国共产党统一战线工作条例》，新时代我国统一战线工作的核心任务是（）。

B 凝聚人心、汇聚力量

D 服务经济发展

5. 题目：依据《中国共产党统一战线工作条例》，中国特色解决民族问题的正确道路的核心是（）。
A 各民族一律平等

6. 题目：依据《中国共产党统一战线工作条例》，新时代我国统一战线工作对象重点是（）。
A 民主党派成员
B 党外代表人士
C 少数民族人士
D 宗教界人士

7. 题目：依据《中国共产党统一战线工作条例》，我国各省（自治区、直辖市）可成立的党外知识分子联谊组织是（）。

8. 题目：依据《中国共产党统一战线工作条例》，统一战线工作任务重的乡镇（街道）党组织应当（）。

A 50%
B 60%
C 65%
D 70%

11. 题目：依据《中国共产党统一战线工作条例》，各级党委帮助民主党派解决的问题不包括（）。
A 机构编制
B 经费办公场所
C 干部交流挂职
D 成员薪酬待遇
正确答案：D

12. 题目：依据《中国共产党统一战线工作条例》，国家机关和国有企事业单位党外知识分子工作重点对象不包括（）。

正确答案：D

13. 题目：依据《中国共产党统一战线工作条例》，党的宗教工作基本方针不包括（）。

14. 题目：依据《中国共产党统一战线工作条例》，我国对做好出国和归国留学人员统战工作的方针是（）。

B 广泛团结、热情服务、积极引导、发挥作用

15. 题目：依据《中国共产党统一战线工作条例》，党中央以及地方党委设置统战部，统战部是党委主管统战工作的（）。

1. 题目：依据《广西壮族自治区国民经济和社会发展第十五个五年规划纲要》，广西城镇化建设的核心是（）。
A 北部湾城市群
B 南宁都市圈
C 平陆运河经济带
D 西江经济带
正确答案：B
解析：广西以南宁都市圈为核心，引领全区新型城镇化建设。

2. 题目：依据《广西壮族自治区国民经济和社会发展第十五个五年规划纲要》，广西区域发展龙头是（）。

B 南宁
C 平陆运河经济带
D 西江经济带
正确答案：A
解析：北部湾经济区是广西区域发展和对外开放的龙头。

3. 题目：依据《广西壮族自治区国民经济和社会发展第十五个五年规划纲要》，广西安全建设总目标是（）。
