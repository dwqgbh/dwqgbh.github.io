中考政治| 2026年中考道德与法治十大时政热点及时事政治资料

潇湘仁德学堂 2026-06-07

本文整理了2026中考十大时政热点及对应考点，同时将2025年5月至2026年4月的重大时政热点汇总整理，供应考师生备考使用。


一、十大时政热点



热点1：2026年全国两会


时政背景：2026年3月，十四届全国人大四次会议、全国政协十四届四次会议在北京召开，审议政府工作报告、“十五五”规划纲要草案，部署全年经济社会发展重点任务，聚焦新质生产力培育、民生保障、乡村振兴、生态文明等核心议题。

材料关键词：全国两会、人民代表大会制度、政治协商、政府工作报告、新质生产力、民生实事、全过程人民民主。

对应考点

1. 政治制度：人民代表大会制度是我国的根本政治制度；全国人大是最高国家权力机关，行使立法权、决定权、监督权、任免权；中国共产党领导的多党合作和政治协商制度是基本政治制度。

2. 发展理念：坚持以人民为中心的发展思想；贯彻创新、协调、绿色、开放、共享的新发展理念；推动高质量发展。

3. 民主政治：全过程人民民主是最广泛、最真实、最管用的民主；人民当家作主是社会主义民主政治的本质特征。

4. 国家机构：政府是国家行政机关，坚持依法行政，全心全意为人民服务。

出题设问
1.两会共商国是，体现我国民主有何特点？
2.政府出台多项民生举措说明了什么？




热点2：2026年中央一号文件

时政背景：2026年2月，《中共中央 国务院关于锚定农业农村现代化扎实推进乡村全面振兴的意见》发布，这是连续第23个聚焦“三农”的中央一号文件，部署粮食安全、乡村产业升级、和美乡村建设、农民增收等27项重点任务。

材料关键词：中央一号文件、乡村振兴、粮食安全、18亿亩耕地红线、和美乡村、农业现代化、共同富裕。

对应考点

1. 国情国策：农业农村农民问题是关系国计民生的根本性问题，坚持农业农村优先发展；实施乡村振兴战略。

2. 国家安全：粮食安全是国家安全的重要基础，严守耕地红线，保障国家粮食安全。

3. 发展理念：贯彻协调发展理念，促进城乡融合发展；坚持共享发展，实现共同富裕。

4. 生态文明：建设宜居宜业和美乡村，促进人与自然和谐共生。




热点3：重大科技成就与科技自立自强

时政背景：2025年7月-2026年4月，我国在航天、量子、人工智能、深海探测等领域取得标志性突破：嫦娥六号完成月球背面采样返回、长征十号运载火箭首飞、“九章三号”量子计算机算力再提升、“奋斗者”号万米深潜突破100次。

材料关键词：嫦娥六号、长征十号、量子计算机、科技自立自强、新质生产力、创新驱动发展、航天精神。

对应考点

1. 发展战略：实施创新驱动发展战略、科教兴国战略、人才强国战略；建设创新型国家。

2. 创新重要性：创新是引领发展的第一动力；科技创新能力已经成为综合国力竞争的决定性因素。

3. 民族精神：弘扬以爱国主义为核心的民族精神和以改革创新为核心的时代精神；践行科学家精神。

4. 核心素养：培养创新精神和实践能力，立志成才，报效祖国。




热点4：《中华人民共和国生态环境法典》实施

时政背景：2026年1月1日起，我国首部生态环境法典正式施行，整合了环境保护法、大气污染防治法、水污染防治法等多部法律，构建了统一、系统的生态环境保护法律体系，用最严格制度最严密法治保护生态环境。


材料关键词：生态环境法典、全面依法治国、良法善治、生态保护、双碳目标、美丽中国。

对应考点

1. 法治建设：全面依法治国是中国特色社会主义的本质要求和重要保障；坚持科学立法、严格执法、公正司法、全民守法。

2. 生态文明：坚持节约资源和保护环境的基本国策；走绿色发展道路，促进人与自然和谐共生；绿水青山就是金山银山。

3. 法律作用：法律规范全体社会成员的行为，保护我们的生活；法律为生态文明建设提供法治保障。

4. 公民义务：自觉履行保护环境的法定义务，践行绿色低碳生活方式。




热点5：文化传承发展与文化自信

时政背景：2025-2026年，我国深入推进中华优秀传统文化传承发展工程：良渚古城遗址申遗成功10周年、非遗保护成果丰硕、“中国传统制茶技艺”等非遗走向世界、各地开展“我们的节日”系列活动，推动中华优秀传统文化创造性转化、创新性发展。

材料关键词：文化自信、中华优秀传统文化、非遗保护、文化传承、文化出海、文明互鉴、社会主义核心价值观。

对应考点

1.中华文化：中华文化源远流长、博大精深；中华文化是中华民族的精神命脉，是我们最深厚的文化软实力。

2. 文化自信：文化自信是一个国家、一个民族发展中更基本、更深沉、更持久的力量；坚定文化自信，事关国运兴衰、文化安全和民族精神的传承发展。

3.文化创新：推动中华优秀传统文化创造性转化、创新性发展；继承革命文化，发展社会主义先进文化。

4. 文明交流：文明因交流而多彩，文明因互鉴而丰富；学习和借鉴人类文明的一切优秀成果，兼收并蓄。




热点6：纪念中国人民抗日战争暨世界反法西斯战争胜利80周年

时政背景：2025年9月3日，纪念中国人民抗日战争暨世界反法西斯战争胜利80周年大会在北京隆重举行，习近平主席发表重要讲话，强调铭记历史、缅怀先烈、珍爱和平、开创未来，弘扬伟大抗战精神。

材料关键词：抗战胜利80周年、伟大抗战精神、爱国主义、民族精神、国家安全、珍爱和平。

对应考点

1. 民族精神：中华民族形成了以爱国主义为核心的团结统一、爱好和平、勤劳勇敢、自强不息的伟大民族精神；伟大抗战精神是中华民族精神的具体体现。

2. 爱国主义：爱国主义是中华民族精神的核心，是中国人民和中华民族维护民族独立和民族尊严的强大精神动力。

3. 国家安全：国家安全是安邦定国的重要基石；坚持总体国家安全观，维护国家主权、安全、发展利益。

4. 时代主题：和平与发展是当今时代的主题；反对战争，珍爱和平，推动构建人类命运共同体。




热点7：民生保障与共同富裕

时政背景：2025-2026年，国家持续加大民生投入：医保药品目录再扩容、异地就医直接结算全覆盖、义务教育优质均衡发展推进、保障性住房建设加快、养老服务体系完善，着力解决群众急难愁盼问题，扎实推进共同富裕。

材料关键词：医保改革、教育公平、保障性住房、养老服务、稳就业、共同富裕、以人民为中心。

对应考点

1. 发展思想：党和政府坚持以人民为中心的发展思想，让人民群众共享发展成果。

2. 发展目的：发展的根本目的就是增进民生福祉；人民对美好生活的向往，就是党的奋斗目标。

3. 社会公平：维护社会公平正义，促进社会和谐稳定；保障公民平等享有受教育权、生命健康权等基本权利。

4. 共同富裕：共同富裕是中国特色社会主义的根本原则；缩小收入差距，让全体人民朝着共同富裕方向稳步前进。




热点8：未成年人保护与网络治理

时政背景：2026年，新修订的《未成年人保护法实施条例》深入实施，国家持续开展“清朗”系列专项行动，整治网络谣言、网络暴力、未成年人网络沉迷，严厉打击侵害未成年人合法权益的违法犯罪行为，为未成年人健康成长保驾护航。

材料关键词：未成年人保护、六大保护、网络治理、清朗行动、网络沉迷、校园欺凌、法治教育。

对应考点

1.特殊保护：未成年人身心发育尚不成熟，自我保护能力较弱，需要给予特殊、优先保护；家庭、学校、社会、网络、政府、司法六大保护共同发力。

2. 网络生活：网络是把双刃剑，要合理利用网络；恪守道德、遵守法律是网络生活的基本准则；提高媒介素养，学会信息节食。

3. 法治观念：增强法治意识，依法自律，自觉抵制不良诱惑；学会依法维护自己的合法权益。

4. 自我保护：增强自我保护意识和能力，掌握基本的自救自护方法。




热点9：中国特色大国外交与人类命运共同体

时政背景：2025-2026年，中国积极开展大国外交：举办上海合作组织峰会、金砖国家峰会，推动全球发展倡议、全球安全倡议、全球文明倡议落地见效，向发展中国家提供人道主义援助，为世界和平与发展贡献中国智慧和中国方案。

材料关键词：人类命运共同体、中国担当、国际合作、全球治理、和平与发展、对外开放。


对应考点

1. 时代主题：和平与发展是当今时代的主题；经济全球化是当今世界的发展趋势。

2. 中国担当：中国是一个和平、合作、负责任的大国；中国积极参与全球治理，主动承担国际责任。

3. 中国智慧：中国着眼于时代发展大势，为全球治理提出中国方案，贡献中国智慧。

4. 对外开放：我国坚持对外开放的基本国策；积极构建人类命运共同体，推动建设持久和平、共同繁荣的和谐世界。




热点10：“十五五”规划开局与中国式现代化

时政背景：2026年是“十五五”规划的开局之年，也是全面建设社会主义现代化国家的关键之年。“十五五”规划明确了未来五年我国经济社会发展的目标任务，聚焦中国式现代化建设，推动高质量发展，为全面建成社会主义现代化强国奠定坚实基础。

材料关键词：“十五五”规划、中国式现代化、高质量发展、新发展格局、民族复兴、共同富裕。

出题设问
1.我国制定“十五五”规划体现哪些教材观点？
2.为什么要以高质量发展推进中国式现代化？

对应考点

1. 中国式现代化：中国式现代化是人口规模巨大的现代化、全体人民共同富裕的现代化、物质文明和精神文明相协调的现代化、人与自然和谐共生的现代化、走和平发展道路的现代化。

2. 发展战略：我国经济已由高速增长阶段转向高质量发展阶段；构建以国内大循环为主体、国内国际双循环相互促进的新发展格局。

3. 民族复兴：实现中华民族伟大复兴是近代以来中华民族最伟大的梦想；新时代我们比历史上任何时期都更接近、更有信心和能力实现中华民族伟大复兴的目标。

4. 党的领导：中国共产党领导是中国特色社会主义最本质的特征，是中国特色社会主义制度的最大优势。

罗老师励志心语

同学们：
      可以这样说，我们老师是这个世界上唯一与你们没有血缘关系，却愿意因你们的进步而高兴，退步而着急，满怀期待，助你们成长成才，舍小家顾大家并且无怨无悔的“外人”。你们可知道：老师前世跟你们无仇，今生与你们无冤，何必要对你们严加管教而让你们心里不爽？只因为老师身上肩负的一份责任，老师想对你们的前途负责，希望你们积极上进，将来有所作为。你们要理解老师恨铁不成钢的良苦用心，积极配合，懂得感恩，励志上进，无愧青春年华！

送给进入中学胸怀梦想的你

进入中学，你需要改变一个观念：你不再是懵懂的儿童，而是逐渐走向成熟的青少年，你应该以一个中学生的标准来严格要求自己。 
进入中学，你需要一个原则：凡是有利于学习的事情，不遗余力；凡是不利于学习的事情，坚决不做。
进入中学，你需要一个健康的学习心态：平心静气，抵制诱惑。   
进入中学，你需要一个坚韧的意志品质：拼搏进取，永不放弃。
进入中学，你需要一个清醒的认识：凡是学校纪律要求的，都是利于大家学习和健康成长的，因此，要自觉遵守学校纪律。
进入中学，你需要谨防中学时代的四大陷阱：打架、赌博、恋爱和手机。

中学生良好学习习惯

1.树立远大理想，做好生涯规划，明确学习目标，制定科学合理的学习计划，一步一步具体落实，绝不拖欠学习账。
2.课前认真预习勾出疑问；上课专心听讲，积极思考，认真做笔记；课后及时复习并限时完成作业。
3.认真对待每一次作业和考试，及时反思总结，更正错题，查漏补缺。
4.学贵有恒。多读、多看、多想、多问、多练，多积累；勤于观察思考，多关注时政新闻，坚持写日记、周记，熟练写作。
5.记知识要高度集中注意力，力求眼看、口念、耳听、手写、心想，理解的基础上记忆才能长久。
6.不说闲话、做闲事、好闲游，善于利用“零星时间”，识记“碎片化知识”。
7.科学规划时间管理，讲究学习方法，注重学习效率，劳逸结合，不搞疲劳战术。
8.谦虚好学，不懂就问；勤于反思，善于总结；整理错题，总结提升。
—THE END—
