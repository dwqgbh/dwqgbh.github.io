/ 信息技术与产品展 /

信息技术与产品展（以下简称“IT展”）是高交会中规模最大的专业产品展，展览面积达30000平方米，是全球新一代信息技术汇聚展示平台，其专业化水平、参展实效、国际化程度、买家数量、著名企业等主要指标在国内同类展览同行中名列前茅。 


本届IT展将扣行业热点趋势，设立数字经济、人工智能、智能制造、智慧生活四大亮点主题展区，全面涵盖了新一代信息技术领域的热点技术、产品。


为展示IT行业在抗击新冠疫情中发挥的出色作用，以及5G应用热点和趋势，IT展继续设立IT抗疫科技专区和5G商用及物联网专区。

—

/ 环保与能源展 /


环保与能源展是高交会的重要组成部分，已成为高交会最受瞩目的品牌专业展项目之一，是战略性新兴产业企业的国家级展示平台，经过多年的倾心打造，得到海内外展商及专业观众的一致赞誉。深圳市水务局作为本展的专业指导单位之一，同时也是水务高新技术主题展合作单位。 


本展将围绕实现碳达峰、碳中和目标，汇聚符合环保和能源产业的发展趋势以及服务于绿色生活、绿色发展的相关应用，重点展示水治理、空气净化、水净化、垃圾处理、智慧环保（物联网、云计算、大数据、移动互联、5G等信息化技术在生态环保建设和治理中的应用）、环保材料、新能源、储能、充电桩、智慧电力、节能应用等领域的最新产品、技术及应用。 

—

/ 建筑科技创新展 /

建筑科技创新展将紧扣发展趋势，展示国内外装配式建筑、智能建筑、建筑节能和建筑工业化等的最新技术与产品应用实例，充分发挥平台的“行业风向标”、“技术风向标”、“创新风向标”特色，整合行业资源，将成为您推广品牌、做好战略布局、赶超竞争对手、抢占市场先机的绝佳机会。


本届建筑展主要展示：装配式技术、近零能耗建筑、智慧建造（物联网、云计算、大数据、移动互联、BIM、5G等信息化技术在建筑设计、施工中的应用）、绿色施工、绿建规划设计、绿色建材、绿色运营、绿色物业、建筑智能化与智能家居、海绵城市、智慧住建（VR看房、智能租赁平台、建筑能耗监测等）、建筑废弃物利用、既有建筑改造等建筑先进技术和产品。 


重点打造亮点展示项目“绿色之家”： 通过集成绿色建筑先进技术与产品，模拟绿色建筑实景，向观众直观展示绿色建筑中的生活和工作场景，引导老百姓感知绿色生活理念，了解绿色建筑价值。

/ 国家高新技术展 /

高交会国家高新技术展由国家部委局院组团展出，按照主题明确、专业领域清晰进行组展布展，突出展示各部委局院的科技重点发展方向、国家重大科技专项、战略性新兴产业等方面所涉及的重点领域、重大工程和重大项目等。


本届国家高新技术展重点展示的内容：我国在优化营商环境，促进国内国际双循环方面取得的成果；我国加强基础研究，加强关键核心技术攻关等方面取得的成果；我国提振工作经济，制造业数字化转型等方面的成果；我国在原始创新和关键核心技术攻关等方面的成果；强化农业科技支撑，农业关键技术攻关，数字乡村建设等方面取得的成果； 深入实施国家知识产权战略等方面的相关成果和服务项目等。


本届国家高新技术展包括6大主题展区和专馆：工业和信息化数字中国专题馆、中国中科院专馆、商务部专馆、科学技术部展区、农业农村部展区、国家知识产权局展区。

/ 创新与科研展 /

创新与科研展是高交会重点展区之一，由省市及港澳台、高校、科研院所及各类创新主体围绕特定主题展示相关高新技术产品，将重点展示战略性新兴产业、重点技术领域、未来产业、疫情催生的新业态等前瞻性、关键性的科研成果、技术和产品。本届创新与科研展将围绕“十四五”规划和各地科技发展重点领域，聚焦产业技术变革和转型优化升级，强化企业创新主体地位，推动产学研合作。截至目前，已有30多个省（自治区、直辖市）、计划单列市和新疆生产建设兵团、近30所高校意向组团参展。         


创新与科研展展示范围： 高端制造、新材料、新一代信息技术、生命科学、绿色低碳、数字经济、空天科技、深地深海、未来产业等领域的最新科研成果和先进技术，科研机构及创新中心与企业研发中心合作、实现产学研有机结合的技术创新成果及与战略性新兴产业、基础研究、产业融合相关的研究成果。

/ 先进制造展 /

随着新一代信息技术的不断应用，我国的制造业的生产模式也在发生着深刻的历史变革。高交会先进制造展继续为广大企事业单位精心打造一个集产、学、研、用于一体的综合性专业展览服务平台，助力产业升级，转变增长方式，借助装备制造振兴规划，再掀智能制造装备采购高潮，快速树立行业品牌形象。


先进制造展紧跟潮流，强势升级，细分为智能制造和工业机器人等专区，重点展示智能装备、工业机器人、智能工厂、工业个性化定制、激光产品、高端装备和工业互联网等相关领域的最新技术、产品和解决方案。

/ 初创科技企业展 /

为更好营造创新创业氛围，进一步激发经济发展活力，高交会期间特设前瞻性、开放性的初创科技企业展。作为力推科技创新品牌的国际化展会，初创科技企业展为企业免费提供技术创新、产品创新展示平台。展会整合专业投融资资源、与高交会系列经典活动配套，提供投融资对接服务，搭建初创科技企业与创投机构的交流合作平台，共同推动初创科技企业的健康、持续发展！

本届初创企业展参展商可享受6大福利：（1）项目配对洽谈活动：11月16日-19日为有融资意向的参展企业安排了项目配对洽谈活动，快速与资方进行项目对接和合作洽谈。 （2）项目融资培训会：每个参展企业提供高交会融资培训会参会名额1位。 （3）项目融资路演会：每个参展企业提供高交会融资路演会参会名额1位，参展企业具有路演优先权。（项目路演需另外收费） （4）评选优秀企业项目：组织科技专家评选团评审，推选参展企业，评选“最具发展潜力科技企业奖”、“最佳创新科技项目奖”等。 （5）获奖企业颁奖典礼：为获奖参展企业举行颁奖典礼，并对投资企业举行项目签约仪式，鼓励并支持企业发展。 （6）投资荟“微路演”：参展企业加入优质科技企业项目智库，通过深圳市投资商会常年免费路演平台，为资本方和项目方进行精准对接。

/ 航天航空科技展 /

航空航天科技展是高交会组委会于2015年新增设的特色专业展区之一，宗旨是积极响应国家航空航天强国战略，充分发挥高交会多年国际高新技术领域品牌展影响力，为我国战略新兴产业的发展提供一个集成果交易、产品展示、高层论坛、项目招商、合作交流于一体的高质量展会服务平台。 


航空航天科技展自创展以来，每年由深圳市航空航天产业协会负责承办，细分为商业航天及北斗应用展区、无人机系统展区、通用航空服务及装备展区等专区，重点展示卫星及应用、商业航天、无人机、航空装备及服务等产业领域的新技术、新产品、新应用。

—

/ 半导体显示展 /

高交会半导体显示展（原光电显示展）作为“高交会”的重要组成部分，集成果交易、产品展示、高层论坛、项目招商、合作交流于一体，自2004年在深圳会展中心成功举办首届以来，已成为中国半导体显示行业公认的国内行业规模最大、影响力最强、水平最高的盛会。


半导体显示展展示范围包括：

◆新型显示区：柔性显示、激光显示、虚拟显示模块（VR）、增强显示模块（AR）、全息显示、电子墨水（Eink）、3D显示、虹膜/指纹识别、量子点技术等。

◆显示区：液晶面板模块（LCD）、有机电激光(OLED)、QLED、Micro-led、Min-Led、LED、TP等。

◆商用显示专区：数字标牌，广告机，会议一体机，视觉图像识别系统，智能显示系统等。

◆材料区：偏光片，玻璃盖板，液晶材料，薄膜胶黏带，涂布，油墨、光学膜、光学玻璃、光学胶、导电银浆、靶材，洁净耗材等 。

◆设备检测仪器区：贴合设备、绑定设备、曝光设备、检测设备、脱泡机、清洁设备、激光设备、测试仪器视觉设备、封装设备等。 

◆半导体显示区：半导体显示设备（刻蚀机、光刻机、PVD等）；半导体显示材料（大硅片、靶材、光刻胶、抛光材料等）；半导体显示代工；半导体显示芯片设计（CPU、GPU、存储芯片、射频芯片，驱动芯片等）；半导体显示封测。

/ 智慧城市展 /

智慧城市展以“数智联动，共建共享”为主题，整合全球前瞻性的新信息技术资源，带来优秀研发成果、行业解决方案和典型案例，将为智慧城市建设者之间、智慧城市建设方案提供者与建设者之间打造深度交流合作的优质平台。

本届高交会智慧城市展设有智慧城市生态圈展区、新基建区、数字经济与产业创新展区、人工智能与智能科技体验展区、智慧城市成果展示等五大主题专区，除了让参展商能充分展示品牌和产品外，更方便参展商精准地触达有需求的专业买家、观众和采购团，促成沟通互动。

—

/ 智慧医疗健康展 /

智慧医疗健康展紧跟医疗行业热点，全面展示互联网医疗（在线问诊、医药电商、中医诊疗）、智慧防疫、5G+医疗、科技+中医、物联网+医疗、医疗机器人、智能穿戴、健康管理、智能养老、康复理疗大健康等智慧医疗健康全产业链最新技术、产品及服务。打造专业的医疗采购贸易平台、最佳的企业形象发布地以及专业信息集散地和学术、技术交流平台。
