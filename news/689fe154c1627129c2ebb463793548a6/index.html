PROGRAM

01

节目征集

// Program solicitation


往期风采展示

节目要求

一、征集对象

深信大全体学生

二、节目类型

1. 语言类（小品、相声、舞台剧等）

2. 歌唱类（独唱、对唱、合唱、音乐剧等）

3. 舞蹈类（民族舞、街舞、拉丁舞、芭蕾舞等）

4. 乐器类（弦乐、管乐、打击、拨弦牙器等)

5.其他特长（戏曲、走秀等）

三、征集条件

1. 有一定舞台驾驭能力和临场应变能力。

2. 擅长或有相关类型表演经验优先。

3. 节目内容健康向上、热情活泼。

四、报名时间

报名时间：2025年10月28日～2025年11月4日

节目初审时间：2025年11月6日




报名事项

有意愿参加节目表演的同学，在报名截止前领队添加以下其中一位负责人并扫码进一审群。

若二维码过期,请扫码联系项目负责人
