【时政专栏】11.7-11.12 时政汇总

漳州华图 2022-11-14

本周时政专栏新鲜出炉~

👇下滑查看👇

★

11月07日

★

1.第五届世界顶尖科学家论坛11月6日在上海开幕。本届论坛由世界顶尖科学家协会、中国科学技术协会、中央广播电视总台主办。主题是“科学向新 共创未来”，共举办21场重要活动。（新闻联播）

2.11月5日，我国最大推力液体火箭发动机首次整机试车圆满成功，研制取得重大突破。该型发动机设计推力500吨级，采用世界上最大的补燃循环发动机推力室，是现役120吨级液氧煤油发动机的四倍。（新闻联播）

3.2022—2023赛季国际滑联短道速滑世界杯盐湖城站比赛当地时间5日进入第二个比赛日，由李文龙、张楚桐、钟宇晨和王欣然组成的中国队，勇夺混合团体接力金牌。这是中国短道速滑队在本赛季世界杯比赛中收获的首枚金牌。（新闻联播）

4.11月6日，《湿地公约》第十四届缔约方大会部级高级别会议通过《武汉宣言》。（央视新闻）

5.11月6日，2022北京马拉松举行，创立于1981年的北京马拉松赛今年迎来第40届。在本届比赛中，获得男、女冠军的中国选手阿奴拜克·库弯和夏雨雨均刷新个人最好成绩。（新华社）

★

11月08日

★

1. 国务院新闻办公室7日发布的《携手构建网络空间命运共同体》白皮书指出，数字经济已成为推动经济增长的主要引擎之一；中国数字经济规模连续多年位居全球第二。（央视新闻）

2. 近日，工信部批复组建国家石墨烯创新中心、国家虚拟现实创新中心、国家超高清视频创新中心等3家国家制造业创新中心，分别落户宁波、南昌和成都三地。（经济日报）

3. 近日，教育部印发《特殊教育办学质量评价指南》，明确了政府履行职责、课程教学实施、教师队伍建设、学校组织管理、学生适宜发展等五方面评价内容。（央视新闻）

4. 11月7日，国家东南区域应急救援中心在广东潮州开工建设，该项目重点担负台风及其次生、衍生灾害应急救援等任务，救援范围辐射广东、广西、海南、福建、浙江、上海等地区，预计2024年建成投用。（新闻联播）

5. 11月7日，2022全球工业互联网大会在辽宁沈阳开幕，大会主题为“赋能高质量·打造新动能”。目前，中国工业互联网产业增加值规模居全球第二，达到6485.92亿美元。（新闻联播）

★

11月09日

★

1. 据联合国《世界人口展望2022》报告预计，到11月15日，全球人口将达到80亿。（经济日报）

2. 11月8日，第十四届中国航展在广东珠海开幕，空军50型武器装备在航展上集中亮相。（新华社）

3. 近日，联合国大会通过决议，指定每年11月18日为“防止儿童性剥削、性虐待和性暴力及促进受害者疗愈世界日”。（央视新闻）

4. 11月8日，2022年世界互联网大会“互联网之光”博览会在浙江乌镇开幕。本次博览会以“共建网络世界、共创数字未来”为主题，设置展览展示、新产品新技术发布等板块，有40个国家和地区的415家企业和机构参展。（新闻联播）

5. 据山东省济南海关统计，今年前10个月，山东省出口农产品1138.8亿元，出口值创历史同期新高，继续稳居全国农产品出口首位。（新闻联播）

★

11月10日

★

1. 生态环境部近日印发《生态环境卫星中长期发展规划（2021—2035年）》。《规划》提出，到2025年，初步建成监测要素基本完备的生态环境卫星体系。到2035年，全面建成响应快速、天地融合的生态环境卫星体系。（新闻联播）

2. 11月9日，2022全国中小企业数字化转型大会在山东济南开幕。大会以“创新引领发展 数字点亮未来”为主题，发布了《中小企业数字化水平评测指标》《中小企业数字化转型指南》，并从新技术、新应用等维度，系统展示中小企业数字化转型创新成果。（新闻联播）

3. 近日，中央财政提前下达2023年农业相关转移支付2115亿元，确保国家粮食安全和重要农产品稳产保供，促进农业强国建设。（经济日报）

4. 国务院关税税则委员会日前发布公告决定，自12月1日起，对原产于阿富汗等10个最不发达国家的98%税目产品实施零关税。（经济日报）

5. 中国载人航天工程办公室：11月9日14时55分，已完成既定任务的天舟四号货运飞船，顺利撤离空间站组合体，转入独立飞行阶段。（经济日报）

★

11月11日

★

1. 第五届中国国际进口博览会10日闭幕，本届进博会按一年计意向成交金额735.2亿美元，比上届增长3.9%。（新华社）

2. 国家卫生健康委、国家中医药管理局、国家疾控局近日印发的《“十四五”全民健康信息化规划》提出，到2025年，我国将初步建设形成统一权威、互联互通的全民健康信息平台支撑保障体系，基本实现公立医疗卫生机构与全民健康信息平台联通全覆盖。（经济日报）

3. 教育部消息，《关于进一步加强新时代中小学思政课建设的意见》于日前发布。意见指出，到2025年，小学思政课专职教师配备比例达到70%以上，初高中配齐专职教师。（央视新闻）

4. 2022年“全国安全用药月”11月10日在京启动，本次活动主题为“安全用药 同心同行”，其间将开展“药品安全网络知识竞赛”“中国药师周”以及“寻找身边最美药师”等活动。（新闻联播）

5. 世界入海口城市合作发展大会11月10日在山东东营举行。大会以“江河奔海·共向未来”为主题，发布了《世界入海口城市发展研究报告（2022）》《山东黄河流域生态保护和高质量发展理论与实践》等重要研究成果。（新闻联播）

★

11月12日

★

1. 国家林草局11月11日消息，中央财政将安排专项资金约360亿元，延长退耕还林还草补助期限。（央视新闻）

2. 11月11日，最高法、最高检、教育部联合发布意见，严格执行犯罪人员从业禁止制度，净化校园环境，切实保护未成年人。（央视新闻）

3. 近日，教育部等七部门印发《农村义务教育学生营养改善计划实施办法》，要求农村义务教育学校食堂不得对外承包或委托经营。（央视新闻）

4. 2022世界工业与能源互联网博览会11月11日在江苏常州开幕。本届工博会以“双网赋能新智造，数实融合向未来”为主题，新能源汽车、智能制造、重大装备等领域的上百家企业、200件最新产品集中亮相。（新闻联播）

5. 11月11日，宝马全新动力电池项目在辽宁沈阳签约，项目投资总额约为100亿元人民币。自2010年以来，宝马沈阳生产基地获得总投资超过830亿元人民币，沈阳已成为宝马集团全球最大生产基地。（新闻联播）
