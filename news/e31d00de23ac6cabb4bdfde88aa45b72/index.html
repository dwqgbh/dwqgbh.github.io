山屿海康养 2023-11-02

EVENT REVIEW


重点事件回顾

2023 第三季度

 山屿海集团 




7月重点事件

KEY EVENTS REVIEW

2023.07

山屿海自营基地健康服务全面升级，提升入住体验

山屿海基地健康服务全面升级，围绕健康促进、健康管理、体脑激活、文娱体验等领域，构建基地全新健康服务矩阵，打造暖心、便捷、多元的全新体验。


2023.07.04

集团党委及3位优秀党员荣获鄣吴政府实干争先表彰

山屿海参与鄣吴镇庆祝中国共产党成立102周年主题大会。集团党委荣获“在湖州看见美丽中国”实干争先先进集体称号，我司胡佟、王玉艳、汪云三人分别被授予“在湖州看见美丽中国”实干争先先进个人。


2023.07.11

山屿海受邀参加浙商研究中心文旅康养委员会揭牌仪式暨第一次全体会议

山屿海作为浙商研究中心文旅康养委员会副主任单位，受邀参加委员会揭牌仪式暨第一次全体会议。浙江省政府原副省长、十二届省政协副主席、党组副书记，浙商研究中心理事长孙景淼，浙江省民政厅原厅长、浙江省民政事业发展促进会会长、浙商研究中心副理事长尚清出席会议并讲话。


2023.07.13

集团董事长熊雄带领开展“喜迎八一、致敬老兵”活动

集团董事长、杰出军旅企业家熊雄带领集团党委、忘不了公益基金会走访慰问抗美援朝老兵，并组织座谈会。安吉县退役军人事务局、鄣吴镇人武部等领导一同参加。


2023.07.14

集团党委隆重召开党员大会，开展红色教育

集团党委隆重召开党员大会，并赴南京深入开展红色教育活动，教育引导广大党员以崭新面貌、一流业绩、更大作为推进山屿海高质量发展。



2023.07.20

四川广安市人大常委会党组副书记、副主任尹黎明一行赴山屿海幸福城考察

四川广安市人大常委会党组副书记、副主任尹黎明一行赴山屿海幸福城考察，对山屿海的运营理念、运营方式、乡村共富成效非常认同，期待山屿海能到广安等西部红色地区，带动当地康养产业发展。





8月重点事件

KEY EVENTS REVIEW

2023.08.01

南宁市工商业联合会、总商会领导一行莅临山屿海考察，探讨康养经验复制推广

南宁市委统战部副部长，市工商联党组书记、常务副主席张献辉率市工商联、广西大学、南宁商会相关领导莅临山屿海，希望借助山屿海康养产业经验优势，在南宁打造全新康养体验。


2023.08.02

安徽宣城市委书记李中一行莅临山屿海调研，肯定集团发展成效

安徽宣城市委书记李中一行到山屿海实地调研，详细了解山屿海康养产业布局及运营情况，高度认可山屿海在乡村振兴，共同富裕，全民健康产业的辛勤付出和取得的成效。


2023.08.03

山东威海市文登区人民政府副区长顾甜甜一行到访山屿海，探讨合作新模式

山东威海市文登区人民政府副区长顾甜甜一行赴山屿海集团实地考察，希望充分发挥山屿海康养产业标杆企业优势，与威海文登区创新合作模式，谱写新篇章。


2023.08.13

集团董事长熊雄受邀出席2023浙江省在沪商会工作联席会议

集团董事长熊雄作为上海市浙江商会副会长、上海市长兴商会会长，受邀出席2023浙江省在沪商会工作联席会议。并代表上海市长兴商会获领“浙江省在沪特色商会”奖项。


2023.08.15

山屿海旅游与杭州新湖绿城物业达成旅游服务战略合作

山屿海旅游与杭州新湖绿城物业达成旅游服务战略合作，双方将在文旅资源推介、线路宣传推广、客源互送等方面开展全方位深度合作。


2023.08.16

卢旺达驻华大使来访山屿海，与集团董事长熊雄共话合作

卢旺达大使詹姆斯·基莫尼奥一行莅临山屿海集团考察，表达了对山屿海的高度信任以及合作愿望，并诚挚邀请集团董事长熊雄赴卢旺达考察。


2023.08.20

集团董事长熊雄应浙江省民政厅邀请赴滇参加南亚博览会，并考察养老服务

集团董事长熊雄应浙江省民政厅邀请，赴滇参加南亚博览会、考察养老服务。考察团由浙江省民政厅党组成员、副厅长方仁表带队，先后考察了昆明市相关养老社区，学习养老服务新做法。


2023.08.26

山屿海健康首期健康沙龙圆满举行，促进多方合作共同发展

山屿海健康主办的“慢病管理新机遇”首期健康沙龙顺利举行，邀请各行业领域数十位高管、专家出席。期待彼此携手合作、场景打通，形成多方协作参与的健康产业发展格局。


2023.08.28

山屿海与海铭康达成健康管理和旅居康养业务合作

山屿海与湖北海铭康咨询服务有限公司就健康管理和旅居康养达成合作，完成健康管理体系进驻。山屿海将通过完成“健康管理”和“健康生活”优势体系双输出，赋能打造高品质健康管理服务。



2023.08.31

广德市政府领导一行莅临山屿海考察，洽谈康养服务

广德市政府民政局党委书记、局长任武祥，桐汭街道办事处主任方荣波一行莅临山屿海幸福城考察，就康养综合体项目运营、社区健康养老服务等话题展开深入交流。




9月重点事件

KEY EVENTS REVIEW

2023.09

山屿海开展“全民健康生活方式宣传月”系列活动

9月是“全民健康生活方式宣传月”，山屿海响应国家相关政策，多渠道举办线上健康答题打卡、健康体验官招募等系列健康活动，强化健康知识科普，提升全民健康素养，深入推进健康中国战略。


2023.09

山屿海健康全新品牌形象广告正式亮相多省市高铁站

山屿海健康全新品牌形象广告开启投放，覆盖上海、浙江、江苏、安徽等省市数十家高铁站服务厅，依托高铁媒体，撬动亿万旅客关注度，为山屿海健康发展积蓄增长势能。


2023.09.05

集团CEO侯欢一行赴山东文登考察调研，开启“相约文登·共赴精彩”新篇章

集团CEO侯欢率考察团赴山东文登调研，文登区委副书记、区长单浩仁接待陪同，介绍文登文旅资源，康养产业发展空间，希望山屿海发挥康养产业优势，以丰富的康养资源、成熟的运营模式，助推文登区经济社会高质量发展。


2023.09.09

山屿海健康汇携手山屿海医生开展肺结节专项科普义诊活动

山屿海健康汇携手山屿海医生开展肺结节专项科普义诊活动，特别邀请复旦大学医学博士刘庆华莅临现场，展开科普分享和一对一见诊，为长者们送去健康福利。


2023.09.12

山屿海大小姐高质量发展专题研讨会召开，正式确认品牌升级

山屿海大小姐召开研讨会，集团董事长熊雄、熊雄商学主理人陆小妹、总裁办主任陶然、山屿海大小姐CEO戚施明及团队骨干参加会议，正式确认品牌升级为“山屿海大小姐”，并明确下季度重点工作。


2023.09.13

集团CEO侯欢一行赴浙江中医药大学交流，商讨更多方面校企合作

集团CEO侯欢一行赴浙江中医药大学参加校企战略合作交流，人文与管理学院院长、党委副书记林敏，党委书记、副院长梁泽华一行接待。基于大健康产业发展趋势和慢病健康管理需求，希望校企双方通过合作，赋能和提升长三角慢病管理行业高质量发展，并辐射全国。


2023.09.14

山屿海健康参加第十四届亚洲营养大会，共话营养学科未来发展

山屿海健康参加第十四届亚洲营养大会，与来自国际营养科学联合会、世界卫生组织、联合国粮农组织等国际组织，以及来自学术组织、研究机构、高等院校的专家学者及行业代表3000余人一同，探讨前沿营养信息，共话营养学科未来发展。


2023.09.20

山屿海党委委员、康养联席CEO田龙一行参加浙江中医药大学产业主题交流座谈

山屿海党委委员、康养联席CEO田龙一行赴浙江中医药大学参加“中医药+康养”产业主题交流座谈。田龙表示，希望与浙江中医药大学进一步加强产教融合，在大健康领域成果产出和转化方面实现双赢。


2023.09.22

山屿海健康第二期健康沙龙圆满举行，共商新机遇背景下合作模式

山屿海健康第二期健康沙龙会举行，与多行业领域高管、专家共商在慢病管理前所未有的机遇期背景下，如何开展深入、紧密合作，落实国家对慢病健康管理的战略规划，加速推动优质管理体系的建设与推广。



2023.09.23

集团董事长熊雄受邀参加河南省光山县首届根亲文化节

集团董事长熊雄受邀赴河南信阳参加光山县首届根亲文化节，县委书记王建平陪同。熊雄希望山屿海能为推动光山全民健康、乡村振兴、高质量发展作出积极贡献。




10月重点事件

KEY EVENTS REVIEW


2023.10.01

“溪语林居·温泉公寓”国庆封顶大吉

国庆当日，幸福城大型康养社区组团项目之一“溪语林居·温泉公寓”封顶完成，项目预计2024年底前精装修交付使用，将成为承载美好旅居生活和活力健康体验的优选之地。


2023.10

山屿海葛洋荣获湖州市两新组织“百名奋斗者”称号

浙江山屿海旅游发展股份有限公司基建部经理、党务工作者葛洋荣获湖州市两新组织“百名奋斗者”称号。在安吉县总工会主办主题演讲比赛中，葛洋作为全县几家企业代表参赛，并荣获三等奖。


2023.10.08

康养杭州区营销中心第三季度总结表彰大会召开

康养杭州区营销中心第三季度总结表彰大会召开，公司高管及全体营销伙伴共同总结成绩，并对第三季度中取得佳绩的个人和团队进行表彰。


2023.10.10

集团董事长熊雄参加康养上海区四季度誓师动员大会

康养上海区四季度誓师动员大会盛大举办。集团董事长熊雄，集团CEO侯欢，党委委员、上海支部书记、集团高级副总裁王振宇，党委委员、康养联席CEO黄波，熊雄商学主理人陆小妹，党委办主任、集团总裁办主任陶然参加，共同谱写山屿海高质量发展新篇章。


2023.10.11

山屿海健康与树兰互联网医院共建健康管理合作签约仪式圆满举行

山屿海健康与树兰互联网医院举行共建健康管理合作签约仪式。山屿海健康将依托对方在医疗线上线下一体化综合运营的布局优势、专业智慧医疗体系共建基础，探索全新增长路径，赋能旅养康医各板块发展。


2023.10.16

企业上市，政府护航。鄣吴镇党委书记梅本炜，安吉金融办主任马权一行调研山屿海

鄣吴镇党委书记梅本炜，安吉县金融办主任马权一行赴山屿海调研集团海外上市工作。就企业上市过程中的具体要求与思路进行深入探讨。安吉县金融办将继续提供精细化服务指导，为企业顺利上市保驾护航。


2023.10.18

集团董事长熊雄率上海市长兴商会考察团赴汶川县调研，并捐赠20万元帮扶资金

集团董事长熊雄带领上海市长兴商会考察团赴汶川县开展项目调研、助力发展工作，并向汶川县捐赠20万元帮扶资金。熊雄表示，希望双方在交流互动中加强了解、增进友谊，在深化合作中取长补短、携手共赢。


2023.10.19

山屿海健康营200期答谢会盛大举行，共庆美好时刻

山屿海健康营200期感恩答谢会隆重举行，数百位朋友共同见证这一具有纪念意义的时刻。未来山屿海将持续打造安全、专业、值得信赖的健康品牌，为每个人构筑健康堡垒，拥抱理想生活。


2023.10.20

山屿海健康亮相2023浙江省医师协会康复医师分会大会

由浙江省医师协会康复医师分会及浙江大学医学院附属第一医院主办的2023浙江省医师协会康复医师分会大会在杭州举行。山屿海健康受邀参会，与数百名参会者一同进行交流探讨，并进行健康品牌宣传展示。


2023.10.29

山屿海银发歌王大型中老年梦想歌唱大赛正式启幕

山屿海银发歌王大型中老年梦想歌唱大赛正式启幕，活动聚焦50+中老年群体，以“线上+线下”同步进行的模式，采用歌唱比赛与真人秀跟拍的方式，选拔出优秀中老年歌唱选手，打造展现文艺风采、促进相互交流、寻找志同好友、实现自我价值的全新乐龄艺术表演舞台，建立中老年生活&歌唱基地标杆新IP。


2023.11.5

山屿海参展第六届中国国际进口博览会，向世界展示健康产业创新亮点

11月5日至10日，山屿海亮相第六届中国国际进口博览会，并受邀参加中国大健康产业未来发展高峰论坛。借“世界之窗”加深与健康产业生态圈的联结，与行业伙伴同心共进，与大健康市场同频共振。
