北一机床召开2022年全面从严治党(党建)暨纪检工作会

北京北一机床有限责任公司 2022-04-02


3月31日，北一机床采用视频会议形式召开2022年全面从严治党（党建）暨纪检工作会。会议认真传达习近平总书记在十九届中央纪委六次全会上的重要讲话精神和京城机电全面从严治党（党建）暨纪检监察工作会精神，总结回顾了北一机床2021年全面从严治党、纪检相关工作，部署2022年重点任务。






京城机电副总经理、北一机床党委书记、总经理李忠波作题为《统一思想深融合强引领，凝心聚力抓改革促发展》的全面从严治党（党建）工作报告，回顾了2021年北一机床全面从严治党各项工作，并对下一步工作开展提出具体要求。

京城机电副总经理，北一机床党委书记、总经理

李忠波作报告

李忠波同志充分肯定了北一机床2021年全面从严治党取得的成绩，他指出：2021年是中国共产党成立100周年，也是公司“十四五”战略开局之年，公司党委始终坚持以习近平新时代中国特色社会主义思想为指导，深入学习贯彻党的十九大及十九届历次全会精神，按照中央、市委、市国资委党委及京城机电党委的部署要求，紧紧围绕中心工作，发挥了党委“把方向、管大局、保落实”的作用，带领全体员工有力推动了北一机床“十四五”战略起好步、开好局。并从以下四个方面进行回顾：

一

聚焦统一思想认识，把握战略发展方向。

二

聚焦压实主体责任，明确战略实践路径。

三

聚焦打造高效团队，提供战略基本保障。

四

聚焦凝聚发展正能量，营造战略文化氛围。

李忠波同志强调，2022年公司党委将围绕北一机床改革调整的经营实际，从以下几个方面持续推进全面从严治党的各项工作。

一 //  持续抓好习近平新时代中国特色社会主义思想学习贯彻。

二 //  持续巩固拓展党史学习教育成果。

三 //  持续抓好管党治党责任落实。

四 //  持续强化党风廉政建设。

五 //  持续凝聚改革发展合力。


北一机床纪委书记李金勇作报告

北一机床纪委书记李金勇作题为《深入推进全面从严治党，为北一机床开创改革发展新局面提供坚强纪律保障》的纪检工作报告。回顾了2021年北一机床党风廉政建设和反腐败工作，分析研判公司反腐败斗争面临的形势。

李金勇同志指出，2021年是“十四五”战略规划的开局之年，公司纪委在京城机电纪委和公司党委领导下，坚持以习近平新时代中国特色社会主义思想为指导，深入学习贯彻党的十九大和十九届历次全会精神、中央纪委五次全会精神和市纪委十二届六次全会精神，认真贯彻落实京城机电全面从严治党工作要点和纪检监察工作要点、政治监督工作要点，结合疫情防控工作整体部署、市国资委规自领域巡察和市国资委党委巡察及巡察“回头看”问题反馈整改，紧紧围绕公司“十四五”战略宣贯、实施和改革调整重点工作，发挥监督保障执行、促进完善发展作用。

报告分别从“监督（同级和下级党组织）贯彻执行上级重大决策部署情况；监督（各级领导人员、领导干部）遵守党章党规党纪、廉洁从业情况；对“三重一大”相关事项决策制度执行情况进行监督；开展专项监督检查情况；开展日常监督的情况”五个方面对2021年纪检工作进行详细总结，同时通报2021年公司纪委开展执纪问责工作情况。

李金勇同志强调，2022年纪委将围绕中心工作积极作为，持续深化不敢腐、不能腐、不想腐体系建设，更好发挥监督保障执行、促进完善发展作用，为公司开创改革发展新局面提供坚强纪律保障。从“加强政治理论学习，持续提高政治站位；聚焦职能职责，压实压紧‘两个责任’；加强廉洁教育，重视教育成果转化；持续强化监督，提升监督质量效果；严格执纪问责，释放震慑强烈信号；聚焦队伍建设，打造德才兼备铁军”六个方面对做好纪检工作作出部署。

 会议要求各单位及时将会议精神进行传达贯彻，持续统一思想、引领行动，立足全面从严治党（党建）和纪检工作的整体要求，对标对表细化措施，融入中心工作逐项安排，责任层层分解倒逼落实。在完成“十四五”战略的阶段性指标任务中、在推进北一持续改革进程中、在当前正在积极推进的精益化管理中发挥各级党组织的主体作用，不断促进北一高质量发展。

 公司全体领导班子成员、中层领导干部、各单位纪检委员参加会议。
