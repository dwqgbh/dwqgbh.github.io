01

中央风向

1、2025年9月10日中国国际服务贸易交易会/第十二届一带一路文旅科技高质量发展大会将首发190多项新产品、新成果


8月27日，国务院新闻办公室举行新闻发布会，介绍中国服务贸易发展和2025年服贸会筹备工作进展情况，并答记者问。北京市副市长司马红介绍，本届服贸会更加突出优质服务供给，展示优质服务品牌，百余家企业将首发190多项新产品、新成果。

第十二届一带一路文旅科技高质量发展大会将于9月10日在北京首钢园区冬奥广场开幕。


2. 国务院印发《关于深入实施“人工智能+”行动的意见》


国务院日前印发《关于深入实施“人工智能+”行动的意见》（以下简称《意见》）。《意见》提出加快实施6大重点行动，强化8项基础支撑能力，包括提升模型基础能力、加强数据供给创新、强化智能算力统筹、优化应用发展环境、促进开源生态繁荣、加强人才队伍建设、强化政策法规保障、提升安全能力水平等。《意见》要求，坚持把党的领导贯彻到“人工智能+”行动全过程，国家发展改革委要加强统筹协调，各地区各部门要结合实际、因地制宜抓好贯彻落实，确保落地见效。要强化示范引领，适时总结推广经验做法，加强宣传引导，广泛凝聚社会共识，营造全社会共同参与的良好氛围。


3. 第四批国家级抗战纪念设施、遗址名录公布


为隆重纪念中国人民抗日战争暨世界反法西斯战争胜利80周年，经党中央、国务院批准，近日，国务院印发通知公布第四批34处国家级抗战纪念设施、遗址名录。各地区、各有关部门要进一步加强抗战纪念设施、遗址的保护管理，做好抗战史料文物和英烈事迹的发掘整理、宣传展示工作，广泛开展群众性拜谒参观、祭扫纪念活动，组织学习抗日英烈的英雄事迹，教育引导广大群众特别是青少年充分认清日本军国主义侵略者犯下的罪行，牢记中华民族抵御侵略、奋勇抗争的历史以及中国人民为世界反法西斯战争胜利作出的巨大民族牺牲和重要历史贡献，大力弘扬伟大的民族精神和抗战精神，进一步增强民族凝聚力、向心力，为以中国式现代化全面推进强国建设、民族复兴伟业不懈奋斗。


4. 全国工商联发布“2025中国民营企业500强”


8月28日，全国工商联在辽宁沈阳发布“2025中国民营企业500强”。 在这份代表中国高水平民营企业的榜单中，一批深耕文化、旅游、休闲消费领域的民营企业崭露头角。此外，榜单中许多知名科技企业和平台型企业也正以投资、合作、生态布局等方式，深度涉足文化和旅游业。如京东集团、阿里巴巴（中国）有限公司、腾讯控股有限公司、北京嘀嘀无限科技发展有限公司（嘀嘀出行）等。如今的文化和旅游业，已不再是单一的“吃、住、行、游、购、娱”链条，而是一个融合科技、文化、消费与城市发展的复杂生态系统，而这些500强中的“老玩家”和“跨界玩家”，正在重新定义文旅的边界与未来。

5. 2025中国国际大数据产业博览会在贵阳开幕


8月28日，由国家数据局主办、贵州省人民政府承办的2025中国国际大数据产业博览会在贵州贵阳开幕。本届数博会以“数聚产业动能 智启发展新篇”为主题，设置数智领航、数智基建、数智服务、数智应用、数智创新、数智体验6大主题展馆，吸引375家中外企业参展；围绕数据要素、数据产业、数字经济、数据基础设施、国际合作5大核心板块，举办26场交流活动。在深入实施“人工智能+”行动的背景下，今年的数博会充分展示数据要素与人工智能融合创新的最新成果，同时举办数据标注、数字经济等供需对接活动。


6. 第十五届中国-东北亚博览会在长春开幕

02

地方资讯


1. 2025北京文化论坛将于9月23日至24日举办


经国务院批准，由中共中央宣传部、北京市委、北京市政府共同主办的2025北京文化论坛，将于9月23日至24日在北京举办。2025北京文化论坛以“传承·创新·互鉴”为永久主题，以“文化和科技融合发展”为年度主题，论坛还将举办重要成果发布专场活动、文化产业投资人大会等配套活动。


2. 第33届广州博览会意向签约总额400.81亿元


为期三天的第33届广州博览会24日闭幕，累计签订经贸合作项目83个，意向签约总额达400.81亿元人民币。本届广州博览会加大采购商精准邀请力度，提升供需匹配度，力促现场成交与合作落地，累计超4000家商协会及企业观展采购，吸引与会观众16万人次。第十一届国际发明展同期举办，吸引了来自全球30多个国家和国际组织参与，展出2000多项中外发明成果。


3. 北京市文化和旅游局联合百度地图发布“音乐地图”


近日，北京市文化和旅游局联合百度地图发布“音乐地图”，上线首批20个“京彩户外音乐会”点位，丰富市民游客演艺体验，增强城市音乐氛围。首批上线的20个音乐会项目分布于北京市11个区，从8月持续至年底，将陆续演出超过300场。演出场地类型多元多样，为市民和游客带来丰富的视听享受。


（图源：文旅北京）


4. 2025湖南文旅产业招商推介活动（文旅科技专场）在京举办


8月21日，由湖南省文化和旅游厅、怀化市人民政府主办的2025湖南文旅产业招商推介活动（文旅科技专场）在北京举办。活动共推介招商项目418个，总投资3386.07亿元。此次推介会为京湘两地的文旅合作搭建了坚实桥梁，吸引北京科技企业及投资机构积极参与湖南文旅产业数字化转型，也为第五届湖南旅发大会的筹备工作奠定了良好基础。


5. 第九届河北省旅发大会主会场活动将于9月15日至17日举办


8月20日，从省政府新闻办召开的第九届河北省旅游产业发展大会新闻发布会上获悉，本届大会采用“主分会场联动”模式，邢台设主会场，石家庄、张家口、保定、邯郸设分会场。其中，主会场活动于9月15日至17日举办，各分会场活动将在9月错时开展。此外，在第九届省旅发大会前后，五市还将联动开展5个系列100余场“太行周末”特色主题活动，全方位展示推介太行山沿线优质文旅资源产品，形成持续性宣传推广热潮。


6. 第五届湖南旅游发展大会将于9月16日至18日在怀化举行


8月7日，第五届湖南旅游发展大会新闻发布会在长沙举行。发布会透露，本届大会将于9月16日至18日在怀化启幕，以“美美与共 合作发展 打造湖南文旅融合发展新高地”为主题，通过创新办会模式、深化区域联动、激活市场活力，为湖南文旅产业在新形势下破局指明路径，推动文旅产业的“逆周期”潜力转化为高质量发展的动能。


7. 甘肃、青海两省多部门共同制定《甘青环线旅游畅行安全守护八项措施》


近期，甘肃、青海两省公安厅联合交通运输厅、文化和旅游厅共同制定了《甘青环线旅游畅行安全守护八项措施》，开展交通安全工作联动、交通隐患排查共治、交通组织优化提升、交通违法联合打击、旅游企业闭环监管、旅游信息数据共享、旅游应急协同处置、安全宣传融合协作等工作，为甘青环线旅游高质量发展提供坚实的安全保障，为游客创造安全、有序、畅通的出行环境。


8. 2025广东文旅推介大会系列活动将于9月开启


8月26日，广东省文化和旅游厅在广州举行2025广东文旅推介大会系列活动新闻发布会。会议宣布将于9月12—14日在广州全面启动广东文旅推介大会暨金秋文旅消费季、广东文化和旅游产业招商暨投融资对接会、广东国际旅游产业博览会等多项活动，以多元化举措推动文旅产业高质量发展。


9. 贵州省文化和旅游厅发布26家严重违法违规旅行社名单


近日，贵州省文化和旅游厅发布全省严重违法违规旅行社名单，旨在进一步规范全省旅游市场秩序，推动旅游产业高质量发展，维护公平竞争的旅游市场环境，严厉打击违法违规经营行为，切实保障旅游者和合法经营者权益。

国际信息


1. 2025年上海合作组织峰会将举行


2025年上海合作组织峰会将于8月31日至9月1日在天津举行，中国国家主席习近平将同20多位外国领导人和10位国际组织负责人聚首海河之滨。习近平主席将主持上合组织成员国元首理事会会议和“上合组织+”会议，并分别在上述两场会议上发表主旨讲话。多国人士热切期待上合组织天津峰会举行，相信峰会将进一步凝聚“上合组织大家庭”合作共识，为上合组织擘画发展蓝图提供重要指引，推动上合组织朝着构建更加紧密命运共同体的目标阔步迈进。

4. 中国—东盟旅游安全论坛在缅甸仰光成功举办

2. 第二十届中国长春电影节开幕式在吉林长春隆重举行


第二十届中国长春电影节由中央广播电视总台、吉林省人民政府主办，长春市人民政府承办，主题为“新时代·新摇篮·新力量·新征程”。设开闭幕式、电影评奖、电影展映、电影交流、致敬“摇篮”、产业融合等六大板块。共征集150部影片参展参赛，其中93部角逐金鹿奖。在本届电影节举办期间，长春市通过构建“一街引领，全城联动”的文旅活动体系，着力打造三大板块50余项特色文旅活动。“电影长街”文旅嘉年华系列活动、“清爽之约”电影消夏系列活动、“光影长春”电影惠民系列活动。


3. Meta即将推出首款面向消费者的智能眼镜Hypernova


Meta计划于9月的年度Connect大会上发布代号为Hypernova的新款智能眼镜。这是该公司首款面向消费者的智能眼镜，配备显示屏，显示屏将安装在合适的镜片上。这款眼镜由Meta与EssilorLuxottica合作研发，预计售价约为800美元。预计这款眼镜将与一款可通过手势控制眼镜的腕带一同亮相。

4. 百度正式发布AI搜索APP「梯子AI」：主打无广告智能搜索


8月26日，据报道，百度旗下的AI搜索App「Tizzy.ai」已完成了前期的测试，并正式更名为「梯子AI」上架。据介绍，梯子AI定位为智能搜索助手，依托百度多个大模型能力开发而成，主打无广告智能搜索，整合深度思考、资源检索及影视娱乐功能，致力于为用户提供「简单搜索，一触即达」的极致体验。



5. 掌阅科技2025上半年净亏损1.6亿元，短剧收入同比增长149.09%成第一大业务


8月27日消息，掌阅科技披露2025年半年报：公司上半年实现营业收入15.26亿元，同比增长14.58%；归属于上市公司股东的净亏损1.6亿元，上年同期亏损4770.72万元；基本每股收益-0.36元。短剧成掌阅科技第一大业务，短剧等衍生业务上半年收入为8.38亿元，较去年同期相比增长149.09%。掌阅科技表示，报告期内，公司持续大力发展短剧等衍生业务，构建了「国内扩张、海外突破」双轮驱动格局，国内外双线发展模式带来短期投入的持续增加，对公司盈利能力形成一定拖累；同时，数字阅读平台业务收入下滑，但短期内容成本仍保持刚性，进一步对公司盈利能力造成负面影响。
