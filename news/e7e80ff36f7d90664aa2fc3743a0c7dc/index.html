【月鉴南充】2025南充七月大事记

方志南充 2025-10-15





九月大事记







8月31日至9月1日

// 南充市选手胡力文（南充文化旅游职业学院教师）、李泉霖（南充市方志馆讲解员）在2025年四川省社科普及大赛中分获二、三等奖。





9月1日

// 南充市启动G318高坪区青莲至隆兴段改建工程，实施交通管制至2027年5月1日，同步发布货车禁行及绕行方案，助力区域路网升级。


// 南充高坪区嘉陵江凤仪湾旅游度假区获评省级旅游度假区。





9月1—2日

// 全省东西部协作工作现场培训班在南充市举行。





9月3日

// 南充市籍烈士文明兴之子文建受邀烈士遗属代表，出席纪念中国人民抗日战争暨世界反法西斯战争胜利80周年阅兵仪式观礼。

9月3日上午，烈士遗属代表文建参加中国人民抗日战争暨世界反法西斯战争胜利80周年阅兵仪式。


// 据四川省中医药管理局公布2024年度三级中医医院评审结论，南充阆中市、南部县、营山县三地中医医院通过三级评审并报请国家备案，正式跻身国家三级甲等中医医院行列。





9月4日

// 由中宣部组织的2025年“活力中国调研行”主题采访团走进南充，中省主流媒体聚焦南充在服务消费、文旅融合等领域的创新实践进行深度调研。


// 南充市吉利四川商用车、四川弘鑫云创智造两家企业入选工信部《2025年5G工厂名录》。

吉利四川商用车有限公司生产车间。南充市融媒体中心记者乐校臣摄


// 南充市21家庭院在四川省天府美丽庭院“她”地图发布活动中成功入选，涵盖巾帼工坊、亲子时光、巾帼雅居等多个主题。





9月4—7日

// 南充市组织“好充食”等名优特产品亮相第十一届四川农业博览会，达成签约及成交总额逾1500万元。

第十一届四川农业博览会上，艺术家进行国家级非物质文化遗产代表性项目川北大木偶表演。





9月5日

// 南充阆中市《客来阆中》环江实景演艺项目入选四川省消费新场景运营激励拟支持名单。





9月8日

// 南充市地理标志品牌“南充丝绸 百年六合”成功入选2025年日内瓦世界地理标志产品品牌分销服务大会。


// 南充阆中市、仪陇县入选“2025中国县域旅游综合竞争力百强县市”。





9月9日

// 在成渝地区双城经济圈建设先进表彰中，南充市经济和信息化局、南充高新技术产业园区管理委员会、四川吉利商用汽车有限公司3个集体，以及许钊、黄海燕、王雄3名个人荣登榜单。





9月10日

// 南充市举行庆祝第41个教师节大会，市委副书记、市长黄志权代表市四大班子向全市教师及教育工作者致以节日问候，并对先进集体与个人进行表扬。


// “发展中国家公共服务与减贫官员研修班”在南充市启动案例教学，来自10国34名官员将通过实地考察，交流学习南充在减贫与乡村振兴方面的实践经验。

9月10日，研修班在南充开展案例教学





9月11日

// 南充市成功举办2025四川—南亚东南亚国家工商发展合作大会“南充行”产业投资推介大会。

9月11日上午，四川—南亚东南亚工商发展合作大会“南充行”正式开启





9月12日

// 南充市两家医院入选四川省首批研究型医院与研究型病房建设单位名单，其中南充市中心医院入选研究型病房建设单位，川北医学院附属医院入选研究型医院培育单位。


// 南充市保宁醋公司、张飞牛肉公司等68家企业被认定为第十二批农业产业化省级重点龙头企业。


// 南充“好充食”品牌企业在杭州举办的2025长三角暨对口地区农产品产销对接洽谈会上与台州企业达成合作，签署金额500万元的农产品采购协议。





9月15日

// 南充市2025年国家网络安全宣传周活动启动仪式在四川省大木偶剧院举行。


// 南充市四川中其盛汽车科技有限公司、四川越创铝业有限公司、四川东阁科技有限公司三家企业入选2025年度四川省规上工业企业研发平台建设专项资金拟支持名单。


// 南充市经济开发区获评“2024年度全省优秀开发区”。





9月15—18日

// 南充市组织26家经营主体携80余款农产品参加全国名特优新农产品产销对接活动（厦门），西充玫瑰花茶获评最受欢迎产品奖，并达成意向销售额超350万元。





9月16—17日

// 省政协副主席林书成到南充市就营商环境工作开展监督性视察，先后深入中科九微、吉利四川商用车等企业及“双蓝护企”工作室进行调研。





9月18日

// 南充市房管局等7部门联合出台《进一步推进住房政策与民生政策联动的若干措施》，推出31项具体举措。





9月19日 

// 南充市中等职业技术学校及党委正式揭牌，市委常委、宣传部部长廖雪梅，副市长欧阳梅共同出席揭牌仪式。





9月20—21日

// 2025年川渝飞盘公开赛在南充市体育中心体育场举行。





9月22日

// 2025年“中国农民丰收节”南充市庆祝活动启动仪式在南部县启幕。


// 南充市开通40路公交线路（南充大学城—西山风景区），并优化调整5路、13路公交线路。





9月23日

// 四川省人民政府公布第八批全国重点文物保护单位和第九批省级文物保护单位保护范围，南充七宝寺（嘉陵区）、南部永安庙大殿（南部县）、南部观音庵大殿（南部县）位列其中。


// 南充市体育总会第一届会员（代表）大会暨成立选举大会在北湖宾馆举行。





9月23日至24日

// 市委书记张冬云到蓬安县、顺庆区专题调研工业企业产能利用提升情况并主持召开座谈会。





9月24日

// 2025年四川省网络普法天府行暨网络举报辟谣宣传周南充站活动顺利举办。


// 南充市高坪区、南部县、营山县三地入选2025年全国县域商业“领跑县”名单。


// 南充至广安铁路建设启动活动在广安举行。





9月25日

// 在四川省“走好新时代的长征路”主题演讲活动中，南充推送的4件作品获奖，市委宣传部、市委党史研究室、团市委获优秀组织奖。





9月26—29日

// 南充成功举办第十四届中国升钟湖国际钓鱼大赛及国际舟钓大赛。国家体育总局监察局原局长、中国钓鱼运动协会会长施泽华致贺词，省政协副主席刘成鸣宣布开幕，市委副书记、市长黄志权致辞，市政协主席廖伦志出席开幕式。


// 南充市人民政府对韩顺清等4名见义勇为个人及杜炳福等1个见义勇为群体（共4人）进行奖励。





9月28日

// 市委书记张冬云在南充会见永荣控股集团有限公司董事长吴华新一行，双方就深化务实合作达成共识。






9月29日

// 南充市嘉陵区之江小学、营山县档案馆获授“四川省民族团结进步模范集体”称号，阆中市博树回族乡乡长马勇、顺庆区委编办副主任贺丹获授“四川省民族团结进步模范个人”称号。



// 南充市组织参加2025年“川超”联赛。同日，南充丝绸源点队以4:0成绩客场战胜巴中红叶队。

南充丝绸源点队客场与巴中红叶队比赛精彩瞬间。郑钦豪 摄





9月30日

// 南充市在烈士陵园隆重举行向人民英雄敬献花篮仪式，张冬云、黄志权、廖伦志、王双全等市领导出席。


// 南充市举行第四届中国（四川）国际熊猫消费节南充站暨2025年“金九银十”消费促进活动启动仪式。


// 南充市南充经开化工园区入选生态环境部公布的化工园区突发水污染事件环境应急三级防控体系建设暨“一园一策一图”试点名单（第二批）。
