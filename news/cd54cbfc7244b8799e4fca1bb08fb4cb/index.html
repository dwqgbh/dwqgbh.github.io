寓你相遇丨智能制造与信息学院2025-2026学年“文明宿舍”评比活动来啦!

智能制造与信息 2025-11-21


宿舍是学生校园生活的“第一社会、第二家庭、第三课堂”，是培养和睦相处能力、塑造良好生活习惯的重要阵地。为营造“干净整洁、温馨和谐、各具特色”的宿舍文化氛围，增强学生集体归属感与创造力，引导学生以“寓你相遇”为缘，在日常相处中传递友善，在空间共建中彰显个性，现依托易班平台策划“寓你相遇”文明宿舍评比活动，通过“线上展示+线下检查+互动评选”形式，挖掘优秀宿舍文化案例，传递青春正能量。  



一、活动组织


主办单位：广东碧桂园职业学院智能制造与信息学院 

承办单位：学院学生工作办公室、易班工作站



二、活动目标 

活动以“相遇”为情感纽带，将“和睦、整洁、特色”三大核心融入学生日常，通过“干净整洁”环境展示，引导学生养成良好卫生习惯与空间管理能力，推动学生宿舍从“物理空间”向“文化家园”升级，培育“团结友爱、自律自理、创意表达”的宿舍文化。



三、活动主题

寓你相遇



四、活动时间

2025年11月22日—12月15日



五、活动对象  

智能制造与信息学院全体在校学生宿舍  



 六、活动平台 

易班APP（主要载体） 

主入口：易班“优课课群”（课群码：7EQXG24C，入群后查看活动详情页）； 

辅助平台：易班轻应用（用于投票互动）。 



七、活动内容与形式    

（一）两大主题赛道（赛道一必选基础项，赛道二自愿选择原则）

赛道一：「寓净·焕新」——宿舍整洁展示

 { 内容要求 } 

展示宿舍日常或“最美瞬间”的干净整洁状态（如书桌收纳、床铺整理、地面清洁、公共区域分工等），需体现“自律管理”与“空间美学”。

 { 活动形式 } 

线下（由学院学生工作室牵头，团委学生会、学生干部组织参与检查）

 { 评选重点 } 

 整洁度（无杂物堆积）、空间利用率（收纳创意）、日常维护痕迹（非临时摆拍）。具体评分细则以附件为准。

赛道二：「寓彩·独韵」——宿舍装饰特色风采展示

 { 内容要求 } 

内容要积极向上，符合社会主义核心价值观。需展示宿舍独特的文化标签或才艺创意。

 { 主题一：文艺风尚型 }

以“文艺与生活美学”为核心，将宿舍成员书法、摄影、合作画画作品作为核心装饰。利用低功率小灯串、手工绘画、粘土摆件、编织物件或挂件、DIY手工作品等营造统一艺术氛围，传递对生活的热爱与对美的追求。如有乐器、音乐特长的同学可以合作录制歌唱表演。

 { 主题二：学风引领型 } 

与学风建设和共同成长为核心，打造专注学习、互相协作的宿舍环境，激发全员的学习动力与责任感。可设计书籍分类标签、图书角、读书笔记、拍摄宿舍学习Vlog等，旨在打造一个宁静致远的学术净地，营造宿舍浓郁的学习氛围。

 { 主题三：绿色生活型 } 

宿舍装饰可融入易成活的花草多肉绿植、环保材料等，通过DIY绿植角、废旧物改造摆件、手绘自然画等形式，打造清新宜居的“宿舍桃源”，传递绿色生活、低碳环保理念。

活动形式：线上（易班APP）

1.图片组：6-9张高清照片（需包含全景、细节特写，如书桌分类标签、床铺折叠技巧、DIY手工作品、阳台绿植布置等）+文字说明（设计理念/合作过程）。

2.视频类：1-2分钟短视频（需添加字幕，展示宿舍装饰过程及成果，如“定格宿舍美好，见证创意绽放”）+文字说明（拍摄理念/合作过程）。

评选重点：创意性、团队协作性、与宿舍生活的关联性、文化特色与创意（温馨和谐、美观大方）。 



八、活动流程

 { 赛道一} 

1.宣传活动：2025年11月22-2025年11月24日，组建宿舍长群，发布活动通知、评分规则，由宿舍长转发宿舍群，组织宿舍成员按照要求进行整改。

2.检查时间：2025年11月30日-2025年12月5日（抽两个时间段进行抽查）。

3.综合评审：12月6日-12月10日，根据学校“标兵宿舍”评比细则进行计算，按照总分进行降序排序，分别推出一等奖（3间）、二等奖（5间）、三等奖（8间）、并颁发电子、纸质证书及奖品。

 {赛道二} 

1.宣传报名：11月22日-11月24日在易班课群发布活动通知、评分规则及投稿模板；班委转发班级群，鼓励宿舍组队报名（以宿舍为单位提交作品）。

2.作品提交：11月24日-12月3日     

2.1. 宿舍长通过易班优课课群“作业/考试”板块提交作品（附件支持png,jpg,jpeg,rar,zip，文件最大50M，作品在截止时间之前可反复提交，但只保留最新上传的作品内容，截止后不可提交。请按要求分类上传多张代表性图片/视频，并配上文字简介）。

2.2.提交文件附件时，文件需注明专业、班级、宿舍号（如“修身楼1座203”）、成员姓名及联系方式。

2.3.截止后关闭提交通道。      

3.初筛审核：12月3日-12月6日     

3.1.组织教师代表、辅导员与学生代表组成初审小组，剔除不符合主题/格式要求的作品，根据参与情况，筛选出若干间宿舍进入线上投票环节。                    

3.2.线上投票：12月6日-12月10日

3.2.1易班轻应用开启“投票”功能，全体在校师生可为心仪宿舍投票（每人每天限投3票，同一宿舍最多投1票）；       

4..综合评审：12月10日-12月13日     

4.1由教师代表、辅导员代表、学生干部代表组成评审团，根据“内容质量（60%）+人气投票（30%）+创意特色（10%）”综合评分，确定获奖名单。    

5.结果公示：12月14日，通过易班课群、学院公众号/官网公示一等奖（3间）、二等奖（5间）、三等奖（8间）、人气奖（3间），并颁发电子、纸质证书及奖品。

十、活动要求

（一）真实性：所有作品需为宿舍日常真实呈现，禁止摆拍造假（如整洁赛道的“临时打扫”将被扣分）；  

（二）文明性：内容需积极向上，避免低俗、负面或侵犯他人隐私的信息；禁止墙壁张贴不文明内容或涂鸦。 

（三）在评选过程中，若宿舍对评选结果存在异议，可在结果公示后的3个工作日内，向学院学生工作办公室提出书面申诉。

（四）获奖的寝室，应在后续的学习生活中持续保持良好的宿舍环境、纪律秩序及文化氛围，发挥模范带头作用。

（五）安全，是美的底线：请避免使用“有安全隐患的装饰”，评比过程若发现违规，将取消一切评优资格！



 十一、注意事项

1. 活动期间安排专人负责易班平台技术维护（如作品上传问题、投票异常处理）； 

2. 评审过程公开透明，评分细则提前公布（可附在课群“资料区”）； 

3. 获奖宿舍需配合提供高清素材（用于后续宣传），并作为“文明宿舍示范岗”参与学院后续活动。
