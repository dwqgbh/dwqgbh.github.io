2026年9月俄罗斯圣彼得堡电子工业展

 RAEDL 2026



展出时间：2026年9月23日~25日


展出地点：俄罗斯-圣彼得堡


展会周期：一年一届


展会介绍：



俄罗斯圣彼得堡电子工业展(RADEL) 自 2001 年起在圣彼得堡举行，是一个集电子元器件、材料、技术、自动化设备、认证、电子仪器仪表制造服务为一体的专业化博览会。历年来，RADEL 展览会上展出了俄罗斯本土和国外电子零件、印刷电路板、仪器、电力供应制造商和无线电测量仪器供应商们广泛应用的成果及专利产品。


RADEL 是俄罗斯和整个东欧地区最具权威、规模最大的电子工业及自动化、芯片等的专业展会，同时也是国防电子产业首选展会。


本展会得到了俄罗斯联邦委员会、俄罗斯联邦国防部、俄罗斯联邦工业贸易部的大力支持，且展会全面覆盖了俄罗斯电子产业的所有重要领域，每年吸引来自独联体各国的大量专业人士参加。


【上届数据】


参展商数量：超过 400家，来自 20 多个国家和地区。

观众人数：约 6800 名专业观众。

展览面积：10000平方米。


【市场分析】


一、宏观市场背景：供应链重构，进口替代空间巨大西方断供，刚需缺口受制裁影响，欧美品牌（英特尔、AMD、英飞凌等）对俄出口暴跌 90% 以上，俄罗斯电子 62% 依赖进口，汽车电子、工业电源、传感器等领域本土产能不足 10%。


政策强力扶持

俄政府出台《电子工业发展国家计划》，2026–2028 年投入超 4280 亿卢布（约 47 亿美元），重点支持工业电子、汽车电子、能源电子、国防电子，进口替代上升为国家战略。


中俄贸易黄金期

2024 年中俄贸易额超 2000 亿美元，中国占俄罗斯电子进口 45%+、汽车电子进口 50%+，人民币结算普及，关税优惠，中国产品性价比与认可度第一。


二、展会核心价值：西北区龙头，B2B 精准对接行业地位RADEL 是俄罗斯西北区最大电子专业展（圣彼得堡为俄第二大工业城），25 年历史，获俄国防部、工业贸易部官方支持，国防 + 工业双渠道。


观众质量（高决策力）

79% 为企业高管 / 研发 / 采购负责人

73% 拥有直接采购决策权

覆盖俄、白俄、中亚、东欧15,000 + 专业观众，90% 为工厂、进口商、系统集成商。


展品范围：



一、电子制造技术与设备

SMT 设备：贴片机、回流焊、波峰焊、丝网印刷、点胶设备。

PCB 制造：钻孔、蚀刻、镀覆、层压、切割设备。

焊接材料：焊锡膏、助焊剂、无铅焊料、焊接工具（烙铁、热风枪）。

电子材料：绝缘 / 导热材料、导电浆料、防静电材料、基板材料。

智能制造：工业机器人、视觉检测、自动生产线、3D 打印电子、EMS 代工服务。

二、电子元器件与组件

半导体：IC、MCU、功率器件、二极管、三极管、传感器、光电 / 显示器件。

无源元件：电阻、电容、电感、滤波器、继电器、连接器、开关、线缆、磁性材料。

PCB 与基板：印制电路板、柔性板、金属基板、载板、PCB 设计与制造服务。

电源 / 电池：各类电源模块、UPS、适配器、锂电池、电源管理芯片。

三、测试测量与仪器仪表

通用测试：示波器、万用表、频谱 / 信号 / 功率分析仪、数据采集器、LCR 电桥。

电工仪表：电流表、电压表、电能表、绝缘 / 接地电阻测试仪。

过程控制：温控器、压力 / 液位 / 流量变送器、阀门定位器。

环境与可靠性：高低温箱、湿热箱、振动 / 冲击 / 盐雾试验箱、EMC 测试系统。

分析 / 实验室仪器：各类理化分析、无损检测、精密测量设备。

四、工业电子与控制系统

工业控制：PLC、PAC、DCS、HMI、工业 PC、嵌入式系统、伺服 / 变频驱动。

通信与网络：工业以太网、现场总线、IoT 网关、无线模块、射频 / 微波器件。

照明与显示：工业照明、LED、显示模组、触摸屏。

关于我们

北京叁贰壹零时代国际展览有限公司(简称：3210时代)是从事全球(海外)国际展览会的专业公司。


自2008年9月成立以来，公司的宗旨是为中国出口企业拓展国际市场铺路架桥，为中国企业走向世界提供多方位的国际展览专业化服务。


企业出海参展



10多年来，我公司组织了上万家企业走出国门开拓海外市场，展会分布在五金工具行业、园艺及户外行业、工业及机械行业;汽车配件行业、商用车及卡车行业，建材卫浴行业、暖通、制冷及厨卫行业；照明及LED行业；食品行业及包装行业、太阳能及光伏行业、新能源、体育用品等行业。


每年成功组织中国出口型外贸企业参加在全球30多个国家和地区的近2000场大型专业化国际展(博)览会。
