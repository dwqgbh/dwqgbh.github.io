航空航天与复合材料专场









报告一：刘松平特级专家



单位：航天材料及工艺研究所

报告题目：

基于Mia Visual 4.0++环境的AI检测与深度评估技术中国航空制造技术研究院

人物介绍

Character Introduction

刘松平，俄罗斯国立莫斯科大学理学博士、研究员、中航工业特级专家、中国机械工程学会无损检测分会副理事长。专心从事复合材料、焊接无损检测与评估技术创新性研发和工程化应用30余年，在超声和射线2D、2.5D、3D高效自动化扫描成像检测技术及其检测仪器设备、换能器、信号处理、智能检测与评估、检测标准及检测应用等方面有独特的自主研究。承担政府和企业资助的重要课题和国际科技合作研究80余项以上，先后获省部级科技发明和科技进步一、二、三等奖20余项，发表论文150余篇，出版《先进复合材料无损检测技术》专著1部，获国家发明专利80余项，编写国家和企业级检测标准100余项，是国际上NDE4.0（无损评估4.0）、智能检测与评估（iNDT&E）主要技术推进者，中国机械工程学会无损检测分会iNDT&E大会主席。







报告摘要

Abstract

随着先进航空航空航天的日新月异，各种复杂结构不断扩大应用，复杂结构造价高、制造周期相对长，多属关键结构和受力结构，其质量要求更高，作为质量控制主要手段的无损检测技术在结构可检性、缺陷检出能力、检测可靠性与准确性、检测成本与效率等诸多方面不断遇到挑战。面向航空航天等领域，研发或采用智能检测与评估技术，提升复杂结构检测精度与检测颗粒度，实现复杂结构的深度检测与智能评估，已成为业内非常关注的焦点。在许多复杂结构无损检测中，超声方法因其环保、易实现自动化、成本可控、检测效率高等，通常是制造场景复杂结构首选的无损检测方法。但由于其复杂的几何外形，内部结构细节复杂变化多样，采用常规超声2D扫描检测方法不能实现复杂结构高准高精高效超声自动扫描检测，基于常规超声2D扫描检测结果也难以真实地揭示检出缺陷的量化信息，特别是常规超声检测设备和仪器，其检测结果仍然依赖人工判别完成，容易造成误判漏检，且检出缺陷人工准确评定和标注难度非常大，效率非常低，容易标注不准、评判不准。本文提出了一种基于Mia Visual 4.0++的智能（AI）检测与深度评估技术。首先介绍了复杂结构超声可视化检测目前存在的主要技术难点与瓶颈；然后，以复杂复合材料结构为例，在超声检测信号高保真解析、2D成像的模型约束、3D洞悉复杂结构超声检出缺陷准确性等，进行方法和模型的解析；最后，以自主研发的工业级超声3D自动扫描可视化检测方法与检测系统为例，介绍了融入自主研发的Mia Visual 4.0++ 智能检测与深度评估方法，给提升复杂结构缺陷检出能力和检测结果高准高精高效可视化AI赋能带来的效果，同时还能实现被检测零件与超声可视化结果的自动赋码，特别适合批量产品的智能检测与评估，以及产品质量的终身管理，从而为复杂结构提供了一种工业级AI检测与深度智能评估方法和手段及应用案例。







报告二：陈颖研究员



单位：航天材料及工艺研究所

报告题目：

运载火箭结构无损检测技术

人物介绍

Character Introduction

陈颖，博士，航天材料及工艺研究所研究员。长期从事无损检测技术研究与应用工作，积极推动无损检测新技术、新方法的应用，致力于航天产品无损检测综合解决方案研究，主持863、国防技术基础和航天型号产品无损检测工艺攻关等多项无损检测技术研究项目，获北京市科技进步一、二等奖。







报告摘要

Abstract

运载火箭主要由发动机、箭体结构及控制系统组成，其结构设计时尽可能追求结构质量最小化，对产品生产和检测提出来较高要求。报告简要介绍了液体火箭发动机、燃料贮箱、箱间段、级间段、有效载荷支架和整流罩等部段的材料、结构及工艺，结合各部段典型产品及制造工艺特点，详细介绍了各种无损检测技术在其研制生产中的应用。






残余应力检测与调控专场









报告三：徐春广教授



单位：北京理工大学

报告题目：

低应力制造技术的医学思考

人物介绍

Character Introduction

徐春广，北京理工大学二级教授、博士生导师，先进加工技术重点学科实验室主任。长期从事超声检测与控制原理和技术的教学研究，提出了缺陷和残余应力等材料性能的智能多维量化超声检测与控制技术，创立了高能声束低应力制造技术原理和方法，是国内残余应力超声检测与调控领域的奠基性人物。兼任国际标准化ISO/TC35/SC3委员、中国电子学会会士、中国机械工程学会无损检测分会副主任委员、中国电子学会电子机械分会副主任委员、第20届世界无损检测大会（WCNDT）副主席、远东无损检测大会理事会主席（FENDT）、中国力学试验仪器装备技术与创新联盟（CMIC）副主席等。牵头制定了1项我国首项超声检测国际标准、68项国家行业企业标准。授权国际国家发明专利277件，出版10部教材专著，发表344篇SCI/EI学术论文；获中国产学研合作一等奖、日内瓦博览会金奖、中国发明创业奖人物特等奖，荣获2024年度中国当代发明家荣誉。







报告摘要

Abstract

残余应力存在于机械构件的全生命周期，俗称机械材料的癌细胞，不集中不破坏不质变，允许其存在。低应力制造技术是对机械构件全生命周期中的残余应力进行有效控制和利用的一门新兴工业技术，就是使残余应力在构件胚料成形阶段就得到抑制，以得到低应力低变形的胚料，如低应力铸造/增材技术；在机械构件加工成形后，对切削/折弯/焊接构件及时消减残余应力以减小变形开裂，使构件健康服役；在构件服役期间，交变载荷产生疲劳残余应力，要及时原位消减疲劳导致的残余应力，延迟服役寿命；对机械构件可以预置压缩残余应力梯度，提升材料性能，使得机械构件更健壮。残余应力在材料内部是时空变化的，超声无损检测和监测是低应力制造技术的基础。给出了大量残余应力对机械构件精密成形与健康服役的医学思考，有助于理解和运用低应力制造技术。







报告四：邱发生副教授



单位：南昌航空大学

报告题目：

电磁声应力无损检测进展

人物介绍

Character Introduction

邱发生，副教授，工学博士/博士后，硕士生导师，赣鄱俊才-主要学科学术和技术带头人。担任《Exploration》（IF=22.5）、《Instrumentation》等期刊青年编委。主要从事电磁无损检测、数字成像及智能无损检测研究。主持/完成国家自然科学基金青年基金、中国博士后科学基金面上项目、江西省主要学科带头人青年项目等国家/省部级课题10项，横向课题2项。在NDT&E International、IEEE T. Instrum. Meas.等领域权威期刊发表SCI/EI论文40余篇。申请国家发明专利13项，授权5项。获2025年第三届粤港澳大亚湾区无损检测论坛青年科学家奖、2025年江西省高层次人才创新创业大赛优秀奖。担任NDT&E International等30余个国际期刊审稿人。指导学生获全国大学生物理实验竞赛一等奖、机械工程创新创意大赛一等奖、全国仪器类毕业设计大赛一等奖等省部级以上奖励30余项。







报告摘要

Abstract

铁磁性材料在制造与服役过程中易产生应力集中，严重威胁结构安全。传统基于单一传感器或单一特征的应力分析方法预测精度低、鲁棒性差。本报告介绍电磁声应力无损检测的最新进展，重点阐述多传感、多特征融合的智能检测框架。首先，结合磁光克尔显微成像技术，揭示了应力作用下磁畴与畴壁动力学的演变规律，并建立了电磁声信号与应力之间的宏观映射模型，为应力检测提供了清晰的物理机制解释。在此基础上，同步获取磁声发射（MAE）与磁巴克豪森噪声（MBN）等多种宏观电磁声信号，提取其时频域多阶统计特征，通过主成分分析进行降维，进而发展了高斯核乘积融合、粒子群优化BP神经网络及PCA+XGBoost等多传感多特征融合模型用于应力检测。所提出的力‑磁‑声耦合模型及多传感融合策略，为铁磁材料应力评估提供了一条高可靠、高精度的新途径。






先进超声及装备专场









报告五：李洋副研究员



单位：北京航空航天大学

报告题目：

设计-仿真-控制-成像全链路协同的现代超声检测系统关键技术研究

人物介绍

Character Introduction

李洋，北京航空航天大学宁波创新研究院副研究员，毕业于北京航空航天大学，工学博士，宁波市高级人才。长期从事超声无损检测评价技术研究，包括：超声无损检测过程仿真、超声C扫描轨迹规划及运动控制、阵列超声实时成像以及自动化超声无损检测系统研发等。主持中国商飞创新基金项目1项，参与国家自然科学基金项目3项，发表SCI论文8篇，发明专利2项。







报告摘要

Abstract

摘要：面向航空航天、核电等领域日益复杂的超声无损检测评价需求，围绕"设计-仿真-控制-成像"全链路协同理念，对现代超声检测系统的关键技术开展了系统性研究。在设计层面，建立了具备CAD/CAM功能的3D软件平台，支持超声检测方案（探头、楔块、波束聚焦法则）的参数化设计以及C扫描仿形轨迹的交互式规划。在仿真层面，开发了多维度3D可视化仿真验证工具链：一方面覆盖声学仿真，包括声束传播路径模拟、稳态与瞬态声场计算以及缺陷响应预测；另一方面覆盖运动仿真，包括多轴联动扫查的可达性分析、奇异点辨识与干涉点检测，为检测方案的可行性提供全面预判。在控制层面，构建了适配不同超声模块的通用化超声控制与数据实时采集软件，以及适配任意轴数、任意构型扫查机构的通用化C扫描实时运动控制系统，实现了超声激励/接收与多轴协同插补运动的高速精确同步。在成像层面，构建了不依赖英伟达等专用GPU并行架构的超声数据实时成像软件，提供多模态自适应全聚焦成像（TFM）、C扫描三维实时成像以及3D数据体渲染等全维度可视化功能。

关键词：超声检测，相控阵超声，仿形轨迹，运动控制，三维实时成像







报告六：周文彬副教授



单位：北京航空航天大学

报告题目：

先进超声检测技术与装备研制

人物介绍

Character Introduction

周文彬，博士，北京航空航天大学机械工程及自动化学院副教授，博士生导师，国家级青年人才。从事超声无损检测评估、制造工艺在线监测与智能控制、轻量化薄壁结构高性能成形制造技术等相关研究。近年来主持并参与了国家自然科学基金青年项目、“人工智能赋能工程科学前沿探索”专项项目、国家科技重大专项、航空航天重点项目等多个国家/省部级项目。在NDT Int、Measurement、Int J Mach Tools Manuf，Int J Mech Sci，Aerosp Sci Technol等无损检测与航空航天领域SCI Q1区高水平期刊发表论文20余篇。担任中国空天动力联合会试验测试与检测技术青年委员会委员和《塑性工程学报》第二届青年编辑委员会委员。







报告摘要

Abstract

针对航空发动机整体叶盘等复杂曲面结构，传统超声检测扫描系统存在可达性差的突出问题。曲面曲率的不断变化，导致检测探头难以维持恒定入射角度与耦合距离，且无法抵达特定点位获取缺陷分布全貌，使得检测效率与精度均无法满足工程实际需求。为此，本研究建立了专门面向复杂曲面结构的七轴联动超声检测系统，尤其适用于航空发动机整体叶盘等复杂构型零件的内部缺陷检测。

该系统由六轴机械臂、变位机、超声探伤仪及激光轮廓仪组成，通过上位机总线实现超声信号与运动指令的传输。变位机可驱动工件回转，将干涉盲区内的待检曲面调整至机械臂作业范围；六轴机械臂携带探头完成局部曲面姿态跟随；激光轮廓仪采集工件表面点云，修正实际工件与理论模型的位姿偏差；超声探伤仪负责扫查过程中激励脉冲的发射与回波信号的接收。

系统结合变位机与机械臂实时反馈的关节角度，依据CAD模型自适应规划扫查路径并生成轨迹指令。借助二者的协同插补运动，超声水浸聚焦探头沿曲面母线及环向执行连续栅格扫查，始终保持最优法向姿态与恒定耦合水距，最终实现复杂曲面内部缺陷的高分辨率C扫描成像，有效解决了传统检测系统的痛点。
