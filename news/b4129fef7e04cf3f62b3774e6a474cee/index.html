【工业AI · 战略合作】西门子在北京 RXD 峰会发布26款新产品并深化云端仿真布局


2026年3月23日，西门子在北京举办首届 RXD 峰会（Real Meets Digital），汇聚逾2000名客户、合作伙伴与行业领袖，展示工业AI如何从实验阶段迈向工厂、基础设施与工业供应链的大规模部署。


西门子CEO Roland Busch 携阿里巴巴集团董事长蔡崇信、宇树科技创始人王兴兴、宁德时代首席制造官倪军共同登台。双方宣布扩大合作，联合向中国客户提供以基础设施即服务（IaaS）形式交付的计算机辅助工程（CAE）能力，通过将西门子仿真软件组合与阿里云先进计算基础设施相结合，为工程团队提供可扩展的云端仿真环境，包括虚拟仿真设备和高性能计算集群。


此次合作还探索将阿里巴巴的通义千问（Qwen）大语言模型引入西门子产品生命周期管理软件，为AI辅助工程工作流和智能产品开发开辟新可能。在产品层面，西门子发布了26款横跨边缘计算、自动化与控制、电气化及AI赋能基础设施的新产品，并将这批新品定位为"在中国为中国研发、同时具备全球适用性"的本土化创新。Forrester 分析师点评，此次峰会是西门子明确将中国定位为工业AI规模化验证场的战略信号，数字主线与数字孪生在其中扮演着关键的协调与信任层角色。

【数字孪生 · 工业AI】西门子 Insights Hub Production Copilot 亮相汉诺威工业博览会


西门子 Insights Hub 与其 Production Copilot 正在证明，数据驱动制造可以变得更简单、更快速，并对最接近操作现场的人员更易用。通过使产线团队以更自然直觉的方式与制造数据交互，企业可以加快决策节奏，提升运营绩效。


Production Copilot 的核心能力在于将机器数据、运营数据与文档数据整合为一个自然语言对话体验，使团队能够更自然地提问、快速获得清晰答案，并以更高信心解决现场问题。与传统 BI 工具相比，其开箱即用的部署模式使制造商可以迅速、安全、规模化地释放现有数据中已有的价值，这使汉诺威工业博览会成为观察西门子如何重塑运营智能未来的绝佳时机。


西门子数字工业软件 Insights Hub 高级副总裁 Dr. Ralf M. Wagner 表示，让制造数据更易于使用，能帮助产线团队更快将洞察转化为行动，推动绩效的可量化提升。该产品将在汉诺威工业博览会（4月20-24日，27号馆）正式展出，并作为西门子 Xcelerator 平台数据智能层的核心能力亮相。

【客户案例】一汽-大众携手西门子 Insights Hub 构建多工厂数字化制造数据底座


一汽-大众（FAW-VW）采用西门子 Insights Hub，以"平台优先、用例其次"为转型原则，建立了从数据采集、数据处理、数据分析到可视化与行动的端到端路径，在多工厂和产线间打通数据孤岛，实现设备层面的快速透明化监控，并创建了可复用的智能制造用例蓝图。


项目分四阶段推进：第一阶段优先打通设备连接，确保数据稳定接入；第二阶段标准化数据处理与上下文化管道，减少人工干预并提升跨产线一致性；第三阶段向全组织推出运营仪表板与实时告警，推动团队从"事后分析"转向"快速感知响应"；第四阶段则将单线可见性扩展至多站点运营智能。


值得关注的是，一汽-大众同步将人才赋能纳入核心转型设计，通过围绕 Insights Hub 构建的培训体系，提升技术工人和一线工程师在数据理解与数字工具使用方面的能力，降低对少数专家的依赖，为多工厂规模化扩展奠定可持续人才基础。此案例是目前中国汽车制造企业中，工业IoT平台落地最具完整参考价值的分阶段实施路径之一。

【工业AI · 数字主线】西门子 NX AI 加工建议 Copilot 与 Designcenter X 联合亮相汉诺威


在汉诺威工业博览会创新中心（Innovation Hub），西门子展示了全新 AI Make Machining Suggestion Copilot，该工具为数控（CNC）编程人员提供多套加工方案选项。现场展示以鞋底外底模具型芯加工为具体场景，通过 AI 推荐加工工序，有效缩短编程时间、减少手动决策数量。


在设计端，Designcenter X NX Copilot 同步亮相，允许设计工程师以自然语言提问，获取实现设计目标的多种可行方案建议。命令预测、几何识别、选择预测等功能，使产品设计与制造流程更易于现代化环境下的快速迭代。


这两款 Copilot 均内嵌于 Siemens NX CAD/CAM 软件，与 Microsoft Azure 结合后，形成云端驱动的AI数字主线，连接端到端的全流程；团队可实时协作、访问高性能仿真计算资源，并在全球范围内安全扩展运营规模。西门子同时表示，AI 能力将进一步拓展至加工策略建议、工艺质量预判等方向，帮助制造商应对质量、效率与可持续性方面的持续需求。

【客户案例 · 数字主线】OPmobility 采用 Teamcenter X，以云端 PLM 贯通全球40个研发中心


西门子宣布，全球可持续出行领域领导者 OPmobility 正式采用 Teamcenter X——来自 Siemens Xcelerator 组合的云端产品生命周期管理（PLM）软件。OPmobility 将以 Teamcenter 作为单一主干系统，提升研发工作效率与项目交期，并推动在快速变化的汽车行业中加速产品设计流程转型。


OPmobility 的产品日趋复杂，现已涵盖电子器件与软件，用于智能外饰与复杂照明系统，以及包含电池和氢能电气化解决方案的能源存储系统。为统一其分布于全球40个研发中心的团队协作，OPmobility 选择将 Teamcenter 部署为全球研发网络的标准化 PLM 骨干，从而显著提升效率、缩短上市周期，并持续受益于 Teamcenter X 的人工智能等最新功能迭代。


西门子数字工业 CEO Cedrik Neike 表示，该项目标志着汽车行业从遗留系统向 SaaS 化 PLM 解决方案转型的前沿实践。这一案例对正在评估云端 PLM 路线的中大型制造企业具有重要参考价值。

【行业活动 · 数字主线】Realize LIVE Europe 2026 开放注册，道达尔、罗尔斯·罗伊斯等标杆客户确认出席


Realize LIVE Europe 2026 将于6月29日至7月1日在荷兰阿姆斯特丹举行，是欧洲规模最大的工业数字化专业聚会，面向制造、工程、运营、IT及生命周期管理专业人员，聚焦如何将 Insights Hub 等连接数据平台转化为跨工厂的可量化运营改善。注册门票涵盖超过400个专题课程及60天 Siemens Xcelerator Academy 访问权限。


多家标杆客户确认参会并分享实践成果：道达尔能源（TotalEnergies）将介绍如何通过 Insights Hub Production Copilot，实现对200余个加油站资产、告警与维护活动的端到端可见性，以及基于角色的灵活仪表板如何降低运营成本；Rolls-Royce 将展示如何在本地私有云中整合 Teamcenter、Opcenter、Simcenter 与 Insights Hub，构建基于模型的数据驱动产品生命周期管理；Vorwerk 则将分享其连接 PLM、ERP、MOM、MES 与 PLC 的端到端 IT/OT 集成制造方案，以及这一整合如何提升产线可靠性并助力工业4.0就绪。

【技术动态 · 数字主线】西门子数字制造软件将在微软展台联合展示 AI 数字主线方案


汉诺威工业博览会期间，西门子数字制造软件将同步出现在微软展台（17号馆），展示以 Siemens NX X Manufacturing 为核心、运行于 Microsoft Azure 云端的 AI 数字主线方案，涵盖产品设计、机器人规划、制造编程与生产规划的完整闭环。


该演示以工业机器人伺服系统零部件的设计与加工为具体场景，展示了 AI 如何在云端贯通从创意到产品的每个环节；团队可实时协作，并通过 Azure 云平台实现全球规模化扩展。此外，微软展台还专设西门子独立展台，访客可与专家深入交流不限于软件的完整 Siemens 技术栈，包括控制器硬件层在内的全套工业AI解决方案。Azure 平台是 Siemens 多款 Copilot 产品的底层算力基础，两家公司的联合展示进一步强化了这一深度绑定关系。
