产品介绍

CES-RU1XX-W4XA


CES-RU1XX-W4XA是一款"小身材、大能量"的新一代工业边缘计算平台，凭借Ultra低功耗处理器+丰富IO+工业级可靠性，特别适合对稳定性、扩展性、部署灵活性有高要求的智能化场景。主要应用于：智慧城市、 智能零售、自助终端、工业自动化、智能工厂、通信基建。


参数：

Intel Core Ultra 5-125U/Ultra 7-155U ,TDP 15W CPU，4 x LAN，4 x COM(up to 6COM)，6 x USB（up to 8USB），1 x HDMI，1 x DP，M.2 2280(NVMe)，10V-36V Wide voltage input，fan-less system


CEFS-MV20-D221-PL32


CEFS-MV20-D221-PL32搭载Intel 12/13/14代酷睿+DDR5高速内存，算力澎湃；6网口+16路隔离DIO+4路智能光源控制，视觉/控制/通信一机集成；PCIe显卡扩展+宽温设计+工业认证，无惧严苛环境。机器视觉、智能检测、边缘计算，一机搞定，赋能产线高效升级！


参数：

支持12/13/14th Gen. Core ™ i3/i5/i7, Pentium®, Celeron® Processor, LGA1700针；搭载H610&H610E芯片；2个千兆网口，4ｘUSB3.0，4ｘUSB2.0，2 x COM，1 x VGA, 1 x HDMI，16in 16out 隔离DIO，4 x POE(可选)，4 x Lighting Port（可选），24V 电源输入。
