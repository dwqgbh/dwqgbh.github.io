扬帆起航展新程

——千家店中心小学2025年秋季开学第一课

    新学期新起点新征程，秋日的朝阳，绽放着温暖的笑脸。

9月4日上午，千家店中心小学全体师生举行了以“砺行筑梦迎新季  扬帆起航展新程”为主题的开学第一课。

NEW START

01

一、 奏唱国歌

// Have a nice day


庄严肃穆的升旗仪式为开学典礼拉开了序幕。奏唱国歌，全体师生面向国旗行注目礼，少先队员行队礼，五星红旗承载着大家对新学期的无限憧憬和希望！




NEW START

02

一、 奏

一、 奏

迎接新生

在孟老师的带领下，一年级的新同学们以崭新的面貌来迎接新的挑战，他们满脸自信，斗志昂扬。他们的加入让千小的队伍更加壮大。


新学期·新起点

开启 新的征程

在新的学期，高年级的哥哥姐姐们对一年级的弟弟妹妹们寄予了期望和嘱托，并送上了对弟弟妹妹们的入学寄语。




NEW START


03

表彰优秀学生

学习小达人：张雪晴、韩林、高如阳、王羽萱、郝雨嫣

绘画小达人：赵子禛、朱佳煜、宋铭轩、胡博薇、赵可儿

写作小达人：陈子轩、翟嘉雯、刘澍轩、张浩然、梁晓冉

运动小达人：谷沐阳、刘俊航、李佳奕、张国淇、李梦歆




优秀学生代表发言

郝雨嫣的发言透露着新学期开学的欣喜与忐忑，也透露着努力学习，立志成为新时代好少年的决心。




04

青年教师代表发言

赵老师愿和千小的全体教师一起携手共进做好“培根铸魂”的基础教育，用行动落实“双减”工作，让精彩从千小起航，共创未来！







05

爱国主义教育

// Have a nice day

为了纪念中国人民抗日战争和世界反法西斯战争胜利 80 周年，思政课闫老师以《铭记抗战历史，砥砺强国之志》为题，为学生带来一堂意义非凡的思政课。




回顾抗战历史

闫老师带领学生回顾波澜壮阔的抗战历史，让他们更加深刻地了解抗战胜利的伟大意义，感受革命先烈们的爱国情怀和英勇无畏的精神。





06

安全教育

// Have a nice day


千家店镇派出所丁旭警官，为全体师生做法治讲座。


抗战胜利

卫生副校长岳副院长强调落实秋季传染病防控的各项举措


抗战胜利

德育主任刘老师为我们讲述了《安全教育第一课》，从交通、食品、消防、心理等多方面进行了强调。





07

校长讲话

// Have a nice day

张校长的谆谆教诲，给我们指明了新学期的奋斗目标和努力方向，真是催人奋进、让我们备受鼓舞，豪情满怀




08

开学第一课

// Have a nice day


为了迎接新学期，引导学生以更好的状态迎接新学期，缓解学生开学初的焦虑情绪，9月4日开学第一天，我校各班班主任开展以“拥抱新学期 开启新征程”为主题的开学第一课活动。

观看9月3日纪念抗战胜利80周年阅兵盛况片段，做开学爱国教育第一课。


开展一次安全教育活动。重点从食品卫生、交通、防溺水、防火防电防诈骗安全等方面对学生进行宣传教育，对学生提出明确要求，让学生有章可循。




开展一次心理健康教育活动。针对如何返校收心，尽快调整心理状态，适应快节奏的学校生活，教师们提出了计策，让学生们尽快步入学习正轨，适应校园生活。



开展一次习惯养成教育活动。从学校常规、课堂常规、行为习惯养成等方面进行了细致的教育，促进同学们逐步养成良好的学习习惯和行为习惯。



开展以“抗战胜利80周年阅兵观后谈：传承爱国精神，勇担时代使命”为主题的爱国主义教育班队会，通过回顾阅兵式亮点，引导学生铭记抗战历史，理解爱国精神内涵，明确新时代青少年的使命与责任。




新学期，新起点，新希望，新梦想。千小的全体师生定会借助开学典礼的东风，全校师生将迈开铿锵有力的步伐，在充满爱的校园里，斗志昂扬地前进。我们将继续紧握奋斗之桨，高昂奋斗之帆，乘风破浪、砥砺前行。
