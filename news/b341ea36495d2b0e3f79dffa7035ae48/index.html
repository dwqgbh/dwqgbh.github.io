创天展品牌 | 在西博城,看见“三城三都”的万千气象

中国西部国际博览城订阅号 2026-04-09


4月7日，市委推进“三城三都”建设工作专题会召开。会议指出，要做强特色品牌，焕新发展动能，推动“三城三都”（世界文创名城、旅游名城、赛事名城和国际美食之都、音乐之都、会展之都）建设取得更大实效，更好以文化赋能经济社会高质量发展。



2025年，“三城三都”六大产业（文创、旅游、体育、餐饮、音乐、会展）规上企业总营收首次突破万亿元，相关领域在权威机构榜单中均位居全国前列。面向2026年，成都将紧扣“五中心五地”发展定位，坚持创新驱动，以强链突破、聚链成群为牵引，推动产业能级、消费品质、城市品牌同步跃升，推进文商旅体展深度融合，绘定“三城三都”建设工作“路线图”。

这里是西博城

（国际展览展示中心、天府国际会议中心）

一座城市面向世界的“超级会客厅”



从次元潮玩到美食盛会

从顶级赛事到国际大展

成都“三城三都”的澎湃活力

正在这里精彩上演


(   世界文创名城   )
01




次元潮玩集结
去西博城赴一场漫展狂欢



数字文创，正成为一张闪亮的成都名片。2025年，成都数字文创核心产业实现总营收4139.7亿元、同比增长8.3%，产业综合实力居全国第一方阵。


每逢节假日，西博城的动漫展都人气爆棚——五湖四海的Coser盛装赴约，古风雅韵与次元潮酷碰撞交融。独特的“坝坝漫展”文化，把成都包容松弛的城市气质拉满。


2026年，西博城将继续作为“世界文创名城”的活力缩影舞台：让创意在这里迸发，热爱在这里相遇。





(   世界旅游名城   )
02




好吃好耍
糖酒会、“村糖会”点燃文旅新场景


2025年，成都文旅市场数据亮眼——签约引进重大文旅项目71个、协议总投资262.58亿元。旅游总花费4337.73亿元、增长6.53%。文化旅游业总体实力保持全国第一梯队，重要节日持续位居全国热门旅游城市前列。


同时，去年9月以来，成都聚焦国庆中秋、元旦春节、春季糖酒会等重要节点，策办了差异化、特色化“村糖会”活动。截至目前，共举办“村糖会”活动246场，吸引游客2470万人次，实现产品销售超6.1亿元，带动相关消费超46.8亿元。



“过完春节过春糖”——第114届全国糖酒会在西博城与世纪城举办，展览规模达32.5万平方米。来自全球40多个国家和地区的6615家参展企业汇聚于此，超41万人次专业观众，既是数字，也是信任。


第114届全国糖酒会现场

2025年10月1日晚，天府国际会议中心“天府之檐”下，一场“大屋檐下的美好生活节”璀璨开启。作为2025年秋季“村糖会”重点活动，音乐嘉年华、世运回顾展、楼宇灯光秀、湖畔茶席……后备厢集市聚起滚滚人流，营造出“最烟火”的节日氛围。

2026年，西博城将继续以“融合创新”建场景，让更多“流量”化为更大“留量”。



(   世界赛事名城   )
03




从世运会到马术世界杯
屋檐之下，荣耀之上



以赛会友，以赛营城。2025年，成都世界赛事名城建设成果丰硕——全年举办包括第12届世界运动会在内的国际、国内高级别赛事78项，亚冠、中超等主场赛事33场，群众性赛事活动6100余场次。全市全年体育产业总规模达1502.6亿元、同比增长11.2%，体育消费总规模达817亿元、同比增长约9%……


2025年8月7日晚，第12届世界运动会在天府国际会议中心震撼开幕。这里，正是亚洲最大的单体木结构建筑——天府之檐。屋檐之下是家，屋檐之下是城。这场汇聚全球顶尖运动员的体育盛会，让成都“世界赛事名城”的金字招牌闪耀世界。

第12届世界运动会现场

2025年12月14日，2025浪琴表国际马联场地障碍世界杯中国联赛总决赛，在中国西部国际博览城圆满落幕。优雅跨越，激情腾跃——西博城不仅办大展，更办大赛。

2026年，西博城将持续助力成都承接国际、国内高等级赛事，推动文商旅体深度融合，成为“赛事名城”不可或缺的竞技高地。


(   国际会展之都   )
04




链动全球：
重大会展从这里走向世界



2025年，成都举办重大会展活动1035场，会展业总收入1613.9亿元，同比增长5.1%。锚定国际会展之都建设目标，2026年，成都将力争全年举办重大会展活动1060场以上，展出总面积超1450万平方米，会展业总收入超1690亿元。


2025年8月29日，第二十八届成都国际汽车展览会在西博城启幕，这场盛会不仅是汽车产业的年度焦点，更标志着西博城正式迎来投运后的第1000场展会。


第二十八届成都国际车展现场


从2017年一期国际展览展示中心开放首展时57万平方米建筑面积的“物理骨架”，到如今承载“千展”、注入鲜活“产业灵魂”的会展极核，西博城的成长轨迹与成都会展业的崛起之路高度契合。


8年来，这里先后承接了全国糖酒会、西部国际博览会、世界大学生运动会等重磅活动，累计带动会展业总收入超1200亿元，间接促成产业合作协议近3500项，吸引直接客流量超2000万人次，成为成都搭建对外开放桥梁的核心载体。


第20届西博会在西博城开幕




2026年，西博城将锚定“国际会展之都”目标，持续办好重大展会，承接国际高等级赛事，做活漫展等文创活动，推进文商旅体深度融合，让更多“流量”化为“留量”，为成都“三城三都”建设和高质量发展贡献核心场馆力量。
