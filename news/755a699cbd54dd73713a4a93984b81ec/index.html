展会信息服务网 2025-05-28

  

2025XME西安国际机床展


举办时间：2025年06月05日—06月08日

举办展馆：西安国际会展中心

展会城市：陕西/西安

主办单位：上海华墨展览服务有限公司

承办单位：上海华墨展览服务有限公司


展会介绍                                   


      安国际机床展是CMES华机展布局的第10城，属于工业领域核心前端的装备制造展会。

      2024CMES华机展|西安国际机床展由陕西省商务厅、陕西省工业和信息化厅指导，陕西省机床工具协会和西安华好会展有限公司联合举办，聚焦航天军工、汽车制造、电子信息等西安地区支柱性产业，借力"一带一路"核心城市的地理位置，重磅打造中西部硬核机床展。

      西安制造"被频频聚焦，2025年西安国际机床展谋势而动，特设"金属切削、金属成形、磨削工量具、机床附件、机器人及自动化、电加工"六大主题展区，35000 m展厅集结600+品牌展商，打造中西部最具影响力的工业会展标杆。

2025年6月5-8日，西安国际机床展将在西安国际会展中心(浐灞)重磅举办。


展品范围                                   


金属切削展区 Metal Finishing Machines area
车、铣、镗、钻、磨、加工中心（卧式、立式、龙门式、组合式）等电火花机床、线切割机床、微孔加工、雕刻机、打标机、液压压力机.......

金属成型展区 Metal forming machine tools area
数控折弯机、数控剪板机、火焰切割机、水切割机、带锯床、剪床等各种钣金数控加工设备；高速精密冲床、伺服压力机、开式压力机、闭式压力机、挤压机、数控转塔冲床、伺服曲柄压力机等；薄板拉伸液压机、单柱/四柱液压机、液压校直机、油压机、液压拉床；激光切割机、激光焊接机、激光打标机、激光雕刻机、激光打(钻)孔机、其他相关激光设备；刀模（激光刀模）、数控转塔冲床模具、折弯模具、多工位级进模等各种五金/冲压模具；钣金表面处理设备（去毛刺机）、钣金检测设备、钣金加工软件等。

工量刃具展区 Work Measure Cutting Tools area
各型面加工刀具、组合刀具、非标刀具、刀具涂层、刀具材料、刀具附件、磨削机械、各类型磨具（固结、超硬材料等）、磨削材料、磨削配件及辅料等；三坐标测量仪、影像投影仪、激光干涉仪、各类千分尺、对刀仪、刀具测量仪、力学性能仪器、表面粗糙度仪等 ；手动、气动、电动工具等

机床附件展区 Machine tool accesory area
各种机床功能部件、机床附件、机床电器等；气动液压元件及装置、数显装置等；润滑油、工业清洁、物流设备等工厂相关配套产品......

智能工厂展区 Intelligent Factory Exhibition Area
自动控制系统、自动检测系统、自动报警系统、自动操纵系统、工业机器人
