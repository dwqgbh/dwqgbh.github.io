敢教日月换新天
——2026年宏观经济分析与新基建“六张网”的全方位解读企业应对策略

课程背景：

特朗普再次强势归来，美联储步入降息通道，是否引发金融动荡？
    关税战全面开打，中国具身智能震撼全球，对世界意味着什么？
    俄乌战争又见变局，伊朗战争风云再起，对中国是利是弊？
    中日关系波云诡谲，九三阅兵震撼全球，中国外交有何动作？
    我们应该如何应对人工智能时代？
    如何理解具身智能时代，它在人工智能时代的基础上有何变化？
    各部门各环节应该做好什么准备？
    如何提前预判未来的技术走势？

课程收益：

我们独家研发的课程为您解决以上所有难题，帮您轻松了解人工智能时代，解析2026年经济环境，提升自己的竞争力，在风云变幻的局势中把握先机。

第一模块 看全球经济——2026年的全球经济趋势

一、大雾迷城：世界格局重新洗牌与中国复兴

1. 百年未有之大变局：逆全球化甚嚣尘上，高通胀高负债并行

2. 美国经政扑朔迷离：美国开启降息通道，关税战与战略收缩

3. 俄乌中东南美大战：地缘政治深刻改变，美伊大战牵动全球

4. 科技战争再起波澜：人工智能双雄对决，AI时代谁能先破局

5. 中国对外强势出击：九三阅兵震撼全球，新式装备开创新局

二、全球数据：经济数据背后的百年变局

1. 经济总量：中美双雄格局初成，印度崛起挑战秩序

2. 人工智能：深度求索横空出世，具身智能时代来临

3. 工业产值：中美日德寡头垄断，新兴经济强势崛起

4. 货币战争：美联储利率新调整，全球新一轮大洗牌

5. 债务通胀：各国负债再创高峰，通胀全球难以消退

三、关税战争：逆全球化与中美关系

1. 关税战：从特朗普上台历次关税记录

2. 贸易战：从2017年以来历次制裁记录

3. 金融战：美联储加息与人民币汇率变化

4. 外交战：从联合国到香格里拉峰会

5. 经济战：中国产业升级与产能过剩之辩

6. 舆论战：无所不在的舆论热点争夺

7. 军事战：南海对峙到台海博弈

8. 其他战场：各种影响对峙

四、中美关系：特朗普掌权下的中美关系未来研判

1. 中美经济研判：新一轮关税战的应对策略

2. 中美科技研判：新质生产力的终极对决

3. 中美产业研判：对新兴产业与未来产业的争夺战

4. 中美金融研判：中国防风险与美国压通胀的博弈

5. 中美军事研判：解放军军力的提升与美国军备的换代

6. 中美外交研判：中国一带一路与川普孤立主义

7. 中美关系研判：经济基础决定上层建筑

一、重要视点：中国今年重点事件解读

1. 全国两会：设定赤字率4%支持经济刺激，保就业促民生

2. 十五五规划：从高速度全面转向高质量

3. 新质生产力：中国千行百业的迭代升级

4. 人工智能：春晚表演震慑全球，具身智能时代来临

5. 延迟退休：应对人口老龄化与各行业结构性冲击

6. 民营经济：政策护航下的民企信心修复与营商环境优化

7. 金融风险：坏账、债务、产能过剩与房地产风险频冲击

8. 经济刺激：经济金融刺激政策密集出台影响几何

二、重点数据：数据背后的中国经济画像

1. 全国两会：设定GDP增长4.5%—5%左右

2. 经济工作会议：积极财政政策与宽松货币政策

3. 经济总量：经济总量再创高峰，新质生产展新篇

4. 通缩风险：物价持续下跌成隐忧，全力刺激消费

5. 省市区域：全国各省区，冷暖几重天

6. 工业数据：转型升级与传统产能的博弈

7. 金融数据：系统性金融风险重点防范化解

三、产业经济：新质生产力与十五五规划

1. 全国两会：安排2500亿特别国债促消费

2. 经济工作会议：苦练内功来应对外部挑战

3. 大国重器：制造业高质量发展与产业升级

4. 大国空巢：人口问题与扩大内需消费升级

5. 大国饭碗：乡村振兴与农业现代化政策解读

6. 虚拟世界：人工智能与东数西算助力信息产业

7. 房产之辩：房地产软着陆与行业泡沫问题

8. 金融战场：守住不发生系统性风险底线对企业风控的要求

9. 砥励奋进：政府过紧日子与积极财政政策的投资方向

四、人工智能：新一轮科技革命浪潮与十五五规划

1. 全国两会：点名6G、具身智能与低空经济

2. 横空出世：美国ChatGPT平地惊雷

3. 百模大战：中国各大厂大模型狂潮

4. 具身智能：春晚表演、失业冲击与产业重塑

5. 千行百业：“人工智能+”行动重塑全球产业格局  

第三模块 看中国基建——中国“六张网”建设与十五五规划

一、重要视点：“六张网”今年重点事件解读

1. 国常会：首次将“六张网”纳入国家顶层部署

2. 十五五规划：从传统基建转向新老结合的基础设施体系

3. 新型电网：源网荷储协同，主配微协同架构重塑电力格局

4. 算力网：东数西算深入推进，全国一体化算力调度

5. 通信网：空天地海一体化，6G与5G-A双轮驱动

6. 地下管网：390万公里城市生命线智能化升级

7. 国家水网：覆盖80%国土，南北调配东西互济

8. 物流网：多式联运降本增效，国家级平台搭建中

二、重点数据：数据背后的“六张网”画像

1. 国常会：政治局会议、国常会相继部署“六张网”建设

2. 投资规模：“十五五”六网总投资预计约25万亿元

3. 新型电网：投资超5万亿元，新增特高压直流15条

4. 算力网：智算规模188万P，建成超70条算力通道

5. 通信网：5G基站超500万个，5G-A覆盖330城

6. 地下管网：建设改造77万公里，带动投资约5万亿元

7. 国家水网：投资突破6万亿元，新增就业300万+

8. 物流网：社会物流总费用与GDP比率降至13.9%

三、产业经济：“六张网”与十五五产业升级

1. 全国两会：扩大有效投资，今年估算投资超7万亿元

2. 经济工作会议：扩内需稳增长的压舱石

3. 新型电网：特高压电力高速+配网最后一公里

4. 算力网：芯片、服务器、光模块产业链受益

5. 通信网：射频器件、光纤光缆、卫星基站联动

6. 地下管网：钢铁、管材、传感器、GIS平台拉动

7. 国家水网：水泥钢材、水利施工、智慧水务协同

8. 物流网：冷链设备、物流机器人、数字平台需求爆发

四、人工智能：AI赋能“六张网”与十五五智慧升级

1. 全国两会：点名“人工智能+”与六张网深度融合

2. 智能电网：AI预测负荷，数字孪生提升调度精度

3. 算力调度：统一监测调度平台，毫秒级响应

4. 智慧水网：天空地水工一体化监测，精准分水

5. 智慧管网：实时监测、动态预警、智慧管控闭环

6. 智能物流：国家级大数据平台，统筹多式联运

7. 通信赋能：AI驱动信息通信，夯实科技底座

8. 新质生产力：六网融合为新质生产力提供硬底座

4. 技术创新驱动：数字化转型与智能化升级
