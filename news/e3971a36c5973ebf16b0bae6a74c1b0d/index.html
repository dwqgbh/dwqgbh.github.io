WORK SUMMARY




前言

Introduction


墨西哥作为拉美地区重要的汽车生产和消费市场，在全球汽车产业格局中占据着举足轻重的地位。墨西哥汽车产量持续增长，出口规模不断扩大，吸引了众多国际汽车企业的关注与投资。其完善的汽车产业政策、优越的地理位置以及丰富的劳动力资源，为汽车行业的蓬勃发展提供了坚实的基础。

墨西哥汽车行业考察旨在深入了解墨西哥汽车市场的现状、发展趋势以及相关政策法规，通过实地走访墨西哥汽车协会、企业及研究机构，与当地行业专家进行交流，探讨中墨两国在汽车领域的合作机会与潜力，为我国汽车企业拓展海外市场、提升国际竞争力提供有益的参考与借鉴。

关税红利+产业导向

Work Review

墨西哥是全球第九大汽车生产国，产业链覆盖整车制造、零部件供应及售后市场。依托北美自贸协定（USMCA）关税豁免优势，墨西哥成为中国车企规避贸易壁垒、辐射北美市场的战略要地，其新能源汽车本土化生产、充电基础设施建设等领域获政策重点扶持，为合作提供政策保障 。

洞察拉美电动化实践

Work Review

涵盖新能源整车制造、出行服务、行业协会、本土化龙头等全链条，既有Vemo、Zacua等本土创新企业的生态构建，也有Avis、Hertz等国际品牌的本地化运营，更有GML等龙头的区域布局，全方位展现墨西哥汽车产业电动化转型路径。

产业与本土融合

Work Review

深入墨西哥城、墨西卡利等产业核心城市，一边考察汽车工厂与运营场景，一边感受当地对新能源汽车的接受度与消费习惯，理解“本土化生产+区域化适配”的市场逻辑，同时体验拉美汽车文化与城市交通的独特魅力。

Vemo

Work Review

考察主题：拉美清洁交通生态构建——一站式新能源汽车、充电基础设施与车队服务整合模式探析

业务定位 / 模式

Vemo 是在墨西哥以“电动出租 / 电动车队 / 清洁出行生态系统”为核心的公司。其业务不仅是运营电动出租车车队，还包含充电基础设施、车队管理、租赁/融资方案等。

它集合了多个子业务模块：
VEMO Conduce：司机端服务，招聘 / 管理电动车司机 
VEMO Impulso：为电动车出行提供融资 / 租赁 /分期方案支持 
VEMO Charging Network：建设并运营充电站网络 
VEMO EV Fleets：向企业客户提供整车 + 充电 + 车队服务整合解决方案 
ZEE：其内部的数据 / 车队管理平台

Electricab

Work Review

考察主题：本土化电动出行落地实践——电动出租车试点运营、车型适配与政策协同机制考察

主营业务 /业务模式

电动出租车服务：在 Mexicali 市运营电动车作为出租车使用
作为示范 /试点项目验证电动车在地方出租车运营的可行性
车队采购 /运营 /充电基础设施配套（项目设计通常要考虑充电支持）

Avis

Work Review

考察主题：国际租车品牌电动化转型——墨西哥市场新能源车队布局、租赁服务体系与用户需求洞察

成立时间 /历史

Avis Car Rental 最初由 Warren Avis 在 1946 年创立（美国），在其成立十周年（约 1956 年）就进入国际扩张，包括墨西哥市场。

主营业务 /业务范围
汽车租赁（短租 /长期租赁 /机场租赁）
车辆订阅服务
特许经营 /加盟运营
在多个国家 /地区有租赁网络，在墨西哥也设有多个网点

Hertz

Work Review

考察主题：出行服务新能源升级路径——电动汽车租赁网络建设、运维保障与市场推广策略研究

成立时间 /公司历史

Hertz 的起源可追溯到 1918 年，当时 Walter L. Jacobs 以一批福特 Model T 汽车创立租车业务。
在墨西哥，Hertz 通过与本地特许经营商合作运作。2009 年，Hertz 与AVASA 签署了在墨西哥的特许经营主合同，使 AVASA 成为在墨西哥运营 Hertz 品牌的主特许商。

主营业务 /业务范围
汽车租赁服务（机场 /市区 /旅游租赁等）
高端 /豪华 /特殊车型租赁
特许经营 /加盟合作
在租赁车队管理、车辆更新、车况维护、保险与客户服务等方面运营支持

AMIVE

Work Review

考察主题：墨西哥汽车产业协同发展——行业资源整合、企业合作对接与市场规范构建考察

AMIVE 是墨西哥的一个旨在推动电动车 /电动交通 (“vehículo eléctrico”) 的协会。其西班牙语名称通常翻译为“墨西哥电动车推广协会”。

它是电动交通生态系统（制造商、政策制定者、服务供应商、充电基础设施提供商等）中的策略性联盟，致力于在墨西哥推广更清洁、更高效的交通方式。

在其公开网页中，AMIVE 地位较新（2023–2024 年活跃），致力于在政策、标准、公共意识、行业对话方面成为代表性平台。

EMA

Work Review

考察主题：新能源汽车产业政策解读——墨西哥电动化转型规划、标准制定与扶持措施深度交流

成立背景 / 目标

EMA（Electro Mobility Association / Electro Movilidad Asociación）是近年来在墨西哥成立的一个新型电动出行 /电动车产业联盟 /协会，它包括多个整车厂商、充电基础设施公司、出行 /车队 /租赁 /电动车生态系统中的参与者。

EMA 的目标之一是推动墨西哥到 2035 年实现 100% 车辆销售为 BEV（电池电动车）的愿景。

它还致力于推动公共政策、标准制定、充电基础设施建设、教育 /公众意识提升，以及高里程车队（如出租车 /物流车队）的电动化。

ELECTROCAR

Work Review

考察主题：电动交通工具技术创新——产品研发、本土化生产适配与市场竞争力分析

主营业务 /服务范围

根据其在维修 /电动车维修领域的出现位置，可以推断其可能的业务范围如下（基于行业常见功能 + Yelp 分类）：

电动车 /电动汽车的电气 /电子系统维修：如电机、控制器、电池管理系统 (BMS)、功率电子、车载充电器等故障维修或替换

传统汽车 /混动车 /电动车的机械维修：轮胎、刹车、悬挂、车体与底盘维护

日常维护 /保养：如检查 /更换液体（冷却液、润滑液等）、更换滤网、电气检查、制动系统检查等

诊断 /故障检测：使用诊断工具检查车载 ECU/电气故障码、系统诊断

可能的改造 /升级 /电动车改装（虽无确凿资料，但在电动车维修店中较为常见）

Electro Mobility Association

Work Review

考察主题：拉美电动化生态协同——行业交流平台搭建、跨区域合作机遇挖掘

成立 /发展历程

EMA 的官方网站显示其使命与愿景，以及其目标在墨西哥推进电动交通。

在行业媒体中，EMA 被报道为在 2024 年正式成立（或至少公开宣布成立）。例如 Aftermarket International 报道：“The Electro Mobility Association (EMA) emerges in Mexico” 作为新成立的联盟。

在 T21 行业新闻中，也报道说 EMA 是由 BYD、Evergo、JAC、SEV、Tesla、VEMO、Volvo 这些企业推动成立。

在 Autocosmos 报道中，EMA 在其成立一年之后已由最初 7 家成员扩展至 21 家公司。

Zacua

Work Review

考察主题：墨西哥本土新能源汽车标杆——纯电微车研发生产、充电基础设施联动与政策红利应用

2017 年。Zacua（全名 Zacua México / Motores Limpios, S.A.P.I. de C.V.）在 2017 年正式推出。

主营业务 /定位

生产与销售电动微型车（micro electric cars），主要用于城市短距离出行。车身小巧，乘坐量通常为两人。车辆设计 /平台部分来源于与法国品牌 Chatenet 的合作（以 CH30 平台为基础改造电动驱动系统）

Giant Motors Latinoamérica

Work Review

考察主题：拉美汽车产业龙头布局——区域化生产基地建设、市场辐射能力与供应链整合策略

Giant Motors Latinoamérica 是一家 100% 墨西哥的汽车企业，成立于 2006 年。它是负责在墨西哥组装、分销、销售中国品牌 JAC 的公司。

主营业务 /经营模式

组装与生产 JAC 品牌的乘用车、商用车、电动车等在墨西哥市场销售的型号。
拥有整车组装厂（Ciudad Sahagún, Hidalgo 州），分销网络（包括 JAC Stores／经销商网络）遍布全国。
销售燃油车和纯电动车型（JAC 的 “Pure Electric” 系列）两条线并行。 
同时负责任务包括后市场（配件、服务）、品牌形象 /经销商体验、用户支持等。

结语

Introduction


墨西哥汽车行业考察，我们深入了解了墨西哥汽车市场的规模、增长趋势、技术创新以及政策环境等方面的情况，与当地汽车企业、协会和研究机构建立了良好的沟通与合作关系，为未来的合作奠定了坚实的基础。

墨西哥汽车行业的快速发展和巨大潜力，为我国汽车企业提供了广阔的市场空间和发展机遇。我们将以此次考察为契机，进一步加强与墨西哥汽车行业的交流与合作，积极推动我国汽车企业“走出去”，提升我国汽车产业的国际竞争力，共同推动全球汽车产业的繁荣与发展。
