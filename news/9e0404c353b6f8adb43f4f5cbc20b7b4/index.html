关于2025“AI+创新应用案例”


2025年是AI应用的落地元年，用户端开始呈现出从多个方向落地AI应用的混合AI应用趋势。同时，更多用户认识到AI技术与业务结合将带来一系列颠覆，但却并不清楚从什么路径落地AI应用，在落地的过程中遇到问题又如何获取帮助。


为深入挖掘AI技术赋能产业升级的创新实践，推动AI技术与实体经济的深度融合，现面向全国企业发起 “AI驱动·智领未来”企业AI应用典型案例征集活动。诚邀各行业标杆企业分享AI落地经验，共建行业生态，共享因AI应用落地带来的业务价值。案例参与者将获得品牌曝光、资源对接机会，优先参与AI技术峰会、产业闭门研讨会等活动，我们也会适时将优秀案例集结成册，让参与者收获更多行业影响力。


本次AI案例征集的目的在于梳理AI技术在企业场景中的成功应用路径，形成可复制推广的解决方案；打造标杆案例库，为政策制定、行业研究及企业决策提供参考；搭建企业间技术交流与资源对接平台，加速AI技术普惠化进程。征集主题将涵盖但不限于智能制造、智慧服务、管理决策、行业创新。

参选单位



华润江中制药集团有限责任公司



公司简介


华润江中制药集团有限责任公司（简称华润江中）前身是江西中医学院（现江西中医药大学）的校办企业，创建于1969年，2019年成为华润集团直管业务单元。公司主要从事中成药、保健食品的研发、生产和销售，拥有一家A股上市公司（江中药业，股票代码：600750），有“江中”“初元”两个中国驰名商标，其中江中健胃消食片连续二十一年获“中国非处方药产品综合统计排名”中成药•消化类第一名。



案例名称



华润江中财务智能灵犀助手



案例简介



华润江中财务智能灵犀助手通过统一的服务入口，集成“问、办、查”全财务服务于单一界面，使用自然语言交互，支持文本/语音输入及上下文理解，消除使用壁垒。同时实现流程自动化与风控前置，从填单环节把控合规风险，降低数据查询、分析的门槛，自动获得关键分析，解放财务人员聚焦高价值分析，赋能业务自助服务，推动财务共享中心智能化转型。



案例特色



一、项目目标


（一）核心目标


当前财务共享场景中存在三大痛点：


一是员工在报账过程中，对制度、流程的疑问需人工解答，响应慢且易出现信息不一致；二是传统表单申请依赖人工录入，耗时且易因手动输入产生错误；三是业务人员查询财务数据需通过报账系统或技术人员协助，数据获取效率低，难以快速支撑决策。


本次项目旨在通过开发集成智能问答、智能申请、智能问数三大核心功能的 AI 智能体，解决上述痛点，实现：优化财务共享流程，减少人工操作成本；提升员工报账及数据查询效率，降低人工错误率；增强财务服务的响应速度与准确性，最终提高财务共享服务水平。


（二）创新点


1. 服务模式创新：接入AI智能体后，我们实现统一智能入口，为员工提供覆盖“问、办、查”全财务服务的统一、便捷交互界面。同时提升财务共享中心服务效率与体验，显著缩短服务响应时间（问答）、减少事务性操作时间（申请）、加速数据获取与分析过程（问数），全方位提升员工满意度。


2. 交互方式创新：接入AI智能体后，问答助手以大语言模型（LLM）为核心，员工使用日常业务语言即可交互，彻底消除系统操作壁垒。同时支持多模态交互融合，员工可以使用文本、语音等多种输入方式；且助手能够做到上下文感知与记忆，根据输出结果进一步引导员工提问，提供连贯服务。


3. 工作方式创新：智能助手提供“零门槛”的自然语言数据探索，业务人员无需掌握技术工具即可直接提问获取结果；更重要的是，它能智能进行初步分析（如趋势对比、异常提示、关键归因），提高数据使用便捷度；同时，在严格的动态权限管控保障让业务人员能够在自身权限范围内实时获得数据。


4. 流程效率与风控创新：智能助手实现了源头填单自动化与嵌入式风控，通过智能填单大幅减少输入错误，并在填单环节实时进行标准校验与附件检查，将合规控制前置；同时，前端数据质量的提升和预审信息的清晰化显著加速后端审核效率；加之知识驱动的标准化服务确保政策解答一致准确，系统性降低操作风险与合规隐患，实现效率跃升与风控强化。


5. 组织与角色转型创新：智能助手驱动了根本性的角色重塑，解放财务人员从重复性任务转向高价值的分析、决策支持与规则优化；赋能业务人员通过问答助手强大的自助服务能力（咨询、填单、查数）提升效率，促进业财融合与数据驱动决策；最终推动财务共享中心智能化升级，成为高效、高价值服务的核心引擎，实现组织效能的质变。


总结：财务智能AI问答通过统一的服务入口，集成“问、办、查”全财务服务于单一界面，使用自然语言交互，支持文本/语音输入及上下文理解，消除使用壁垒。同时实现流程自动化与风控前置，从填单环节把控合规风险，降低数据查询、分析的门槛，自动获得关键分析，解放财务人员聚焦高价值分析，赋能业务自助服务，推动财务共享中心智能化转型。


二、项目方案


（一）场景描述


本项目聚焦财务共享服务场景，具体覆盖四大核心场景：


1. 智能问答场景：创建财务知识库，利用AI理解制度、操作手册等内容，强化AI理解能力。员工在报账、系统流程办理中遇到疑问（如报销标准、操作流程）时，可通过AI智能体直接提问，7*24小时在线，替代传统 “找专员咨询” 模式。


2. 智能报账场景：智能体自动填写申请、报账单，提供一站式报账服务。


3. 智能问数场景：业务人员需查询财务数据（如部门月度支出）时，通过向智能体提问（如 “XX 部门 Q3 差旅费总额是多少？”），智能体直接返回数据结果并提供简要分析，替代传统 “查询 - 筛选” 流程。


4.智能预测场景：结合历史数据、对标企业、行业趋势以及外部监管等要求，对公司关键运营指标、现金收益等实现有效预测，助力公司决策


（二）预期效果


1. 一站式无缝服务体验：员工登录报账3.0界面或移动端，即可在显眼位置（如侧边栏/悬浮窗）唤醒智能助手，无需切换多个平台。且支持多维度查询，场景无缝跳转至分析报告生成，实现服务闭环。

• 预期效果： 消除传统多系统切换的割裂感，提高智能助手使用率100%，提升服务效率30%+，用户满意度显著提高。


2. 多模态智能交互：支持语音、文本两种输入方式，同时结果以结构化卡片、分步引导（流程说明）等多样化形式呈现。

• 预期效果： 降低使用门槛，提效50%（较传统表单/查询操作），尤其优化外勤人员、高管等群体的操作体验。


3. 7×24小时无人值守服务：即时响应，全天在线实时处理咨询、填单与数据查询需求，极端情况下自动推荐人工客服。

• 预期效果：解决传统财务服务“5×8小时”限制，全球业务团队可随时获取支持；减少80%非工作时间积压工单，加速业务流转。


4. 智能引导与意图澄清：通过语义分析预判用户潜在需求，在员工模糊提问时主动追问关键参数，并给出下一步提问建议。

• 预期效果：问题解决率提升至95%+，减少因信息不全导致的重复提问。


三、设计方案


（一）功能设计


1. 智能问答功能：财务智慧交互中枢

• 深度整合报账制度、报销流程、税务政策等核心知识体系，构建全维度结构化财务知识库，实现专业信息的系统化沉淀与高效调取，为业务支撑筑牢数据根基；

• 搭载精准语义解析引擎，可高效响应 “招待费报销有哪些限制？” 等具体提问，依托大模型生成规范且精准的答复，同步智能关联对应制度原文作为权威依据，确保每一项解答都有章可循、有据可查，彰显专业严谨性；

• 具备强大的上下文理解与多轮交互能力，针对复杂业务问题可层层深入拆解，动态适配用户需求递进，实现从初步解答到深度解决的闭环，全面提升财务咨询响应的专业性与高效性。


2. 财务报账事前申请单：智能单据生成枢纽

• 深度融合语音识别技术与大模型智能理解能力，构建 “语音输入 - 语义解析 - 单据生成” 的全流程自动化体系，用户通过自然语音描述报账事由、金额、涉及项目等关键信息，即可触发高精度语音转文本及语义逻辑解析，自动匹配单据模板完成填写，从根本上简化手动录入流程，推动事前申请从 “手动填报” 向 “语音直达” 跨越，彰显智能技术对业务效率的颠覆性提升；

• 依托大模型的持续学习与迭代能力，不断优化复杂业务场景的理解精度，针对模糊表述或特殊业务需求，可通过上下文语义分析做出合理判断，确保自动生成的单据内容准确合规，有效降低人工填写失误导致的退单率，切实强化智能工具对业务规范性的支撑作用。


3. 智能问数功能：财务数据洞察引擎

• 深度对接财务数据中台，实时调取全量结构化财务审批数据（涵盖操作次数、驳回次数、驳回率等核心指标），构建数据直达的高效通道，实现业务数据的即时获取与动态同步；

• 在精准返回数据结果的基础上，同步触发智能分析机制，自动生成维度清晰、结论明确的解读报告 —— 例如：“作为共享初审人员，上季度累计操作 64 单，驳回 7 次，驳回率 10.94%，平均审核时长 1.08 天，审核准确率 96.49%”，通过数据与解读的无缝衔接，让数据价值从 “呈现” 直接跃迁至 “洞察”，为业务优化提供精准决策支撑。


4. 基础支撑功能：财务智能生态基座

• 深度打通华润江中财务数字化平台与财务数据中台，构建全域数据互联通道，实现系统间数据的无缝流转与实时同步，为上层功能提供坚实的数据支撑底座，筑牢一体化运营根基；

• 搭载多模态交互引擎，全面支持文本、语音等多样化交互形式，同步适配 PC 端与移动端全场景使用需求，打破设备与交互方式的壁垒，让操作体验更灵活高效，彰显技术适配的前瞻性；

• 内置全链路操作日志追踪系统，对所有交互行为进行精准记录与存档，为功能迭代优化提供数据依据，同时满足合规追溯要求，实现技术支撑与风险管控的双重保障，凸显系统的可靠性与规范性。


5. 数字人交互功能：财务智能交互新界面

• 深度打通上述四大核心功能与数字人交互系统，构建 “功能 - 数字人” 一体化交互中枢，将智能问答、单据生成、数据查询等能力全面赋能至数字人载体，实现从 “文字 / 语音指令” 到 “数字人可视化交互” 的体验升级；

• 数字人具备拟人化沟通能力，可通过自然语言对话承接用户需求：例如用户询问 “招待费报销限制” 时，数字人同步调取智能问答知识库，以语音 + 文字 + 表情的生动形式输出答案，并展示关联制度原文；用户发起单据生成需求时，数字人通过语音交互引导补充关键信息，实时反馈单据生成进度；查询数据时，数字人同步呈现数据结果与解读报告，通过可视化图表动态演示指标变化趋势；

• 支持多场景适配，在 PC 端、移动端及线下智能终端均能提供一致的数字人交互体验，结合上下文记忆功能实现跨设备连贯对话，让财务智能服务从 “工具调用” 转变为 “伙伴式沟通”，进一步降低操作门槛，提升用户对财务系统的接受度与使用粘性，构建更具温度的财务智能交互生态。


四、项目评估


评估指标


1. 业务价值：智能申请覆盖率≥80%，表单错误率≤2%；智能问数月均使用量≥500 次，数据查询效率提升≥80%；智能问答月均解决问题≥500条，替代 80% 的人工问答工作量。


2. 落地可行性：与现有财务数字化平台、数据中台的兼容性≥95%。


3. 创新性：大模型在财务场景的多模态处理（文本 + 表格 + 数据）深度应用，技术方案在行业内具备可复制性。


4. 用户满意度：通过问卷调查，员工对智能体的整体满意度≥90 分（100 分制）。


5. 费用消耗：项目上线后，因效率提升与错误减少，每年降低财务共享服务成本≥15%。



申请理由



华润江中财务智能灵犀助手项目通过集成智能问答、智能申请、智能问数三大核心功能，以AI智能体、AI数字人交互方式重构财务共享流程，系统性解决传统模式下人工响应慢、表单错误率高、数据查询效率低等痛点。项目将带来多重可量化收益：覆盖80%以上申请场景并降低表单错误率至2%；提升数据查询效率80%以上，月均服务量超500次；替代80%人工问答工作量。其创新性体现在财务场景多模态处理技术的深度应用，兼容性达95%以上，预计年降本15%且用户满意度超90分，兼具业务价值与行业可复制性，是财务数字化转型的高效实践。

CIO时代：

CIO时代成立于2003年，由北大CIO论坛创始人姚乐博士带领论坛骨干创建。成立20年来，CIO时代以传播“新技术、新商业、新管理”知识为使命，专注CIO人群的培养和技术专题培训，为CIO提供数字化相关的资讯和专业研究内容，打造CIO领域专业、精准、多维度的垂直媒体平台，致力于成为“个人和组织数字化业务、管理和技术知识的赋能者”。


新基建创新研究院：

新基建创新研究院是CIO时代旗下的智库研究机构，汇聚院士、政府领导、央国企、医疗、教育、能源、制造等十多个行业专家组建智库，拥有千余位智库专家阵容。研究院的主要目标是建立政府、科研机构、高校和行业数字化转型从业者之间的合作，发挥政、产、学、研的桥梁、纽带作用，促进政府与企业间、企业与企业间的交流合作，促进数字产业落地。研究院希望将数字化的优秀实践提炼成理论方法，进行推广应用；助力提升行业数字化转型效率，促进数字中国建设。
