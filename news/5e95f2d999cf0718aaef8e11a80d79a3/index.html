圈粉又圈单!“深圳智造”闪耀2026CES国际消费电子展

深圳市音响行业协会 2026-01-13

2026年1月6日-9日在美国拉斯维加斯举行的

全球科技界年度盛事

——2026年国际消费电子展（CES2026）上

一股来自深圳的科技创新力量席卷全场


超过350家深企集中亮相

在具身智能、新能源、智能穿戴、

智能影像等核心赛道上

展出众多全球首秀的“黑科技”与尖端解决方案

从能自主完成长程复杂任务的机器人

到打破价格壁垒的消费级AR眼镜

再到实现能源自主供给的“光储+”生态

深圳企业正以扎实的

技术内功与前瞻的产品布局

在全球顶尖科技舞台上

彰显“硬核实力”与定义未来的能力


智能穿戴：交互入口“新标配”


智能眼镜是今年CES的另一大焦点，全球的多款智能眼镜集中亮相，智能眼镜正全力竞逐下一代交互入口。

其中，深圳AR品牌雷鸟创新（RayNeo）现场展示的全球首款支持eSIM功能的双目全彩AR眼镜“雷鸟X3 Pro Project eSIM”，可实现“无手机依赖”全天候使用体验，吸引大量观众驻足了解。



据介绍，X3 Pro Project eSIM在轻便的一体机身内，加入了eSIM通信模块，使得AR眼镜首次真正具备脱离手机的能力，无需通过手机或WiFi，即可独立完成包括通话、实时AI对话、实时翻译、在线流媒体播放等功能。值得一提的是，这款智能眼镜还搭载等效43英寸的3D空间屏幕、高通骁龙AR 1计算平台，以及“RayNeo AR应用虚拟机”，支持微信、抖音、B站等多款主流应用，让用户在AR眼镜与智能手机间无缝切换。


雷鸟创新创始人兼CEO李宏伟表示，AR眼镜是下一代通用计算平台，它就是下一代“智能手机”，雷鸟X3 Pro Project eSIM的全球首秀，也清晰指向雷鸟创新打造“下一代个人计算平台”的长期战略。



光峰科技则再次携蜻蜓G1、蜻蜓G1 mini、彩虹C1三款LCoS AR光机产品亮相CES。记者从光峰科技了解到，此次参展不仅要展现中国在半导体显示领域的硬核创新实力，更以突破性技术方案推动消费级AR眼镜价格下探至1500元以内，加速AR眼镜从“小众尝鲜”向“日常必备”转变。


价格下探与体验升级的双重突破，将推动AR眼镜在导航、翻译、社交分享、办公辅助等高频场景中实现广泛应用，最终推动人机交互进入“视觉化、场景化”的全新阶段。“来看产品和洽谈的客户很多，我们的产品在国际市场上非常有竞争力。”光峰科技参展工作人员连线记者时说。
