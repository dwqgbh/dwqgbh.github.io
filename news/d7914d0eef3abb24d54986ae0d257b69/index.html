会展陈先森 2026-06-06

重磅｜2026 工博会・国芯展全新独立大展全球招商启动！

📅展会时间：2026.10.12–10.16

📍展馆：上海・国家会展中心（5.2 馆）

✨展会主题：芯联工业 智创未来

核心理念：芯智融合・生态共生・交易赋能

🔥国产替代加速，万亿芯赛道迎来黄金窗口期

“十五五” 新质生产力战略加持，集成电路作为工业粮食、智造底座，AI 算力、汽车电子、工业母机、具身智能、新能源全产业链芯片需求爆发，国产芯片、设备、材料迎来规模化落地黄金机遇。

依托中国工博会国家级超大平台，2026 国芯展正式从展区升级为独立专业大展，打通「芯片设计→晶圆制造→先进封测→设备材料→终端落地」全链路，聚焦芯片对接工业全终端，打造展示 + 论坛 + 精准采购 + 评奖 + 投融资五位合一行业顶级平台。

🏛权威阵容加持，工博会官方旗舰 IC 展

指导单位：上海市经信委、上海市发改委

主办：东浩兰生集团

承办：东浩兰生会展集团、上海市集成电路行业协会

背靠工博会 23 万 + 全品类工业专业观众资源，上届工博会全网曝光23.86 亿次、3577 家媒体全程报道，亚洲 133 个国家采购组团到场，境外韩 / 日 / 马 / 泰 / 印等优质买家云集。

本届硬核数据

✅精选优质展商：100 + 头部企业

✅开幕高峰论坛：800 + 政企院士大咖参会

✅闭门行业年会：500 + 产业资本、龙头高管

✅定向邀约：120 + 整车 / 工控 / AI 头部采购集团

✅五大专项评奖赛道，可冲刺 CIIF 中国工业奥斯卡大奖

📍五大主题展区，全产业链一站式亮相

1｜集成电路设计展区

EDA 工具、IP 核、处理器 / 存储 / 电源 / 车规 / AI 芯片、MCU、射频芯片；代表：紫光展锐、华大九天、新思、瑞芯微、全志等

2｜晶圆制造与先进工艺

逻辑 / 特色工艺、IDM、化合物半导体、硅片；代表：中芯、华虹、长江存储、合肥晶合、沪硅产业等

3｜先进封装展区

晶圆级 / 系统级 / Chiplet 封装、AI & 车规封装、封装材料与设备；代表：长电、日月光、安靠、盛合晶微等

4｜半导体设备 & 材料展区

刻蚀 / 沉积 / 检测设备、光刻胶、特种气体、湿电子化学品、靶材；代表：中微、盛美、上海新阳等

5｜应用集成展区

设工业子岛 + 消费融合岛，聚焦汽车电子、机器人、工业母机、AI 算力、新能源、智能家居，芯片 + 嵌入式方案实景落地展示。

🎯「1+1+N+X+1」五维生态活动矩阵

▫️1 场千人高峰论坛：中国集成电路生态高峰论坛，政策发布、签约合作、全球连线、资本对接（800 人参会）

▫️1 场行业闭门年会・上海之夜：华东 IC 年度高端晚宴，政企、院士、投资、整车龙头面对面

▫️N 场细分论坛：车规芯片、工业算力、设备材料、机器人用芯、ESG 等十余场专题研讨会

▫️X 场一对一供需对接：120 + 终端大厂分行业专场闭门采购（汽车 / 工控 / 机器人 / 消费电子）

▫️1 项 IC 专项大赛：设计 / 制造 / 封测 / 设备材料 / 应用五大赛道评奖，获奖享订单、基金、展位多重福利

💎参展五大硬核收益

1️⃣ 海量精准客流：共享工博会 20 万 + 工业采购，车企 / 自动化 / 新能源 / 机器人原厂直采；

2️⃣ 权威品牌背书：央视、人民日报等全媒矩阵曝光，快速提升行业品牌地位；

3️⃣ 政企资本对接：直面各地经信、产业基金、园区资源，落地项目融资；

4️⃣ 新品首发高地：工博会官方新品发布专区，优先全行业曝光；

5️⃣ 奖项赋能：参评 CIIF 专项奖，获奖企业优先对接龙头订单、享受次年展位优惠。

📢招商范围

芯片设计、晶圆代工、封装测试、半导体设备、电子化学品、EDA/IP、功率 / 车规 / AI 芯片、嵌入式方案、检测机构、科研院所、产业园区、创投机构。
