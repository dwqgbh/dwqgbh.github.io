第一次!广交会整体搬上“云”端

广东发布 2020-06-22

在欧派家居2万平方米的豪华展厅，可以看到25款风格各异的橱柜、衣柜。除了直播展示外，欧派还精心设计了专访意大利家居设计师、探秘AI工厂、体验全自动生产等新颖环节。

精心布置的“云展厅”背后，少不了配套政策服务的支持。欧派家居财务负责人王欢说，“受疫情影响公司资金流陷入紧张状态，税务部门了解情况后主动上门服务，提供政策辅导，出口退税“随到随审”服务，1天就拿到退税款531万元，我们利用这笔资金可以更好地投入到技术研发、线上直播间、VR展厅打造等活动中去。”

广交会首次整体搬上“云”端，这个崭新的国际贸易平台打破了时空界限，也对配套服务提出了更高的要求。税务、市场监管、海关等部门都为此设置了“云”端定制服务礼包，为“云”上广交会的举办保驾护航。

主播正在直播中展示自家橱柜的产品质量。

//

税务“云辅导”

引导企业用好出口退税政策

//


“为了抗击经济压力，这次广交会线上举办对我们公司非常关键。”广州市虎头电池集团有限公司财务总监温丹玲坦言，上半年新冠疫情席卷全球，各国纷纷采取措施，封关、锁国、闭市，由此引发市场低迷、外汇紧缺、货币严重贬值、国际物流不畅、货物清关障碍等，导致市场走访和国内外各类展会不能如期安排、部分营销促销活动延后。受严峻的外贸形势影响，“2020年上半年我们公司外贸业务同比下降了20%”。

对于新形势下的参展方式，温丹玲认为将是虎头电池集团一次新的机遇。为此，虎头电池集团做了不少新探索，比如对重点产品提炼卖点和搜索关键词，综合考虑各销区时差和生活习惯等因素，做好各类产品直播时间段安排统筹等。

在温丹玲看来，以往线下参展时，传统优势企业通过展位的地理位置、规模、装饰展示企业的实力。而在“云”端模式下，企业需要走上在线平台，以往线下参展的优势被抹去，在标准化、模板化、统一化的平台上，通过对线上内容的美化包装，重新吸引客商关注。

在参展的过程中，虎头电池集团也得到税务部门的帮助指导。为助力参展企业做好广交会参展工作，广州税务部门组建了“首席税务服务官”团队“一对一”“手把手”辅导企业即时办理出口退税业务，“线上+线下”提高服务和辅导质效，确保税收优惠政策稳定推行。广州税务部门还与广交会财务人员建立线上沟通平台，掌握参展商名单，收集参展企业关注的各项税费优惠政策，为网上广交会开展做好“税务力量”储备。

“今年的展厅从往年的‘线下’搬到了‘线上’，税务部门对我们的服务也从往年的实体展厅现场辅导变成了‘云’辅导，为企业提供了便利，也提高了效率，让我们进一步享受国家减税降费的红利。”温丹玲说。

首次参加广交会的广州世观进出口贸易有限公司，也收到了越秀区税务局的“云端定制政策礼包”。该局税务人员通过“税视通”方式与企业视频连线，了解企业参展产品、市场需求等情况，为企业推送一揽子相关出口退税政策，并结合线上参展的特点，对企业开拓市场提出意见建议，引导企业用足用好出口退税政策，综合运用多领域便利政策，优化整体外贸经营模式，推动外贸业态发展。

“税务部门的服务太及时了。”广州世观进出口贸易有限公司财务负责人张敏仪点赞道，“我们今年是首次申请出口退税，之前担心疫情可能会影响退税进度，后来税务部门通过‘税视通’和我们在线核对资料，还可以‘容缺办理’，我们申请的第一笔退税，全流程都在网上就办好了，方便又快捷。”

//

海关“云会诊”

助力企业开拓海外市场

//


为适应线上广交会需求，广州海关专门推出八项服务举措。“相比实体展会的驻会监管服务模式，这次海关服务更具特色，很多措施具有首创意义。”广州会展中心海关副关长徐越说。

其中，广州海关尝试打破线上广交会展览品的“时空限制”。对线上广交会参展商和采购商为成交磋商而产生的样品，包括在线上广交会结束后进出境的，视同本届广交会的展览品，按规定享受展览品进出境的优惠措施。

“像这样的视频直播，也是一次全新尝试。”徐越说，海关总结历年来在广交会设立现场服务咨询台的经验，首次将实体展会的驻会咨询台“搬上网”，设立“广交会海关之窗”，外接广交会官网，内联“广州海关12360”热线，在线提供咨询服务。

同时，开设海关直播间，通过动漫视频、海关业务专家在线直播等方式，“面对面”进行政策宣讲。结合往届广交会常见问题，由专家团队来“把脉会诊”、答疑解惑，帮助企业学好、用好政策，化解经营风险。

该关还搭建了技术中心在线服务平台，采用“线上申请、线下邮递”方式，为参展企业提供出口产品的技术问题咨询、认证测试、检验检疫、验货鉴定、能力验证等技术服务，帮助出口企业应对技术贸易壁垒，助力企业开拓海外市场。

结合广交会对参展企业的资质审核，研究利用国际贸易单一窗口、海关信用管理信息化系统提供企业信用评价支持，帮助参展商和采购商了解拟交易企业的信用信息，帮助提高订单“撮合”的成功率。

“我们想方设法为参展商减负，化解疫情影响。”徐越说，广交会线上举办公告发布前已有52家企业约4.2万件展品发货，广州海关设立“绿色通道”，“一对一”在线指导企业办理通关手续，压缩展览品通关时间，并引导已入境展品存放至展馆方仓库，协调参展商与展馆方沟通减免仓储保管费。

//

商品“云维权”

为知识产权保驾护航

//


“云”端广交会上，参展商最大的担忧之一，便是知识产权保护问题。不少参展商此前都有疑问，发生了线上知识产权侵权纠纷，应如何应对处理？日前，在第127届广交会指挥中心的知识产权纠纷投诉接待站，几十名工作人员正紧张地忙碌着，高效有序地调查处理线上知识产权纠纷。

，
对于参展商和采购商来说，发现涉嫌侵权行为后，只要登录广交会官网，在“服务专区-知识产权保护和贸易纠纷处理”模块提交知识产权投诉、申诉、撤诉申请和有关电子证据材料，就可以进行线上维权了。

事实上，早在参展企业集体“上云”前，省市场监督管理局（知识产权局）已会同广交会主办方，建立起“线上+线下”相结合的投诉处理新模式，组织了知识产权保护专家队伍，为广交会护航。

“今年，办展方式发生重大变革，展会知识产权保护工作也面临着全新的挑战。”据省市场监管局（知识产权局）副局长何巨峰介绍，本届广交会从线下搬到“云上”，由于网上展品体量大、受关注程度高，知识产权纠纷数量可能上升；同时，商品在线上展出，缺乏实物比对和纠纷双方现场交流的环节，直接对网上展品或直播视频涉嫌侵权的判定难度加大。

结合网上举办展会的特点，省市场监管局报请国家知识产权局统筹协调全国各省市知识产权管理部门开展展前知识产权侵权风险排查，同时迅速部署全省各地开展展前排查工作，有效提升参展企业知识产权保护意识，净化展会知识产权保护环境，防范和降低知识产权侵权风险；商请国家知识产权局派出权威专家来粤，现场指导并全程参与网上广交会知识产权保护工作，专门增设资深专家组成的纠纷处理决策组，确保判定意见的专业性、准确性和权威性；指导广交会主办方制定出台《广交会关于网上参展内容涉嫌侵犯知识产权投诉及处理的暂行规定》，对网上广交会知识产权纠纷处理的结构设计和流程进行再造，优化知识产权纠纷处理程序，降低纠纷处理成本，提高纠纷解决效能。

“目前来看，线上广交会的纠纷投诉数量保持平稳，纠纷双方当事人比较理性，纠纷解决顺畅有序。”何巨峰表示，本届参展商知识产权保护意识明显提升，上线后被投诉的参展企业能够积极主动行使答辩、举证、异议等权利，维护自己合法权益，同时，有的被投诉方自行将涉嫌侵权产品“下架”，有的纠纷双方能够主动协商解决。

为更好应对知识产权纠纷投诉，参展企业应注意哪些事项？何巨峰建议，首先是自查自纠，排除高风险侵权展品，避免自身产品侵犯他人知识产权；其次，做好自有知识产权备案工作，对自有的知识产权权利证书和其他证据材料进行全面整理，并按要求报广交会主办方备案，提高应对知识产权纠纷投诉的效率。“一旦出现知识产权纠纷投诉，参展企业应按照广交会主办方要求，积极行使答辩救济权利，主动履行举证责任，防止因应对不当而出现不利后果。”
