/ 期刊简介/

About the Journal

《智能制造》杂志创刊于 1994 年，由机械工业联合会主管、机械工业信息研究院主办，国家制造强国建设战略咨询委员会智能制造专家委员会指导，为中国期刊方阵双效期刊，正奋力打造成为中国智能制造领域智库型传播与服务平台。
历史上，国家科委实施以“甩图板”为口号的全国“CAD 应用工程”，国家科委工业司和机械部科技及质量管理司，大力协同推进 CAD 应用工程，由此创刊。之后，科技部、经贸委等部门成立“全国制造业信息化工程协调领导小组”。制造业信息化成为《国家中长期科学和技术发展规划纲要》的重点方向，为响应政策要求，将刊名调整为《CAD/CAM 与制造业信息化》（《智能制造》杂志前身）。凝聚政府主管部门、科研院所、行业协会的专家学者，制造业大中型企业、大专 院校的丰富资源，洞察产业发展，把脉行业趋势，传播国内外智能制造领域政策 信息、行业新闻、科技成果和产业实践，重点聚焦智能装备、汽车制造、工业互联网、ICT 技术应用、工业机器人、智能工厂、航空航天、新能源装备、石化冶 金、能源电力等重点技术和领域，助力中国智能制造发展。
在各级领导和各界读者的支持下，现已形成《智能制造》杂志、智造网、智能制造 IMS 微信公众号和视频号、智能制造官方微博等全媒体平台，全网粉丝数量超过100万，成为中国智能制造领域颇具影响力的窗口。
为方便更多作者高效快捷投稿，现发布《智能制造》2025年度征稿计划：

/ 主要栏目设置/

Main Column

《智能制造》全年对外征集以下稿件：
1.智能装备
装备制造业领域的智能装备新技术、新工艺、装备智能化升级改造应用的典型案例等。
2.智能工厂
智慧工厂解决方案的典型案例等。
3.软件应用
大数据、工业互联网、人工智能等在制造业的实践应用。
4.智慧物流
智能供应链与智慧物流典型案例。
5.创新技术应用
智能制造的新技术、新应用的研究与实践。
6.人才培养
智能制造领域的高等教育人才培养理论研究、课程培养方案策略及实践案例等。

/ 特别征集 /

Special Collection

本年度主题策划栏目选题分别为前瞻科技趋势、工业母机、海洋工程装备及高技术船舶、先进轨道交通装备、智能汽车和智能机器人。

/ 投稿须知/

Instructions for Authors

1. 本刊以尊重原创、倡导学术创新、杜绝学术不端为征稿基本原则。
2. 稿件字数要求5000字（含图/表）以上为宜。本刊不接收一稿多投稿件，在稿件受理期间，不得另投他刊。
3. 论文的排序：中文标题、作者姓名、工作单位（每位作者一个工作单位，包括邮政编码和所在省、市）、中文摘要、中文关键词、中图分类号、文献标志码、引言、正文、结语、参考文献、基金项目、作者简介，具体格式可参考投稿网站的作者中心中投稿须知及论文模板。
