2025年4月第三周汽车与机器人行业资讯速递

研发工程师视角 2025-04-19


🚗 汽车行业要闻 

 
1. 德国研究：电动车故障率首超燃油车 
   - 事件：德国ADAC最新数据显示，电动车故障率仅占救援请求的1.2%，首次低于燃油车，可靠性优势凸显。 
   - 点评：技术成熟度及动力机构的简单，提升加速电动车普及，但需警惕充电基建与维修配套的跟进速度。 电动车智能化增提高了系统软件的故障率，虽然不会导致车辆趴窝，但消费者感知很明显。

2. 哪吒汽车深陷危机，创始人公开致歉 
   - 事件：方运舟承诺4月30日前解决配件供应问题，并寻求融资；公司销售体系停摆。 
   - 点评：新势力洗牌加速，现金流与供应链管理成生死线，行业或现更多兼并重组。 

3. 知豆汽车“复活”，微型车市场再起波澜 
   - 事件：知豆彩虹Pro上市，售价3.89万起，续航282km，主打性价比与个性化。 
   - 点评：低价电动车或成下沉市场突破口，但品牌信任重建仍是挑战。 


4. 央视曝光新能源消费纠纷 
   - 事件：免费权益变付费、智驾功能夸大宣传、维修难等问题频发，行业监管趋严。 
   - 点评：政策需细化责任界定，车企应加强用户教育，避免“技术跃进”透支信任。 

5. 特斯拉保险费用飙升，维修成本成主因 
   - 事件：Model Y保费涨幅达29%，美国市场因维修成本高企，保险公司倾向报废车辆。 
   - 点评：智能化与高集成设计推高维修门槛，车企自建维修网络或成趋势。 


🤖 机器人行业动态 


1. 全球首场机器人半马开赛，宇树G1亮相 
   - 事件：宇树G1机器人完成适赛化升级，4月19日参与半程马拉松，展示运动性能。 
   - 点评：人形机器人应用场景向极限拓展，运动控制技术突破是关键。 

2. Hugging Face进军实体机器人领域 
   - 事件：开源AI巨头收购Pollen Robotics，计划以开源模式推动机器人技术普及。 
   - 点评：AI与机器人融合加速，开源生态或降低行业技术门槛。 

 

3. 戴盟发布全球首款多维触觉传感器 
   - 事件：DM-Tac W传感器覆盖4万感知单元/平方厘米，推动灵巧手精细化操作。 
   - 点评：触觉感知技术突破将赋能工业自动化与医疗机器人，国产替代空间广阔。 

 

4. 上海成立具身智创基金，5.65亿布局产业链 
   - 事件：15家企业联合投资，聚焦具身智能上下游技术研发与商业化。 
   - 点评：资本加码机器人赛道，政策与市场双驱动下，国产供应链有望崛起。 


🔍 行业趋势观察 
- 汽车智能化监管收紧：工信部要求车企禁用“智驾”宣传，改称“辅助驾驶”，夸大宣传或面临刑事风险。 
- 固态电池技术角逐：宁德时代领跑中国固态电池竞争力榜单，太蓝、清陶紧随其后。

 
- 机器人“数据+技术”双驱动：天猫养车与九识智能合作无人车维保，推动服务标准化。 



📊 数据快报 
- 一季度中国新能源车销量占比超40%，出口东盟汽车零配件增速超20%。 
- 理想ONE推出终身质保转移计划，回馈首批用户。凯迪拉克推出限时购车获终身免费保养。伴随各车企竞争加剧，终身质保及免费保养或将成为标配 。
