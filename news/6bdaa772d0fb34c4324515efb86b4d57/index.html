数聚产业动能,智启发展新篇——2025数博会圆满结束

首席数据官群管 2025-08-31

 

一、主题引领：数据要素与产业升级的深度共振

 

2025年8月28日至30日，以“数聚产业动能 智启发展新篇”为主题的中国国际大数据产业博览会在贵阳盛大举办。这一主题紧扣国家“东数西算”战略和数字经济高质量发展要求，首次将“数据要素市场化配置改革”贯穿全流程，设置六大主题展馆、26场交流活动和34场特色活动，吸引了来自31个 国家的1.6万余名嘉宾和375家中外企业参展 。

 

核心价值维度：

 

1. 战略高度：呼应“数字中国”建设目标，将数据要素提升至与土地、劳动力、资本同等重要的生产要素地位，推动数据从资源向资产、资本的价值转化。

2. 产业广度：覆盖算力基础设施、数据交易、人工智能、低空经济等25个行业领域，形成“技术展示-场景应用-生态构建”的完整闭环。

3. 国际深度：设立“数字丝绸之路”供需对接专区，促成国内外企业现场接单，推动中国数字技术标准和解决方案的全球化输出。

 

二、成果荟萃：技术突破与产业协同的双重跨越

 

数博会期间，一系列标志性成果集中释放，勾勒出中国数字经济创新发展的清晰脉络：

 

1. 战略级成果发布：

 

- 政策与标准：发布《数据产业图谱（2025）》《AI CITY城市智能体前瞻性研究报告》等纲领性文件，首次系统性提出数据要素流通的“技术-场景-生态”三维评估体系。

- 算力基建：全国智算总规模达78万PFlops，八大枢纽节点集聚60%以上新增算力，华为云整体算力规模同比增长250%，昇腾AI云服务客户从321家激增至1714家 。

- 数据交易：贵阳大数据交易所展示数据交易全生命周期服务体系，推动北京、深圳、贵阳三地数交所战略签约，建立跨区域数据流通机制 。

 

2. 企业级创新突破：

 

- 华为：发布新一代分布式云架构，实现“空间换算力、带宽换算力、能源换算力”的系统级优化，助力贵州建成人工智能大模型公共服务平台，向70余家政企开放API接口。

- 中国电信：推出“5G+工业互联网”解决方案，在矿山监管、智慧农业等领域实现数据实时采集与设备智能诊断，推动贵州数字产业规模突破2500亿元。

- 广汽集团：与华为联合发布首款搭载华为云车机的埃安UT车型，实现车载系统与云端算力的无缝协同，标志着汽车产业向“软件定义汽车”时代加速迈进。

 

3. 生态级协同效应：

 

- 产学研融合：国家信息中心联合华为等30余家机构发布《区域算力网—高速互联篇研究报告》，提出“城市-区域-全国”三级算力网络建设路径 。

- 国际合作：设立国际供需对接合作展区，促成特斯拉赛博越野车首秀、诚迈科技高性能信创产品发布等跨国合作项目，推动64项前沿成果全球首发。

 

三、企业赋能：从技术展示到价值创造的范式转变

 

数博会作为全球数字科技企业的“竞技场”，正在重塑企业创新发展的底层逻辑：

 

1. 技术验证与场景落地的双向加速：

 

- 华为昇腾：通过数博会平台展示多模态生成式AI技术，吸引能源、金融等领域12家头部企业达成联合研发协议，推动AI模型在生产调度、风险预测等场景的规模化应用。

- 中科星图：发布GEOVIS iFlight低空智航平台，整合空天遥感数据与无人机调度系统，在贵州实现火情监测响应时间缩短至15分钟。

 

2. 生态构建与资源整合的乘数效应：

 

- 数据要素市场：数博会期间达成数据标注、算力租赁等合作项目100个，引资超463亿元，其中数据标注领域签约金额达7.2亿元，推动全国首个“数据要素服务创新中心”落地贵阳 。

- 资本对接：华为开发者大赛吸引2000余名开发者参赛，促成37个优质项目获得风险投资，平均单个项目估值增长3.2倍。

 

3. 品牌塑造与标准输出的战略价值：

 

- 中国电信：通过数智服务馆展示数据加密、隐私计算等安全技术，中标贵州省“可信数据空间”建设项目，推动企业从通信服务商向数字安全解决方案提供商转型。

- 万联易达：发布产业互联网大模型，为制造业企业提供智能排产、供应链优化等服务，客户平均生产效率提升28%，成本降低15%。

 

四、数交所突围：数据要素市场化的贵州样本

 

全国21家数据交易所齐聚数博会，贵阳大数据交易所的实践为破解数据流通难题提供了系统性方案：

 

1. 全链条服务体系：

 

- 登记确权：建立数据资产登记、产权认证、价值评估的标准化流程，累计完成数据产品登记3.2万件，覆盖气象、电力等25个行业领域 。

- 交易流通：创新“数据信托”模式，引入区块链存证和智能合约技术，实现数据交易全流程可追溯，2024年平台交易额突破80亿元，同比增长120% 。

 

2. 跨区域协同机制：

 

- 标准共建：联合国家发展改革委价格成本和认证中心成立“数据要素价格形成与认定创新实验室”，发布数据定价方法论和交易规则白皮书。

- 生态共赢：与上海、山东等地行业协会签署战略合作协议，推动公共数据、算力资源等跨区域流通，2025年上半年跨区域交易占比达45% 。

 

3. 场景化价值释放：

 

- 政务数据开放：通过“贵州省公共资源（数据要素）交易中心”平台，开放交通、医疗等领域脱敏数据集，支撑23个智慧城市项目建设 。

- 行业应用创新：与金融机构合作开发“数据资产质押融资”产品，为中小微企业提供基于数据价值的信贷支持，累计发放贷款超20亿元 。

 

五、政策风向：从战略擘画到实施路径的精准锚定

 

国家数据局领导的系列讲话，为未来一年数据领域发展划定清晰路线图：

 

1. 数字经济发展目标：

 

- 规模跃升：预计2025年底数字经济增加值达49万亿元，占GDP比重35%，数据要素对经济增长贡献率突破15% 。

- 算力布局：坚持“东数西算”战略，推动算力设施“利当下、谋长远”发展，到2026年建成全国一体化算力网，实现算力资源跨区域调度效率提升50% 。

 

2. 数据要素市场化改革：

 

- 高质量数据集建设：通过发放“语料券”补贴供给端、需求端和交易端，培育“为优质数据买单”的市场共识，2025年计划建设100个重点领域高质量数据集 。

- 安全合规体系：建立数据分类分级、跨境流动等标准规范，推动隐私计算、联邦学习等技术在金融、医疗等敏感领域的规模化应用 。

 

3. 产业生态培育路径：

 

- 集聚区建设：部署数据产业集聚区试点，重点发展算力服务、数据标注、AI训练等细分领域，2025年培育10家百亿级数据企业 。

- 绿色低碳发展：推动算力中心采用液冷技术、可再生能源供电，目标2026年全国绿色算力占比超60%，单位算力能耗下降18% 。

 

六、未来展望：数字文明新篇的中国叙事

 

2025数博会不仅是一场科技盛宴，更是数字文明时代的战略转折点：

 

1. 技术范式迭代：

 

- AI原生：多模态大模型从实验室走向生产一线，预计2026年将有70%的企业核心业务系统接入AI能力，推动全要素生产率提升3-5个百分点。

- 算力普惠：通过“算力券”等政策工具，降低中小企业算力使用门槛，目标2027年实现算力服务成本下降40%，覆盖率提升至85% 。

 

2. 产业变革机遇：

 

- 低空经济：中科星图等企业的技术突破，将推动无人机物流、城市巡检等场景商业化落地，预计2028年市场规模超5000亿元。

- 元宇宙应用：数智体验馆展示的全息投影、VR/AR技术，正在重构文旅、教育等领域的用户体验，预计2026年相关市场规模突破2000亿元。

 

3. 全球治理参与：

 

- 标准输出：中国主导的“数据要素流通国际标准”进入ISO立项阶段，有望在2026年成为全球数据跨境流动的重要参考 。

- 生态共建：“数字丝绸之路”供需对接专区促成的跨国合作项目，已带动沿线国家数字经济增速提升2.3个百分点，形成“中国方案”的全球示范效应。

 

2025数博会的成功举办，标志着中国数字经济发展进入“数据要素驱动、场景价值引领、全球生态协同”的新阶段。从贵阳这座“中国数谷”出发，数据正在重塑产业竞争格局，算力正在重构经济地理版图，而这场由技术创新、政策突破和生态协同共同驱动的数字革命，终将书写人类文明发展的新篇章。正如国家数据局局长刘烈宏在开幕式上所言：“当数据成为新的生产要素，当算力成为新的基础设施，我们正在见证一个比工业革命更深刻的变革时代。”
