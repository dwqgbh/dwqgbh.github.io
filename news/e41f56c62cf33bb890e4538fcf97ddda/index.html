2026重庆新能源与新型储能展览会

西部高交会 2026-06-15

承办单位：鼎诺国际会展（北京）有限公司

展会主题：绿动新重庆・能创新未来・储汇万亿能

招商咨询：曾先生 19802738018（微信同号）

一、展会背景

1. 产业背景

新能源与新型储能产业是双碳战略核心载体、能源转型关键支撑、新质生产力重点赛道，是重庆 “33618” 现代制造业集群体系万亿级特色优势产业，覆盖光伏、风电、氢能、储能电池、系统集成、充换电设备、能源管理等全链条，贯穿新能源汽车、工业节能、智慧电网、绿色建筑、低空经济等多元场景，市场需求爆发式增长。

国家战略顶层赋能，产业迎来黄金发展期。国家 “十五五” 规划、《新型储能制造业高质量发展行动方案》将新型储能、新能源装备列为重点培育的高潜力赛道，政策密集加码、资金持续投入。重庆紧扣成渝双城经济圈、国家综合能源基地、西部绿色低碳高地三大战略，出台《重庆市新能源及新型储能产业发展行动计划》，构建 “风光水火储一体化、源网荷储协同化” 全生态，全力打造西部新能源与新型储能产业第一高地。

产业规模高速扩容，重庆稳居西部龙头地位。重庆已将新能源及新型储能明确为全市6 大千亿级特色优势产业集群，产业规模持续扩大。2025 年铜梁区新型储能产业产值达165.7 亿元，海辰储能单厂产值破百亿。全市可再生能源装机占比达45.6%，已建成156 万千瓦电化学储能项目、120 万千瓦抽水蓄能项目。成渝协同发力，成为全国重要的绿色能源产业增长极。

龙头集聚技术领跑，储能赛道引爆新增长。重庆构建 “铜梁 + 江津 + 璧山 + 两江新区 + 西部科学城” 产业布局，集聚海辰储能、厚生新材料、兴恒、金汇能、科达、阳光电源、隆基绿能、亿华通等龙头企业。海辰储能重庆基地为全球灯塔工厂，长时储能产品规模化出口；厚生新材料锂电池隔膜产能27 亿㎡/ 年，全球领先。重庆依托汽车、电子、材料等优势产业，形成 “材料 — 电芯 —PACK— 系统集成 — 场景应用” 完整产业链，国产化替代全面提速。

成渝协同共建能源走廊，链式效应持续放大。成渝联合打造 “成渝新能源与新型储能产业集群”，串联重庆铜梁、江津、璧山与成都郫都、宜宾、绵阳等核心片区，协同布局储能电池、光伏组件、氢能装备、能源管理系统等赛道。重庆侧重储能制造、系统集成、终端应用；四川聚焦锂矿资源、光伏硅料、水电配套，形成资源 + 制造 + 应用互补协同格局。

2. 展会概况

2026 重庆新能源与新型储能展览会，作为重庆高交会核心专题展，定于 2026 年 10 月 16 日 —18 日在重庆国际博览中心盛大举办。展会与先进材料、智慧显示、集成电路、低空经济、人工智能与机器人等十余场主题展会同期联动，共享国家级展会政企资源、专业客流、媒体曝光与产业资本流量。

展会立足成渝、辐射西部、面向全国，全域覆盖新型储能、光伏风电、氢能燃料电池、充换电设备、能源管理系统、新能源汽车配套、绿色低碳技术等核心领域，集中展示前沿技术、创新产品、核心部件与整体解决方案，打造西部规格最高、产业链最全、落地性最强的新能源与新型储能产业交易对接盛会。

3. 展会定位

以 “绿动新重庆，能创新未来，储汇万亿能” 为核心定位，立足国家级科创平台、赋能西部绿色能源产业升级，重点布局三大战略板块：新型储能全产业链创新成果、风光氢储多能互补技术、新能源 + 千行百业落地方案，推动新能源与储能技术在西部地区的交流、对接与规模化落地。

4. 总体规模

本届展会展览总面积预计40000㎡，将汇聚国内外新能源、储能领域顶尖企业、央企国企、上市龙头、科研院所及重点实验室，吸引数万名政企采购、能源运营商、终端制造、系统集成、科研机构、投资机构、渠道商等专业观众到场对接，打造西部年度规模最大、规格最高、落地性最强的新能源与新型储能产业成果展示与交易平台。

二、展会核心优势

✅ 万亿赛道风口正盛，西部增量空间广阔

新能源与储能是国家战略级绿色低碳产业，2025 年中国市场规模超10 万亿元，储能装机量五年增四十倍，国产化替代与场景落地需求爆发。重庆作为西部绿色能源核心枢纽，已形成千亿级新能源与储能产业集群，新能源汽车产业规模超 8000 亿元，工业、电网、建筑、交通等场景需求集中释放，是西部最大的新能源与储能消费市场与产业集群地。

✅ 西部全产业链展会，差异化优势无可替代

成渝地区是西部新能源与储能产业高地，但长期缺少覆盖储能电池 + 风光设备 + 氢能装备 + 充换电设施 + 能源管理 + 场景方案的一体化专业展会。本次展会精准补齐行业空白，是企业深耕西部绿色能源市场、抢占万亿赛道的核心唯一窗口，赛道稀缺性显著。

✅ 产业集群强势集聚，龙头企业引领发展

重庆构建 “铜梁 + 江津 + 璧山 + 两江新区 + 高新区” 产业布局，集聚海辰储能、厚生新材料、阳光电源、隆基绿能、亿华通等龙头企业，铜梁区已聚集12 家以新型储能为核心的规上工业企业、全球灯塔工厂、国家级储能产业集群，产业链完整度达 75% 以上，产业集群势能全面释放。

✅ 政策红利密集叠加，地方支持力度领先

重庆出台《新能源及新型储能产业发展行动计划》，设立绿色能源专项扶持基金，对重大储能项目最高补贴2000 万元、首台套产品奖励最高 500 万元、储能示范项目补贴最高 300 万元。符合条件的特装参展企业可申请最高 50% 展位费补贴（以所在产业园区实际扶持政策为准），叠加税收减免、场景试点、研发奖补等红利，大幅降低拓市成本。

✅ 全产业链全景覆盖，一站式供需精准对接

展会贯通 “资源开采 — 材料制备 — 电芯制造 —PACK 集成 — 储能系统 — 风光设备 — 氢能装备 — 充换电设施 — 能源管理 — 场景落地” 全产业生态，实现上下游企业精准配对、批量采购、配套洽谈、技术合作一站式落地，高效打通研发、制造、采购、应用、服务完整供需闭环。

✅ 多展同期联动，全域高端资源互通共享

与高交会全系主题展会同台联动举办，汇聚先进材料、智慧显示、集成电路、低空经济、新能源汽车、人工智能与机器人等跨行业优质采购商、核心研发团队、企业决策层与产业资本，打破单一行业壁垒，拓宽企业配套合作、项目落地、场景试点、渠道拓展多元路径。

✅ 国家级平台权威背书，品牌层级强力拉升

依托重庆高交会五部委联合主办的国家级权威平台，展会公信力、行业影响力、产业号召力稳居西部同类展会首位，助力企业塑造绿色低碳 + 硬核科技双重品牌形象、提升行业话语权，快速切入西部绿色能源产业核心供应链体系。

三、展品范围

本届展会围绕新能源与储能全产业链，按照 “源 — 储 — 网 — 荷 — 用” 完整生态链条设置展区，同时新增 \\“新型储能全产业链”、“风光氢储多能互补”、“新能源汽车能源配套”、“零碳园区与绿色建筑”以及“成渝新能源与储能协同创新成果”\\ 五大特色专题展区，集中展示成渝地区在绿色能源领域的产业实力与创新成果。

1. 新型储能全产业链（专题展区）

  • 储能电池与电芯：磷酸铁锂、三元、锰铁锂、钠离子、液流电池、固态电池、储能电芯（海辰储能、厚生新材料重点展示）；

  • 储能系统与集成：集装箱式储能、户用储能、工商业储能、电网级储能、虚拟电厂、储能 BMS、EMS 系统；

  • 储能核心部件：PCS 逆变器、变压器、开关柜、散热系统、消防设备、储能电缆、连接器；

  • 储能材料与设备：正负极材料、隔膜、电解液、铜箔、铝箔、储能生产设备、检测设备。

2. 光伏与风力发电

  • 光伏组件与系统：单晶 / 多晶组件、TOPCon/HJT/IBC 高效组件、BIPV 光伏建筑一体化、光伏逆变器、支架、跟踪系统；

  • 光伏材料与设备：硅料、硅片、玻璃、银浆、光伏胶膜、光伏生产设备、检测仪器；

  • 风力发电设备：风机整机、叶片、齿轮箱、发电机、控制系统、风电运维设备、海上风电装备。

3. 氢能与燃料电池

  • 氢能制备与储运：电解水制氢设备、绿氢 / 灰氢 / 蓝氢技术、储氢瓶、加氢站设备、氢气压缩机、输氢管道；

  • 燃料电池与系统：氢燃料电池电堆、燃料电池发动机、车载氢系统、备用电源、热电联供系统；

  • 氢燃料电池汽车：氢燃料乘用车、商用车、叉车、重卡、加氢车。

4. 充换电设备与能源配套

  • 充电桩 / 站设备：直流快充桩、交流慢充桩、超级充电桩、光储充一体化充电站、换电站设备；

  • 车网互动（V2G）：V2G 充电桩、双向逆变器、车网协同控制系统、新能源汽车储能应用；

  • 智能电网设备：储能变流器、SVG 无功补偿、智能电表、电力物联网设备、配电自动化系统。

5. 新能源汽车能源配套（专题展区）

  • 动力电池系统：新能源汽车动力电池、电池 PACK、BMS 管理系统、热管理系统；

  • 车载储能与节能：车载储能电池、轻量化电池箱体、电池回收设备、新能源汽车节能技术；

  • 汽车电子电气：车载充电机、DC/DC 转换器、高压配电盒、车载电源管理系统。

6. 零碳园区与绿色低碳（专题展区）

  • 分布式能源系统：微电网、离网系统、风光储一体化、综合能源服务平台；

  • 绿色建筑节能：光伏屋顶、储能空调、节能门窗、建筑能源管理系统、绿色建材；

  • 工业节能与减排：工业储能、余热余压利用、节能电机、能源审计与管理软件。

7. 成渝新能源与储能协同创新成果展区（专题展区）

  • 成渝两地协同创新成果、产学研合作项目、跨区域产业链配套成果、重点企业新品首发、绿色能源示范项目案例。

产业链配套企业

渠道商贸与代理商

西南各省市新能源设备代理商、储能经销商、工程集成商、跨境贸易商、系统集成商、绿色能源产品贸易商。

科研院校与创新机构

新能源科学、储能工程、电力系统、材料科学、氢能技术相关高校院系、重点实验室、产业研究院、国家级科研院所、技术成果转化平台、绿色能源创新团队（重庆大学、重庆理工大学、西南大学、明月湖实验室、储能与智慧能源创新中心等）。

五、参展回报

🔹 抢占万亿绿色赛道，精准对接刚需订单

🔹 卡位西部产业高地，构建区域竞争壁垒

依托重庆西部新能源与储能产业高地优势，抢占双碳目标下能源转型窗口期。重庆已形成千亿级新能源与储能产业集群，拥有全球灯塔工厂、国家级产业集群，全方位彰显西部绿色能源产业硬核势能，助力企业快速切入增量市场，率先建立西南区域品牌壁垒与渠道优势。

🔹 对接政企政策红利，享受多重产业赋能

精准对接重庆绿色能源专项扶持基金、重大项目最高 2000 万元补贴、首台套最高 500 万元奖补、示范项目最高 300 万元支持等多重红利。符合条件的参展企业可申请展位费最高 50% 补贴，高效对接政府产业项目、园区招商、技术改造与成果转化工程，降本增效。

🔹 展会平台加持，全方位拔高品牌层级

🔹 一站式渠道布局，低成本深耕西部市场
