智能感知领军者机智人科技即将亮相2025高交会,多款激光雷达产品引领行业创新

Witty Sensor WR机智人 2025-11-13

深圳人工智能与机器人展（第二十七届高交会）11月14-16在深圳国际会展中心举办，深圳人民政府主办，展览面积40万平方，5000+品牌展商，40万+专业观众，100+国家和地区；本届展会将集中展出人工智能及机器人全产业链的前沿科技成果，涵盖人工智能基础层，人工智能技术，人工智能集成应用，元宇宙、VR/AR虚拟现实、3D全息，工业机器人及协作机器人，建筑机器人，服务机器人、特种机器人、核心部件等。

深圳市机智人科技有限公司，这家深耕机器人感知技术多年的国家激光雷达标准制定者，将在2025年深圳高交会上展示其最新技术成果， 作为国内机器人感知技术领域的先行者，深圳市机智人科技有限公司自2014年起就建立了早期机器人技术交流社区ROS维基技术交流社区；基于对机器人技术的深刻理解，该公司先后为多所大学和研究机构提供多款移动机器人 ；其产品已供应给中科院、清华大学、同济大学、浙江大学和深圳大学等知名院校 。凭借在机器人技术领域的专业能力，机智人科技进入激光雷达领域，开发出WR （Witty Robotics）系列WS(Witty Sensor)激光雷达 。目前，公司激光雷达产品已有三大系列，20多款成熟型号，已经在机器人公司、AGV搬运和低速自动驾驶机器人等领域树立了良好口碑 。

1.公司发展历程

  •  公司核心团队在2014年底就建立了中国最早期的机器人技术交流社区ROS维基技术交流社区，展现出深厚的技术积累和行业影响力 。

  • 机智人科技成立于2017年3月24日 ，作为一家服务型机器人研发商，基于人脸识别、语音交互、声源定位等系统，以及机器人操作系统，研发了通用服务机器人、银行服务机器人、搬运机器人等产品

  • 2017年11月，机智人科技参与制定国内激光雷达标准初审制定，成为起草国家标准的发起代表企业 ³，彰显了其在行业内的技术领导地位。

  • 随着公司不断发展，2019年机智人科技获得欧盟TUV颁发的CE证书，产品质量获得国际认可 。

  • 2020年11月23日，公司再次受邀作为中国电器工业协会《移动机器人振动与冲击试验条件制定方法》与《机器人控制部件可靠性强化试验方法》团体标准第二次工作组会议标准起草成员单位 。

  • 2023年，公司投入多线面阵激光雷达的研发，目前公司的首款150线的面阵激光雷达产品深受市场好评。

3. 核心产品线

  • 机智人科技的激光雷达产品以其高精度和可靠性著称，涵盖了导航、避障和3D感知等多个领域。

    导航型激光雷达

  • 机智人®WS-15/25PCD-H2超精单线导航型激光雷达测程远、精度高、分辨率高，具备采样速度快、抗光干扰性强、体积小巧和点云细密的特点 。

  • 它基于TOF原理实现高精度导航，在15米/25米范围内可精准导航，270°的扫描角度，0.1/0.15角分辨率 。

  • 用户反馈点云数据十分稳定，在物体弧线处的粗细十分均匀。玻璃处扫描较为笔直，没有大量的重影，可以模拟呈直线。

  •  多椅环境下依然可以扫描成直线边界清晰，可用于有玻璃墙面和狭窄空间场景，保证人流拥挤处的路径规划的精确性和安全性 。

     避障型激光雷达

  • 机智人®WS-15/25PNP/NPN-H3超精单线避障激光雷达，同样具备超精单线的优势，在15米或25米距离内实现精准避障 。

  • 该雷达在空旷或狭窄环境下，通过高扫描、低分辨角0.3°、高点云数据，能敏锐感知周边环境，16组防区为设备提供可靠的避障支持 。

  •  它合理的区域检测设置，可独立输出信号，有效助力各类设备避开复杂障碍物，保障运行安全 。

    环境适应型激光雷达

  • 公司的WR-DF防烟雾型激光雷达表现出色，具备IP65防护等级，工作温度范围**-30℃~50℃**，能够适应恶劣环境条件 ³。其检测角度达270°，检测分辨角为0.36-0.012°，检测误差仅±2cm，响应时间为42ms 。

  • WR-H-DM抗磁防尘型激光雷达其检测角度达270°，检测分辨角为0.36 - 0.012°，检测误差仅±2cm，响应时间为42ms 。质量稳定可靠，抗磁能力较强，高磁环境（100~300GS）下能够正常输出数据 。用户反馈抗粉尘能力较强，使用125目粉末撒在光学窗口上，依然有正常的数据输出 。

4.全新产品亮相

  • 据近期展会信息显示，机智人科技将携多款前沿产品亮相2025高交会，展示其在激光雷达领域的最新技术突破，深圳市机智人科技有限公司致力于通过高性能三维感知方案，推动机器人、智能网联汽车、智慧交通等领域的规模化落地，让精准感知成为智能世界的标配。其中产品展示如下;

 144线全新纯固态面阵激光雷达WS-100/20PCD-ET3 3D LiDAR

  • 深圳市机智人科技有限公司即将推出公司新一代面阵激光雷达产品WS-100/20PCD-ET3，该产品以120°×90°超广视场角，0.625°×0.625°的高精度角分辨率，以及70米(90%反射率)的探测能力，为机器人、智能驾驶、工业自动化等领域带来三维环境感知体验。

  • 新品拥有120°水平视场角和90°垂直视场角，增加了扫描覆盖范围，可一次性捕捉更广阔的场景信息，避免扫描盲区。无论是复杂城市场景中的交叉路口监测，还是物流机器人对动态障碍物的快速响应，均能实现“一眼尽收眼底”，显著提升系统决策效率与安全性。

  • 0.625°×0.625°的角分辨率赋予雷达毫米级的细节捕捉能力，可清晰分辨细小物体、边缘轮廓及低反射率目标(如黑色轮胎、暗色衣物)。即使在10%低反射率条件下，仍可稳定探测25米内物体，为低反射环境提供可靠感知保障。

  • 0.625°×0.625°的角分辨率赋予雷达毫米级的细节捕捉能力，可清晰分辨细小物体、边缘轮廓及低反射率目标(如黑色轮胎、暗色衣物)。即使在10%低反射率条件下，仍可稳定探测25米内物体，为低反射环境提供可靠感知保障

3D系列全固态面阵激光雷达

  •  机智人®WS - 20/30PCD - ET3 3D LiDAR采用全固态面阵技术，拥有150线的高分辨率，120°*50°视场角，540,000/S点云量输出，0.33角分辨率 。

  •  这款产品可快速、精确地构建周围环境的三维模型。无论是在复杂的工业自动…还是对环境感知要求极高的人形机器 它都能提供详细、精准的环境信息，助力设备实现更智能、更耐用和更高效的决策与操作。

    高性能导航与避障雷达

  • WS-10/20PCD-C3单线导航型激光雷达在复杂环境下表现卓越，可以清晰分辨细脚凳等反射率低的材质。

  • 该产品用于自主式移动机器人导航、移动机器人和平台的防撞与定位等场景，为用户提供稳定可靠的环境感知能力。

  • WS-06/10PNP/NPN-H5避障激光雷达则通过高扫描、低分辨角、高点云数据，能敏锐感知周边环境，为设备提供可靠的避障支持。

5.应用领域

  • 机智人科技的激光雷达产品已经在多个领域展现出广泛应用前景。在工业自动化领域，其产品可用于AGV搬运和自动驾驶机器人等。

  • 在室外环境中，WR-H-DM抗磁防尘型激光雷达适用于高磁环境下电磁干扰的场景，如电厂、变电站等。

  • 在安防、测量和扫描领域，公司的激光雷达技术在物体外形扫描、港口设备防撞及定位，重型设备室外防撞等方面均有应用。对于智慧物流和智能制造场景，这些激光雷达产品能够为移动机器人提供精确的环境感知能力，助力实现更智能、更高效的操作与决策。

6.展会邀请函

  • 深圳市机智人科技有限公司欢迎各位参观者亲临机智人科技展位，深入了解这些创新产品的技术细节与应用前景。


机智人科技将展示其全系列激光雷达产品。包括导航型、避障型和3D系列的多款型号,这些产品展现了激光雷达技术的最新方向，将为访客带来智能感知技术的前沿,机智人科技正以强大的产品矩阵和前瞻性技术战略，推动中国激光雷达技术向前发展，助力机器人产业突破商业化落地的最后一公里。
