中贸全球展 2022-04-22

JIANGSU

江苏


13个设区市跨境电商综试区全覆盖

“电商出海”，正成稳外贸生力军



01



Cross-Border E-Commerce

 # 政策红利加快产业集聚 #










“跨电综试区的政策红利将加快跨境电商产业在泰州集聚。国务院的批复刚刚出炉，就有一批跨境电商企业有意向在泰落户。”泰州市商务局局长黄红旗说。


南京海关监管通关处处长张睿介绍，近年来，南京海关在省内跨境电商综试区城市中全面推广零售进口试点政策，全面开展零售进出口以及B2B监管全模式，企业可以享受简化申报、清单核放、便利转关、创新退货监管等便利化措施。目前全省13个设区市全部获批综试区，南京海关将进一步加大支持力度，助力跨境电商为我省传统产业转型升级、稳定外贸外资基本盘、促进贸易高质量发展等方面做出更大的贡献。


据了解，最新一批跨境电商综合试验区在试点要求上有两大侧重点：一是注重跨境电商对传统产业转型升级的作用，利用数字赋能助力传统产业转型升级，促进产业数字化发展；二是从稳外贸稳外资的角度出发，利用外贸新业态打造外贸增长引擎，促进稳住外贸外资基本盘和外贸高质量发展。


镇江市制造业基础不断夯实，拥有国家级外贸转型升级基地3个、省级出口基地1个，江苏省重点培育和发展的国际知名品牌21个，已经形成眼镜、五金工具、汽车零部件等多个适合发展跨境电商的制造业集群。镇江市商务局相关人士说，发挥丹阳“世界眼镜之都”优势，打造眼镜跨境电子商务进出口交易平台；建立眼镜产品研发中心，通过跨境电子商务平台，推进研发新产品，培育高品质、高附加值的眼镜产品自主品牌。

Cross-Border E-Commerce

 # 稳外贸呼唤新业态提升 #










2022年，“外贸新业态提升行动”这句话频频出现在我省相关部门的各种文件里。而跨境电商综合试验区作为跨境电商发展的重要载体，创新形成了一大批可复制推广的经验做法。过去一年，江苏制定全省跨境电商高质量发展20条，新培育认定34家省级跨境电商产业园、13家省级公共海外仓。同时，开展“产业带+跨境电商”，深化与大型电商平台的合作，支持国家级、省级外贸转型升级基地应用跨境电商加快数字化转型，张家港塑饮机械、南通体育用品、常熟服装等基地积极打造线上产业专区，超百家传统外贸企业触网上线，开展跨境电商业务。


去年11月，中国（苏州）跨境电子商务综合试验区首单“跨境电商+保税展示”业务在苏州吴中落地，苏州市民可以在家门口现场采购全球商品，这一模式下消费者线上扫码下单，在核对个人身份证信息后，最快10分钟左右即可完成物流配送提货。


去年年底，南通综合保税区顺利完成了江苏省内首批7票跨境电商零售进口商品退货业务，这也是我省首家跨境电商零售进口退货中心仓，解决了长期以来跨境电商退货慢、退货难的问题，提升了消费者“跨境购”体验。


江苏智贸邦电子商务有限公司副总经理李丽说，新业态的打造，跨境电商生态圈的逐步建设，会帮助企业解决在转型跨境电商时遇到的一些如人才招聘难、平台对接难、服务企业对接难等问题，“我们期待能够有机会在跨境咨询、店铺运营、实操培训、平台对接、品牌规划、资源共享等各方面都能帮到意向或已经转型跨境电商的企业，实现品牌出海。”


我省正在强化政策创新指引，其中，“加快发展外贸新业态新模式的若干措施”已印发，16项举措将保证外贸新业态新模式的提升。例如，深入推进“江苏优品·数贸全球”专项行动，新增一批省市重点培育的跨境电商产业园和公共海外仓；支持跨境电商、外综服、市场采购贸易等新业态深度融合发展，加快推广应用“市采通”平台。

Cross-Border E-Commerce

 # 跨境电商机遇多市场大 #










泰州润元机电科技发展有限公司致力于研发、生产、销售户外运动产品，产品主要销往美国大型商场连锁超市及各专业连锁零售店。同时在Amazon、ebay平台进行跨境电子商务销售。作为2016年江苏省首批四家跨境电子商务公共海外仓试点企业之一，企业在正常贸易出口的同时，大力推进跨境电商海外仓业务，目前海外仓服务企业已增加到13家，跨境电商海外仓业务将成为公司的重要增长点。


“我们一方面注重跨境电子商务公共海外仓实操，另一方面引领泰州及省内更多企业特别是同行企业境外发展电子商务、自创品牌销售，自主拓展境外终端市场，实现业务模式转变、升级与转型。”该公司董事长赵立新说。


中国外运长江有限公司扬州分公司总经理助理朱建平表示，就目前来看，跨境电商行业仍处于高速发展阶段，因为全球疫情的影响，人们的消费习惯正在改变，跨境线上交易变得越来越频繁，依赖度也越来越高。对于跨境电商企业来讲，要找准自己的产品定位，选择优质的交易平台和物流供应商，一切以客户需求为导向来调整经营策略。


“从国际贸易关系、跨境电商平台政策、国内政策来看，跨境电商行业在往品牌化、长期主义、合规化等方向发展。”李丽认为，作为跨境电商企业一定要密切关注相应的政策发布，项目在做到阳光、合规的同时，需着重提升企业的品牌打造、产品研发、店铺运营、供应链管控等综合能力。


赵立新说，过去一年跨境电商行业实现爆发式的增长，被称为“跨境电商元年”，在今年、明年真正有实力的公司将脱颖而出。欧美发达国家前景广阔，从跨境电商海外目的地分布来看，美国和法国基础设施完善，网购环境成熟，已经成为我国跨境电商的主要出口目的地；同时，俄罗斯、巴西等新兴市场蓬勃发展，电子商务发展基础广阔，发展潜力巨大，有望成为中国出口跨境电子商务的增长点。与此同时，《区域全面经济伙伴关系协定》（RCEP）生效实施也是跨境电子商务新的发展机遇。

 # 展会助力 助推企业电商出海 #




江苏省人民政府办公厅2022年3月20日发布《关于做好跨周期调节进一步稳外贸的若干措施》中提及：千方百计稳住国际市场份额。积极应对疫情影响，创新参展办展方式，线上线下结合开拓国际市场。持续推进“江苏优品·畅行全球”“江苏优品·数贸全球”等系列贸易促进活动，深化“线上展会+跨境电商”模式，实施“一国一展”“一业一展”，支持企业利用线上国际展会和对接会开拓国际市场。组织企业利用广交会、华交会、高交会等境内国际性展会拓展内外贸销售渠道。支持各地举办跨境电商交易会、选品会，引导企业参加跨境电商展会。


2021年，江苏省商务厅及省内各地商务主管部门共组织超400个展位参加2021中国跨境电商交易会（秋季）（以下简称＂跨交会＂），取得了较好的参展效果。2022年，江苏省商务厅继续将跨交会纳入“跨境电商专题展会”中，加大扶持力度，鼓励江苏省企业通过参加跨交会，借助跨境电商出海模式，抢先机，抓机遇，稳出口。

展会简介：

中国跨境电商交易会是为链接中国制造与全球消费，整合飞速发展的跨境电商生态资源而举办的跨境电商交易会，实现外贸界、跨境电商界及会展界的资源整合，服务于全球贸易，帮助企业开辟更广阔的市场。展会不仅实现了交易的撮合功能，而且带来多样的平台和全链的服务，其增值功能是其他同类型展会所不兼具的差异化优势。中国跨交会以“外贸新动能 · 数字新未来“为主题，以“打造跨境电商国家级品牌和平台，服务外贸创新发展”为定位，致力于优化优质供货商、跨境电商采购商及平台资源；升级高端论坛与活动，打造中国跨境电商行业的龙头盛会。
