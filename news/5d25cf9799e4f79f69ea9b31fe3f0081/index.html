出团须知 | 2025年德国汉诺威工业博览会

西麦克展览公司第一事业部 2025-03-27


尊敬的参展商：

      西麦克国际展览有限责任公司2025年德国汉诺威工业博览会展团即将出发，为确保参展顺利，请您仔细阅读以下内容。




展会情况




展会概览



      德国汉诺威工业博览会，拥有76年的悠久历史，是世界上展出面积最大的工业展览会。该展会不仅展示技术含量极高的产品，而且被誉为连接全球工业设计、加工制造、技术应用和国际贸易的关键平台。经过几十年的发展，它已经成为“全球工业贸易领域的旗舰展”、“世界工业贸易的晴雨表”以及“全球工业技术发展的风向标”。今年，德国汉诺威工业展以“工业可持续发展注入活力”为主题，聚焦“人工智能在工业领域的应用”，并展示工业领域的创新产品、开创性产品以及相关解决方案。

      2025年合作伙伴国：加拿大

      加拿大正引领着通往更加光明未来的道路。通过在人工智能、量子技术、电动汽车、先进材料、清洁能源、自动化以及机器人等领域的深厚专业知识，开发创新解决方案，旨在塑造一个更加可持续的世界。我们先进的制造能力与世界领先的研究成果，与我们在自然资源方面的传统优势相得益彰。合作伙伴国加拿大有来自265家企业参展。

参展商数量：全球约为3500家，其中，中国展商数约900家。

参展商: 3月31日-4月4日 7:00 - 19:00 ‍

专业观众：3月31日-4月4日 9:00 - 18:00

展会期间，凭证件可免费搭乘汉诺威市内公共交通。





展会详情



展馆：Messegelände HANNOVER

地址：Messegelande, D-30521, Hannover


展期主要活动

初创科技企业舞台

( 2号馆 D10 )

如何将产品推向市场？哪种融资方案最适合？初创科技企业正在推出哪些革命性的解决方案和商业模式？——答案将在此一一揭晓。

可持续发展论坛

( 3号馆 C71 )

工程零件和解决方案的核心阶段

从经典的金属加工、轻量化结构和增材制造领域到当前的回收和循环经济问题。论坛形式包括用户演示和小组讨论。另一个亮点是传统的买家日，将于展览会第三天（4月2日，周三）举行。

动力传动舞台

( 5号馆D32 )

动力传动舞台是驱动技术、气动和液压技术知识传递和交流的中心。

RoboTalk机器人应用舞台

( 6号馆D51 )

机器人应用舞台将重点展示机器人、物流和自动化以及这些技术间的相互作用。

工业4.0舞台

( 8号馆D17 )

工业 4.0舞台为自动化和智能生产的众多主题提供了答案和创新解决方案。

能源4.0舞台

( 12号馆 E68 )

以能源4.0助力工业4.0发展，从发电（高效生产、可再生能源、氢能、工艺和废热）到最优利用（智能电网、能源效率、可靠的基础设施、人工智能管理）再到高效存储等（电池技术、发电站、车队管理、碳捕获）。

贸易与投资舞台

( 12号馆 B68)

汉诺威工业博览会的贸易与投资舞台是一个独特的论坛区域，专注于能源和经济政策、投资支持和全球经济等诸多问题。

氢能燃料电池舞台

( 13号馆 D40)

可再生能源中生产绿色氢及其衍生物、二氧化碳中性工业生产和燃料电池的应用。

欧盟委员会舞台

( 13号馆 E66)

欧盟委员会负责提出、实施和执行欧洲立法。委员会与各行业合作，帮助欧洲企业迎接绿色和数字化转型，创造条件提高竞争力，支持成为未来全球领导者。该论坛区邀请著名的欧盟委员会决策者和专家、行业领袖、协会代表和公司发言人。

5G与工业无线舞台

( 14号馆 H06)

5G 和无线技术——工业驱动力和应用

无线通信系统为灵活、可扩展及从物联网行业的特定需求出发从而进行量身定制，提供了绝佳机会。

数字化转型舞台

( 17号馆 E44 )

跨技术和跨行业交流的舞台。在这里，您将深入了解生产和数字化、人工智能AI、能源和可持续性等领域。

注意：签证有效期天数 ≠ 申根停留天数

注意事项




布展




布展：3月30日 8:00-18:00‍

‍‍1.请先清点展品。如展品回运需进行空箱存储，请将空箱放置在展台内。

2.核对楣板、展具、喷绘及相关服务预定是否就位。

3.严禁对展台进行改装。请勿在墙板上使用不可清理的胶带，不允许在墙板上打孔和上钉。若墙板有无法清洁还原的污损，需承担损失。

4.展位需要使用德标转换插头，展台内插座仅用于为手提电脑或手机充电，不可使用大功率电器。‍

‍5.布展垃圾请放置在通道上，布展结束后将由保洁公司进行摊位清理。若有需要签字的服务单据，请联系我司所在展馆负责人。



撤展



撤展‍：4月4日 18:00后

‍1.严禁提前撤展。根据主办方要求，撤展须在4月4日18:00后撤展。提前撤展的展商将被收取展位费20% (最少1,000 欧元) 的罚金。

2.撤展时请将您海报从墙板上揭下，将展品从摊位道具上取出，展具恢复成布展时效果。

‍3.若有需要您签字的服务单据，请您联系我们的所在展馆负责人。

‍4.撤展时请您保管好随身携带的贵重物品，防止护照、展会记录本、名片、电脑等物品被盗。

⚠️主办方对任何展品的遗失或损坏不承担任何赔偿责任。

垃圾处理：‍

1.撤展后垃圾处理：‍参展商须妥善处理留存在展台的展品，多余的宣传材料、包装等。如仍有需要的，请回运至国内，之后遗漏在展台和/或展览场地发现的任何物料，都将被视为垃圾处理，由此产生的处理费用及其他增值附加费将由您承担。

2.‍德国垃圾车按照体积收取垃圾处理费，建议参展商尽可能‍缩小包装面积（如将木箱、纸箱、泡沫箱等拆开、折叠、压扁后整齐存放；尽可能进行垃圾分类；‍将垃圾整齐堆放在您展台边缘；‍请勿在未妥善处理的情况下随意丢弃垃圾，以免产生高昂的垃圾处理费。
