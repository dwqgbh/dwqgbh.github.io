直击上海进博会,消费品牌最新趋势与拓展计划都给你扒来了!

赢商网 2022-11-09


来源 | 赢商网（ID：winshang）

作者 | 于金平、何丽思

吴婷婷对本文亦有贡献

头图来源 | 赢商网



根据情绪定制的入浴剂、功能性化妆水即时定制仪、闪穿科技、元宇宙展台……第五届进博会消费展区，各家品牌都带来十足的诚意。

11月5日至10日，在上海国家会展中心，第五届进博会如约而至。消费品展区面积超过7万平方米，包含绿色智能家电及家居、美妆及日化用品、时尚潮流及珠宝、体育用品及赛事四个主题板块，绿色低碳、可持续发展的理念贯穿整个展区。

图源：进博会官方

▶▶ 智能家电及家居专区规划展览面积1万平方米，以“家”为载体，围绕“绿色”“智能”两大主题，按照家庭生活场景分区布局，打造智联全屋、乐趣厨房、洁净空间、舒适起居、娱乐及影音共5个生活乐活区。参展品牌中，连续五年参展的宜家以“家 因你而生”为主题，搭建一处占地360平方米的“宜家公园”，松下电器、博西家电等集团均带着新品与中国消费者见面。

宜家展区，图源：宜家中国资讯中心官微


▶▶ 美妆及日化用品专区规划为三大核心品类，分别是“面部护理及彩妆”、“身体护理及日化用品”和“专业美容”，围绕“绿色·可持续”“美妆‘黑科技’”两大时下热点主题关键词，欧莱雅集团、COTY、资生堂集团、雅诗兰黛集团、宝洁、联合利华、爱茉莉太平洋、逸仙电商等品牌和集团再次集结，展示品牌历史和对于可持续理念的思考。

图源：进博会官方

▶▶ 时尚潮流及珠宝板块规划展览面积3万平方米，展品范围覆盖服装服饰、鞋履、箱包、珠宝、艺术品等中高端消费类展品。展品围绕“绿色”“可持续”等主题，展区集聚了中高端消费品行业优质企业，既有首次参展的加拿大鹅，又有迅销（优衣库母公司）、INDITEX（ZARA母公司）、H&M、Tapestry（Coach母公司）、开云、LVMH、历峰集团、D&G等熟悉的面孔。

图源：进博会官方

▶▶ 今年也是进博会连续第三年设置体育用品及赛事专区，本届专区以“体育科技赋能健康可持续生活”为主题，展览面积近7000平方米，不少与体育相关的国际品牌在展会上带来全球首发、亚洲首秀的新品。斯凯奇带来闪穿科技，亚瑟士带来了SUPERBLAST跑鞋的全球首秀。此外，耐克、阿迪达斯、NBA等品牌在进博会上均有亮眼表现。


三大趋势掀起消费新浪潮


// 个性化和定制化

随着各类企业数字化转型的深入，精准需求引导下的定制化供给服务正日益走俏。进博会的美妆及日化用品专区上的展商对个性化和定制化情有独钟。爱茉莉太平洋、欧莱雅集团、雅诗兰黛集团等都在进博会上展示了其对定制化产品的探索。

据爱茉莉太平洋介绍，本届进博会上爱茉莉太平洋带来了入浴剂个人定制服务「Mind-linked Bathbot」、功能性化妆水即时定制仪「Formularity」、个人定制服务「BESPOKE NEO」等多款黑科技加持的定制化产品和服务。以入浴剂个人定制服务「Mind-linked Bathbot」为例，消费者感受不同的色彩和香气，产生不同的情绪，头戴式设备捕捉并获取个性化数据，由此推荐愉悦身心的疗愈“泡澡球（bath bomb）”产品。据悉，该产品尚未进入中国市场。

图源：爱茉莉太平洋官微

带来情绪定制产品的还有欧莱雅集团，此次进博会，欧莱雅集团的YSL圣罗兰“电波穿香室”，可以用来跟踪、分析消费者闻到不同香味时的情绪反应，进行个性化香水推荐。至于体验者是否满意，仍需欧莱雅集团收集体验后数据进行分析和优化。此外，资生堂集团、花王等展商也带来了独家的黑科技定制技术或产品。

图源：欧莱雅中国咨询中心

定制化的浪潮不仅体现在美妆及日化用品专区，在汽车展区，多个品牌推出了车漆颜色定制服务，也有展商针对中国市场新的消费热点推出定制产品。意大利的商用车企业依维柯在进博会上重点展示了房车产品，能够满足消费者的露营需求。

定制化和个性化产品虽然可以带来新的增长，但对于品牌方来说，定制化意味着生产流程的增加和成本的上升，而且，定制化产品能够多大程度上满足个性化的需求，以及消费者在多大程度上为个性化产品买单，仍是个问号。

// 科技

对于科技的重视，进博会本身的展区设置可以体现。今年首设了5000 平米人工智能专区，元宇宙、NFT、虚拟现实、VR等概念和技术在科技展区得到了集中展示。

在消费展区，科技也是主题之一。前文提到的各家推出的定制化产品之所以能够实现，均是因为在大量数据背后有科技加持。

今年进博会，体育专区倡导“体育科技赋能健康可持续生活”，亚瑟士、斯凯奇、耐克、阿迪达斯等均有尖端科技产品亮相。例如，斯凯奇展出了Hyper Burst气泡科技、Slip-ins闪穿黑科技、GODRI吸湿速干科技等专攻不同功能的科技，提高鞋服产品的功能性。专业跑鞋品牌亚瑟士展区特别开辟“动力科技站”板块，展示了品牌FF BLAST中底科技矩阵的核心产品。

图源：斯凯奇新闻中心官微

此外，时尚潮流及珠宝展区，优衣库专为进博会打造了全球首个高2.5米巨型HEATTECH保暖内衣，展示其吸湿发热黑科技；ZARA打造了一个巨型魔方，每一面均由多个LED屏幕组成，屏幕上不间断播放ZARA最新的产品系列；H&M在进博会商展示了元宇宙设计服饰,为H&M打出数字时尚的标签。

图源：优衣库官方

虽然均有科技加持，但不同板块的展商对于科技的理解和研究方向不尽相同。美妆及日化用品专区展出的科技更多在于推动定制化和个性化产品的打造；在绿色智能家电及家居专区，科技的重要性更多体现在推动产品创新、智能化方面；在体育专区和时尚潮流及珠宝，科技的意义是为产品增强科技感，优化日常使用，提升生活质量。

除此之外，在可持续、零碳的理念下，科技又被赋予了新的使命。

// 绿色可持续

在“零碳进博、零塑办博”的理念下，展商都寻求在可持续和绿色上讲好故事。不仅在产品层面实现零碳可持续，甚至上升到供应链及企业发展的战略层面。

在零碳产品上，分化为两个趋势：用替换装减少包装的资源浪费，例如资生堂线下展出了16款替换装明星单品，用替换装产品展示可持续发展理念。

图源：赢商网

另一个趋势是使用回收废物生产产品，降低生产过程和包装产品的碳排放，生产出更多可降解的产品。诸如在科思创展区，特步发布的全新低碳环保概念跑鞋减碳率最高可达81%，假设特步基于质量平衡法生产5000万双鞋，一年可减碳约26,600吨，相当于每年植树造林约147.8万棵。

图源：特步官方

此外，在战略层面，多个展商在进博会宣布其可持续发展计划或者生产计划。

进博会上，联合利华中国六星零碳工厂揭幕仪式在展台举行，从“让可持续生活触手可及”的企业使命出发,联合利华不断升级绿色生产技术,树立业内绿色“智”造标杆；

图源：赢商网

加拿大鹅提出可持续发展目标，包括：到2025年实现净零碳排放；到2025年将其90%的材料过渡到优选纤维和材料（PFMs）；将可持续解决方案纳入整个企业的包装中。

事实上，绿色可持续从纸面上的数字正在向可感知的层面迈进，未来有望成为各品牌的核心竞争力。


“新老”品牌抢抓中国市场


作为国际社会进一步观察中国的一扇窗口，参展商无疑是看好中国市场的发展潜力，无论是连续多年参展的老朋友，还是初次参展的新面孔。而它们在中国市场的渠道布局与拓展计划也展现了其在中国市场的发展“野心”。

▶▶ 优衣库：深耕线下渠道，保持每年80-100家新店的速度

在中国市场拥有数百家供应链合作伙伴、超900家直营店的优衣库，将继续加大深耕中国线下渠道的力度，剧透露，2022年1月至今，优衣库新开了50多家门店，年内还将继续新开约20家店，未来将保持每年80-100新店的速度。

赢商大数据显示，在24城5万方以上购物中心内，优衣库门店已覆盖28个省，200多座城市，主要集中于广东省、上海、江苏、北京、浙江、山东等地区，TOP5省份占比达52.42%。当然，四川、湖北、湖南、陕西等华中地区也是其第二梯队市场，省份门店数量均在30家左右。


▶▶ ZARA：优化升级门店，建立线上线下一体化的多功能店铺模式

ZARA以本土消费者的需求为导向，深入探索挖掘本土元素，基于此，其将大力投入对线上线下的智能化创新升级，建立线上线下一体化的多功能店铺模式，并持续加速本土化体验升级，不断开拓新渠道。在过去几年中，ZARA也在持续对线下门店进行翻新升级。今年，ZARA开出了上海太阳宫门店，武汉汉街门店等全新门店。

图片来源：ZARA官方

▶▶ 斯凯奇：深耕中国，加大中国投资

自2008年以合资模式进入中国市场以来，斯凯奇已在中国大陆地区开设近3000家线下门店并在多个电商平台发展线上销售业务。未来，斯凯奇将深耕中国，加大中国投资。除了一二线城市之外，还会抓取县级城市的空白点。

图片来源：斯凯奇官方

▶▶ 耐克：在中国打造互联互通市场

未来，计划通过线上、线下在中国打造一个互联互通的市场，而通过互联互通，耐克有一定比例的门店是通过经销商来服务中国消费者。今年年初，耐克与中国所有的单一品牌店合作商都签订了耐克市场一体化合作伙伴协议，大家都承诺将拥抱彼此，深化合作，百分之百连接会员产品和服务。

图片来源：赢商网

▶▶ 加拿大鹅：2023财年在中国新开4家专门店

中国已经成为加拿大鹅专门店布局最多的国家，共有22个专门店。据了解，Canada Goose加拿大鹅计划在2023财年新开设四家专门店。其中西安SKP、天津万象城、青岛海信广场等三家专门店已经开业，第四家专门店位于成都，也将在年底前启幕。此外，Canada Goose加拿大鹅在中国市场还拥有三家线上商店。

图片来源：加拿大鹅官方

▶▶ 宜家：53亿加码中国市场

2023财年，宜家将继续加大对中国市场的投资，预计将投资53亿元进一步加快数字化转型，加快线下门店运营、建设，还会投资风能、光伏等可再生能源。

图片来源：赢商网

▶▶ 乐高：建设并扩大工厂业务，2023年要新开80家店

中国对乐高来说是一个非常具有战略意义的市场。目前，集团在中国开设的乐高品牌零售店总数已从2018年的60家增长至超过380家，覆盖的城市数量也从18个拓展至100多个。2022年1月，乐高集团宣布扩大其位于浙江省嘉兴的工厂业务，以支持在本地区市场的长期发展。未来，乐高集团将继续拓展在华品牌零售业务，计划于2023年在中国大陆新开设80家品牌零售店。

随着中国电子商务行业的蓬勃发展，乐高集团将于今年年底在抖音平台正式开设官方店铺，进一步强化其在主流电商平台上的业务布局。此外，乐高集团于近期在上海开设了中国首个客户服务中心，为本地消费者提供更便捷的支持服务。

图片来源：赢商网

▶▶ ABH：今年首进中国市场，独家进驻丝芙兰中国

ABH（Anastasia Beverly Hills）拥有多项黄金比例画眉法专利，是美国首屈一指的眉部产品品牌。此次首次亮相上海进博会，是ABH深入挖掘中国消费者美妆需求的首次尝试，对品牌开拓在中国市场有着积极深远的意义。

据了解，品牌于2022年6月21日独家入驻SEPHORA丝芙兰中国市场。其实，ABH和丝芙兰在国际美妆市场早已展开长期持续的合作关系，此次独家入驻丝芙兰中国，正式开启ABH在中国市场布局的新篇章。

▶▶ 闻献 DOCUMENTS：主攻线下门店，明年计划开店8家

由欧莱雅在中国市场设立的首家投资公司——上海美次方投资有限公司投资的中国本土高端香水香氛品牌闻献 DOCUMENTS 此次也是首次亮相上海进博会。

据了解，闻献 DOCUMENTS是创立于2021年的中国高端香水香氛品牌，以线下门店作为主要销售渠道。截至今年9月，已在上海淮海路、上海静安嘉里中心和北京SKP-S开设三家品牌店，明年的开店计划是8家。
