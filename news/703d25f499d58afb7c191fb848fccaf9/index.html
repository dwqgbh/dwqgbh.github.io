CHIC 2026大湾区(广州)国际服装及时尚生活展览会

展会资讯前哨 2026-06-01

1

CHIC

展会概况

CHIC 2026大湾区(广州)国际服装及时尚生活展览会是粤港澳湾区首个聚焦全球当代服装及时尚生活方式的国际性展览会，将于6月10-12日在广交会展馆-D区隆重举办！
作为大湾区首个聚焦全球服装与时尚生活方式的超级联展，本届集结超1000家展商、覆盖80+时尚品类，汇集海内外众多优秀服装、箱包、鞋履、眼镜、智能穿戴、配饰、珠宝、婚纱礼服等时尚生活品牌及产品，以时尚趋势发布、新品发布会、采购对接会及互动打卡等多元亮点活动，四展六馆同期联动，一站式打通设计、采购、贸易全链路。

提前预注册享免费门票*

2

CHIC

展品范围

服装系列

从商业跑量到设计溢价，

一站横向比选


女装 / 男装 / 童装 / 国风服饰 / 运动装 / 内衣 / 

牛仔 / 婚纱礼服

服装始终是时尚采购的基本盘。本届联展服装板块覆盖从大众基本款到高定设计款的完整光谱：做下沉市场跑量批发的，这里有产业集群直供，成本优势明确；做品牌差异化选款的，这里有国风设计师品牌、运动功能线、牛仔洗水工艺产品，风格辨识度高；做婚纱礼服集成店的，从轻纱到重工、从秀禾到改良旗袍，可直接对接源头工厂与设计师工作室，面料、版型、工艺现场实看实物。

一届展会，横向比对不同定位的产品，真正实现一站式采购。

时尚配饰

小配件撬动大生意


鞋帽 / 围巾 / 腰带 / 首饰 / 手套 / 袜子 / 

宠物服饰 / 娃衣

配饰是门店连带率的关键变量。本届联展配饰板块涵盖从头到脚的全品类覆盖，同时纳入宠物服饰、娃衣等新兴增长赛道，为选品人提供差异化切口。无论你是饰品集合店组货、服装品牌配饰线采购，还是直播电商选品、社群团购找款，这里支持小单快反、灵活混批、源头直采。


用一双袜子拉新客，用一条围巾拉高客单，用萌宠娃衣打开全新消费圈层——配饰虽小，组合得当就是盈利亮点。

箱包鞋履

源头直供，起订更灵活


时尚皮包 / 皮具 / 男鞋 / 女鞋 / 童鞋 / 运动鞋 /

时尚潮鞋及功能鞋履

全品类箱包鞋履集中呈现，覆盖通勤、运动、户外、商务、休闲等全场景。直接对接源头工厂，缩短供应链，降低采购成本，起订量更友好，交期更可控。从基本跑量款到高溢价设计款，不同价位、不同工艺现场横向比选、集中比价，一趟行程锁定鞋包板块的核心供应商，把成本与选款节奏牢牢握在自己手里。


珠宝眼镜

从造型点缀到智能穿戴，

一条赛道两重增量


眼镜 / 太阳镜 / 珠宝 / 腕表

珠宝与眼镜是时尚零售中客单价最高、客户粘性最强的品类之一。本届联展珠宝眼镜板块覆盖时尚首饰、珍珠、银饰、K金、彩宝、腕表等珠宝全线，以及光学眼镜、太阳镜、智能眼镜、镜片镜架、眼健康服务等眼镜全链。


对买手店而言，一组轻奢珠宝就能拉高整店调性；对眼镜集合店而言，从时尚成镜到智能穿戴到眼健康管理，一条展区配齐三条增长曲线。高客单、高复购、高专业壁垒——这正是珠宝眼镜品类的不可替代之处。

生活美学

跳出服饰做增量，

把氛围感变成复购率


香氛 / 家居时尚 / 文创礼品

服装店的顾客为什么逛了就走了？买手店的复购率为什么上不去？答案往往不在服装本身，而在空间体验。本届联展生活美学板块集中呈现香氛、设计感家居、文创礼品等高毛利非服饰品类，为服装门店、买手店、生活方式集合店提供场景化组货方案——用一盏香氛延长顾客停留时间，用一件文创小物撬动连带消费，用空间氛围感把一次性顾客变成常客。

这不是锦上添花，而是门店差异化的核心战场。

展会亮点

来到这场“生活方式展”，你可以：

按生活场景一站式选品——效率更高，组货更准

直连源头企业与品牌——减少中间环节，拿到更有竞争力的合作条件

一次性掌握跨品类趋势——提前布局下一季的组货方向

参与生活方式主题商贸对接会——直接找到与你的场景互补的合作伙伴

参与行业论坛与趋势发布——获取真实的市场信息与消费者洞察

同步接触国内+国际资源——为下一步品类拓展和渠道升级留足空间

为什么是广州？

坐拥万亿级时尚产业集群，服装产业基础雄厚；

国际贸易与消费中心双重驱动，千年商都贸易网络通达全球；

依托大湾区世界级制造与开放门户优势，助力展会高效链接世界市场；

产业、渠道、物流高度集中，一次观展触达全产业链；

亚洲顶级会展综合体赋能，广交会展馆专业服务与规模效应双重保障。

4

CHIC

同期活动

本届联展包含2026大湾区（广州）国际服装及时尚生活展览会、2026大湾区（广州）国际眼镜展览会、2026大湾区（广州）国际皮具箱包及成品鞋类展览会、2026大湾区（广州）国际珠宝配饰及原创婚纱展览会在内四展同开，侧重品牌企业与设计师资源的集聚，通过多品类联动，为采购商打造一站式采购、全链路对接、全球化贸易的优质平台。
三大主题馆各有锋芒——18.2馆衣合空间聚焦原创设计与多元风格、19.2馆霓裳江南演绎温婉国风与精工织造、20.2 馆粤海千服彰显岭南潮牌与产业集群实力，三大馆区同频共振，勾勒出中国时尚向上生长的鲜活图景。

   同期举办

多展联动，一次到场，覆盖服装及生活方式全产业链资源。

大湾区（广州）国际眼镜展览会

聚焦“科技+经典+时尚”三维场景，融合AI眼镜与潮流配饰，打造智能时代眼镜新体验。

大湾区（广州）国际皮具箱包及成品鞋类展览会

依托狮岭皮革与广州鞋业两大产业集群，集中展示全品类皮具鞋类及原创设计与智能制造。

大湾区（广州）国际珠宝配饰及时尚婚纱展览会

荟萃全球珠宝、饰品与婚纱礼服，通过国际时尚秀与论坛，推动产业与时尚生活的创意融合。
