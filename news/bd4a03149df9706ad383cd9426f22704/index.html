2026深圳（国际）具身智能创新展览会

具身 AI 无界・开创全球新局

•时间：2026 年4月9-11日

•地点：深圳会展中心（福田）1号馆

•同期展会：第十四届中国电子信息博览会、第107届中国电子展

一、展会背景

当前，具身智能已成为人工智能从“感知智能” 向 “行动智能” 跨越的核心方向，融合AI大模型、多模态传感、高精度运动控制等技术，可赋能机器人在工业、医疗、家居、特种作业等场景实现 “自主感知 - 决策 - 执行” 闭环。

据行业数据显示，2025年全球具身智能核心产业规模预计突破千亿元，中国凭借完整产业链优势与政策支持，正成为全球具身智能创新的重要策源地。本次展会落地深圳 —— 中国人工智能与机器人产业核心城市，依托其产业集群优势、完整产业链与创新活力，旨在打通 “技术研发 - 部件供应 - 整机制造 - 场景应用” 全产业链环节，推动全球资源深度链接与协同发展。

本次展会联合第十四届中国电子信息博览会（简称“CITE”）实现强强联合：CITE 最初由工业和信息化部、深圳市政府联合主办，历经13年发展已成为亚洲版 “CES”，规模达10万㎡、展商1500+家，华为、中国电子、高通、TCL 科技等龙头企业常年参展。两展在市场推广、买家邀请、媒体传播等方面协同赋能，2026年将成为中国最具影响力的具身智能展会之一。

展会核心定位：既是观察中国电子信息及具身智能产业发展趋势的“风向标”，也是全球了解中国相关产业实力的重要窗口，本质是 “全球科技竞争 + 国家战略驱动 + 产业升级需求” 的叠加产物。

四、展会核心优势

1.  全产业链生态聚合，覆盖新兴与基础领域

以10万㎡展示面积、30 + 主题展区，构建 “材料 - 核心部件 - 整机 - 场景应用” 完整产业链，同步联动电子信息产业资源，打破技术与应用壁垒。 

2.顶尖企业与前沿技术密集首发

2025年展会已吸引华为、联想、联发科、TCL华星光电等1500 + 全球领军企业参展，2026 年将聚焦具身智能核心技术首发（如人形机器人运动控制、多模态大模型适配），成为行业技术创新 “首发平台”。 

3. 国际化与市场化深度融合

设“国际创新合作展区”，联动 50 + 国家和地区的 6.3 万专业观众互动； • 联合俄罗斯、中东、东南亚等海外商会举办 500 + 场供需对接会，助力企业出海拓展全球市场。 

4. 政策导向与学术智库双重赋能

作为工信部与深圳市政府支持的平台，同期打造“2+8+N” 会议体系：

• 高层论坛：开幕峰会聚焦“全球AI存算新生态”，邀请蒋国飞等专家解析产业革新路径；

• 垂直研讨：50+场论坛覆盖具身智能、6G通信标准、AI伦理等热点，潘毅院士等 500 + 嘉宾参与； 

• 创新评奖：揭晓具身智能细分领域“创新金奖”，树立行业技术标杆。  

5.  场景化体验与产业协同创新 

设“跨领域融合展区”，模拟工业装配、医疗手术、家庭服务等真实场景，直观展示具身智能落地效果，推动 “技术方 - 应用方” 现场协同。 

6.  深圳本土创新资源强力支撑

依托深圳5.81% 研发投入占比（全国第一） 与 1880 亿元研发经费，展会成为本地创新成果的集中展示窗口，助力产业链本土化。 

7.  专业服务与精准营销体系

• 依托 120 万观众数据库与 200 + 媒体合作网络，实现展前 - 展中 - 展后全周期传播；

• 现场设 “采购商俱乐部”，为 5000 + 团体观众提供定制化对接服务；

• 线上广告覆盖百度、抖音等平台，年曝光量超 1000 万次，强化品牌影响力。

五、招展范围

1.  具身智能核心技术企业

     AI模型研发企业、传感器制造商（视觉/力/触觉传感器）、机器人操作系统开发商、运动控制技术企业。

2. 具身智能机器人制造商

    人形机器人、服务机器人（医疗/家庭/教育）、工业协作机器人、特种机器人（消防/救援/农业）生产企业。

3. 应用场景相关企业

     智能制造（汽车/电子制造）、医疗康养机构、物流仓储企业、教育培训机构、智能家居企业（展示落地案例）。

4. AI 眼镜展区

    消费级/工业级 AI眼镜、核心硬件技术（光学显示方案、芯片与模组、交互技术、轻量化材料）。

5. 科研机构与高校

展示具身智能前沿研究成果、实验室项目，促进产学研合作。
