// 第131届中国进出口商品交易会将于4月15-24日在线上举办

第131届中国进出口商品交易会（广交会）将于4月15至24日在线上举办，展期10天。本届广交会以联通国内国际双循环为主题，展览内容包括线上展示平台、供采对接服务、跨境电商专区三部分，在官网设立展商展品、全球供采对接、新品发布、展商连线、虚拟展馆、新闻与活动、大会服务等栏目，按照16大类商品设置50个展区，境内外参展企业2.5万多家，并继续设立“乡村振兴”专区，供所有脱贫地区参展企业集中展示。本届广交会围绕提升贸易对接成效和用户体验，持续优化提升线上平台功能，多措并举便利展客商互动交流和贸易成交。




// 2022中国跨境电商交易会线上系列分享活动即将开始

等待开幕的2022中国跨境电商交易会没有停下脚步，即将推出线上系列分享活动（第一期）——TikTok直播。2022年4月12日15:00-17:00，杭州红猪科技COO DC.Wang将做客中国跨交会视频号，届时，将与各位分享关于TikTok的见解。




// 郦江科创再获“政府采购公共座椅首选品牌”称号

近日，由《政府采购信息》报、政府采购信息网主导的2020年度政府采购家具优秀供应商系列评选名单正式揭晓，郦江科创被授予「2021年度政府采购公共座椅/排椅首选品牌」称号，获得了政府、市场、行业的高度认可。

// 恒林股份三次收购永裕家居股份

2022年3月，恒林股份披露公告称，公司召开董事会议审议并通过了《恒林家居股份有限公司发行股份购买资产并募集配套资金预案》及其摘要等相关议案。根据预案，该公司拟通过向陈永兴、朱小芬、安吉安裕等发行股份的方式，购买其持有的永裕家居52.60%股权。交易完成后，永裕家居将成为恒林股份的控股子公司，持股比例为94.72%。值得注意的是，这已经是恒林股份三年中第三次收购永裕家居的股份。

// AAUA与广州美术学院集美研究院开展交流会


近日，在亚洲城市与建筑联盟和广州美术学院集美研究院的邀请下，海珠区黄符伟副区长、昌岗街道办负责人走进广州美术学院集美研究院和广东省集美设计工程有限公司。广州美术学院集美研究院院长陆玮琦先生和常务副院长刘冰先生热情接待。此次交流会探讨了海珠区如何借力广州美术学院和亚洲城市与建筑联盟的优质学术和专业资源建立高水准的艺术创意园区。



// 中国纤维板产业总生产能力呈增长态势

2021年度，全国建成投产14条纤维板生产线，新增生产能力336万立方米/年。截至2021年底，全国376家纤维板生产企业保有纤维板生产线425条，分布在24个省市区，总生产能力为5355万立方米/年，净增生产能力179万立方米/年，在2020年底基础上增长3.5%，平均单线生产能力进一步上升到12.6万立方米/年。中国纤维板产业总体呈现企业数量及生产线数量下降、而总生产能力及平均单线生产能力增长态势。


 

// 2021年涂料行业利润总额预计降4%至303亿元

中国涂料工业协会对2021年中国涂料产业发展的关键数据进行了披露，2021年我国涂料行业企业总产量约3800万吨，较去年同期预计同比增长16%；主营业务收入预计超过4600亿元，较去年同期增长16%；利润总额预计可达303亿元，较去年同期降低约近4%。受上游原材料涨价、下游需求紧缩、行业内卷三方面影响形势日益严重，亏损情况明显。
