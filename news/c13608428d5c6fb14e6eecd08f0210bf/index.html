【中俄双语】中国空间站核心舱将在珠海航展首次公开亮相

人民网俄文版 2018-10-25

记者从23日召开的第五届载人航天（国际）学术大会上了解到，中国“天和”号空间站核心舱将首次以1：1实物形式（工艺验证舱）参加第十二届珠海航展。这将是我国空间站工程首次在公众面前亮相。

 

По информации, полученной 23 октября в ходе Пятой международной научной конференции пилотируемой космонавтики в Сиане, на Чжухайском авиасалоне будет показана модель базового модуля будущей космической станции Китая «Тяньхэ» в масштабе 1:1 (технологический отсек). Таким образом, один из проектов китайской космической станции впервые будет представлен общественности.

 

据中国载人航天工程办公室主任郝淳介绍，中国空间站额定乘员3人，乘组轮换时最多可达6人，建成后将成为我国长期在轨稳定运行的国家太空实验室，基本构型包括核心舱、实验舱I和实验舱II，每个舱段规模20吨级。核心舱有3个对接口和2个停泊口。对接口用于载人飞船、货运飞船及其他飞行器访问空间站，停泊口用于两个实验舱与核心舱组装形成空间站组合体，另有一个出舱口供航天员出舱活动。

 

По словам руководителя канцелярии Программы пилотируемой космонавтики Китая Хао Чуня, китайская космическая станция будет вмещать трех членов экипажа, в период смены космонавтов максимальное количество может достигать шести человек. После строительства космическая станция станет национальной космической лабораторией, которая будет стабильно функционировать на протяжении длительного времени. Станция будет состоять из базового модуля, первого и второго экспериментального отсеков, масса каждого достигает 20 тонн. Базовый модуль оснащен тремя стыковочными узлами и двумя портами. Стыковочные узлы будут использоваться для стыковки пилотируемых космических аппаратов, грузовых кораблей и других космических судов, порты будут предназначены для формирования цельной космической станции, соединят два лабораторных модуля и базовый модуль. На станции также установлены иллюминаторы для выхода космонавтов в открытый космос.

 

核心舱将于今年年底转入正样研制阶段。按计划，我国空间站将于2022年前后完成在轨组装建造。

 

В конце этого года начнется стадия практической разработки базового модуля. По плану, сборка китайской космической станции завершится к 2022 году.
