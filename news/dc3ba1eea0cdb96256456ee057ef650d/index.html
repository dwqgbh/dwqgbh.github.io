《粤港澳大湾区文学评论》2025年第1期目录点评,“80后”作家林培源小辑

当代文学批评 2025-02-16




目 录


习近平总书记关于文艺工作重要论述研究

文明互鉴作为建构与强化文化自信和文化主体性的必由之路——以“文化自我”和“文化主体”两个概念为切入点 / 金惠敏

这种“内部文件”式的文章就别发出来了吧？


前沿观察

论“新国学”理论建构中的语言学维度 / 谭桂林

“新国学”是啥时候冒出来的，查了发现是王富任2005年提出的，本文是从语言学维度进行的阐发。在我看来，一切新国学都是新瓶装旧酒，文章已经刊发，瞅了一眼，催眠佳作。


思想圆桌｜海外华文文学研究的现实与未来

栏目名中“现实”疑为现状之误，看编后记并没有错，感觉怪怪的。

历史化与再出发：由新世纪海外华文文学研究引发的思考 / 白 杨

引起的思考是什么：历史化与再出发，说了等于没说，当然如果是对华文文学研究的历史回顾倒是可以一看；并且“由**研究引起的思考”这样的表述实在少见。

华文文学理论建构暨学科潜质 / 沈庆利

第一次产生断句疑问，是华文文学、理论建构还是华文文学理论、建构？暨字也用得颇不寻常，一般用在标语或横幅上，文章标题用及或与即可。

从中国故事到中国叙事——数智时代海外华文文学创作与研究的新动态 / 戴瑶琴

从中国故事到中国叙事，是个好题目，如果只关注“研究”新动态，可能更容易出好文章，不知写得如何？

“摆渡”的场景：新世纪海外华文文学影视改编的反思与重构 / 许文畅

文学影视改编可以反思，如何重构呢？


粤港澳文学批评溯源

批评的“矛盾论”与文学政治批评——黄药眠的文学批评观念与方法 / 李圣传

用副题即可，主标题真让人费解，不知所云。


粤港澳作家专栏

“今日方知我是我”：论鲍十小说创作的内在精神理路与主体探求 / 张 栋

看编后记是对鲍十小说的全面梳理，还挺感兴趣，但看这题中“内在精神理路与主体探求”，让人望而生畏。


作家作品评论

青年作家专辑 林培源（贺仲明 申霞艳主持）

异乡人，请带上你的乌托邦 / 陈培浩

他以文学定义“潮汕”——林培源论 / 杨 玲

讲故事与写小说（创作谈） / 林培源

林培源，1987年生，潮汕人，同样成名于新概念，创作丰富，目前是广东一高校老师，希望这一组文章不要太枯燥，能让人对他有点深入了解。

世界意识与世界书写——以徐则臣为例 / 李徽昭

怎么个世界书写，好奇。

浩渺的宇宙：张枣诗歌的空间意识 / 夏 至

情境叙事、欲望象征以及“浮士德难题”——论莫言《鳄鱼》的戏剧张力与思想深度 / 谢腾飞

莫言向来有感觉没有思想，不知新作思想有多大深度，再看题目中并列的几个名词，估计脑子也是一团糊涂。


新作快评

静默的回响——邱华栋长篇小说《空城纪》读札 / 曾 攀

小说家的建筑史诗——论邱华栋的长篇小说《空城纪》 / 李德南

《空城纪》，说是历史小说，更感觉是史料堆砌，历史意义大于文学意义，跟王安忆《天香》情况有点类似，不过写的是西域各小国。


广东文学振兴

在“大湾区”建设背景下对粤东西北诗歌发展的思考 / 向卫国

粤东西北，只没有东？编后记说是基层作者，文章很接地气，不知真假，反正标题一点不接。


香港文学研究

论《文艺季》（1962—1963）的“纯文艺”探索 / 王艳芳

过于冷僻，没啥兴趣。


《戏剧叙事学》研究小辑

表意摹仿及其理论意义 / 袁国兴

摹仿叙事学与新世纪中国叙事学新探 / 张 羽

《戏剧叙事学》网上没找着资料，作者袁国兴，1953年生，华南师范大学文学院教授，戏剧研究专家，不敢评论。


创作谈

小说家的秘密工作——长篇小说《风叩门环》创作谈 / 盛 慧

《风叩门环》讲孀居农妇赵老太与儿媳玉珍被一笔数目可观的抚恤金绑在一起，她们勾心斗角，也依靠取暖。作品通过描摹空巢老人被遮蔽的生活图景与乡村女性孤独的精神世界，进行了深刻的人性探问。

编后记

 

“习近平总书记关于文艺工作重要论述研究”刊登金惠敏《文明互鉴作为建构与强化文化自信和文化主体性的必由之路——以“文化自我”和“文化主体”两个概念为切入点》。作者长期研究文艺学，文章也是结合文艺学理论进行的深入阐发，具有较强的创新性。

“前沿观察”刊登谭桂林《论“新国学”理论建构中的语言学维度》。王富仁先生当年提出“新国学”思想，引发学术界较多讨论和关注。本文从语言学角度予以阐发，是之前从来没有过的，也是对“新国学”理论的进一步推进。  

“思想圆桌”讨论的话题是“海外华文文学研究的现实与未来”。海外华文文学学科与中国当代文学学科关系非常密切，粤港澳地区更与海外华文文学渊源深厚。由中国海外华文文学会长白杨教授领衔来深入思考学科研究的现实与未来，具有很好的启示意义。

本期“粤港澳文学批评溯源”刊登李圣传讨论黄药眠文学批评观念与方法的文章。黄药眠先生的文艺理论造诣深厚，学术界对他的研究却较为匮乏。本文深入研究对象的思想观念，是颇具功力的用心之作。

“粤港澳作家专栏”刊发张栋评论鲍十小说的文章。鲍十近年来创作颇丰，但却没有引起评论界的必要关注。本文全面梳理鲍十小说创作轨迹，讨论其创作特色，对于我们全面认识鲍十创作具有很好的价值。

“青年作家专辑”评论的作家是青年作家林培源。林培源是潮汕人，属于成名较早的“80后”作家，同时兼具作家和学者的双重身份。对林培源的评论也能够促进我们对这一代作家的一些思考。

“作家作品评论”刊发李徽昭、夏至、谢腾飞等几位中青年批评家的文章，评论对象中有莫言近作《鳄鱼》，也有张枣和徐则臣这样的中间作家。文章的论述都很敏锐深入。

“广东文学振兴”栏目刊发向卫国《在“大湾区”建设背景下对粤东西北诗歌发展的思考》。这是一位基层作者，讨论的也是广东基层文学的发展问题，文章很接地气。

本期开始，新设“新作快评”栏目，讨论近期问世的新作品。本期讨论的是邱华栋的长篇小说《空城纪》。李德南和曾攀都属于“80后”批评家，两篇文章风格不同，但都体现了他们对作品的深入思考。

本期还设置了“《戏剧叙事学》研究小辑”。袁国兴教授潜心多年，出版了颇具创新意义的学术专著《戏剧叙事学》。文章之一是作者对问题的进一步阐发，之二则是作为同行学者的张羽教授的阅读思考。都能够帮助我们更好理解这部著作。
