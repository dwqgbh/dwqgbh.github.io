deepin 亮相 elexcon2025 深圳国际电子展,全面展示 RISC-V 生态建设新成果

深度操作系统 2025-08-28

2025年8月26-28日，为期 3 天的 elexcon2025 深圳国际电子展暨嵌入式展于今日在深圳会展中心圆满落幕。本届展会以“All for AI, All for GREEN”为主题，汇聚了全球 400 多家领先技术企业，吸引了数万名专业工程师与产业决策者到场交流。值得一提的是，本届展会特别为 RISC-V 开源社区搭建了专属展示平台，以切实行动大力支持 RISC-V 开源生态的建设与发展。

作为国内领先的桌面操作系统社区，deepin（深度）社区助力"甲辰计划"，首次在大会亮相并深度参与了此次盛会，通过一系列硬核展示和互动活动，全面彰显了其在 RISC-V 生态建设中的重要贡献和技术实力。

deepin 对 RISC-V 架构的支持并非一日之功。在展会现场，deepin 全面展示了其在操作系统研发、RISC-V 架构适配与开源生态建设方面的多项核心成果。
自 2022 年 2 月起，deepin 就建立了对应 SIG，推进 RISC-V 架构的适配工作，现已成功支持了大量主流的 RISC-V 硬件和开发板。
deepin on RISC-V
在软件方面，deepin 也完成了对 RISC-V 开源软件生态的适配，提供了超过 27,000 个软件包，并为 RISC-V 开发板提供了内核、GPU、VPU、NPU 等驱动解决方案，确保了这些关键组件能够长期、及时、良好地维护。

deepin(深度)社区展位

作为中国桌面操作系统的核心力量，deepin 积极响应国家战略，深度参与“甲辰计划”，开放了社区参与通道和“甲辰计划”实习生招募，全力投入 RISC-V 开源新生态建设。迄今，deepin 操作系统已成功适配了几乎所有可公开获取的桌面级 RISC-V 设备，并提供了关键的 GPU、NPU、VPU 等硬件加速支持。deepin 23 稳定版及 deepin 25 预览版均已为 RISC-V 平台提供官方镜像并持续更新。

通过在硬件适配、软件生态构建、社区协作及战略规划上的不懈努力，deepin 已成为 RISC-V 生态的重要贡献者，并有力推动着 RISC-V 桌面操作系统的普及与应用。

未来，deepin 将继续关注 RISC-V 技术的发展，与广大开发者一起，为推动包括 RISC-V 在内的开源技术生态的健康发展贡献力量。
