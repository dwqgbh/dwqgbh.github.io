1. 工信部+国资委联合启动"人形机器人与具身智能实景实训专项行动"

事件概述：6月9日，工业和信息化部办公厅、国务院国资委办公厅联合印发《关于联合开展2026年度人形机器人与具身智能实景实训专项行动的通知》（工信厅联科〔2026〕XX号），正式启动国家级专项行动。核心目标：

到2026年底，人形机器人等重点产品在一批代表性场景中率先完成应用验证和常态化部署，开启"作业模式"，凝练形成百个以上高价值应用场景，带动形成万台级规模落地能力

覆盖北京、天津、上海、江苏、浙江、山东、湖北、湖南、广东、四川等10个重点省市

行动重点部署四大任务：实景实训空间建设、创新应用联合体培育、作业技能攻关、应用部署验证。面向工业、特种、服务等领域，推动真实场景训练→模型优化→规模部署的闭环。

为什么值得关注：

万台级落地是确定性订单，央企/国企将成为采购主力，这是B端大客户最直接的切入窗口，"常态化部署"而非"试点"，意味着持续性租赁/维保订单，而非一次性买卖，10个重点省市已明确。配套实训空间建设、技能培训需求同步爆发，培训业务迎来政策东风

北京工贸技师学院已率先开标439.69万元实训中心建设，各地类似项目将陆续释放

来源：工信部官网 / 科技日报 / 新华网 

2. 2026世界杯成全球最大机器人试验场：人形机器人从"演员"变"打工人"

事件概述：当地时间6月11日，2026 FIFA世界杯在美国、加拿大、墨西哥三国开幕。FIFA官方确认，这是人类体育史上最大规模的人工智能与机器人技术真实试验场：

波士顿动力 Atlas

（人形机器人）：在多个场馆执行赛事运营辅助、球迷互动等任务

波士顿动力 Spot

（四足机器狗）：已入驻达拉斯国际广播中心、纽约/新泽西体育场，执行安全巡逻、实时场地监测

中国机器人企业同步加速出海：

优必选

Walker S2：已部署于德州仪器半导体产线、ROSSMANN欧洲零售店、日立电梯工厂；2025年海外收入4.75亿元，产品进入50+国家

宇树科技

G1：日本羽田机场担任地勤（行李装卸/货物转运）、波兰华沙街头驱赶野猪；2026年计划出货量1-2万台

越疆科技

Nova系列：咖啡机器人已在20+场景运营，服务地中海可口可乐、阿联酋、新加坡等

2026年前4个月：各类机器人累计出口814.5万台，货值157.9亿元，创历史新高

为什么值得关注：

世界杯是全球级场景验证，将大幅加速客户对机器人"实际工作能力"的认知，降低销售说服成本

中国机器人出口激增（前4个月157.9亿元），说明国产品牌全球竞争力已获认可，销售时可强化"国产替代"信心

机场、零售、工业巡检等出海场景，为国内同类场景的销售/租赁提供了可复制的案例库。摩根士丹利预测2026年中国人形机器人销量达2.8万台（同比翻番），市场规模持续放大

来源：36氪 / 腾讯新闻 / 中国能源新闻网 🔗🔗

3. 智元机器人全球线下首店落地上海，开启"零售综合体部署态"新模式

事件概述：6月12日，京东MALL上海七宝店正式开业（上海首店，京东全国第28家门店）。店内核心亮点之一：智元机器人全球首个线下体验店同步亮相。这并非简单的产品展示，而是智元打造的中国首个"零售综合体部署态"落地标杆店：

消费者可近距离观看、实操人形机器人、机器狗等前沿产品

门店打造两座全国首创特色场馆：银龄馆（面向老年人）、视障人士体验馆，建筑面积近5万平方米，汇聚全球200家品牌、超20万款商品。试营业期间日均客流过万，周末峰值近2万人次，营业额累计突破1.6亿元

为什么值得关注：

"机器人卖机器人"的零售模式首次在大型商业综合体规模化落地，证明消费端接受度正在提升

对租赁/销售创业者：商超、购物中心、科技潮品店将成为新的租赁需求场景（互动引流、体验营销）

上海标杆落地将带动全国"机器人零售店"业态扩张，零配件销售和培训需求同步增长

银龄馆、视障体验馆等细分场景的出现，说明服务机器人在养老/无障碍领域有明确增量

来源：中新社 / 快科技 / 腾讯新闻 

4. 京东+机时租达成全面合作：机器人全链路服务生态成型

事件概述：6月11日，京东与头部开放式机器人租赁平台机时租正式达成全面合作。双方围绕以下环节构建全链路服务体系：

设备租赁：人形机器人、四足机器人、AI陪伴机器人等前沿设备

全球供应链物流：京东物流支持

机器人维修售后：京东8大维修中心+5年10万工程师计划

保险服务：设备使用保障

设备回收：废旧设备回收链路

二次开发：数据采集与定制化开发

京东服务目前已与行业绝大多数头部机器人企业达成深度合作，全国布局8大机器人维修中心，并首创"机器人救护车"+120服务体系（1分钟响应、30分钟到场）。

为什么值得关注：

京东进场意味着售后维保赛道正式成为巨头必争之地，独立维保公司需要尽快建立差异化能力（如本地化响应、特定品牌深度、培训+维保一体化）

京东"机器人救护车"1分钟响应/30分钟到场的服务标准，正在抬高行业服务门槛，你的团队能否对标？

设备回收链路打通，意味着二手翻新/再销售成为可行商业模式，适合资金实力较强的创业者布局

数据采集+二次开发服务需求增长，如果团队有技术能力，可切入数据服务高附加值环节

来源：京东物流黑板报 / 荣格工业资源网 

5. 长三角机器人展览会开幕：无锡锁定具身智能200亿营收目标

事件概述：6月11日，2026长三角机器人及自动化展览会暨无锡具身智能机器人产业链伙伴大会在无锡举行。大会由中国机电一体化技术应用协会主办，以"能力进阶·生态聚变"为主题，汇聚智元机器人、开普勒机器人、中国移动等头部企业。核心亮点：

无锡正式发布目标：2026年具身智能机器人产业营收突破200亿元

无锡已提前布局国内首个支持10万卡规模组网的智算集群

中国移动工业具身智能创新基地正式揭牌

开普勒机器人披露：已建成国内唯一工业实景人形机器人专属训练场（2024年提前布局），精准匹配工信部专项行动导向

无锡市委书记杜小刚出席，显示地方政府对具身智能产业的高度重视

为什么值得关注：

无锡200亿目标意味着长三角地区的产业配套、供应链、人才培训需求将爆发，如果你在江浙沪地区，应积极参与当地生态

开普勒"国内唯一工业实景训练场"的落地，说明数据服务/训练场运营成为新基建需求，如果你的团队有场地资源，可转型为区域训练服务提供商

各地政府都在争抢具身智能产业落地，应关注本地是否有类似政策窗口，提前对接智算集群+机器人产业链配套，意味着算力租赁+机器人租赁的组合服务模式可能兴起

建议关注：各地产业园、开发区是否有免费场地/补贴用于机器人展示中心或实训基地入驻

来源：无锡日报 / 新浪财经 / 无锡市人民政府 

6. 宇树IPO注册进入倒计时：预计6月18日批复，7月上旬上市

事件概述：宇树科技科创板IPO持续高速推进：

6月1日：通过上交所上市委审议（历时73天，创科创板2026年最快纪录）

6月2日："闪电"提交注册申请

当前状：处于证监会20个工作日注册审核周期中，6月18日（周四）被预判为注册批复的高概率窗口（端午节前最后一个工作日）

若批复顺利，预计最早7月上旬正式登陆科创板，成为A股"具身智能第一股"拟募资42.02亿元，用于智能机器人模型研发、本体研发及智能制造基地建设

为什么值得关注：

宇树上市后品牌溢价将进一步提升，配件供应能力将增强，利于维保业务长期规划

宇树上市后股价波动将引发媒体持续报道，相当于免费的市场教育，有助于降低客户决策门槛

英伟达已将宇树H2 Plus选为机器人参考设计"身体"，宇树将成为Cosmos 3生态标杆，未来应用场景更丰富

来源：东方财富 / 新浪财经
