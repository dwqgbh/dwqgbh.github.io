燃情大湾区!峨眉传奇2026年度盛典:高额奖金、金腰带荣耀,崛起之路与传奇少年共铸搏击传奇!

峨眉传奇 2026-02-07


2026峨眉传奇·传奇少年广州粤港澳大湾区选拔赛

与金腰带争夺战

峨眉传奇崛起之路广州粤港澳大湾区站

2026赛程规划来了！

1分钟带你看懂峨眉传奇大湾区赛事：怎么打？奖多少？怎么往上冲？

1. 赛事级别：

•A级赛（世界冠军赛）：最高级别，世界高手过招。

•B级赛（崛起之路）：职业俱乐部联赛，成人打，打得好直接进A级赛。

•C级赛（传奇少年）：青少年比赛，培养搏击小将，打得好能进全国总决赛，甚至有机会进A级赛。

2. 比赛时间地点：

•2026年3月开始，每月一场，全年10场！

•首场：3月28-29日，广州黄埔区人为峰广场·峨眉传奇大湾区职业拳场。

3. 奖金和金腰带：

•传奇少年（C级赛）：

◦常规腰带赛：冠军最高拿1000元现金 + 金腰带 + 证书。

◦年中决赛：冠军最高拿1500元现金 + 金腰带 + 证书。

◦年终总决赛：冠军最高拿2000元现金 + 金腰带 + 证书。

•崛起之路（B级赛）：职业赛打得好，不仅有奖杯证书，还能推荐去国内外职业比赛，有出场费！

4. 怎么晋级？

•传奇少年：分站赛冠军亚军 -> 年中决赛 -> 大湾区年终总决赛 -> 全国总决赛 -> 优秀小将有机会直通A级赛！

•崛起之路：B级赛打得好，直接受邀参加A级赛，跟世界高手过招！


序言：峨眉传奇三级赛事体系，铸就中国搏击新篇章


作为中国搏击界的领军品牌，“峨眉传奇”始终致力于构建科学严密的“金字塔”型三级赛事体系，旨在实现从青少年启蒙到世界顶级职业竞技的全方位覆盖。这一体系由三大核心级别构成：


•A级赛事（世界格斗冠军赛）：代表国际顶级水平的“荣耀巅峰”，汇聚全球格斗精英。

•B级赛事（崛起之路）：作为职业俱乐部联赛，是连接基础与顶尖的桥梁，为职业选手提供晋升通道。

•C级赛事（传奇少年）：作为峨眉传奇生态的核心基石与造血细胞，肩负着为中国搏击事业培养后备军的重大使命，是青少年搏击梦想的起点。


2026年，峨眉传奇将携“崛起之路”与“传奇少年”两大核心赛事板块全面登陆粤港澳大湾区，为不同年龄段的搏击爱好者提供专业竞技平台，共同书写属于他们的传奇篇章。


2026大湾区赛事日程与地点


今年的比赛将从2026年3月正式开始，第一场比赛定在3月28日至29日，地点是广州市黄埔区人为峰广场·峨眉传奇大湾区职业拳场。之后，每个月都会有一场精彩的比赛，一直持续到12月，全年总共有10场硬仗。在这期间，“崛起之路”和“传奇少年”会轮流进行，让大湾区的搏击界一直充满活力。


传奇少年（C级赛事）：点燃青少年搏击梦想


“传奇少年”系列比赛是专门为17岁及以下的青少年搏击爱好者设计的。我们希望通过不同层次的比赛，全面提高小选手们的格斗技术和心理素质。比赛包括自由搏击、泰拳、拳击这三种主流规则，全程都会佩戴全套护具，确保小选手们的安全。


  赛制与奖励概览

新秀赛：面向17岁及以下选手，进行3回合（每回合1分30秒，休息40秒）的比赛，全程佩戴全护具。新秀赛不分输赢，所有参赛者都将获得奖杯和证书。报名费为380元。


冠军赛：同样面向17岁及以下选手，赛制与新秀赛相同，为3回合（每回合1分30秒，休息40秒），全程佩戴全护具。冠军赛将分出胜负，获胜者将获得奖杯和证书。报名费为380元。


腰带奖金赛：面向17岁及以下选手，比赛升级为3回合（每回合2分钟，休息1分钟），全程佩戴全护具。腰带奖金赛将分出胜负，报名费为600元。


  丰厚奖金与晋级之路

“传奇少年”比赛不仅提供专业的竞技平台，还设置了非常吸引人的奖金和清晰的晋级路线，鼓励每一个参赛的少年去追求卓越，成为更好的自己。


常规腰带奖金赛：

•四人赛：冠军将获得500元奖金，以及金腰带和证书。其余三名选手将获得奖杯和证书。

•八人赛：冠军将获得1000元奖金，以及金腰带和证书。其余七名选手将获得奖杯和证书。


年中决赛（8月）：

•四人赛：冠军将获得800元奖金，以及金腰带和证书。其余三名选手将获得奖杯和证书。

•八人赛：冠军将获得1500元奖金，以及金腰带和证书。其余七名选手将获得奖杯和证书。


年终总决赛：

•四人赛：冠军将获得1200元奖金，以及金腰带和证书。其余三名选手将获得奖杯和证书。

八人赛：冠军将获得2000元奖金，以及金腰带和证书。其余七名选手将获得奖杯和证书


  晋级路径

年中决赛（8月）：在3月到7月期间，只要在腰带奖金赛中拿到冠军或亚军的选手，才有资格报名参与年中决赛（详见后续推文年中决赛竞赛规程）。

大湾区年终总决赛：全年所有腰带赛的冠军和亚军选手，将齐聚大湾区年终总决赛，争夺大湾区年度总冠军的最高荣誉。

全国总决赛：在大湾区年终总决赛中表现出色的选手，有机会最终进入“峨眉传奇·传奇少年”全国总决赛，和全国最厉害的少年选手一较高下。

直通A级赛：在全国总决赛中表现特别好，并且年龄也符合A级赛要求的少年选手，有机会被邀请参加“峨眉传奇”A级赛事，实现从少年到职业选手的华丽转身！

崛起之路（B级赛事）：职业俱乐部联赛，通往荣耀巅峰

“崛起之路”是峨眉传奇的B级赛事，主要是职业俱乐部之间的对抗。它是连接基础比赛和顶尖比赛的桥梁。这个比赛为17-30岁的职业搏击选手提供了一个高水平的竞技平台，目的是选拔出最有潜力的搏击人才，输送到更高级别的比赛。除此之外的百姓擂台主要针对业余爱好者，不论年龄和拳种都可以报名参与，赛区将根据选手综合实力进行配对。

  赛制与晋级机制

百姓擂台：面向成年选手，进行3回合（每回合2分钟，休息1分钟）的比赛，全程佩戴全护具。参赛者将获得奖杯和证书。报名费为380元（含保险）。

职业赛：面向17-30岁的职业选手，进行3回合（每回合3分钟，休息1分钟）的比赛，不佩戴护具。参赛者将获得奖杯和证书。只需缴纳100元保险服务费。如果选手能连赢三场，将有机会被推荐参加国内外职业比赛，并获得出场费！

拳击金腰带赛：面向成年选手，进行3回合（每回合2分钟，休息1分钟）的比赛，全程佩戴全护具。参赛者将获得奖杯和证书。报名费为600元（含保险）。

  晋级路径

直通A级赛：在“崛起之路”B级赛事中表现出色的选手，有机会直接被邀请参加“峨眉传奇”A级赛事，和世界级高手同台竞技，争夺搏击界的最高荣誉！

2026年“峨眉传奇”大湾区赛事，无论是“传奇少年”为青少年搭建的梦想舞台，还是“崛起之路”为职业选手提供的晋升通道，都将是搏击界的一场盛事。我们真心邀请大湾区乃至全国的搏击健儿们，勇敢地站上擂台，用汗水和激情书写属于自己的传奇，一起攀登中国搏击的最高峰！
