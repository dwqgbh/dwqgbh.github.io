设备工程学院2026年专业介绍

武汉工程职业技术学院 2026-06-03

 设备工程学院


校企共建杜操测量技能大师工作室


学院简介


      设备工程学院（原土木工程学院、智能建造与设计学院）成立于1984年，现有专任教师30余人，其中教授1人、副教授12人，在校生近1200人，现有安全技术与管理、绿色低碳技术、供热通风与空调工程技术、建筑设备工程技术、建筑工程技术、建筑室内设计等6个专业，其中建筑设备工程技术、建筑工程技术是省级品牌特色专业，供热通风与空调工程技术是学院重点专业。学院积极与本地优质企业深化产教融合、校企合作，先后创建武汉天华BIM协同创新工作室、宝钢建科钢结构深化设计工作室、杜操测量技能大师工作室，近三年，学生获湖北省职业院校技能大赛、湖北省大学生职业规划大赛等15项荣誉，应届毕业生就业率稳定在93%以上。


专业介绍


建筑工程技术

（湖北省特色专业）

培养目标 …

       培养从事建筑施工技术及建筑施工合同管理、进度管理、质量管理、安全管理、技术资料管理和成本控制等工作的高素质技术技能人才。

核心课程 …

      建筑施工技术、建筑施工组织、工程项目管理、工程资料管理、建筑工程计量与计价、建筑信息模型应用等。

就业去向 …

     建筑施工企业、建筑装配化企业、建筑设计院、建设单位、监理公司、钢结构建筑及安装公司、建设行业管理部门、测绘工程公司等。



安全技术与管理

培养目标 …

      安全管理、安全评价技术、防火与防爆技术、电气安全技术、消防工程技术、安全生产事故调查与案例分析等。培养从事安全管理、安全评价和安全技术咨询等工作的高素质技术技能人才。

核心课程 …

      安全管理、安全评价技术、防火与防爆技术、电气安全技术、消防工程技术、安全生产事故调查与案例分析等。

就业去向 …

      安全员、安全生产管理工程技术人员、安全评价工程技术人员等。




绿色低碳技术

培养目标 …

      培养从事碳排放管理、绿色体系评价、清洁生产、低碳技术应用等工作的高素质技术技能人才。

核心课程 …

      企业碳核查技术、低碳技术与应用、碳排放权交易与绿色金融、企业清洁生产审核、绿色生命周期评价、工业节水管理技术等。

就业去向 …

      碳排放管理员、环境保护工程技术人员等，碳排放管理、绿色体系评价、清洁生产、低碳技术应用等岗位（群）。





建筑设备工程技术

（湖北省特色专业）

培养目标 …

     培养从事中小型建筑设备工程施工图设计与建模、施工与造价、运维与管理等工作的高素质技术技能人才。

核心课程 …

     建筑给排水、建筑电气、供热工程、制冷技术、通风与空调工程、建筑信息建模应用等。

就业去向 …

      建筑给排水、供热、通风空调、建筑电气等安装工程施工、造价、管理、运行与调试等岗位（群）。



供热通风与空调工程技术

培养目标 …

       培养从事中小型暖通空调工程设计、施工与造价，暖通空调系统智能化运维等工作的高素质技术技能人才。

核心课程 …

      供热工程、供热通风与空调施工技术、工种操作实训等。供热工程、通风与空调工程、制冷技术、建筑给排水、BIM技术应用、建筑设备施工技术等。

就业去向 …

     供热、通风、空调、冷库、给排水、供配电与照明等方向的设计、生产、施工、造价、运维技术人员。



建筑室内设计

培养目标 …

       培养具备数字化思维、可持续设计理念及AI辅助设计、智能生成优化能力，掌握智慧人居设计理念的复合型装饰设计人才。

核心课程 …

      涵盖AI辅助效果图制作、智慧人居场景规划、室内/商业空间设计、家居定制与参数化设计、装饰构造、施工图深化设计、装饰施工技术等。

就业去向 …

      服务于各类装饰设计公司、智慧康养中心、文化创意公司、AI设计赋能平台及智慧人居解决方案企业等。
