2022年一建机电学霸笔记:机械设备(一)

233网校一级建造师考试 2022-08-15


考点一：设备基础的种类及应用

【经典例题】

沉井基础是用钢筋混凝土或（ ）制成的井筒式基础。

A .混凝土

B .预制的钢筋混凝土柱

C .工字钢

D .大口径钢管

参考答案：A
参考解析： 本题考查的是各种基础的定义，沉井基础是用混凝土或钢筋混凝土制成的井筒式基础。如冶炼、石油化工工程的烟囱和火炬。

下列机械设备基础中，适用于需要减少基础振幅的大型设备的是( )

A .扩展基础

B .联合基础

C .桩基础

D .沉井基础

参考答案：C
参考解析： 桩基础是由承台、桩组成的基础形式，可分为预制桩和灌注桩两大类，适用于需要减少基础振幅、减弱基础振动或控制基础沉降和沉降速率的精密、大型设备的基础。如透平压缩机、汽轮发电机组，锻压设备。本题答案为C。
考点二：设备基础混凝土强度的验收要求

①基础施工单位应提供设备基础质量合格证明文件，主要检查验收其混凝土配合比、混凝土养护及混凝土强度是否符合设计要求。如果对设备基础的强度有怀疑时，可请有检测资质的工程检测单位，对基础的强度进行复测。

②重要的设备基础应做预压强度试验，预压合格并有预压沉降详细记录。如大型锻压设备、汽轮发电机组、大型油罐。

考点三：设备基础位置、标高、几何尺寸的验收要求

设备基础位置和尺寸的主要检查项目：

基础的坐标位置；不同平面的标高；平面外形尺寸；凸台上平面外形尺寸；凹槽尺寸；平面的水平度；基础的垂直度；预埋地脚螺栓的标高和中心距；预埋地脚螺栓孔的中心线位置、深度和孔壁垂直度；预埋活动地脚螺栓锚板的标高、中心线位置、带槽锚板和带螺纹孔锚板的平整度等。

检查基础坐标、中心线位置时，应沿纵、横两个方向测量，并取其中的最大值。

设备基础混凝土强度的验收，主要检查验收其( )是否符合设计要求。

A .外观观感

B .混凝土配合比

C .混凝土养护

D .基础的坐标位置

E .混凝土强度

参考答案：BCE
参考解析： 设备基础混凝土强度的验收要求
(1)基础施工单位应提供设备基础质量合格证明文件，主要检查验收其混凝土配合比、混凝土养护及混凝土强度是否符合设计要求。
(2) 如果对设备基础的强度有怀疑时，可请有检测资质的工程检测单位对基础的强度进行复测。(3)重要的设备基础应做预压强度试验，预压合格并有预压沉降详细记录。如大型锻压设备、汽轮发电机组、大型油罐。
考点四：机械设备安装的一般程序（重要考点）

机械设备安装的一般程序：施工准备→设备开箱检查→基础测量放线→基础检查验收→垫铁设置→设备吊装就位→设备安装调整→设备固定与灌浆→设备零部件清洗与装配→润滑与设备加油→设备试运转→工程验收

记忆口诀：机械设备安装的一般程序：施工准备——设备开箱检查——基础测量放线——基础检查验收——垫铁设置——吊装就位——设备安装精度调整——设备固定与灌浆——设备零部件清洗与装配一润滑与设备加油——设备试运转——工程验收。(想放茶点就调酱，不加食盐)

机械设备安装的一般施工程序中，设备基础检查验收的紧后工序是( )

A .基础放线

B .设备开箱检查

C .安装精度调整与检测

D .垫铁设置

参考答案：D
参考解析： 机械设备安装的一般程序：
施工准备→设备开箱检查→ 基础测量放线→基础检查验收→垫铁设置→设备吊装就位→设备安装调整→设备固定与灌浆→设备零部件清洗与装配→润滑与设备加油→设备试运转→工程验收。

机械设备安装的一般程序中，设备试运转的紧前工作为（）

A .设备固定与灌浆

B .设备零部件清洗与装配

C .润滑与设备加油

D .工程验收

参考答案：C
参考解析： 机械设备安装的一般程序：
施工准备→设备开箱检查→ 基础测量放线→基础检查验收→垫铁设置→设备吊装就位→设备安装调整→设备固定与灌浆→设备零部件清洗与装配→润滑与设备加油→设备试运转→工程验收。

机械设备安装程序中，安装精度调整与检测的紧前工作是(  )。

A .设备固定与灌浆

B .垫铁设置

C .吊装就位

D .设备装配

参考答案：C
参考解析： 机械设备安装的一般程序：
施工准备→设备开箱检查→ 基础测量放线→基础检查验收→垫铁设置→设备吊装就位→设备安装调整→设备固定与灌浆→设备零部件清洗与装配→润滑与设备加油→设备试运转→工程验收。
