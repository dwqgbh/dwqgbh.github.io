2025苏州机床展,3月19-22日举办,附门票+交通指南

展讯网会展平台 2025-01-10

2025苏州国际机床展将于3月19-22日在苏州国际博览中心举办。苏州国际机床展将以4万平米展览面积，汇聚700+先进机床品牌，全方位多维度展示金切机床、成形机床、机床附件、工量刃具、工业自动化及机器人等加工领域的创新产品及应用解决方案！


2025苏州机床展时间+地点+门票

展会时间：2025年3月19-22日

展会地点：苏州国际博览中心

展馆地址：苏州工业园区苏州大道东688号


预约苏州机床展门票


展览范围

❖金属切削机床Metal-cutting Machine Tool

五轴、龙门、立式、卧式等加工中心，深孔钻、型材加工中心、组合机床、复合机床、雕铣机床、车床、磨床、钻床、铣床、车床...


❖金属成形机床Forming Machine Tool

钣金及冲压自动化设备：开卷校平、上下料、冲床压力机、弯管机、表面处理、钣金辅助及周边设备、工厂数字化系统软件...


激光加工自动化设备：激光平板切割、激光切管、激光器及元器件、通风环保安全、激光辅助设备...


连接与激光焊接工艺：激光焊接机、冲压连接、铆接、粘接及螺接、压接（无铆连接）、螺柱、电渣...


❖机床附件Machine Tool Accessories

数控机床核心功能技术：主轴、丝杠、导轨、电机、转台、分度盘、动力头、刀塔刀库、数字化制造、数控机床...


检测设备：三坐标、光学影像测量仪、测量软件、系统、接触式、扫描式测量臂、X光检测设备、红外测试设备...


工业零件：工业材料、模具、模具配件、三维扫描及测量...


❖工量刃具Measuring and Cutting Tools

加工刀具、组合刀具、非标刀具、刀具涂层、磨削机械、磨具磨料、工具磨床、夹治具、工业耗材、工业测量...


❖机器人及自动化Robotics and Automation

机械传动及专业设备：丝杆、导轨、线性模组、减速机、轴承..


机器人及自动化设备：机器人/协作机器人及应用方案、智能焊接、打磨抛光、机器人应用方案...


运动控制技术：运动控制、伺服系统、连接器、传感器、工业数字化软件...




参展品牌

明星展区，王牌相见。作为2025明星机床展，本届展会以4万㎡展会规模，集结哈斯、迪恩、马扎克、牧野、西铁城、现代威亚、津上、发那科、西部电机、荣田精机、蔡司、海克斯康、纽威、国盛、威达、豪迈、海德曼等全球机床领域内的700+国内外知名品牌参展。（*以上排名不分先后）




会议活动

同期会议：

SMT国际机床与智能制造高级研修班

航空零部件精密加工技术及应用论坛

交通指南
苏州国际博览中心的详细地址为：江苏省苏州市工业园区苏州大道东688号
A、地铁：
地铁1号线，“文化博览中心”站，3号口出。
B、公交：
搭乘公交812路或219路，直达苏州国际博览中心
C、机场：
浦东机场—地铁2号线—虹桥机场2号航站楼—沪宁城际高铁—苏州园区站—乘公交258路—“博览中心北站”—苏州国际博览中心
D、火车站：
苏州火车站→苏州国际博览中心
公车529路转812路可到达/公车529路转219路可到达/公车游2路转28路可到达
