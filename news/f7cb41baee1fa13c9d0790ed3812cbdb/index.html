热点01 2026年全国两会与政府工作报告

【热点衔接】

2026年3月，全国两会顺利召开，国务院总理李强作政府工作报告。报告回顾2025年成就：“十四五”圆满收官，国内生产总值达140.19万亿元、增长5%，粮食产量1.43万亿斤，新质生产力稳步发展，人工智能、量子科技等领域研发应用走在世界前列，民生保障持续加强，城乡居民基础养老金标准提高，脱贫攻坚成果不断巩固；部署2026年重点工作：以“十五五”开局为契机，聚焦高质量发展，推动科技创新与产业融合，扩大高水平对外开放，推进乡村全面振兴，强化民生保障，协同推进降碳减污扩绿增长，守护粮食安全，防范化解重点领域风险，推动中国式现代化迈出新步伐，同时重申坚持“一国两制”，推动构建人类命运共同体。

【对接教材考点】

（一）八年级核心考点

1. 国家机构：全国人民代表大会的性质（最高国家权力机关）、职权（立法权、决定权、监督权等），国务院的性质（最高国家行政机关）及职能，两会体现的国家机构运行原则；

2. 公民权利与义务：公民的监督权（两会代表委员反映民意，体现公民行使监督权）、劳动权、受教育权、物质帮助权（养老金提高等民生举措对应）；

3. 宪法与法治：宪法是根本活动准则，全国两会审议法律草案（如生态环境法典）体现科学立法、严格执法、公正司法、全民守法的法治要求，法治对国家治理的作用；

4. 国家利益与国家安全：维护国家利益（粮食安全、经济发展、对外开放等举措），公民维护国家安全的责任与义务；

5. 社会建设：社会公平正义，政府关注民生、解决民生问题，推进乡村振兴体现对弱势群体的关怀，践行尊重和保障人权的宪法原则。

（二）九年级核心考点

1. 富强与创新：高质量发展、新发展理念、创新驱动发展战略、科技自立自强（新质生产力相关）；

2. 民生与公平：以人民为中心的发展思想、共享发展成果、共同富裕、民生福祉（养老金、就业、教育等）；

3. 改革与开放：全面深化改革、扩大高水平对外开放、海南自贸港建设、经济全球化；

4. 生态文明：绿色发展、人与自然和谐共生、节约资源和保护环境的基本国策、碳达峰碳中和；

5. 国家治理：中国特色社会主义制度优势、全过程人民民主、法治政府建设；

6. 国家安全与统一：粮食安全、资源安全、“一国两制”、国家主权与领土完整；

7. 中国担当：人类命运共同体、独立自主的和平外交政策、反对霸权主义和单边主义。

热点02 党的二十届四中全会与“十五五”规划

【热点衔接】

2025年10月20日至23日，中国共产党第二十届中央委员会第四次全体会议在北京举行，全会最重要的成果是审议通过了《中共中央关于制定国民经济和社会发展第十五个五年规划的建议》。全会充分肯定“十四五”时期我国发展取得的重大成就，明确“十五五”时期是基本实现社会主义现代化夯实基础、全面发力的关键时期，部署了重点任务：坚持党的全面领导，以高质量发展为主题，加快高水平科技自立自强、培育新质生产力，巩固壮大实体经济，推进乡村全面振兴、区域协调发展，扩大高水平对外开放，强化民生保障，推进美丽中国建设，筑牢国家安全屏障，统筹发展和安全，推动中国式现代化迈出新步伐，为2035年基本实现社会主义现代化奠定坚实基础。2026年3月，全国两会审议通过《中华人民共和国国民经济和社会发展第十五个五年规划纲要（草案）》，将全会部署转化为具体行动纲领。

【对接教材考点】

（一）八年级核心考点

1. 国家机构与党的领导：中国共产党的领导是中国特色社会主义最本质的特征，党的二十届四中全会体现党对国家发展的全面领导；全国人民代表大会的职权（审议批准“十五五”规划纲要，行使决定权）。

2. 宪法与法治：宪法是根本活动准则，全会部署符合宪法精神，“十五五”规划实施需坚持依法治国，推进法治政府、法治社会建设。

3. 公民权利与义务：“十五五”规划中民生保障、教育科技人才发展等举措，保障公民的受教育权、劳动权、物质帮助权；公民有维护国家利益、参与国家建设的义务，可通过合法途径为“十五五”规划建言献策。

4. 国家安全：全会强调筑牢国家安全屏障，体现总体国家安全观，公民有维护国家安全（经济安全、科技安全、生态安全等）的责任与义务。

5. 社会公平正义：“十五五”规划推进区域协调发展、乡村全面振兴、民生改善，体现国家致力于维护社会公平正义，践行尊重和保障人权的宪法原则。

（二）九年级核心考点

1. 党的领导与制度优势：中国共产党的领导是中国特色社会主义制度的最大优势，全会部署彰显社会主义制度能集中力量办大事的显著优势。

2. 富强与创新：高质量发展、科技自立自强、培育新质生产力，对接创新驱动发展战略、科教兴国和人才强国战略，体现创新是引领发展的第一动力。

3. 改革与开放：全面深化改革、破除市场壁垒，扩大高水平对外开放、高质量共建“一带一路”，对接改革开放的重要意义，完善社会主义市场经济体制。

4. 民生与共同富裕：强化民生保障（养老、教育、就业等）、推进乡村全面振兴、区域协调发展，践行以人民为中心的发展思想，推动全体人民共同富裕。

5. 生态文明：推进美丽中国建设、协同推进降碳减污扩绿增长，对接人与自然和谐共生、绿色发展道路、节约资源和保护环境的基本国策。

6. 中国梦与中国式现代化：“十五五”规划目标对接中国梦的内涵，推动中国式现代化发展，体现国家致力于实现第二个百年奋斗目标、基本实现社会主义现代化。

【热点衔接】

2025年9月3日，纪念中国人民抗日战争暨世界反法西斯战争胜利80周年大会在北京天安门广场隆重举行，盛大九三阅兵同步开展。习近平总书记出席大会并检阅受阅部队，45个方（梯）队依次亮相，80面抗战英模部队荣誉旗帜承载历史荣光，多数国产现役主战装备首次亮相，空中梯队组成“80”字样和“正义必胜、和平必胜、人民必胜”条幅，14道彩烟寓意14年艰苦抗战历程。此次阅兵既是人民军队奋进建军百年的崭新亮相，也是中华民族捍卫世界和平的坚定宣示，同步开展抗战纪念章颁发、抗战主题展览等系列活动，缅怀革命先烈，传承伟大抗战精神，凝聚实现中华民族伟大复兴的磅礴力量。

【对接教材考点】

（一）八年级核心考点

1. 国家机构与党的领导：中国共产党是中国特色社会主义最本质的特征，抗战胜利的历史和阅兵实践证明，中国共产党是全民族抗战的中流砥柱，是国家发展的领导核心。

2. 宪法与法治：宪法规定国家尊重和保障人权，缅怀抗战先烈、保护抗战纪念设施，是对英烈权利的维护，体现法治精神，践行尊重和保障人权的宪法原则。

3. 公民权利与义务：公民有维护国家利益、国家安全的法定义务，阅兵彰显的国防力量的背后，是全体公民履行维护国家主权、安全、发展利益义务的体现；公民有受教育权，通过纪念活动和阅兵，接受爱国主义教育、国防教育。

4. 国家安全与国家利益：阅兵展示我国国防实力，筑牢国家安全屏障，体现总体国家安全观，维护国家主权、领土完整和发展利益，彰显国家利益至上。

5. 社会公平正义与责任：缅怀抗战先烈、致敬英雄，是对英烈奉献的认可，维护社会公平正义；每个公民都有责任传承抗战精神，守护国家和平与发展。

（二）九年级核心考点

1. 民族精神与文化自信：伟大抗战精神（天下兴亡、匹夫有责的爱国情怀，视死如归、宁死不屈的民族气节等）是中华民族精神的重要组成部分，阅兵和纪念活动传承红色基因，增强民族凝聚力和文化自信。

2. 爱国主义：纪念抗战胜利、举行九三阅兵，核心是弘扬爱国主义精神，爱国主义是民族精神的核心，是激励中国人民奋勇前进的精神动力。

3. 国家安全与国家发展：阅兵展示我国国防和军队现代化成就，体现我国坚持总体国家安全观，统筹发展和安全，为国家高质量发展保驾护航。

4. 中国担当与人类命运共同体：纪念抗战胜利、彰显和平理念，体现中国热爱和平、反对战争，积极推动构建人类命运共同体，为世界和平与发展贡献中国力量。

5. 中国梦与民族复兴：传承抗战精神、凝聚民族力量，是实现中华民族伟大复兴中国梦的强大精神动力，彰显中国人民实现民族复兴的坚定决心。

6. 制度优势：九三阅兵的成功举办、抗战胜利的历史成就，彰显中国特色社会主义制度能集中力量办大事的显著优势，体现中国共产党领导的优越性。
