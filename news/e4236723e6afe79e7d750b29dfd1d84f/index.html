/ 期刊简介/

About the Journal

《智能制造》杂志是由中国机械工业联合会主管，机械工业信息研究院（机械工业出版社）主办，国家智能制造专家委员会指导的国家级双效科技期刊。
《智能制造》杂志致力于打造智能制造领域智库型全媒体平台。期刊内容以智能制造为基础，关注制造业、信息化、智能化及广泛的科技领域。报道最新行业动态，发布产业前沿信息，与广大读者共同分享智能制造领域的新概念、新知识，探讨交流新观点、新技术和新方法。
为方便更多作者高效快捷投稿，现发布《智能制造》2024年度征稿计划：

/ 主要栏目设置/

Main Column

《智能制造》全年对外征集以下稿件：

1.智能装备

装备制造业领域的智能装备新技术、新工艺、装备智能化升级改造应用的典型案例等。

2.智能工厂

智慧工厂解决方案的典型案例等。

3.软件应用

大数据、工业互联网、人工智能等在制造业的实践应用。

4.智慧物流

智能供应链与智慧物流典型案例。

5.新技术新应用

智能制造的新技术、新应用的研究与实践。

6.人才培养

智能制造领域的高等教育人才培养理论研究、课程培养方案策略及实践案例等。

/ 特别征集 /

Special Collection

本年度主题策划栏目选题分别为新型工业化、工业大数据、人工智能、绿色智造、工业母机和智能制造试点示范。

/ 投稿须知/

Instructions for Authors

1. 本刊以尊重原创、倡导学术创新、杜绝学术不端为征稿基本原则。

2. 稿件以5000~6000字左右（含图/表）为宜。本刊不接收一稿多投稿件，在稿件受理期间，不得另投他刊。

3. 论文的排序：中文标题、作者姓名、工作单位（每位作者一个工作单位，包括邮政编码和所在省、市）、中文摘要、中文关键词、中图分类号、文献标志码、引言、正文、结语、参考文献、基金项目、作者简介，具体格式可参考投稿网站的作者中心中投稿须知及论文模板。

4. 稿件应观点明确、数据准确、论述严谨、语句精炼，规范编写法定计量单位、科技名词等。

5. 来稿请注明第一作者或通信作者姓名、详细通信地址、邮政编码、联系电话及E-mail地址。
