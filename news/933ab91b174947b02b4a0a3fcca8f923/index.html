报名指南:2025粤港澳大湾区低空经济高质量发展大会

中国广州发布 2025-12-22


在党中央、国务院关于发展低空经济的战略部署指引下，粤港澳大湾区依托独特优势，积极打造低空经济发展“先行样板”，持续释放发展势能，加速推进商业化进程。现定于2025年12月25日（星期四）在广州市中国进出口商品交易会展馆广交会堂，举办2025粤港澳大湾区低空经济高质量发展大会。


本次大会以“创新引领、安全筑基、湾区先行”为主题，主要内容包括领导致辞、粤港澳大湾区城市代表、著名院士、领军企业、国际组织和机构代表作主题发言，低空经济重磅成果发布，创新成果展示，专题报告，圆桌对话，项目签约等活动。拟邀请政府部门、低空经济领域核心企业、投资机构、国际组织、行业协会、科研院校以及外国驻穗领团代表等参会，共商低空经济发展大计，共谋低空经济安全、高效、可持续发展之路。


现诚挚邀请您报名参会出席2025粤港澳大湾区低空经济高质量发展大会。

谨此诚邀！




一、大会主题


创新引领、安全筑基、湾区先行



二、组织机构


（一）指导单位:

广东省发展和改革委员会、广州市人民政府、香港发展低空经济工作组、澳门经济及科技发展局


（二）主办及承办单位：

广州市发展和改革委员会、海珠区人民政府、中国经济信息社、工信部电子五所、广东省低空经济产业发展有限公司、广州低空经济产业发展有限公司、大湾区低空经济联盟（香港）、无人系统工程师协会（澳门）、广州市低空经济与航空航天产业协会



三、大会时间及地点


（一）大会时间

2025年12月25日（星期四）


（二）大会地点

广州市中国进出口商品交易会展馆

广交会堂

Parallel 

Session A

低空安全与科技创新专题报告会



聚焦低空安全治理与科技创新的深度融合，围绕空防、公共、飞行、信息四大安全领域，构建系统化低空安全保障体系展开探讨。


Parallel 

Session B

粤港澳大湾区经济发展

全球视野与高校行动专题报告会



主题：粤港澳大湾区低空经济发展：全球视野与高校行动

 一个兼具 “全球前瞻性、学术权威性、产业现实性、协同导向性” 的高端闭门研讨会，致力于产出引领湾区行动的思想共识与实体化合作机制。



Parallel 

Session C

低空经济创新应用场景

及要素支撑专题报告会


主题：低空赋能 百业焕新

围绕不同场景的开发应用，需要确保人财物等要素支撑以实现低空经济的商业化闭环，真正迎来低空经济的增长爆发。将围绕公共服务、生产作业、物流配送、交通运输等场景应用的创新经验和各项支撑要素展开研讨。


Parallel 

Session D

低空运营与政府治理现代化专题报告会



主题：低空赋能 高效治理

政府是低空技术在公共应用场景的主要提供者和需求方，政府通过开放公共应用场景、制定激励政策、完善法规标准等方式，为低空技术的落地和创新提供强大支撑，同时低空技术也赋能政府治理现代化。


Parallel

 Session E

低空文旅产业发展趋势

专题报告会


紧扣国家文旅产业升级与低空经济发展战略需求，规范有序发展低空旅游，推动地方结合实际建立低空旅游“行业+地方”联合监管机制，在确保安全的前提下，支持具备资质和条件的企业提供低空旅游项目，结合各地旅游资源地域性、季节性特点开发特色低空旅游线路和产品，探讨空中游览市场管理，引导低空旅游健康有序发展。
