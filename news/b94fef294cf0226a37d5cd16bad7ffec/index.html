智造新纪元 | 2026走进东风奕派-先进制造创新技术交流日

ATC新能源三电 2026-01-08




活动价值


-走进东风奕派-



邀请领导：奕派制造体系部门专业领导


与会对象：奕派ME及体系内零部件企业制造体系人员



重点需求方向



走进东风奕派先进制造创新技术交流日-需求清单



01


智能制造与数字化：聚焦工厂级的数字化平台、数据流与智能决策系统

1. 制造数据中台与质量分析：集成跨工序（冲、焊、涂、总）的检测数据，实现SPC自动生成、趋势预警与自学习修正。

2. 工艺生命周期管理：集成管理特殊特性、PFMEA、控制计划等核心工艺文件，并与同步工程（SE）流程结合，实现经验复用。
3. 数字孪生：基于产线及设备三维模型，实现生产仿真、工艺验证与在线监控。
4. “一车一档”数据体系：构建覆盖焊装、涂装、总装等全过程的车身唯一身份数据档案。

5. 先进制造技术规划体系：建设前瞻性技术洞察、评估与导入的标准化工作体系。
6. 开箱工艺：并行装配与空间效率的技术应用

02


人工智能与机器视觉应用：聚焦AI算法及视觉技术在制造各环节的落地场景

1. 平台化AI视觉检测系统：可应用于冲压件、焊点、漆面、总装零部件的多场景外观缺陷检测。

2. AI智能大模型与自主开发：寻求基于AI大模型的视觉算法自主开发能力及相关培训服务。
3. AI驱动工艺优化：在冲压、焊装等工艺中，应用AI进行参数优化、质量预测与工艺设计。
4. AI装备智能决策：赋予装备基于视觉与环境感知的自主决策与调整能力。
5. 柔性检测机器人：集成多传感器的移动机器人，可灵活切换工具，执行高精度在线测量与检测任务。

03


先进连接与焊接技术：聚焦车身连接工艺的创新与质量保障

1. 激光焊接技术：侧围流水槽激光焊、激光飞行焊的工艺控制要点（型面、夹具、设备）。

2. 钢铝混合车身产线设计：满足60JPH节拍的高效混合材料连接产线规划。
3. 连接质量无损检测：焊点等连接部位的先进无损检测技术（如超声波相控阵）。
4. “一站式”高精度总成工作站：集成多种工艺，实现高精度白车身分总成的制造。

04


机器人自动化与柔性装配：聚焦新型机器人应用与高柔性自动化解决方案

1. 人形机器人/具身智能应用：在装配、搬运等场景的工业应用案例与解决方案。

2. 高柔性机器人装配单元：适用于多车型混线的自适应装配机器人及末端执行器技术。
3. 智能输送与定位：支持多车型柔性生产的AGV系统、高精度自适应输送平台及大总成输送技术。
4. 自动化装配工艺：前悬架自动化装配、电池包自动拧紧等关键总成装配技术。
5. 无序上件技术：基于3D视觉的散乱零件自动识别、抓取与上件技术。

05


先进冲压与材料工艺：聚焦冲压效率、质量与新材料的应用

1. 高速冲压与模具技术：全伺服高速冲压线集成技术、智能化模具及先进模具技术应用。

2. 冲压工艺仿真与AI设计：国产钣金成型软件应用、基于AI/自动化的冲压工艺与模具设计。
3. 新材料冲压应用：铝合金覆盖件冲压工艺、新型自润滑板料的性能与应用研究。

4. 冲压在线质量检测：冲压件生产过程中的实时视觉或光学检测技术。

06


智能涂装与表面工艺：聚焦涂装质量、效率与环保

1. 高外观油漆材料：提升漆面视觉品质的新材料。

2. 漆面智能修整：油漆外观缺陷的自动识别与机器人打磨技术。
3. 环保与资源回收：涂装车间油性废溶剂的再生处理技术等。

07


总装与电装测试技术：聚焦整车最终装配、检测与下线流程

1. 自适应技术：具备Z向自适应调整功能的机器人技术。

2. 关键工艺在线检测：总装关键工位的在线监控与管理系统。
3. 整车功能自动化测试：电器功能耐久测试、智能驾驶功能的自动化检测方案。
4. 整车自动入库：整车下线后至仓储区域的自动化运输与调度解决方案。

08


设备智能运维与物流：聚焦设备可靠性、能耗优化与物流效率

1. 预测性维护：基于数据分析的重点车间关键设备健康管理及故障预测。

2. 设备能耗智能监控：对水、电、气等能耗的实时监测、分析与预测模型。
3. 物流仿真与优化：工厂物流仿真模拟技术，以及AGV集群的绿色节能路径规划算法。
4. 车间环境控制：焊装车间整体除尘及涂装车间能耗控制系统设计方案与应用
