今日新闻(2025.11.16)

登瀛议事厅 2025-11-16

（1）、全国时政民生新闻

一、时政要闻

1. 第三轮第五批中央生态环境保护督察全面启动，将对北京、天津、河北3省（市）及5家中央企业开展1个月例行督察，同时对8省（市）开展大运河生态环境保护专项督察。

2. 国台办回应美国对台军售，敦促美方停止纵容“台独”分裂势力，慎处台湾问题。

3. 工信部明确2025年重点工作，包括实施十大行业稳增长方案、推进设备更新和消费品以旧换新、培育先进制造业社群等。

4. 2025中欧地理标志合作国际会议在浙江青田召开，聚焦地标产品保护、推广与中欧合作新模式探讨。

二、民生热点

1. 中央气象台发布寒潮蓝色预警，中东部地区自北向南将大幅降温，局地降温超14℃，需注意防寒保暖。

2. 网信部门从严整治AI仿冒公众人物直播营销乱象，已清理违规信息8700余条，处置相关账号1.1万余个。

3. 多家航司发布通知，12月31日前出行的涉日航线客票可免费退改。

4. 广东“粤车南下”政策正式实施，符合条件的广东私家车可经港珠澳大桥驶入香港自动化停车场。

5. 广东12月1日起禁止销售不符合新国标3C认证的电动自行车，保障出行安全。

6. 市场监管总局通报食品安全整治成效，截至9月查处违法案件超19万件，中小学食堂“互联网+明厨亮灶”覆盖率达98.5%。

7. 江苏上调汽车焕新补贴标准，最高补贴1万元，促进消费升级。

8. 上海要求学校落实校领导每日陪餐制，公开安排并与学生同餐同食，守护校园食品安全。

三、社会与文体

1. 第38届金鸡奖颁奖，易烊千玺成最年轻影帝，宋佳获影后，《好东西》摘最佳故事片奖。

2. 史上最长双11落幕，天猫剔除退款后成交额创新高，实现四年来全周期最好增长。

3. 神舟二十号乘组带回的近100斤太空实验样品交付科学家，开展后续研究￼￼￼。

4. 全国冬小麦播种过八成，粮食主产区抢抓农时推进秋冬种。


（2）、全国经济新闻

一、宏观政策与调控

1. 央行加码中期流动性投放，11月17日将开展8000亿元6个月期买断式逆回购操作，当月合计加量续作5000亿元，为连续6个月注入中期流动性，助力稳定资金面与政府债券发行￼。

2. 市场监管总局公布《互联网平台反垄断合规指引（征求意见稿）》，列举算法共谋、“二选一”等8类垄断风险示例，意见征集至11月29日，进一步规范平台经济竞争秩序。

3. 国家首次系统部署场景培育开放，在数字经济、清洁能源等领域推出项目清单，加速新兴技术产业化，助力产业链上下游企业发展。

二、经济数据与运行

1. 10月国民经济延续稳中有进态势：规模以上工业增加值同比增长4.9%，社会消费品零售总额达46291亿元、同比增长2.9%，城镇调查失业率连续两月下降至5.1%。

2. 1-10月制造业投资同比增长2.7%，占全部投资比重升至25.6%，较去年同期提高1.1个百分点，投资结构持续优化。

3. 三季度末银行业普惠型小微企业贷款余额36.5万亿元、同比增长12.1%，普惠型涉农贷款余额14.1万亿元、较年初增加1.2万亿元，金融支持实体经济力度加大。

4. 1-10月全国铁路完成固定资产投资6715亿元，同比增长5.7%，基建投资稳步推进。

三、资本市场与行业动态

1. A股呈现“哑铃策略”配置特征：高股息价值股（中证红利指数年内涨超15%）与高成长科技股（科创50指数年内涨近10%）受资金青睐，半导体、AI算力等细分领域龙头涨幅超50%。

2. 科创板、科创成长层首批新公司上市，资本市场支持创新力度持续加码，助力科技企业融资发展。

3. 我国陆相页岩油年产量突破150万吨，能源供给能力进一步提升；北京6G实验室发布十大技术进展，含256TR U6G基站原型机等核心成果。

4. 三星上调存储芯片价格30%-60%，国内小米、OPPO等厂商暂缓采购，行业供需格局出现调整。

四、消费与区域经济

1. “粤车南下”政策11月15日正式实施，符合条件的广东私家车可经港珠澳大桥驶入香港自动化停车场，促进粤港澳大湾区交通与消费联动。

2. 消费品以旧换新成效显现，10月相关商品销售保持较快增长，叠加假日经济带动，市场消费持续扩大。

3. 全国农资保供线上平台正式开市，助力稳定农业生产资料供应，保障秋冬种顺利推进。


（3）、盐城时政民生新闻

一、时政要闻

1. 盐城海事局发布航行警告：11月17日至19日每日0时至24时，黄海中部部分海域将进行实弹射击，期间禁止任何船舶驶入，相关区域已明确坐标范围。

2. 亭湖区专项督查组进驻并公布举报方式：亭湖区纪委监委、检察院、审计局、财政局4个专项督查组已进驻工作，公布统一监督电话0515-88399000和电子邮箱tqjjwxcb@163.com，明确工作期间不接待来访、不办理个人诉求等事项。

3. 第十届盐城国际工博会即将举办：2025中国盐城第十届国际工业博览会暨机床展览会定于11月18日至20日在盐城国际会展中心举行，将汇聚工业装备领域知名企业，展品涵盖数控机床、工业机器人等多个板块。

二、民生热点

1. 历史文化名城保护办法12月施行：《盐城市历史文化名城保护办法》将于12月1日正式实施，旨在守护“瓢城”历史记忆，加强本地历史文化资源的保护与传承。

2. 沉积十年消防问题启动整改：盐城12345平台督办解决沉积十年的消防隐患问题，针对整改“空转”现象推进专项整治，保障居民居住安全。

3. 河道保洁错位问题积极整改：针对河道保洁责任划分不清、作业错位的问题，相关部门已启动整改措施，确保河道整洁，改善水环境质量。

4. 深秋特色景观引游客观赏：盐城市湘江路3000棵银杏形成金色长廊，千亩菊海绽放深秋盛景，成为市民休闲打卡的热门地点，丰富本地文旅体验。

三、经济与发展

1. 1556家企业入选省级科技型中小企业：盐城市1556家企业成功入选江苏省2025年第三批入库科技型中小企业，助力本地科技创新与产业升级。

2. 盐都推进高质量发展提速：盐都区聚焦产业升级，强化发展动能，通过培育新质生产力、优化营商环境等举措，点燃高质量发展“强引擎”。


（4）、盐城经济新闻

一、重大项目与产业升级

1. 盐城经开区四大硬核项目落地：奇瑞汽车产业园、一汽超级工厂、珩创三期、阿里云算力中心四大项目正式官宣落地，形成“制造+算力+新材料”的产业组合，预计年绿电需求达8.5亿千瓦时，契合国家绿色工业园区定位，为新能源、智能制造、数字经济赛道注入强劲动能。

2. 盐南高新区20亿新项目启动：南海未来城解放南路东、长海路南地块项目开工，总投资20亿元，规划建筑面积16.08万平方米，含住宅、商业及配套设施，总户数410户，建设期3年，将进一步完善区域配套功能。

二、经济运行与发展质效

1. 前三季度GDP增长5.6%超全省全国水平：盐城市前三季度实现GDP 5864.1亿元，同比增长5.6%，其中第一产业增长4.2%、第二产业增长5.1%、第三产业增长6.3%。规上工业增加值增长7.5%，汽车制造业、动力电池产业产值分别增长26.7%和31.3%，消费升级类商品销售增速超20%。

2. 跻身江苏经济“中坚力量”：在全省13个设区市经济梯队中，盐城以5864.1亿元GDP总量成为支撑全省经济的“中坚力量”，与徐州等城市协同发力，筑牢江苏经济“基本盘”。

三、招商合作与政策支持

1. 市长带队赴沪深化合作：市长严汉平参观进博会并拜访SK On、蒂升电梯、青山集团等重点企业，与申能集团签约深化新能源、绿色金融合作，同时对接上海科技大学推进“政产学研用”合作，承接进博会溢出效应拓展发展空间。

2. 园区推行总部经济招商优惠：盐城部分园区实施注册式招商政策，企业无需实体搬迁即可享受增值税、企业所得税等70%-80%的财政返还，按月兑现，园区提供免费注册地址和一站式代办服务，降低企业经营成本。

四、企业培育与市场动态

1. 聚焦“四上”企业培育提质：市长严汉平调研盐南高新区“四上”企业，鼓励金驿通光电、呈像应急装备等企业强化技术创新，推动物联网、智能制造、氢能等领域企业迭代升级，健全梯度培育机制，确保企业应报尽报、应统尽统。

2. 半导体企业因原材料涨价调价：盐城矽润半导体有限公司宣布自11月1日起部分产品涨价10%，原因是铜、银等原材料价格上涨导致成本增加，10月30日前订单按原报价执行，后续按新价格清单结算。
