匠心职通车 | 智能制造学院2026年度就业岗位专项推送(第5期)

智领匠心 2026-02-03

山东南海气囊工程有限公司专注港口与航运系泊靠泊装备的研发、生产、销售及技术服务，产品涵盖港口船舶工程气囊、护舷、橡胶浮体等多类产品及深海探测智能设备。

公司秉持“专注研发、创新浮力技术、探索海洋科技、坚持高端制造”战略，设专业研发中心，提供系统化定制化解决方案。业务遍及7大洲95个国家和地区，业内规模稳居国内第一，“南海制造”享誉全球。

依托良好发展势头，公司以山东研星海洋科技股份有限公司为总公司，下设多家子公司，推出“引进来”“走出去”“大破局”“产业化”的“四步走”战略规划，开启多元化、创新发展新时期，以“争做全球第一”为目标，助力中国海洋强国建设。

职位名称：研发部技术人员（机械设计专业）

岗位职责：

1. 负责机械设备的研发、设计及优化工作；

2. 参与新产品的研发、试制及生产过程的技术支持；

3. 对现有设备进行技术改进，提高设备性能及生产效率；

4. 协助团队完成其他技术相关工作。

任职资格：

1. 机械设计或相关专业，本科及以上学历；

2. 熟悉常用的设计软件如CAD、SolidWorks等，能独立完成设备设计工作；

3. 具备较强的数据分析能力，能进行设备的性能评估及优化设计；

4. 具有独立解决技术难题的能力，较强的团队合作及沟通能力；

5. 对机械设备设计、制造有浓厚兴趣，有相关项目经验者优先。

福利待遇：

我们提供具有竞争力的薪资待遇、五险、年终奖以及完善的晋升机制。

期待您能以丰富的专业知识和实践经验，为我们的研发团队带来新的创意和价值。

招聘人数：2人

职位名称：机械工程师助理

岗位职责：

1. 协助机械工程师完成设备的设计、优化及维护工作；

2. 负责相关机械设计图纸的绘制与校对，保证图纸准确性；

3. 对生产设备进行调试与故障排除，保障设备正常运转；

4. 收集、整理机械设备的运行数据，分析设备性能并进行改进。

任职资格：

1. 机械设计、制造相关专业，本科及以上学历；

2. 能够熟练使用CAD、SolidWorks等设计软件，绘制精确的机械图纸；

3. 具备基本的机械零件加工、装配及调试能力，能够独立完成简单设备的维修工作；

4. 能够收集并分析设备运行数据，提出改进方案；

5. 具有良好的团队协作和沟通能力，能够与团队成员有效配合。

福利待遇：

我们提供具有竞争力的薪资待遇、五险、年终奖以及完善的晋升机制。

期待您能以丰富的专业知识和实践经验，为我们的研发团队带来新的创意和价值。

招聘人数：1人

职位名称：PE滚塑材料工艺工程师

岗位职责：

1. 制定PE滚塑工艺规程，优化工艺以提升壁厚均匀度与良品率，降低生产损耗；

2. 对接生产现场提供技术指导，解决原料熔融不充分等异常；

3. 编制归档工艺文件与模具图纸，参与模具选型、验收及维护，确保工艺适配；

4. 协同研发提工艺优化建议，联合生产开展培训，收集反馈推动工艺与模具改进。

任职要求：

1. 熟悉PE材料特性与滚塑成型原理、参数调控，能优化工艺、解决壁厚不均等问题；

2. 了解滚塑设备结构，可操作PLC温控系统等设备，会用CAD、SolidWorks等软件设计模具与图纸，精通Office编制技术文档；

3. 能独立完成滚塑模具下料排版、图纸输出，可设计工装夹具，优化生产辅助工具；

4. 本科及以上学历，材料成型与控制工程、高分子材料、化工工艺、模具设计等相关专业优先。

福利待遇：

我们提供具有竞争力的薪资待遇、五险、年终奖以及完善的晋升机制。

期待您能以丰富的专业知识和实践经验，为我们的研发团队带来新的创意和价值。

招聘人数：3人

山东工程职业技术大学
