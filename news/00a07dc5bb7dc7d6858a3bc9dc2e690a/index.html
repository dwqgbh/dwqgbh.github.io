出海展会信息:国内7-12月线下展会

出海展会信息 2026-06-22

一、7月展会

1. 2026全球跨境电商交易博览会（GCBE 2026）

  • 时间：7月9—11日

  • 地点：杭州大会展中心

  • 行业：全品类跨境选品、跨境平台、海外仓、国际物流、支付合规、独立站、数字营销、外贸工厂OEM/ODM

  • 简介：浙江省商务厅、杭州市人民政府联合支持，以“在杭州·看见跨境未来”为主题，展览面积达10万平方米，汇聚2000余家展商。覆盖长三角服装、家居、3C、美妆产业带，汇集亚马逊、Temu、SHEIN、阿里国际站、Shopee、Lazada等全平台。同期举办金砖国家跨境电商峰会、跨境电商多维生态主题峰会、全国跨境电商产业带大会等系列论坛。

2. 第119届中国日用百货商品交易会（跨境电商生态专区）

  • 时间：7月23—25日

  • 地点：上海

  • 行业：日用百货、家居用品、厨房用品、清洁用品、跨境电商选品

  • 简介：中国日用百货行业历史悠久、影响力广泛的展会，设有跨境电商生态专区，是日用百货品类跨境电商选品的重要平台。

二、8月展会

1. 2026中国（广州）跨境电商交易会（秋季）

  • 时间：8月15—17日

  • 地点：广州

  • 行业：跨境电商全产业链、源头工厂、跨境平台、物流仓储、支付合规

  • 简介：由广交会承办方中国对外贸易中心集团有限公司与荟源国际展览有限公司联合主办。以“外贸新动能·数字新未来”为主题，预计吸引约5万名专业采购商到会。首次设立海外仓及物流专区、新兴平台专区、百亿卖家采购对接区。参展企业涵盖供应链、平台、仓储、物流、资本、财税、营销、软件、运营、人才等跨境电商全生态链。

2. VIV Select China 2026亚洲国际集约化畜牧展览会（上海）

  • 时间：8月19—21日

  • 地点：上海世博展览馆

  • 行业：畜牧养殖、饲料加工、动物营养、养殖设备、畜牧供应链

  • 简介：阔别多年VIV主办方将展会搬回中国一线城市上海，以“链接全球 共筑一站式畜牧商贸平台”为主题，将充分发挥VIV全球资源优势以及强大影响力。

3. 第34届广州博览会

  • 时间：8月21—23日

  • 地点：广州琶洲·中国进出口商品交易会展馆

  • 行业：综合贸易、大科技、消费品、工业品、农产品

  • 简介：由广州市人民政府主办，以“智创未来、乐享生活”为主题，规划总展览面积约22万平方米，预设22个展馆。覆盖大科技、消费品、工业品等全品类，是华南地区重要的综合性国际贸易展会。

4. 2026中国国际纺织纱线（秋冬）展览会（yarnexpo）

  • 时间：8月25—27日

  • 地点：上海国家会展中心

  • 行业：纺织纱线、纤维、面料、针织、服装辅料

  • 简介：全球领先纤维纱线商贸平台，展览面积达27000平方米，全球展商数量逾500家。将与中国国际纺织面料及辅料（秋冬）博览会、中国国际服装服饰博览会（秋季）、中国国际针织（秋冬）博览会同期同地举办。

5. 2026中国（郑州）跨境电商交易会（CCEC）

  • 时间：8月28—30日

  • 地点：郑州国际会展中心

  • 行业：保税跨境电商、进出口货源、航空物流、海外仓、母婴、家居、3C、美妆、中亚/欧洲外贸供应链

  • 简介：由中国国际电子商务中心和中国国际贸易学会跨境电商与物流专业委员会联合主办。以“买全球 卖全球”为主题，规划展览面积3万平方米，1200个标准展位。依托郑州航空港综试区，重点覆盖中欧班列沿线采购商，主打备货、保税仓储、跨境进口分销一体化对接。

三、9月展会

1. 2026深圳全球跨境电商展（CHWE秋季展）

  • 时间：9月3—6日

  • 地点：深圳福田会展中心

  • 行业：跨境爆款选品、源头外贸工厂、平台招商、跨境物流、海外仓、财税合规、DTC独立站、东南亚/欧美/中东市场专区

  • 简介：出海网主办的跨境电商行业展会，华南老牌跨境B2B大展，8万平方米展区，2000家参展企业。聚焦下半年海外旺季备货，开设Temu、TikTok Shop、美客多、沃尔玛专场招商对接会。

2. 2026中国建博会（上海）

  • 时间：9月5—8日

  • 地点：上海虹桥·国家会展中心

  • 行业：建筑材料、家居装饰、建筑五金、智能家居

  • 简介：中国建博会系列展在上海的重要场次，覆盖建筑建材全产业链，是长三角地区建筑装饰行业的重要采购平台。

3. 2026中国国际服务贸易交易会（服贸会）

  • 时间：9月9—13日

  • 地点：北京首钢园区

  • 行业：服务贸易、数字贸易、跨境电商、科技出海、金融科技

  • 简介：全球首个国家级、国际性、综合型服务贸易交易会。本届首次设置“出海专区”、中小企业展览专区及“出海服务推介路演区”。已确认挪威、澳大利亚等40余个国家和国际组织将以国家或总部名义设展办会，沃尔玛、西门子、礼来、强生等250余家企业机构参展。

4. 第七届中国（福州）跨境电商交易会（秋季）

  • 时间：9月12—14日

  • 地点：福州

  • 行业：鞋服箱包、家居日用、户外用品、跨境服务商、东南亚跨境、RCEP外贸供应链、跨境直播

  • 简介：东南沿海核心跨交会，10万平方米规模，3000家展商，联动福建泉州、莆田、石狮鞋服产业带。面向东南亚、日韩采购商，配套跨境品牌出海孵化论坛。

5. 2026中国（重庆）跨境电商交易会

  • 时间：9月15—17日

  • 地点：重庆

  • 行业：西部制造货源、汽摩配、家居五金、农产品出海、中欧班列物流、东南亚跨境、跨境保税

  • 简介：西南地区唯一大型跨境专业展，2.5万平方米，800家展商。辐射成渝、云贵川外贸工厂，主打中亚、东南亚陆路出海渠道对接。

6. 2026深圳跨境电商展览会（CCBEC 2026）

  • 时间：9月16—18日

  • 地点：深圳国际会展中心（宝安）

  • 行业：全品类跨境供应链、3C数码、美妆个护、家居户外、母婴玩具、跨境包装、国际物流、海外仓、合规服务

  • 简介：大湾区规模最大跨境展，展览面积达12万平方米，3000家展商。珠三角电子、家居、服装头部工厂集中参展，覆盖全球全主流跨境平台。已确认参展企业包括亚马逊全球开店、TikTok Shop、eBay、美客多、希音、Shopee、沃尔玛等。设立“跨境电商物流专区”、“POD（按需打印）专区”、“消费类安防爆品专区”三大特色专区。

7. 第五届全球数字贸易博览会（数贸会）

  • 时间：9月23—27日

  • 地点：杭州大会展中心

  • 行业：数字贸易、人工智能、丝路电商、数智出行、数智文娱、数智医疗、数智空间

  • 简介：由商务部和浙江省人民政府联合主办。以“数字贸易遇见算力未来”为年度主题，展会面积达17万平方米。设置“1+6+1”展览架构：1个精华主题展区，丝路电商、人工智能、数智出行、数智文娱、数智医疗、数智空间6个特色产业展区，以及1个数贸创新融合展区。首次设立Token出海专区。

8. 2026东盟跨境电商博览会

  • 时间：9月27—29日

  • 地点：南宁

  • 行业：东盟跨境贸易、农产品出海、轻工日用品、东南亚本土平台、边境跨境物流、RCEP进出口、跨境直播电商

  • 简介：面向越南、泰国、马来西亚、印尼等东盟国家专属外贸展会，依托西部陆海新通道，边境贸易、跨境小包、本土仓配服务商集中参展。

四、10月展会

1. 第140届中国进出口商品交易会（广交会·第一期）

  • 时间：10月15—19日

  • 地点：广州琶洲·广交会展馆

  • 行业：电子家电、照明、五金工具、新能源设备、车辆配件、工业制造、跨境B2B大宗外贸

  • 简介：中国规模最大、历史最悠久的综合性国际贸易盛会，全球210国海外采购商到场。特设跨境电商供应商专区，传统工厂对接海外品牌、跨境大卖、线下批发商。

2. 2026中国品牌出海博览会

  • 时间：10月20—22日

  • 地点：海南国际会议展览中心

  • 行业：品牌出海、跨境电商、国际采购、品牌展示、产业招商

  • 简介：国内首个聚焦品牌出海的高规格大型展会。总规划展览面积达9万平方米，设置七大主题展馆，预计汇聚超1000家优质展商，吸引超5000家专业采购商及超10万人次专业观众。格力电器、居然智家、汇天等多家企业已完成签约。

3. GGE跨境出海博览会暨第12届OEM/ODM外贸源头工厂展

  • 时间：10月20—23日

  • 地点：深圳国际会展中心（宝安）

  • 行业：跨境定制代工、家居、3C、礼品、户外、美妆、箱包、跨境供应链解决方案、海外品牌代工对接

  • 简介：年末收官型外贸选品展，聚焦定制化出海，大量中小型外贸工厂、自有品牌企业参展，对接欧美独立站卖家、线下连锁品牌采购商。

4. 第140届广交会·第二期

  • 时间：10月23—27日

  • 地点：广州琶洲·广交会展馆

  • 行业：家居用品、礼品装饰、建材家具、餐厨、纺织家居、轻工消费品、跨境零售货源

  • 简介：家居、礼品类跨境核心采购期，海外商超、跨境精品卖家集中观展，适合家居、礼品、家纺外贸工厂拓展全球渠道。

5. 第十八届深圳国际跨境电商交易博览会（ICBE 2026）

  • 时间：10月28—30日

  • 地点：深圳福田会展中心

  • 行业：跨境电商全产业链、品牌工厂、平台服务、物流支付

  • 简介：由扩展集团主办，是深圳地区跨境电商行业的重要展会之一。

6. 2026广州国际跨境电商进出口贸易展览会

  • 时间：10月30日—11月1日

  • 地点：广州保利世贸博览馆

  • 行业：跨境电商全品类、进出口贸易、平台服务、供应链

  • 简介：立足粤港澳大湾区全球供应链核心枢纽，创新采用“会+展”双轮驱动的办展模式。特别策划“创新形式精品采买大会”，致力于为中外企业搭建集国际贸易、商机分享、资源对接于一体的高能级新平台。

7. 第140届广交会·第三期

  • 时间：10月31日—11月4日

  • 地点：广州琶洲·广交会展馆

  • 行业：母婴玩具、服装纺织、文具、健康休闲、特色农产品、日化、时尚配饰、快时尚跨境货源

  • 简介：服饰、母婴、日用快消跨境核心采购专场，覆盖东南亚、欧美快时尚品牌、TikTok、SHEIN等服饰类平台采购团队。

1. E-Pack Tech上海国际跨境电商包装技术展

  • 时间：11月3—6日

  • 地点：上海新国际博览中心

  • 行业：跨境专用包装、可降解物流包装、印刷标签、缓冲包装、电商包装设备、跨境冷链包装

  • 简介：跨境供应链细分专业展，服务食品、美妆、家居跨境商家包装履约需求，汇聚全球包装材料、自动化包装设备厂商。

2. 2026深圳国际跨境电商交易博览会（ICBE 2026 秋季）

  • 时间：11月4—6日

  • 地点：深圳福田会展中心

  • 行业：年终跨境备货全品类、平台年终招商、海外仓、跨境财税、独立站建站、欧美/中东/拉美新兴市场专区

  • 简介：华南年末压轴跨境专业展，聚焦黑五、圣诞海外旺季备货，全球跨境平台年终招商专场密集举办，供应链工厂集中对接大卖批量订单。特设四大主题展区：跨境电商进出口品牌和生产厂家展区、跨境平台展区、跨境服务商展区、综试区/园区/政府/集聚地展区。邀请亚马逊、新蛋、沃尔玛、阿里巴巴国际站、速卖通、eBay、Wish、Shopee、Lazada、Coupang等平台齐聚现场。

3. 2026中国（浙江）跨境电商进出口交易会

  • 时间：11月25—27日

  • 地点：义乌国际博览中心

  • 行业：跨境电商进出口、日用百货、五金工具、机电设备、电子电器、文化用品

4. 高交会配套·跨境电商与数字出海专区

  • 时间：11月26—28日

  • 地点：深圳国际会展中心（宝安）

  • 简介：国家级高新技术展会配套跨境板块，主打3C、智能家电、智能家居高新产品出海，对接海外科技品牌、线上数码大卖。

5. 2026第七届杭州国际跨境电商博览会

  • 时间：11月27—29日

  • 地点：杭州国际博览中心

  • 行业：跨境电商全品类、源头工厂、平台服务、物流支付、合规服务

  • 简介：已成功举办六届，是国内极具影响力的跨境电商全产业链盛会。预计汇聚1600家源头工厂、80000人次专业观众。展品涵盖消费精品（日用百货、家居用品、母婴玩具、美妆个护、宠物用品、户外用品、礼品文创、服饰服装、皮具箱包等）和硬核智造（3C电子、家用电器、汽摩配件、五金建材、医疗器械等）以及配套生态（跨境电商平台、代运营、物流、营销、支付、海外仓等全产业链服务商）。

六、12月展会

1. 2026中国义乌外贸工厂展览会暨跨境电商采购交易会

  • 时间：12月6—8日

2. 2026中国国际电商产业（越南）博览会
