2026年3月8日钢铁产业链信息情报(第158期)

钢铁生产运营管理 2026-03-08

1、习近平：充分发挥政治建军特有优势 凝心聚力推动国防和军队现代化行稳致远。中共中央总书记、国家主席、中央军委主席习近平3月7日下午在出席十四届全国人大四次会议解放军和武警部队代表团全体会议时强调，“十五五”时期要如期实现建军一百年奋斗目标、高质量推进国防和军队现代化，必须坚持好、运用好、发展好政治建军这个重要法宝，毫不动摇坚持和加强党对军队绝对领导，充分发挥政治建军特有优势，凝心聚力推动国防和军队现代化行稳致远。在认真听取代表发言后，习近平发表重要讲话。他指出，党的十八大以来，党中央领导人民军队以前所未有的决心和力度深化政治整训、推进政治建军，取得重大成效。军队是拿枪杆子的，军中绝不能有对党怀有二心之人，绝不能有腐败分子藏身之地，必须坚定不移推进反腐败斗争。“十五五”一开局就要立起从严监管硬规矩，紧盯资金流向、权力运行和质量管控等关键环节，加强重大项目监管，强化军地融合监督，确保在监管前提下搞建设。要推进军费预算管理改革，搞好军费供需动态平衡，强化经费使用全链条管控和绩效评估，把每一分钱都用在刀刃上。习近平强调，完成“十五五”时期国防和军队现代化目标任务，归根到底要靠各级党组织来领导和推进。要全面加强我军党的领导和党的建设，选准配强高层党委班子，增强基层党组织自主抓建能力，把各级党组织建设得更加坚强有力。要坚持党管军事、党管干部、党管行业，提高科学决策能力，在攻坚重大任务、化解突出矛盾、解决发展难题过程中发挥关键作用，把党的领导优势转化为发展优势。习近平指出，要加强革命化专业化人才队伍建设，坚持不懈用党的创新理论铸魂育人，打牢官兵听党话、跟党走的思想根基，确保现代化武器装备掌握在革命化人才队伍手中。要完善促进人才发展的制度和条件，体系推进联合作战指挥、新型作战力量、高层次科技创新、高水平战略管理“四类人才”培养，实现人的能力素质同强军实践协调发展。习近平强调，要大力传承和弘扬我党我军优良传统，推进新时代强军文化建设，加强先进典型培养和激励，让红色基因代代相传，让新风正气更加充盈。要把基层作为赓续优良传统的沃土，积极探索新形势下基层建设规律和有益做法，重视抓好管根本、利长远、增后劲的工作，把部队建设和战斗力基础打得更加牢靠。

2、兰格钢铁智策：需求释放有限 钢市震荡趋弱。2026年第10周（2026.3.2-3.6）兰格钢铁全国绝对价格指数为3467元，较上周下降0.1%，较去年同期下降5.1%；其中，兰格钢铁长材绝对价格指数为3343元，较上周下降0.1%，较去年同期下降3.8%；兰格钢铁型材绝对价格指数为3415元，较上周上升0.3%，较去年同期下降5.9%；兰格钢铁板材绝对价格指数为3520元，较上周下降0.1%，较去年同期下降5.9%；兰格钢铁管材绝对价格指数为3875元，较上周下降0.1%，较去年同期下降5.9%。从兰格期现智策数据来看，黑色商品整体收涨，其中主力焦煤、焦炭和铁矿石均涨超1%，强于螺纹和热卷，后者涨幅较小。螺纹主力05合约收3088，日涨8点，较上周五收盘价涨11点，周结算价3074，较上周涨22点。最新持仓179.8万手，日内减仓3.8万手，较上周五减仓15万手，持仓兴趣持续下降，限制了行情的反弹力度。目前周线第二周反弹回升，日线有效向上突破3080一线，下一步有望继续冲击3100关口，上方或回到3110-3120区间。从兰格钢铁智策数据来看，供给端：由于品种盈亏和部分区域限产的影响，钢厂产能释放力度由强转弱，铁水产量有所减少，但品种产量则长升板降。需求端：由于元宵佳节后市场刚刚开市，市场成交渐渐由停滞期转向恢复期。成本端：由于矿石价格小幅上涨，废钢价格小幅震荡，焦炭价格维持平稳，使得生产成本支撑力度持续减弱。因此兰格钢铁智策预计在经济增长目标明确、宏观政策密集发力、央行保持流动性充裕、供给释放由强转弱、终端需求逐渐恢复、成本支撑持续减弱的影响下，预计下周（2026/3/9-3/13）国内钢市或将震荡趋弱。

3、本周（3月2日—3月6日）钢市期螺震荡，现货徘徊；假期结束，等待变局。接下来，预计终端需求增加，投机需求静默，期货区间整理，现货易涨难降。本周（3月2日—3月6日），西本资讯监测的数据显示，截止3月6日，全国69个主要市场25mm规格三级螺纹钢平均价格为3412元/吨，周环比上涨1元；高线HPB300φ6.5mm平均价格为3574元/吨，周环比上涨1元。本期，全国多数市场建筑钢材价格窄幅整理：其中，北方成交较弱，相对稳定；南方需求平缓，低位震荡。本周西本钢材指数徘徊，成本指数调高，显示长流程钢厂利润空间收缩；本周螺纹钢期货主力合约震荡趋高，对现货有示范作用。期货方面，本周黑色系主力合约区间波动。其中，热卷2605合约周五夜盘收在3244元，较上周五收盘3222元上涨22元；螺纹钢2605合约周五夜盘收在3099元，较上周五收盘3074元上涨25元；从全周走势情况看，原料端矿焦上移，成材端螺卷呼应。本期，全国35个主要市场样本仓库钢材总库存量为1410.64万吨，周环比增加105.61万吨，增幅为8.09%。主要钢材品种中，螺纹钢库存量为640.61万吨，环比增加70.04万吨，增幅为12.28%；线盘库存量为99.58万吨，周环比增加10.05万吨，增幅为11.23%；热轧卷板库存量为384.36万吨，周环比增加24.31万吨，增幅为6.75%；中厚板库存量为136.09万吨，周环比减少3.17万吨，降幅为2.28%；冷卷板库存量为150.03万吨，周环比增加4.41万吨，增幅为3.03%。据西本资讯历史数据，当前库存总量较上年同期（2025年3月6日的1348.37万吨）增加62.27万吨，增幅为4.62%。分品种看，本期五大钢材品种中，中厚板库存环比下降，其它品种库存环比回升。

4、焦炭首轮提降落地，价格仍有下降预期。“两会”期间，部分钢厂接受自主减排通知，进行减产检修，焦炭需求走弱，下游钢厂采购减少，导致焦企端出现累库现象，4日钢厂对于焦炭采购价下调，湿熄降幅50元/吨，干熄降幅55元/吨，并于6日落地执行。（1）从供应端看，本周Mysteel统计独立焦企全样本：产能利用率为73.95%，减0.41%；焦炭日均产量63.94万吨，减0.35万吨；焦炭库存110.30万吨，增2.48万吨；炼焦煤总库存949.45万吨，减49.41万吨；焦煤可用天数11.2天，减0.51天。本周Mysteel统计全国230家独立焦企样本：产能利用率为72.29%，减0.54%；焦炭日均产量50.39万吨，减0.38万吨；焦炭库存63.20万吨，增1.01万吨；炼焦煤总库存796.15万吨，减33.31万吨；焦煤可用天数11.9天，减0.4天。上游降价的煤种在扩大，降幅也在增加，焦企生产成本有所下降，焦炭减产限产意愿并不强烈，供需不平导致焦炭库存迅速累积，多数焦企存在去库需求，因此焦炭价格仍有下降预期。（2）从需求端看，Mysteel调研247家钢厂高炉开工率77.71%，环比上周减少2.51个百分点，同比去年减少1.80个百分点  ；高炉炼铁产能利用率85.32%，环比上周减少2.13个百分点  ，同比去年减少1.22个百分点；钢厂盈利率38.1%，环比上周减少1.73个百分点，同比去年减少15.15个百分点；日均铁水产量  227.59万吨，环比上周减少5.69万吨，同比去年减少2.92万吨。受环保限产影响，本周铁水产量明显下滑，对焦炭需求大幅减弱，下游需求支撑不再，叠加成材端累库严重，成材销售不畅，下游需求不足以支撑焦炭现有价格，为寻求利润钢厂对焦炭价格进行提降从而压缩成本。

5、中国扩大必和必拓铁矿进口限制，全球定价体系迎调整。综合彭博社、路透社、《澳大利亚金融评论报》消息，2026年3月，中国矿产资源集团（CMRG）进一步扩大对必和必拓铁矿石的进口限制，要求贸易商停止采购其新货，此举源于双方2026年长协合同谈判长期陷入僵局。自2025年9月起，CMRG已先后叫停必和必拓金布巴粉、金宝粉的采购，此次限制范围进一步覆盖Mac细粉、纽曼粉/块矿等旗舰产品，且2月起采购其海运货需获许可，相关申请至今未获批复。截至2月26日，中国主要港口金宝矿库存达980万吨，较去年9月底激增457%，库存积压严重。为应对合作僵局，必和必拓已转向东南亚市场，于2025年12月和2026年1月向马、越出口金布巴矿石，同时自2025年四季度起，将30%对华现货交易转为人民币结算。此次限制背后是铁矿石定价权的博弈，CMRG正推动摒弃普氏指数，改用Fastmarkets、Mysteel等定价基准，力拓、FMG已率先调整——力拓试行Fastmarkets指数，FMG采用Mysteel与阿格斯指数均值，而新基准价格多数时段低于普氏指数，或致澳矿商收入缩水。澳洲政府对此高度关注，因铁矿价格每波动10美元，将影响其本财年约5亿澳元税收。

6、根据中国汽车工业协会的预测，2026年我国汽车出口将达到740万辆，同比增长4.3%，相较于2025年增加约30万辆。但有机构预测，今年我国汽车出口有机会突破800万辆整数关口。2025年，我国汽车出口高达709.8万辆，同比增长达到21.1%。虽然增速相较于2023年的57.8%低，但较2024年的19.3%，仍增加1.8个百分点，且明显高于汽车国内销量6.7%的增速，显示出未来我国汽车出口仍然具备较大增量空间。虽然目前外部环境多变，但总体趋势仍在朝着向好方向发展，诸如加拿大批准每年进口4.9万辆中国汽车，法国、德国等主要经济体开始强化与中国商贸关系等契机，都有利于今年中国汽车出口。2025年，新能源汽车出口261.5万辆，同比增长103.7%，占据2025年汽车出口总量的36.8%，占比较2024年的21.9%和2023年的24.5%都有明显的提升。2025年，传统燃油汽车出口448.3万辆，同比下降2%，占据2025年汽车出口总量的63.2%，占比较2024年的78.1%和2023年的75.5%都有明显的回落。应该来讲，中国汽车的核心优势在新能源汽车，新能源汽车未来在中国汽车出口占比的提升态势仍将延续，今年有望达到400万辆；同时，燃油车的出口占比虽然未来有持续下滑空间，但毕竟基数相对较大，至少今年应该还能维持在400万辆的规模。车企进一步向海外市场发力的目标十分清晰。其中，比亚迪2025年海外销量达105.4万辆，2026年目标预计在130-160万辆，同比增长24.3%–50%；奇瑞2025年海外销量达134.4万辆，2026年目标预计在150万–160万辆，出口占比接近50%；上汽2025年海外销量107.1万辆，2026年目标预计在150万辆，增长约40%。同时，长安、长城、吉利、东风、小鹏和零跑2026年的出口目标合计也达到370万辆左右。根据中国汽车工业协会的分析，今年有利于汽车出口的因素，一是车企对于海外市场的深耕，二是新能源、智能化、产品力提升、性价比以及供应链保供优势。

7、2月份比亚迪英国大涨83%：正取代特斯拉成为英国新能源汽车技术的定义者。在中国车企继续扩大英国市场份额的2月份，比亚迪的销量同比大增83.0%至2,154辆，大幅度领先于大盘。比亚迪的这一销量超过了Cupra、雪铁龙等传统的豪强。英国汽车制造商与贸易商协会认为，2月份是传统的销售淡季，但比亚迪却实现了极其罕见的高增长，这反映了英国消费者对新能源车型、特别是插电式混合动力的强劲需求，比亚迪成为推动英国汽车市场电气化转型的核心力量。在比亚迪的注册销量中，2月份PHEV的销量为1,076辆，同比增长134.0%；纯电动车的销量为968辆，同比增长40.9%。行业机构与媒体更关注比亚迪与特斯拉之家的较量。英国《卫报》在分析中说，尽管2月份特斯拉以2,422辆的销量领先，但其销量暴跌了45.0%，与比亚迪的暴涨形成了鲜明的对比。2026年1-2月，比亚迪在英国乘用车市场的销量为6,175辆，同比增长121.3%。特斯拉的累计销量为3,140辆，同比跌幅为40.9%。比亚迪以绝对的优势超过了特斯拉。特斯拉老旧的产品线，在面对比亚迪海豹、Atto3等车型的快速迭代时，显得日益乏力。比亚迪正在取代特斯拉，成为英国新能源汽车技术的定义者。News Automotive的CEO表示，每4位英国购车者就有1位选择电动车，比亚迪通过极具吸引力的定价和先进的技术储备，正快速蚕食传统燃油车和早期电动车品牌的份额。英国的专业汽车媒体在评论中称，比亚迪已经从注重性价比的品牌转型为高科技选项的品牌。按照现在的发展趋势，2026年会是英国消费者真正认可中国汽车品牌的元年。

8、福田汽车近日发布了2026年1-2月产销数据快报，其中福田重卡销量2.4万辆，同比增长47%，延续了自去年以来的强势增长势头。自2025年以来，福田重卡实现了销量的持续增长，市场份额不断攀升，并在多个细分市场“全面开花”。而身为福田汽车的重要战略板块，旗下欧曼重卡更是在行业转型浪潮下走在了市场前列。据悉，2026年1-2月，欧曼重卡在燃气牵引车、载货车及新能源等领域均实现销量同比增长。根据第一商用车网掌握的最新数据，2026年1-2月，福田累计销量2.4万辆，同比增幅高达47%；其中，作为福田重卡的主销品牌，欧曼重卡1-2月累计销量1.5万辆，同比增长40%，保持着强劲增长势头。2月，欧曼重卡还在多个细分市场“全面开花”。其中，欧曼燃气牵引车销量同比增长18%；载货车销量同比增长66%；星辉品系重卡-销量同比增长83%；专用车销量同比大涨548%。而在新能源领域，欧曼银河新能源重卡更是做到了上市即热销，2月销售近千辆。成绩的背后，恰恰展现出欧曼重卡作为国内重卡领域的领军品牌，始终以福田汽车前瞻战略为指引，深度践行“全面国际化、全面新能源化、全面智能化”的转型理念。同时，针对新能源领域，欧曼重卡则积极投身电动化转型浪潮，经过多年发展，其自主“三电”系统无论能耗还是可靠性，都赢得了用户的广泛青睐与赞誉，精准破解不同运输场景行业痛点，全面助力物流运输向绿色化、智能化转型。欧曼重卡销量的增长，更离不开其对产品的布局。截至目前，欧曼重卡打造了丰富的产品矩阵，如银河9、银河7、银河5、银河3等系列车型；欧曼重卡还凭借持续的研发投入和场景化精准匹配，力推星辉品系重卡，凭借在技术路线上的多元化布局，以极致高效为用户运输保驾护航。

9、新增产线与竞争优势：国内第10条阿维迪ESP产线在晋钢开工建设，将重点生产取向及无取向电工钢、深冲钢、热成形钢等先进钢种。2026年3月5日，晋钢控股集团举行了1750 ESP产线项目开工仪式，标志着该项目正式迈入全面建设新阶段。该产线是普锐特冶金技术在国内提供的第10条、全球第14条阿维迪ESP产线。本条产线采用全无头模式运行，可生产宽度1750毫米，厚度0.7–12.7毫米的高品质带钢，在阿维迪ESP经典配置基础上实现重大升级，集成多项创新功能，可满足多元化、高端化产品生产需求。该产线由四个大压下机架与五个精轧机架组成，针对特殊钢种生产进行专项优化设计：采用分体铁芯式电磁制动器(EMBR)、先进防鼓肚连铸机辊列、优化二次冷却系统及粗轧前初级除鳞装置，显著提升带钢表面质量与产品性能。同时，立辊轧机、地下卷取机夹送辊打磨装置、在线浸入式水口(SEN)更换系统、精轧机架在线工作辊更换系统等多项专利技术的应用，大幅提升电工钢生产灵活性、带钢边部质量与生产连续性，为高端钢种稳定量产提供坚实保障。项目将重点生产取向及无取向电工钢、深冲钢、热成形钢等先进钢种，并同步开发高硅电工钢、超低碳钢等高端新材料。凭借优异的延展性、可塑性与成形性能，相关产品广泛应用于汽车制造、家电生产等重点领域。本次项目同步搭载普锐特冶金技术完整工业4.0方案，集成全流程质量控制系统TPQC、资产寿命专家ALEX及智能排程系统Scheduler，实现精准质量评估、设备预测性维护与柔性化生产排产，有效提升生产线运行可靠性、减少停机时间、提高整体生产效率，助力晋钢实现更高水平的精益化、智能化制造。

10、晋钢DX51D+Z-JD高端家电镀锌板凭借优异的加工性、耐蚀性、适配性、环保性，为家电制造提供优质解决方案。DX51D+Z-JD高端家电镀锌板采用连续热镀锌工艺，搭配精准温控与冷却技术，形成均匀致密的防护屏障，提供长久保护，大幅延长构件使用寿命，使家电产品能从容应对潮湿浴室、高温厨房等复杂环境。经过精工打磨，板面平整、光洁无瑕，以细腻板面撑起家电的外观质感。DX51D+Z-JD高端家电镀锌板将屈服强度成功达到300MPa，这一数据远超同类产品平均水平，优化的屈服强度使材料在冲压成型过程中降低回弹率，显著提升加工精度；改善的延展性使复杂曲面成型合格率提升，为家电外观设计释放更多可能。DX51D+Z-JD高端家电镀锌板突破传统板材0.5-1.5毫米的常规规格区间，将厚度覆盖范围扩展至0.3-2.0毫米。不仅实现了从微型电子元件到大型家电结构的全覆盖，更构建起精准适配的解决方案体系。0.3-0.5毫米超薄规格满足精密电路板支架的轻量化需求，1.2-1.5毫米中厚规格适配洗衣机内筒的结构强度要求，1.8-2.0毫米厚板则为冰箱压缩机舱提供可靠支撑。这种阶梯式的厚度分布，便于家电制造商实现全产品线材料的一站式采购。DX51D+Z-JD高端家电镀锌板可延长家电整体生命周期，减少资源消耗与废弃物产生，彰显企业可持续发展担当。

12、冶炼技术：锌在高炉中的行为研究。（萍安钢铁公司）锌在高炉中的行为。锌是一种银白色重金属元素，2 0℃时的比重为7.133 g／c m3。Z n的熔点为419℃，沸点为9 07℃，自然界中常以红锌矿( Z n O ) 和 闪锌矿 ( Z n S )形态存在，有时也呈碳酸盐或硅酸盐形式，在烧结矿和球团矿中主要以铁酸锌形式存在。高炉生产中锌存在于两个循环：第一个循环是高炉内部的小循环；第二个循环是烧结、高炉生产环节间的大循环。①高炉内的富集。含锌矿物进入高炉后生成固态 ZnO，随炉料下降过程中被 C 、CO和F e还原，化学反应式分别为 ：ZnO+ C=Zn ( g ) + CO ( g ) - Q ； Z n O + C O( g ) = Z n ( g ) + C O 2 ( g ) - Q ； Z n O + F e ( L ) = Z n ( g ) + F e O ( L ) 。在高炉下部1000℃以上的高温区 ，Z n O还原出来的Z n 全部被汽化为锌蒸汽分散于煤气中并随之上升，锌蒸汽到达高炉上部低温区时冷凝而再被氧化形成Z n O细 粒 ，一部分随煤气炉顶逸出炉外，一部分附着于下降的炉料时就再次进入高温区重新被还原和汽化，周而复始，形成了锌在高炉内的循环富集现象。② 锌在烧结、高炉系统间的循环。在炉内高温区汽化形成的锌蒸汽，随煤气上升温度降低，冷析成微细颗粒或再被氧化成Z n O ，若粘附于上升的粉尘，由荒煤气带出炉外，进入煤气清洗系统会沉积在管道或除尘灰中。富含锌元素的高炉煤气除尘灰被回收，用于烧结配料，含锌的烧结作为炼铁主原料重新进入高炉，形成了锌在烧结、高炉系统间的循环。

13、冶炼技术：锌对高炉危害研究。（萍安钢铁公司）锌负荷从2.0kg/t上升到3.0kg/t左右，各种危害突出显现，具体如下：①对产量、消耗的影响：当锌的富集加剧，高炉内粘结也严重，炉况顺行及煤气利用变差，锌负荷每上升0.1kg/t，燃料比约上升1.68kg/t。为维持生产，消除结厚，每月左右炸瘤一次，造成无计划休风，对产量影响较大。②影响原燃料的冶金性能。渗入铁矿石和焦炭孔隙中的锌蒸汽沉积氧化成Z n O后，一方面由于体积的膨胀会增加铁矿石和焦炭的热应力，破坏铁矿石和焦炭的热态强度，并导致烧结矿和球团矿的低温还原粉化指数变差，焦炭的反应性增加，反应后强度降低；另一方面锌蒸汽也会堵塞铁矿石和焦炭的孔隙，恶化高炉料柱的透气性 ，给高炉冶炼带来不利影响。休风时观察风口扒出的焦炭，粉碎比较严重。③对高炉操作的影响：（1）炉喉及炉身上部砖衬上的锌瘤破坏炉料的下降和煤气流的上升，破坏炉料和煤气的正常分布，导致炉况失常。（2）锌瘤滑落时又引起风口灌渣及损坏，锌负荷明显上升后，风口损坏大幅增加。（3）上升管和下降管的锌瘤会堵塞煤气通道，导致炉顶压力异常。④ 对高炉热分布的影响ZnO+ C=Zn + CO -Q （1）Zn+CO2= ZnO+ CO+Q （2）在高温区通过式（1）吸收热量，在低温区通过式（2）放出热量，热量就从高温区转移到了低温区，导致渣铁温度降低，渣的黏度升高，从而不利于高炉顺行和脱硫。因渣铁温度易降低，炉缸工作易失常，不能有效开展低硅冶炼。④K 、N a 、Z n 、P b等有害元素的叠加效应。高炉砖衬面对的不仅仅是单纯的某一种有害元素，而是K、Na、Zn、Pb等有害元素的综合作用，并且扫描电镜检测结果显示，这些有害元素对砖衬的侵蚀存在叠加效应。所造成的破坏远远超过任何一种单一元素，这是高炉砖衬侵蚀机理的特点之一。
