研修主题一


解读2025工业新动向

了解市场新需求



2025汉诺威工业展

德国汉诺威工业展是工业4.0 的发源地，是当今规模最大的国际工业展览会，被誉为“世界工业贸易的晴雨表”和“全球工业技术发展的风向标”。涵盖集成自动化和动力传动、数字化工厂、能源、工业零部件、空压及真空技术、研究与技术。


展会亮点：

●集成自动化及动力/传动控制展

 电气传动 / 电气工程 / 机械工程 / 机电动力传输 / 流体动力传动（液压和气压传动） / ……

●工业零部件与分承包技术展

转包和分承包 / 材料：钢、铝和其他金属，陶瓷，聚合物（工程材料） ，复合材料等各种不同的工业用材料及化合物等/ ……

●能源展

能源的供应 / 能源的传输与分配 / 传统能源的开发利用 / 可再生能源 / ……

●数字化工业展

产品开发（PLM/CAD） / 生产和过程计划 / 视觉呈现 / 仿真技术 / 电脑图像模拟 / 机械工程、自动化 / 订单处理和技术化销售 / 工业软件 /  网络解决方案与服务/……

●研究与技术展

基础科学研究 / 应用科学研究和技术 / 技术转让 / 技术数据库 / 商业与科技 / 结合、吸引投资 / ……



研修要点：

◆黄昌夏导师全程讲解汉诺威工业展（2天），了解德国及世界产品的发展和市场的具体需求，调整改进产品的结构，为生产高质量的产品奠定基础；

◆联系全世界技术领域和商业领域，寻求合作方向，扩大贸易订单……



随团导师




黄昌夏 老师


黄昌夏导师，80后先进制造业网红，西门子中国研究院高级总监、智能制造首席顾问，在高科技、先进制造业企业有近20年的从业经验，曾主导西门子公司多项战略的制定与落地，在数字化工厂、工业互联网、人工智能、企业技术战略、智能制造市场发展与生态演进等领域具有独树一帜的见解，曾为航天科工、中国烟草、郑煤机、华为、中信集团、康明斯、三一集团、小罐茶、遨博智能等多家国际国内著知名企业提供过服务，曾经参与撰《中国汽车产业数字化蓝皮书》、《美国制造创新研究院解读》等著作。国之重器系列丛书《智能制造术语解读》核心编委之一，业界人称黄小邪。



研修主题二


《德国工业4.0与创新课程》


亚琛工业大学



亚琛工业大学（RWTH Aachen University）位于德国亚琛市，是德国最大的理工大学之一，也是欧洲最著名的工科大学之一。它被誉为“工程师的摇篮”和“欧洲麻省理工”，在全球多个大学排名中均表现优异。该校是IDEA联盟、德国精英大学联盟和德国理工大学联盟（TU9）的重要成员，与全球多所知名高校和企业建立了紧密的合作关系。同时该校的电气工程等专业实力雄厚，吸引了欧洲机电一体化研究中心等众多知名公司坐落于亚琛。

研修主题三


空中客车：航空工业世界领军企业玩转数字化制造



空中客车


凭客户为中心的理念、商业知识、技术领先地位和制造效率，跻身行业前沿，牢固全球约一半的民用飞机订单。


德国汉堡工厂，是全世界三大飞机制造中心中最大之一，负责组装、涂装空中客车A320全系列客机，A380巨无霸也在此初步组装！


研修要点

◆参观空客A320+A380制造现场，感受大飞机制造技术与管理流程；

◆了解设计、供应链、制造、运输、飞机运营等环节上向环保型企业迈进理念与创新




研修主题四


奔驰：历久弥新的汽车业智能制造践行者



奔驰汽车


历经百年风雨，永葆强大生命力。每年限产五十五万辆，在滚滚利润面前奔驰公司为何不动心？采取少而精战略的奔驰公司，如何将其产品真正打造成“物依稀为贵”？百年质量承诺如何为其品牌添金？


研修要点

◆参观智慧生产流水线--工业4.0中的工艺流程开发；

◆体验奔驰完美的技术水平、过硬的质量标准、推陈出新的创新能力，如何书写新巅峰！




研修主题五


德国克拉斯农机公司——农机中的奢侈品



德国克拉斯CLAAS工厂


德国克拉斯（CLAAS），始建于1913年的家族性农机企业，农机圈子里“高品质”的代名词，欧洲最大、世界第四的跨国农业机械制造巨头。


经过几代人的努力，CLAAS如今拥有先进的厂房和制造设备，一流的制造工艺和独立的零配件供应系统，保证用户随时随地享受高质量的产品和优质、及时的售后服务。


研修要点

◆参观CLAAS工厂，0距离体验"高端制造“，感受严谨管理体系与强大的科研开发体系的超级震撼力！




研修主题六


置身创意无限的未来空间

研究创新设计产品的创新思路


工业设计从业者膜拜的圣地

——红点工业设计博物馆

红点奖被称为工业设计界的“奥斯卡”，是国际公认的全球工业设计顶级奖项之一，与德国“IF奖”、美国“IDEA奖” 并称为世界三大设计奖。

红点大奖展示馆更是众多工业设计从业者膜拜的圣地！馆内超过4000平方米的展出区域有超过1000件的作品进行展出，其中包括汽车交通、医学、厨卫、虚拟科技、电子信息产品等领域，尤其是近几年来（2018-2019）上市并获奖的工业产品。





总结课程：《德国制造给予中国制造的启示》



企业如何转型升级？如何避免败局？这些困扰企业家多年的问题，都会在德国之行后有新的思路。




游历德国七座核心城市+荷兰首都


汉堡，德国第二大城市，也是德国最重要的海港和最大的外贸、金融中心，被誉为通往世界的大门。



不莱梅，优势产业包括汽车汽配、食品加工、航天航空、航运物流、贸易、新能源等行业。戴姆勒奔驰（Daimler Mercedes-Benz）是不来梅最大的企业，员工数超过12500人。



汉诺威，工业制造业高度发达，欧洲最大的旅行社组织TUI的总部也设在这里。汉诺威的旅游景点有：世界上最大的展览中心，汉诺威展览中心。



比勒费尔德，座拥有大型集市和美丽框架建筑的典型贸易城市，充满着令市民们令人称道的艺术嗅觉。


多特蒙特，从前是德国西部的工业中心之一，现在是高科技工业园区，"绿色都市"。市内接近有一半地区是布满水道、林地、耕地和绿色地带和宽敞的公园。



埃森，德国钢铁工业先驱克虏伯的发迹之地。如今，德国几家最大的能源公司鲁尔煤炭公司，RWE电力公司，鲁尔天然气公司等企业的总部就设于此。



亚琛，亚琛 (Aachen)，与比利时和荷兰接壤，是著名三国交界城市以及旅游胜地，并以温泉（欧洲中部最热的温泉）著名，亚琛是是工业与科技中心，成立于1870年的亚琛工业大学即位于这里，其是世界顶尖理工类大学之一，也是德国精英大学。



荷兰（阿姆斯特丹），亚琛 (Aachen)，与比利时和荷兰接壤，是著名三国交界城市以及旅游胜地，并以温泉（欧洲中部最热的温泉）著名，亚琛
