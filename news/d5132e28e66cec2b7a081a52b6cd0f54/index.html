2026年6月机器人行业新品盘点

人形机器人资本冲锋号 2026-06-23


2026年6月国内机器人行业新品密集落地，覆盖人形机器人整机、轮式/工业特种机器人、具身智能大模型、核心零部件、消费级及商用机器人全赛道。

行业呈现C端消费加速渗透、工业场景务实落地、模型与操作系统构建生态壁垒、核心零部件实现量产突破四大核心趋势，标志着国内机器人产业从技术演示阶段全面迈入商业化、生态化、量产化发展新阶段。


01  宇树科技 × 英伟达「H2 Plus」

>> Isaac GR00T 参考人形机器人 <<

1. 宇树科技 × 英伟达「H2 Plus」 Isaac GR00T 参考人形机器人

时间：6月1日宇树官网正式发布；英伟达CEO黄仁勋同步于GTC Taipei将其宣示为Isaac GR00T参考人形机器人；预计2026年底由宇树正式推向市场。

核心参数：

  • 基于Unitree H2本体：身高约182cm，带电池重约70kg，续航约3小时

  • 本体关节电机数31个自由度（单腿6/单臂7/腰部3/头部2）

  • 搭配双Sharpa Wave触觉五指灵巧手，单手22主动自由度，全身合计75自由度；单指尖触觉像素超1000个，压力灵敏度达0.005N，负载可达30kg

  • 手臂最大关节峰值扭矩120N·m，腿部最大360N·m；手臂额定负载约7kg、峰值约15kg

  • 搭载NVIDIA Jetson AGX Thor T5000机载计算模块（Blackwell架构，AI算力2070 FP4 TFLOPS，14核Arm CPU，128GB统一内存）

  • 感知：头部5MP大视场角仿人双目相机（140°×102°）、可选手腕相机、IMU；支持以太网、Wi-Fi 6、蓝牙5.2

特点：英伟达把本体（宇树H2）+灵巧手（Sharpa Wave）+算力（Jetson Thor）+软件栈（Isaac GR00T/Isaac Sim/Isaac Lab/Isaac ROS）集成到单一参考设计中，定位面向高等教育机构和大学研究人员。


02  优必选旗下品牌「优世界U1」

>> 全尺寸超仿生人形机器人 <<

时间：6月2日登陆京东开启预售（定金3000元/台，可退）；预计6月30日正式发布。

核心参数：

  • 男款：身高183cm/重42kg；女款：身高168cm/重35.2kg

  • 整机搭载88个高精度关节自由度

  • 搭载“养成系”情感大模型，支持本地加密存储记忆

  • 仅限成年人购买使用

预售数据：预售10天预订量近4000台；上线14天预订超4000台；定金总额逾千万元。京东页面显示累计预订约4500+台量级。

特点：全球首款面向C端的全尺寸超仿生情感陪伴机器人，主打“陪伴+外观定制+记忆养成”心智定位。


03  智元机器人「远征A3」

>> 全球首个全尺寸双足人形机器人自主乒乓球对抗 <<

时间：6月15日官方宣布。

核心要点：

  • 在无遥控、无预设脚本、无人工干预条件下，远征A3自主完成视觉感知→轨迹预测→全身运动规划→精准击球的全闭环控制

  • 技术由智元×北京大学仉尚航团队完成，依托人形机器人乒乓运动控制算法SpikePingpong

  • 乒乓球球速可超5m/s

意义：成为全球首个全程自主决策、完成乒乓球对抗的全尺寸双足人形机器人，标志着人形机器人从实验室加速走向真实落地场景。底层能力（高速动态感知→预测→全身协调→实时修正）可迁移至柔性装配、动态分拣等工业场景。


04  松延动力「OpenHarmony版 N2」

>> 开源鸿蒙消费级人形机器人 <<

时间：6月13日亮相华为开发者大会HDC 2026，6月15日正式发布。

核心要点：

  • 业内首款基于开源鸿蒙（OpenHarmony）的消费级人形机器人

  • 依托鸿蒙微内核+分布式软总线，可充当家庭智能中枢，联动照明、空调、安防、影音等设备

  • 华为常务董事余承东于主题演讲中将其列为“开源鸿蒙走进千家万户”六大核心品类中机器人方向的唯一代表

  • 同步启动“百人百机”开发者共创计划：甄选100位开发者/团队，每人提供一台N2机器人用于本地开发，配套原生SDK，支持Ubuntu+OpenHarmony双生态

  • 松延动力Bumi（小布米）定价9998元，首批500台48小时售罄

特点：战略意义不在单个硬件参数，而在操作系统生态入口——如果鸿蒙人形路线跑通，“机器人=又一个鸿蒙终端”的叙事将重构消费品类边界。

16 星灿智能「Xcan Model X」

>> 可适配脑机接口出行陪伴机器人 <<

时间：2026年6月13日，第十二届中国（上海）国际技术进出口交易会

脑机接口适配层：通过接入高精度非侵入式脑机接口系统（外部BCI头环/解码器对接，非星灿自研解码），用户只需"想象迈步"或"专注目的地" → 系统在毫秒级完成意图识别 → 调动运动控制系统执行。

专注将L4 级空间智能与具身智能技术下沉至家庭与康养场景，打造 “空间智能 + 具身交互” 的 AI 出行陪护机器人，精准解决老年人、行动障碍人群出行与照护刚需。

自动驾驶能力（本体侧）：具备 L4级自动驾驶能力叙事，搭载自研 XcanSense 5D多模态感知系统，支持从室内（居家/病房）→室外（社区/道路）全域自主导航，无图自动驾驶（不依赖预制地图），实现点对点路径规划与动态场景预测性避障。

自适应姿态平衡：斜坡自平衡 + 座椅高度自动升降，应对床↔椅高度差场景；动态场景主动避障，0.3秒内完成避障决策。


具身机械臂（可扩展）：辅助开关门、取物、辅助进食等生活操作

AI健康陪伴系统：24h无感监测 + 主动预警 + 情感交互，配套 Xcan Care 订阅服务

特点：解决行业真实的 "有脑无身"痛点——脑电波再准，没有安全可靠的物理载体去执行，患者依然困在原地。Model X的差异化不在"更像轮椅"，而在于把 L4级空间智能底盘 + 5D感知安全否决权 + BCI标准化API 封装成一个可量产的交付件，让任何BCI厂商都能"插上就跑"。


17 傲意科技「ROH-AP003」

>> 3D磁触觉灵巧手 <<


时间：6月17日在2026张江具身智能供应链大会发布。

核心指标：

  • 搭载工业级3D磁触觉传感（Tactaxis™），可感知0.1N力（≈蝴蝶落手重量）

  • 剪切力方向分辨至约1°，采样频率150Hz

  • 国内首个实现量产的3D磁触觉灵巧手

  • 通过核心部件国产化+供应链系统化布局压成本，已跻身国内灵巧手出货量第一梯队

特点：是国内首个实现量产的3D磁触觉灵巧手。触觉反馈是机器人灵巧操作的核心瓶颈之一。0.1N精度叠加量产能力，傲意科技正在推动国产灵巧手从“能做”向“好用”跨越。

20 它石智航「DexHand」灵巧手

>> 聚焦高难度精细操作场景<<

时间：6月8日亮相ICRA 2026。

硬件架构：21自由度刚性准直驱方案（非绳驱），1:1等比例复刻人手掌骨+指节拓扑，重点攻克拇指CMC对掌关节与MCP掌指关节的多轴空间交汇，消除并联关节运动盲区；仅用三种电机+三种减速器做标准化量产适配
感知配置：指腹/指尖侧集成弹性触觉传感阵列；指尖搭载超高分辨率微型摄像模块（文献/复述口径 240Hz+，可分辨约 0.03–0.05mm 级微观纹理与物性差异）
背书背景：它石A1机器人已于 2026.3.10 以 1小时内105次有效装配、线束规格0.5mm 的成绩，获吉尼斯世界纪录认证（"机器人在一小时内装配亚毫米级线束最多次数"），是中国具身智能在工业精密操作领域首个吉尼斯荣誉

特点：“手脑一体”灵巧手全球首秀，聚焦高难度精细操作场景。不做独立硬件SKU，而是 SenseHub（数采）→ AWE 3.0（具身大模型）→ DexHand（执行终端）三位一体的"手脑一体"闭环——官方原话"硬件即数据、数据即硬件"，从设计源头消除传统"人手→机器人重定向"的翻译损耗


21 灵心巧手「直驱系列020/030」

>> 直驱见"控"<<

时间：6月5日 ICRA 2026

本次ICRA首发：直驱系列 O20 / O30

Linker Hand O20：
16个全主动自由度，融合高性能无刷空心杯电机 + 蜗轮蜗杆关节模组
整手额定负载20kg，全掌开合仅需 0.8秒
核心卖点：蜗轮蜗杆自锁特性 → 抓取后保持状态静态电流仅0.45A，支持24小时不间断运行——解决的不只是"抓得牢"，而是"抓完不烧电、不掉、不需反复纠偏"的部署前提
Linker Hand O30：
20个全主动自由度（O20基底 + 新增4个主动驱动自由度）
专为复杂灵巧操作模型开发 / 强化学习设计，规划开源
面向算法开发者需要清晰、独立的关节级控制通道的场景——即"具身智能研究与真实部署之间的关键硬件接口"
意义：全球唯一实现腱绳（L30系）-腱绳见"柔"、连杆（L20/O6/L6）-连杆见"力"、直驱（O20/O30系）-直驱见"控"。三大主流技术路线全覆盖的灵巧手企业；口号不是"仿人"，而是做具身智能时代的操作基础设施——硬件矩阵 + Open TeleDex遥操作 + LinkerSkillNet技能库 + Linker Genesis大模型

  • 联合北京亦庄共建数据公司，启动百万小时真实数据计划

行业核心观察

四大趋势定调产业新阶段


观察一：人形机器人“C端化”加速

6月至少有4款面向家庭/消费级的人形机器人亮相或预告（优必选U1、越疆陪伴机器人、节卡π仔、元点M1）。2026年正在成为人形机器人从B端走向C端的“破晓时刻”。

观察二：工业场景仍是“真落地”主战场

天工3.0、阿童木天将一号、具福福泰W1、开普勒机械麒麟等产品全部指向工业制造、仓储物流等真实作业场景。能“真干活”的机器人，比能“真表演”的机器人更早拿到订单。

观察三：操作系统与模型层成为新战场

松延N2切入开源鸿蒙生态、星海图开源G0.5——行业竞争正从“硬件参数战”升级为“生态与模型战”。谁能在操作系统和基础模型层建立壁垒，谁就掌握了下一代话语权。

观察四：灵巧手赛道进入“量产+家族化”阶段

傲意、星动、拾玥、它石智航、灵心巧手五家企业在6月集中发布灵巧手产品——从“单品突破”走向“产品矩阵”，从“实验室demo”走向“量产交付”。灵巧手正在成为具身智能产业链中最先完成国产替代的环节之一。
