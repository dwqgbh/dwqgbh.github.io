2026 年 6月20日新闻班车

江湖如闻 2026-06-20

工作路上皆寂寞，唯有关注不迷路。

2026年06月20日 权威官媒热点资讯汇总

信息来源：新华社、央视新闻、人民日报、国防部、中国天气网、证券时报、农业农村部等官方平台

一、头条板块（6条）

1. 2026南博会今日圆满收官，展会累计签约总额达277.4亿元，跨境贸易、绿色农业、文旅投资项目占比超七成，西南对外开放通道持续拓宽。
2. 农业农村部数据：全国夏粮收割进度突破八成，豫鲁皖主产省基本完成小麦收割，各地启动小麦最低收购价执行预案，筑牢全年粮食稳产基础。
3. 商务部公告：6月20日零点起，对超年度进口配额的澳大利亚牛肉加征55%额外关税，稳定国内肉牛产业与生鲜市场物价。
4. 太原卫星发射中心今日成功发射试验二十五号卫星，长征六号运载火箭精准送星入轨，用于新型对地观测技术验证试验 。
5. 深中通道东人工岛主体结构全面完工，水下互通隧道成型，项目通车后深圳至中山通勤时长压缩至半小时内 。
6. 中央网信办启动为期三个月“清朗·整治账号名称信息乱象”专项行动，严打仿冒官方、引流黑灰产、违规昵称四大类问题。

二、政治板块（6条）

1. 中国常驻联合国副代表发言，呼吁国际社会加大人道主义援助投入，人道援助不得附加政治条件，均衡帮扶全球受灾地区。
2. 外交部严正驳斥日本右翼美化731细菌部队行径，指出历史修正主义严重威胁亚太和平稳定，国际社会应共同抵制 。
3. 国务院户籍制度优化新政落地，居民教育、医保、社保等基础公共服务可在常住地一站式办理，惠及2.5亿流动人口。
4. 第三届中非经贸博览会首批海外展品运抵长沙，本届展会将于6月29日至7月2日举办，聚焦中非产业链深度合作 。
5. 纪检监察通报：合肥市原副市长何逢阳涉嫌严重违纪违法，接受纪律审查和监察调查，持续推进全面从严治党、常态化反腐。
6. 多地召开东西部协作工作推进会，部署产业转移、人才交流、乡村帮扶项目，扎实推进共同富裕建设。

三、经济金融板块（6条）

1. 陆家嘴论坛配套政策落地，证监会扩大科创板第五套上市标准覆盖AI大模型企业，支持量子、具身智能等硬科技企业融资上市 。
2. 央行一季度数据出炉：科技型中小企业贷款余额4.03万亿元，同比增20.9%，金融资源持续向科创领域倾斜 。
3. 国家发改委明确6月底前足额下达千亿设备更新补贴，新增户外、日化消费品以旧换新补贴，拉动制造业与内需消费双升级。
4. 多家银行推出世界杯主题场景金融服务，发行限定收藏银行卡，联动餐饮、出境消费推出专属优惠，拓宽零售金融流量渠道。
5. 多地中小银行收到监管通知，要求2026年底前压降存量理财业务，规范中小机构资管业务风险管控。
6. 英国央行今日维持基准利率不变，美欧国债收益率波动，全球资本市场受海外货币政策影响震荡调整。

四、军事板块（5条）

1. 国防部发布2026军队院校招生计划，全军22所院校招收1.75万名普通高中毕业生，新增军医5+3一体化临床医学培养项目 。
2. 北约防长会后发布联合声明，宣布升级核军备规划，引发国际社会对全球军备竞赛加剧的普遍担忧。
3. 解放军多支防汛抢险前置力量驻守南方多雨流域，全天候备勤应对暴雨、山洪、地质灾害抢险救援任务。
4. 美方表态不会向中东派遣地面作战部队，呼吁冲突各方依托外交谈判化解分歧，中方持续呼吁地区全面停火止战。
5. 军方发布边境端午执勤纪实短片，展现边防官兵坚守国土、全天候战备值守日常，全网转发热度居高不下。

五、社会民生板块（6条）

1. 二十余省市发布2026暑期托管优化方案，大幅削减学科补习，新增户外研学、体育美育、安全科普免费公益课程。
2. 市场监管总局持续整治儿童智能手表乱象，央媒集中评论批评厂商堆砌付费娱乐功能，诱导未成年人非理性消费。
3. 四川景区国道收费问题完成全面整改，明确过境自驾车辆无需购票，保障群众正常道路通行权益。
4. 全国多地中医院端午义诊持续开展，艾草、祛湿中药材、食疗香囊平价供应，群众咨询调理需求大幅上涨。
5. 全国铁路7月1日实施三季度列车运行图，优化城际、旅游线路班次，提升暑运、暑期出游运力保障 。
6. 应急管理部联合财政部预拨救灾补助，支援两广、湖南、江西等暴雨区域开展道路清障、群众转移安置工作。

六、科教板块（5条）

1. 国产CPO、高端光模块产业技术迭代取得突破，多地出台算力中心、AI硬件研发专项补贴，加速新质生产力落地。
2. 全国中小学开展端午传统文化主题教学，普及屈原历史、非遗龙舟、传统中医药养生科普内容。
3. 航天部门公布下半年商业卫星发射计划表，多颗新型遥感、通信卫星将陆续组网，完善国产太空观测体系。
4. 文博会数字科技展区热度走高，VR数字演艺、空间计算、数字孪生技术集中展出，科技文旅融合产品集中首发 。
5. 高校端午开放航天、芯片科普实验室，面向中小学生开展公益研学，普及航空航天、半导体基础科学知识。

七、文体板块（6条）

1. 2026美加墨世界杯持续激战，多场小组赛爆冷，国内体育平台观赛流量、相关话题阅读量稳居文娱榜单首位。
2. 重庆两江四岸端午无人机灯光秀连续两晚上演，融合龙舟、国风元素，单场线上直播观看量破千万。
3. 三峡博物馆中意文物特展持续热展，235件中外古代神兽文物集中展出，端午假期参观客流持续走高。
4. 全国百城举办端午国风巡游、汉服游园市集，国潮服饰、传统手工艺品消费热度显著提升。
5. 多地城市同步举办端午龙舟联赛，汨罗、广州、扬州赛场直播全网刷屏，民俗文旅带动周边餐饮住宿增收。
6. 端午档传统文化题材院线影片持续上映，假期累计票房同比大幅上涨，国风影视内容广受年轻观众喜爱。

八、气象灾害板块（6条）

1. 中央气象台6时发布强对流蓝色预警，广东西部、广西北部、湘赣皖浙北部有大到暴雨，局地特大暴雨，伴随雷暴大风、冰雹 。
2. 华北、黄淮、东北多地出现35℃以上持续性高温，全国大范围高强度紫外线预警生效，户外晒伤风险偏高。
3. 内蒙古东部局地预报10级以上雷暴大风，气象部门提醒牧区、农户提前加固棚圈、户外设施。
4. 南方汛期地质灾害风险升级，山区、河谷地带禁止露营、垂钓，防范山体滑坡、河道涨水险情。
5. 海事部门发布沿海风浪预警，近海游船、龙舟赛事严格落实安全管控，恶劣天气立即停止水上活动。
6. 中疾控发布高温健康预警，老年人、婴幼儿、户外劳动者、慢性病群体为中暑高危人群，需常备防暑药品。

九、养生板块（5条资讯+专属养生知识）

养生资讯

1. 中医专家提醒端午湿热叠加高温，粽子糯米难消化，老人、糖尿病人、脾胃虚弱人群少食。
2. 全国各大药店艾草、菖蒲、赤小豆、薏米等祛湿药材销量暴涨，平价养生物资供应充足。
3. 夏季蚊虫进入高发期，疾控科普艾草、薄荷天然驱虫方法，减少化学驱蚊用品使用。
4. 医院消化科接诊积食、肠胃不适患者增多，多为过量食用粽子、生冷冰饮引发。
5. 高温天空调病就诊量上升，医生建议空调温度不低于26℃，定时开窗通风除湿。

今日专属养生知识

仲夏湿热时节养生核心为清热养心、健脾祛湿。每日饮水量维持2000-3000ml，少量多次饮用温水，出汗多时搭配淡盐水补充电解质；饮食遵循多酸少咸、适度吃苦，常食冬瓜、苦瓜、绿豆清热，赤小豆薏米煮水代茶饮祛除体内湿困；避开10:00-16:00高温时段外出，大汗后切勿冲冷水澡、猛喝冰水；夜间睡觉护住腹部薄毯，空调开除湿模式，每日按揉曲泽、阴溪二穴，缓解夏季心慌、燥热失眠。
