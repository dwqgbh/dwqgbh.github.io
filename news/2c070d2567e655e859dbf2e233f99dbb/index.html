卓立湾大 博领未来 !大湾区大学2026年卓越博士后招聘公告

硕博招聘网 2026-06-01

为深度融入粤港澳大湾区国际科技创新中心建设，打造大湾区高层次青年人才蓄水池，大湾区大学正式启动“卓越博士后”人才资助计划。本计划旨在吸引全球范围内极具发展潜力的优秀博士，聚焦前沿基础研究与关键技术创新，在新型研究大学成长沃土上深耕学术、展现才智和共创湾区新篇！

一、薪酬待遇

1.  综合年薪45万元起（含地方政府资助），资助期2年；入选广东省海外博士后人才支持项目，年薪不低于60万元。

2. 除年薪外，学校足额缴纳社会保险与住房公积金；

3. 课题组根据科研业绩发放绩效奖励。

二、福利保障

1. 资助期内每月提供1500元租房补贴；

2. 享受带薪年假、年度健康体检、校内餐厅及工会活动等福利；

3. 按属地政策协助办理落户、子女入园入学；

4.  可申报大湾区及东莞市人才支持政策，享受购房补贴、租房补贴、生活补助、科技成果转化贡献奖励、创业扶持、境外人才个人所得税优惠等，可申领优才卡享受属地就医、行政办事、文体服务等优才服务。

三、科研支持

1.  提供工作开展所需科研条件，支持申报国家自然科学基金、中国博士后科学基金以及省、市各级科研项目；

2. 获得中国博士后科学基金资助或省级以上重点博士后资助项目，地方政府给予相应配套资助；

3.  支持依托学校申报香江学者计划（湾区项目）、澳门青年学者计划（湾区项目）等大湾区博士后人才交流专项；

4. 支持申报东莞市博士后国内外学术交流项目资助；

5. 共享学校高端科研设施、国际化学术生态与跨学科研究平台。

*数学专业可依托国家留学基金委国际人才培养项目前往俄罗斯科学院斯捷克洛夫数学研究所短期访学。

四、职业发展

1. 进站即聘为副研究员，支持对外开展交流与合作；

2.  留莞工作的，东莞市提供相应生活补助；入选广东省海外博士后人才支持项目，留粤工作的，广东省给予40万元住房补贴；

3.  优秀出站博士后，优先推荐至东莞市内科研院所、高等院校、链主企业、上市公司、央国企、国家专精特新“小巨人”等重点用人单位及校内招聘岗位；

4. 依托粤港澳大湾区产业与人才优势，搭建海内外多元化职业发展通道。

*有关人才政策按现行政策执行。

五、项目概况

1. 首期规模：2026年计划引进约30名优秀博士，分上、下半年两个批次招录；

2. 资助期限：2年，进站后须全职在站开展研究工作；

3. 学科方向：物质科学、理学、信息科学技术、先进工程等核心领域；

4.  培养优势：与中国科学技术大学、清华大学深圳国际研究生院、俄罗斯科学院数学研究所联合培养，共享顶尖科研平台与学术资源。

六、申请条件

1. 认同大湾区大学创新开放、科学严谨的学术价值理念，具备优秀的团队协作能力与学术潜力；

2. 年龄在35周岁以下，即将获得博士学位或获得博士学位时间不超过3年；

3. 扎实的教育背景与国际视野，在相关领域取得代表性原创研究成果；

4. 承诺进站后全职在大湾区大学从事博士后研究。

*已入站且未参与过校级评审的博士后，可申报本次卓越博士后项目；各级评审中未入选者，不纳入本次申报范围。

七、申报与评选安排

（一）批次时间

第一批次：个人申报截至2026年5月20日，学院推荐截至5月30日，学校评审6月；

第二批次：个人申报截至2026年10月20日，学院推荐截至10月30日，学校评审11月。

（二）申请材料

1.《大湾区大学卓越博士后项目申请书》（扫码获取）；

2.近三年代表性科研成果（整合为单一PDF文档）；

3.学历学位证书及验证报告（应届博士生可提交答辩决议书/学位授予证明）；

4.相关领域专家推荐信（含博士导师）至少3封，由推荐人直接发送至大湾区大学意向合作导师邮箱。导师简介及邮箱请登录大湾区大学官网（https://www.gbu.edu.cn/menu/3）查询。

（三）提交与评审

1.  材料提交：将申请书、科研成果、学历学位材料压缩打包，邮件主题命名为“卓越博士后申请+意向PI+姓名+博士招聘网”，发送至大湾区大学意向合作导师邮箱。

2. 评审流程：意向合作导师择优推荐→教研单位择优推荐→专家评审→公示录用。
