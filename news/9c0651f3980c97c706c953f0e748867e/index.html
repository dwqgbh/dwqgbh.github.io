2025大湾区深圳半导体展10月重磅来袭!核心亮点+活动论坛一览

广东展讯网 2025-09-28

2025湾区半导体产业生态博览会（深圳半导体展）将于10月15-17日在深圳会展中心（福田）举办。展会设立晶圆制造、封装测试、化合物半导体、汽车半导体等六大专业展区，汇聚应用材料、蔡司、北方华创、中微公司等国内外领军企业。


同期举办20余场技术论坛，邀请超200名行业专家，共同探讨半导体前沿趋势与深圳发展机遇，致力构建产业链融合创新平台。

展会亮点

2025湾芯展深圳半导体展将于10月15-17日在深圳会展中心举办，以"芯启未来，智创生态"为主题，打造半导体全产业链盛会：


1、规模领先：6万㎡展区，600+龙头企业参展，覆盖IC设计、晶圆制造、封测全链条


2、强大阵容：ASML、北方华创等国际国内领军企业齐聚，设4大主题展区


3、高端活动：20+场论坛，含IWAPS光刻研讨会、TOP20闭门会等顶尖学术会议


4、生态服务：深芯盟1400家会员赋能，提供晶圆厂对接、标杆参访等特色服务


5、行业盛典：颁发"湾芯奖"中国半导体至高荣誉，举办湾区半导体之夜


6、预计吸引6万专业观众，构建"展览+论坛+对接"五位一体产业平台。







四大展区


🌟晶圆制造展区：晶圆制造及 IDM 厂商、制造 / 量测设备、先进材料、零部件 / 间接耗材、绿色厂务及配套设施


🌟化合物半导体展区：碳化硅 / 氮化镓等材料、制造 / 量测设备、衬底 / 外延、零部件 / 间接耗材、射频 / 功率器件


🌟IC 设计展区：EDA 软件 / 服务、芯片设计 IP / 服务、存储 / AI 芯片、其他设计服务


🌟先进封装展区：封装测试厂商、封装测试设备、封装材料、玻璃基板及工艺
