2026CIIF数控机床与金属加工展——数控驱动,精工制造

展网未来 2026-05-28


中国国际工业博览会（简称“中国工博会”或“工博会”）是我国以中国冠名的、历史最长的国家级工业展会，自1999年创办以来，每年秋季在上海举办，已成功举办二十余届。

作为服务先进制造业领域展示交往活动的企业间沟通桥梁，中国工博会集“展示、交易、评奖、论坛”四大功能于一体，历经二十余年“专业化、市场化、国际化、品牌化”运作和发展创新，坚持对接国家战略，发展实体经济，已经成长为我国工业领域面向世界的重要展示窗口和经贸交流合作平台。

实现了“西有德国汉诺威工博会，东有中国工博会”的定位目标，成为了世界知名、中国最具影响力的工业品牌展，成为了一张面向国际展示中国工业高质量发展成果的金名片，有力促进了中国工业与全世界的交流、交往和交融。

300,000m²展览面积

9大专业展，覆盖制造业全产业链

3,000家展商同台竞技

中国工博会CIIF2026数控机床与金属加工展

时间：2026年10月12-16日

地点：国家会展中心（上海）


汇展全球智能机床设备聚焦金属加工尖端科技

金属切削机床

  1. 加工中心
    ：立式 / 卧式 / 龙门 / 五轴联动加工中心、车铣复合中心、多主轴 / 多工位加工中心、并联机床
  2. 车床
    ：精密数控车床、车削中心、自动车床、管螺纹车床、重型卧式车床
  3. 铣床 / 镗床
    ：数控铣床、龙门铣床、落地镗铣床、坐标镗床、钻镗中心
  4. 齿轮 / 专用机床
    ：滚齿机、插齿机、磨齿机、剃齿机、珩齿机；制齿加工线
  5. 磨 / 超精加工
    ：外圆 / 内圆 / 平面磨床、工具磨床、珩磨、研磨、抛光机、超精机
  6. 钻削 / 攻丝 / 锯切
    ：数控钻床、深孔钻、攻丝中心、数控锯床、带锯 / 圆盘锯
  7. 制造单元与自动化
    ：柔性制造系统（FMS）、机床上下料机器人、桁架机械手、自动检测线


金属成形机床（钣金 / 锻压 / 型材加工）

  1. 板材加工
    ：数控激光切割机、等离子 / 火焰切割机、水刀切割机、数控转塔冲床、剪板机、折弯机、板料旋压机床
  2. 管材 / 型材
    ：管材折弯机、弯管中心、型材弯曲机、矫直机、线材成型机
  3. 冲压 / 压力机
    ：高速精密冲床、液压 / 机械压力机、多工位压力机、步冲轮廓机、伺服压力机
  4. 锻压设备
    ：数控模锻锤、热 / 冷锻压力机、螺旋压力机、辗环机、

机床零部件与辅助设备

  1. 功能部件
    ：精密主轴单元、直线导轨、滚珠丝杠、刀库 / 刀塔、联轴器、分度盘、回转工作台
  2. 辅助设备
    ：液压 / 气动系统、冷却过滤系统、排屑机、油雾收集器、润滑装置
  3. 机床附件
    ：卡盘、夹头、顶尖、刀架、防护罩、刀具、磨料磨具与工夹具
  4. 刀具
    ：硬质合金刀具、高速钢刀具、陶瓷 / 立方氮化硼（CBN）/ 金刚石刀具；数控刀柄、刀具预调仪、刀具管理系统
  5. 磨料磨具
    ：砂轮、砂带、抛光轮、金刚石 / 立方氮化硼磨具、涂附磨具
  6. 夹具 / 工装
    ：精密平口钳、三爪卡盘、气动 / 液压夹具、组合夹具、非标工装、真空吸盘

中国作为全球最大的机床消费国和生产国，已成为推动全球工业发展的关键力量。
亚威、葛世（嘉意）、立为、蓝昊、力锋等知名展商，重点展示了以智能装备与数据互联为核心的钣金加工全流程自动化产线，推出无人化、柔性化的智慧工厂解决方案。同时，大族、华工、海天、RAS、爱克、奔腾、金方圆、扬力、邦德、迪能、瑞铁、创鑫、东海、谊科等行业领先企业也携多款王牌及首发产品重磅登场，展品涵盖激光高速切割机、高速激光切管机、三维五轴激光切割机、智能激光切割单元、自动化折弯单元、纯电折弯机以及柔性折弯中心等前沿设备。近20家国际钣金品牌强势集结于3H馆，包括村田、希格盟德、法赛、特马液压、亚特兰、文拉、戴蔚、丹兹、兰特克、富美成、旭光、哈格、英视、诺亘、海迈、斯伯克、帕斯德等外资企业联袂参展。展品涵盖焊接平台、卷板机、弯管机、折弯模具、压铆中心、传动系统、传感器、精密钣金检测系统、折弯光栅、钣金软件套料系统及相关软件等，完整呈现钣金加工全产业链的先进设备与技术解决方案。

中国工博会聚焦新工业新经济，强化高端产业引领功能，荟聚顶尖专家菁英，引领产业科技创新风尚，全力构建开放、创新、包容的产业生态，覆盖从制造业基础材料、关键零部件，到先进制造装备、

整体解决方案的智能绿色制造全产业链，是国内展览规模最大、参展企业最多、展示领域最广、国际化程度最高的工业展会之一。
