【大摩】中国工业:2026年展望–人形机器人

我们对A股的思考 2026-02-20

主要观点

  1. 2026年将是人形机器人商业化的关键一年
    2026年将是人形机器人集成商努力实现商业化并建立生态系统的关键时期。零部件供应商将率先从行业初期的增长中获利，同时由于行业的快速发展和众多催化剂，资本市场预计将非常活跃 。

  2. 中国在全球机器人市场中占据主导地位，并成为创新中心
    目前，中国不仅是全球最大的机器人市场，也正在演变为一个创新中心，推动着全球范围内的成本效率提升和下一代机器人的发展 。中国机器人的国产化进程正在加速，不同类型机器人的国产化率从高到低依次为：无人机 > 服务机器人 > 移动机器人 > 协作机器人 > 传统工业机器人 。同时，机器人供应链的国产化也在加速，到2024年，大部分关键部件（如电池、激光雷达、电机等）的国产化率已达到较高水平（超过40%）。

  3. 行业发展重点从“身体”转向“大脑”
    继2024年关注原型机和2025年关注“身体”（硬件）之后，2026年行业和资本市场的焦点将转移到机器人的“大脑”，即其智能和软件系统的开发上 。

  4. 近期出货量预测上调，规模化应用浪潮预计在2028年到来
    考虑到2025年高于预期的销售数据以及2026/27年持续进行的应用验证，报告上调了近期的出货量预测。预计中国的人形机器人交付量将在2026年翻倍至2.8万台，但主要为小规模的验证和试用订单 。经过数据积累和商业价值验证后，预计从2028年开始将出现一波规模化应用浪潮 。

  5. 投资策略上优先考虑领先的零部件供应商
    报告认为，在行业发展的初期阶段，零部件供应商将是首先实现盈利的环节 。因此，投资重点应放在那些拥有技术优势和市场领先地位的零部件公司上，这些公司将直接受益于人形机器人行业的增长 。

事实依据

  1. 巨大的市场潜力
    报告预测，到2050年，全球机器人市场规模将达到25万亿美元，其中中国市场将占据5万亿美元 。人形机器人作为一个重要分支，其全球市场规模到2050年预计可达7.5万亿美元，其中中国市场约占1万亿美元 。

  2. 强烈的市场采用意愿
    根据AlphaWise的调查，尽管目前人形机器人的采用率还很低，但企业部署意愿强烈。62%的受访企业表示有计划在2025-2027年间部署轮式或腿式人形机器人项目 。这一强烈的采用意愿为行业的快速验证和初步采用提供了支持 。

  3. 2025年下半年订单激增
    2025年下半年，中国市场宣布的人形机器人相关订单总额超过20亿元人民币 。尽管这些订单大多数用于研发、数据收集或娱乐目的，而非真正的商业需求，但这表明市场正在为大规模部署进行积极准备和数据积累 。

  4. 硬件趋于成熟，成本持续下降
    报告指出，人形机器ンのハードウェアは量産準備が整っており、将来的には継続的なイテレーションとコスト削減が見込まれます 。2026年には、中国で製造される人形ロボットの部品表（BoM）コストが、規模の経済とプロセス最適化により前年比で16%減少すると予測されています 。

陈述总结

本报告对2026年中国工业领域的人形机器人行业进行了展望，认为2026年将是行业商业化的关键一年 。在这一年，集成商将致力于推动产品的商业化落地和生态系统建设，而行业的关注焦点将从2025年的“身体”（硬件）转向“大脑”（智能与软件）的开发 。

报告强调了中国在全球机器人市场中的领先地位，并预测到2050年，中国机器人市场规模将达到5万亿美元 。人形机器人作为其中的重要部分，长期增长潜力巨大。基于2025年下半年超过20亿元人民币的订单公告，报告上调了近期出货量预测，预计2026年中国交付量将翻倍至2.8万台 。不过，目前多数需求仍集中在研发、数据收集和娱乐领域，真正的规模化商业应用预计将在2028年启动 。

AlphaWise的调查显示，企业对人形机器人的采用意愿非常强烈，62%的受访者计划在未来三年内进行部署 ，这为行业的快速发展提供了坚实基础。同时，硬件技术已为量产做好准备，且成本将持续下降，预计2026年物料清单（BoM）成本将下降16% ，为大规模采用创造了有利条件。

在投资策略上，报告认为零部件供应商将率先从行业增长中获益，并看好那些在谐波减速器、滚珠丝杠等核心部件领域具有领先地位的公司 。

关键数据

  • 市场规模预测：
     到2050年，中国机器人市场预计将达到5万亿美元 。全球人形机器人组件市场到2040年将达到7800亿美元 。
  • 出货量预测：
     预计2026年中国人形机器人交付量将翻倍，达到2.8万台 。
  • 成本下降：
     预计2026年，中国的人形机器人物料清单（BoM）总成本将同比下降16% 。
  • 企业采用意愿：
     AlphaWise调查显示，62%的企业计划在2025-2027年期间部署人形机器人 。
  • 劳动力替代：
     预计未来5年内，约11%的工作岗位可能被机器人替代，10年内这一比例将上升至27% 。

推荐资产标的

  • 绿的谐波 (leaderdrive 688017.SS)：
     在国内集成商中拥有领先的谐波减速器份额，并正在与美国和韩国的集成商合作。公司正将其产品线扩展到行星减速器、轴承、丝杠和模块 。
  • 恒立液压 (Hengli 601100.SS)：
     据信是北美一家集成商的主要丝杠供应商，其墨西哥工厂为此提供产能。它也为大多数国内集成商提供丝杠和线性执行器 。
  • 汇川技术 (INOVANCE 300124.SZ)：
     已推出全系列智能机器人组件和模块产品供客户试用和购买。智能机器人是公司的关键战略重点，预计将成为长期增长的主要驱动力 。
  • 双环传动 (Shuanghuan 002472.SZ)：
     正在为人形机器人开发一种新型减速器，目前正与一家关键客户进行测试 。
  • 极智嘉 (Geekplus 2590.HK)：
     AMR（自主移动机器人）市场的领导者，具有强大的全球影响力。推出轮式人形机器人是其完善无人仓库解决方案的关键一步 。
