机器人行业周报2026年6月1日 - 6月7日

具身智能机器人顾问 2026-06-07

📌 本周核心动态

1. 【IPO里程碑】宇树科技科创板过会并"闪电"提交注册，73天创最快审核纪录

事件内容6月1日，上交所上市审核委员会审议通过宇树科技科创板IPO首发申请。6月2日，宇树科技便"闪电"提交IPO注册，全程从受理到过会仅73天，刷新2026年以来科创板IPO审核速度纪录。宇树拟募资42.02亿元，按发行后估值约127亿元，英伟达等机构联合加持。上交所现场问询重点聚焦商业化进展、技术差异及具身智能核心研发布局。

值得关注的原因

  • 📈 宇树上市后产品迭代和渠道扩张必然加速，建议密切关注其代理政策变化，抢占先机；
  • 🔧 募集资金中20.2亿投入智能机器人模型研发、11.1亿投入本体研发，将催生新一代硬件产品，带动配件及维保需求提前布局。

来源：新京报 / 百家号 

2. 【行业地震】人形机器人价格集体"砸穿地板"，万元时代正式开启！租赁成本同步腰斩

事件内容2026年6月，中国人形机器人迎来历史性价格拐点：松延动力Bumi（小布米）定价9998元，成为行业首款万元级消费型人形机器人；宇树G1降至8.5万元起，宇树R1消费级入门版2.99万元起；加速进化K1系列限时促销2.99万元；优必选12.8万元起。与此同时，二手市场崩盘——去年花30-80万购买的宇树H1、傅利叶GR-1等机型，二手行情仅剩3-6万元。租赁日租金也从2025年最高1万元暴跌腰斩。

值得关注的原因

🏪采购成本断崖式下降：整机价格从数十万元降至万元级别，租赁业务资产投入门槛大降，ROI周期大幅缩短，是启动和扩张租赁业务的黄金窗口期，但功能和品质价格成正比，能否满足市场需求是关键；

  • ⚠️ 价格战下品牌分化加剧，建议关注有核心技术壁垒的品牌（智元、宇树、松延动力等），避免采购即将被淘汰的旧款机型；
  • 💡 客户教育成本下降——"万元机器人"概念易于向中小商家推广，可针对性开拓商超开业、婚礼庆典、校园活动等中低频租赁场景。

来源：百度百科 / 重庆日报 / 百家号 / 快科技 

3. 【供应链风口】特斯拉Optimus V3完成设计冻结，6-7月启动大规模供应链审厂

事件内容据雪球6月报道，特斯拉Optimus V3已于2026年5月完成系统级设计冻结，计划6-7月启动大规模供应链审厂，直接决定Q3产能爬坡阶段的供应商名单及初始份额。与此同步，特斯拉得州超级工厂专属Optimus工厂已破土动工，北区首座钢结构主体落成，规划年产能最高1000万台。特斯拉正将加州弗里蒙特工厂Model S/X产线全面改造为Optimus产线，冲刺年底量产。

值得关注的原因

🔩配件供应链机会：特斯拉Optimus量产将催生海量零部件需求，中国供应链企业有望进入其配套体系。关注伺服电机、减速器、传感器等核心零部件的国产替代机会，这些也是售后维保的核心耗材；

  • 🏭 千万台级产能规划意味着售后维保市场将指数级增长。即使仅覆盖中国市场的1%，也是百万台级的维保和配件需求；

来源：雪球 / 新浪财经 / IT之家

4. 【人才培训】京东服务宣布5年培养10万工程师，机器人售后培训赛道正式引爆

事件内容5月28日世界智能博览会期间，京东服务宣布计划未来5年培养10万名工程师，全面覆盖机器人、智能家居售后维修等服务领域，通过完整的职业培训体系打造蓝领"多面手"。京东将与职业院校合作建立培训基地，输出标准化机器人维护课程体系和认证机制。此举标志着大型平台正式入局机器人售后培训市场。

值得关注的原因

🎓 京东10万工程师计划从侧面印证了机器人维保培训市场规模巨大且亟待填补。对从事机器人培训业务的创业者来说，这是明确的赛道上行信号；

    来源：电器杂志 

    5. 【竞争变局】小鹏IRON机器人核心高管闪电离职，量产关键期生变数

    事件内容6月5日，网易报道小鹏IRON人形机器人核心高管施晓鑫闪电离职。何小鹏此前多次强调2026年是IRON"决胜年"，计划Q3发布新一代量产版、年底实现高阶版本量产并在小鹏门店试商用，2027年向海内外商业客户交付。此次高管离职正值量产冲刺关键期，外界对其能否按时完成量产目标产生疑虑。

    值得关注的原因

    • 🔀 小鹏IRON量产节奏的不确定性，可能为其他品牌（宇树、加速进化、松延动力）腾出窗口期，尤其是在二线城市的渠道布局上；
    • 🏪 小鹏门店体系庞大，若IRON试商用计划延期，可转向与其门店合作开展其他品牌机器人的展示和租赁，填补其机器人品类空白；

    来源：网易 

    6. 【机器狗机会】宇树机器狗日租金低至78元起，IPO后渠道扩张将加速

    事件内容随着宇树IPO过会及人形机器人价格战蔓延，机器狗产品线同步受益。当前市场上宇树Go2系列机器狗日租金已低至百元起（Go2 Air基础版），Go2 Pro日租800-1500元，工业级B2日租更高。

    值得关注的原因

    • 🐕 机器狗是当前商业化最成熟的足式机器人品类，已广泛应用于巡检、安防、教育、娱乐等场景。日租百元起的入门价可覆盖大量中小客户；
    • 🔄 机器狗配件（电池、关节模组、传感器）耗损率高，维保频次高于人形机器人，配件销售和维保是稳定的持续性收入来源。

    来源：百家号 / 今日头条
