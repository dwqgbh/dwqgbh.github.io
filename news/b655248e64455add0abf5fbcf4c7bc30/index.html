2025第三届中国丘陵山区农业机械博览会暨中国(永康)农林装备博览会

农资会 2025-02-08





为贯彻落实党的二十大精神，深化“科技强农”“机械强农”等 相关行动，立足浙江特别是永康制造业、永康市丘陵山区小型农机装 备产业集群、金台丘陵山区农机装备集群等资源优势，发挥行业组织、 科研机构和广大农机科学家群体的智慧，推进农机装备补短板、 强弱项，提升农作物耕种收综合机械化水平，加快推进丘陵山区农业 机械化向全程全面高质高效发展，为全面推进乡村振兴、加快建设农 业强国提供有力的装备支撑。经研究决定，举办第三届中国丘陵山区 农业机械博览会暨中国(永康)农林装备博览会(以下称“永康农林 装备博览会")。

永康农林装备博览会以搭建沟通交流平台，聚集行业资源，提升 产业服务理念，展示农业领域新质生产力发展成果，强化农机产业全 链条产学研推用合作，推动全社会优质资源力量进入丘陵山区农机装 备领域，以持续推进丘陵山区农林装备高质量发展为主线，打造 以浙江为首的特色鲜明的国际、国内的农林装备展示交易盛会。

永康农林装备博览会已成功举办两届，每届展览规模5万平方米， 600余家展商参展参会，成为浙江省内近年来参展农机综合性企业 多，展示机具全、配套活动丰富、专业观众人数多、规模  的农林装备博览会之一，得到了全社会和农机行业的广泛关注。

真诚欢迎并邀请各有关单位积极参展参观，现将2025永康农林装 备博览会有关事项函告如下：

一、组织单位

(一)主办单位：

中国农业机械化科学研究院集团有限公司 中国农业机械学会

浙江中国科技五金城集团有限公司

(二)承办单位：

中国包装和食品机械有限公司

浙江中国科技五金城会展有限公司

(三)支持单位：

浙江省农业机械学会、浙江省农业机械工业行业协会、浙江省农 机装备创新研发推广联盟

二、博览会构成及具体活动安排

博览会采取“1+1+6”模式，“1+1”即为主题博览会和1个主题 大会，“6”代表6个辅助活动。具体活动安排如下：

(一)主题博览会：

展会时间：2025年3月6-8日

展会地点：浙江省永康市永康国际会展中心

展会规模：60000平方米

参展对象：国内外丘陵山区农机装备生产企业、智能装备生产企 业、科研院所、高校、高新技术企业及技术研发机构、零部件企业等。

展馆设置：

(1)综合品牌馆：展示展销国内外品牌动力机械、水稻种收机械、 粮油作物机械、畜牧水产养殖机械等农业综合机械。

(2)数智创新馆：展示展销国内外先进的智能装备、数字化信息 化技术平台、农用无人机、无人农场管理系统、农用导航软件、农业 传感器等及科研院所创新产品展示交易区。

(3)浙江品牌馆：展示展销浙江地区头部企业农机装备产品。

(4)丘陵山区机械馆：展示展销国内外先进的果菜茶菌机械、园 林机械、喷灌灌溉设备、设施农业装备、园艺工具等。

(5)特色机械馆：展示展销植保机械、机电、水泵及零配件。

(6)新产品、新技术成果展示交易区：展示展销科研院所、高校 及高新技术农机企业新科研成果等。

(7)室外机械展示区：展览展示轨道运输、无人驾驶、大型高端 农业机械展示及试乘试驾等。

(二)主题活动

中国丘陵山区农机产业高质量发展大会

(三)六大辅助活动：

2025中国农机行业形势分析研讨会

第三届中国农机品牌大会暨第17届全国农机用户满意品牌颁奖活动

农机企业新品发布会

科研院所农机研发成果专场推介及对接会

农机购置与应用补贴政策培训活动

农机装备试乘试驾及现场演示活动

三 、展位形式及收费标准

1.标准展位(6m×3m):8000      元/个/展期，双开口加收20%; 

2.室内光地展位：360元/平方米/展期(36 m2起 租 ) ;

3.室外光地展位：260元/平方米/展期(36 m2 起 租 ) ;

特别说明：为扩大博览会的展会规模及影响力，博览会组委会将对参展企业 进行展位费全额补贴(详情请见补贴通知)。

四、参会人员

1.邀请农业农村部、商务部等领导，相关行业协会、学会、 商会负责人参会。

2.邀请浙江省领导及相关厅级领导，金华市领 导及相关部门领导参会。

3.邀请各省、自治区、直辖市农业农村部门领导，供销总社系统、 农机科研院所等主要负责人，大专院校的专家、学者参会。

4.邀请农业装备产业技术创新战略联盟企业成员及部分行业企业 负责人、研究中心负责人参会。

5.邀请国内、外农业机械企业，农业产业化重点龙头企业， 农业农机专业合作社，农机经销商、农垦农场及用户参会。

6.邀请科技日报、经济日报、农民日报、中国农机化导报、农业  机械杂志、中国农业信息网、中国农业机械化信息网、 农机360网、农机通、农机1688、浙江电视台、浙江之声以及今日头条、百度、新浪等新媒体进行报道。

五、保障措施

(一)保障。制定保障、消防管控和应急处置预案，落 实会场、展馆、宾馆、餐厅、车辆、应急区域和相关场所的和应 急保障措施，确保、有序办展。

(二)技术服务。组委会在博览会官方网站及微信公众号上为参 展代表团和企业免费提供网上产品信息及企业形象展示服务。

(三)新闻宣传。设立博览会新闻中心，组织新闻媒体进行采访 报道。同时，利用新媒体线上同步直播或线上发布博览会情况。
