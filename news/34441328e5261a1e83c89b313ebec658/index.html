【关注】2024中国一东盟博览会旅游展区27-28日免费开放,公交出行更方便!(内有独家“剧透”)

最美象山 2024-09-25


9月27日-28日

2024中国一东盟博览会旅游展区

来啦！

里面有什么？

热腾腾的独家“剧透”来啦





2024中国一东盟博览会旅游展区将在桂林市国际会展中心举行，为做好会展期间公交服务保障工作，方便市民参观出行桂林市国际会展中心，桂林轨道集团公交公司拟在2024年9月27日-28日会展期间，开通4条公交专线、1条定制公交线路，并调整K2路。具体如下：


01

K2路：分时段服务桂林市国际会展中心


1.线路走向：线路沿原线运行至金融大厦后，左转世纪西路，经三元路、山水大道至终点桂林日报社，经金融大厦、临桂大道西（吾悦广场）站沿原线返回。


2.沿途站点：

公交东环车场→东环市场→普陀路→朝阳路口→三里店广场北口→辅星路口→穿山桥→安新小区→上海路口→机场路口→翠竹路→黑山路口→鼎富大厦→桂康路口→金山广场→金山广场西口→世纪东路→世纪东路环湖路口→世纪东路中→博元小学→新城国奥小区→山水大道（行政中心）→发改住建大厦→青莲路中→人和路→桂林图书馆→公园北路→金融大厦→桂林日报社。


3. 临时服务会展中心发车时间：9:00、11:00、13:00、15:00


4. 票价：2元/人，票证通用。

02

1号线：辅星路口—桂林日报社

1.线路走向：由辅星路口甲天下广场始发，经漓江路、翠竹路、鲁山路、两江大道、西城大道、世纪西路、三元路至桂林日报社。


2.沿途站点：辅星路口→穿山桥→安新小区→黑山路口（万象城）→金水路口→西城北路北→临桂大道西（吾悦广场）→桂林日报社。


3.服务时间：9:00辅星路口→桂林日报社（首班车）；12:00桂林日报社→辅星路口；14:00辅星路口→桂林日报社；17:30桂林日报社→辅星路口（末班车）。


4. 票价：2元/人，票证通用。

03

2号线：商贸城—桂林日报社

1.线路走向：由商贸城始发，经中山北路、中山中路、中山南路、翠竹路、鲁山路、两江大道、西城大道、世纪西路、三元路至桂林日报社。


2.沿途站点：商贸城→北极广场→乐群路口→十字街中心广场→阳桥→桂林站→机场路口→西城北路→临桂大道西（吾悦广场）→桂林日报社。


3.服务时间：9:00商贸城→桂林日报社（首班车）；12:00桂林日报社→商贸城；14:00商贸城→桂林日报社；17:30点桂林日报社→商贸城（末班车）。


4. 票价：2元/人，票证通用。

04

3号线：金山广场西口—桂林日报社

1.线路走向：由金山广场西口始发，经金山路、世纪东路、西城大道、公园北路、三元路至桂林日报社。


2.沿途站点：金山广场西口→世纪东路→世纪东路环湖路口→世纪东路中→世纪东路西→西城中路北→桂林日报社。


3.服务时间：9:00金山广场西口→桂林日报社（首班车）；12:00桂林日报社→金山广场西口；14:00金山广场西口→桂林日报社；17:30桂林日报社→金山广场西口（末班车）。


4. 票价：2元/人，票证通用。

05

4号线：万福广场-桂林日报社

1.线路走向：由十一中学（万福广场）始发，经凯风路、瓦窑西路、崇信路、中山南路、上海路、翠竹路、鲁山路、两江大道、西城大道、公园北路、三元路至桂林日报社。


2.沿途站点：十一中学（万福广场）→凯风路北→瓦窑口→崇信路→桂青路→翠竹路→金水路口→西城北路南→临桂大道西(吾悦广场)→桂林日报社。


3.服务时间：9:00十一中学（万福广场）→桂林日报社（首班车）；12:00桂林日报社→十一中学（万福广场）；14:00十一中学（万福广场）→桂林日报社；17:30桂林日报社→十一中学（万福广场）（末班车）。


4. 票价：2元/人，票证通用。

06

定制公交线路

1.线路走向：开通雁山大学城定制公交线路，由桂林理工大学雁山校区始发，途经南宁理工学校（桂林校区）、桂林卫校、桂林学院、广西师范大学（雁山校区）、桂林旅游学院等高校，经雁山中心环线、321国道、万福路、万平路、凤凰路、山水大道、三元路至桂林日报社。


2.服务时间：9:00-15:00


3.票价及购票方式：15元/人，微信公众号预定，依托桂林公交微信公众号，以按需响应的模式，由乘客在计划出行前一天17:30前，通过桂林定制公交平台订购次日出行车票，购票达到40人及以上线路正常运行，班次正常发车；如订购人数未达到预定人数，系统将提前一天下午17:00进行退票，乘客可采取其他交通方式。



温馨提示



参观会展的市民在桂林日报下车后，往北步行约150米即到桂林国际会展中心，返回市区的乘客也在桂林日报社站上车。
