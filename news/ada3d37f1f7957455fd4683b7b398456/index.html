第139届广交会境内外办证全攻略(2026)

琶洲会展 2026-04-02

展期：4.15-4.19（一期）、4.23-4.27（二期）、5.1-5.5（三期）

核心规则：一人一证、实名核验、严禁转借；境内采购商需交易团邀请码，境外采购商线上自助注册


一、境内采购商证（最核心）


✅ 申请条件（必须满足）


- 境内注册企业、非本届参展商、主营进口/批发/零售/电商、经营范围匹配展区

- 年营收门槛：≥5000万元（2025年度），提供纳税申报表/审计报告证明

- 名额分级（按营收）：

- 普通（5000万+）：3个付费名额

- VIP（2亿+）：3免费+3付费

- 重点VIP（10亿+）：5免费+5付费 


💰 费用标准


- 免费证：全届通用、免费

- 付费证：300元/天 + 50元/张工本费；连续多天办1张（只收1次工本费）；不连续/跨期需办多张、工本费另算

- 补办：200元/张，限1次；审核通过/付费后不退费、不可撤销 


📅 申请时间（关键节点）


- 一期：截止4.10（4.10前申请可免费邮寄）

- 二期：截止4.19

- 三期：截止4.27

- 名额有限、额满即止，建议提前7天提交 


📝 必备材料（电子版+盖章）


1. 境内采购商参会申请表（交易团模板）

2. 营业执照（经营范围含批发/零售）

3. 2025年营收证明（税局盖章申报表/审计报告）

4. 参会人员身份证正反面、蓝底/白底证件照（40×50mm）

5. 企业派员证明（盖章） 


🛠️ 办理流程（4步走）


1. 企业向属地交易团（商务局） 提交材料→审核→获取邀请码（无邀请码无法注册）

2. 登录广交会官网采购商平台（https://www.cantonfair.org.cn）→填邀请码→注册/激活账号→上传材料

3. 选择免费/付费名额、选展期/天数→缴费（付费）→等待终审

4. 审核通过：4.10前申请免费邮寄；之后→展馆D区登录大厅/自助机现场领证/激活；凭电子证+人脸入场 


二、境外采购商证（含港澳台）


✅ 申请条件


- 境外企业/机构采购负责人、港澳台采购商、驻华外企代表；个人游客不可办

- 无需邀请码、线上自助注册、免费（全届通用）


📝 必备材料


- 护照/港澳通行证/台胞证（有效期≥6个月）

- 证件照（白底/蓝底、清晰免冠）

- 境外企业名片/注册证明（可选，加速审核）


🛠️ 办理流程（3步）


1. 官网/广交会境外采购商APP→注册账号→填个人/企业信息→上传证件+照片

2. 在线人脸核验→等待审核（1-3个工作日）

3. 审核通过：生成电子采购商证；现场可在办证点/自助机打印实体证；凭电子证/实体证+护照入场


陪同证（境外采购商专属）


- 名额：1境外采购商每期最多2名（1助手+1翻译）；港澳台/中国护照采购商仅可办助手证

- 费用：300元/天 + 50元工本费；翻译需提供CATTI/外语专业毕业证

- 流程：境外采购商注册后，在系统内添加陪同、提交材料、缴费、审核领证


三、参展商证（境内/境外员工）


✅ 适用对象


本届参展企业员工、展位工作人员；不可兼办采购商证


📝 材料


- 境内：身份证、证件照、近3个月社保证明/在职证明、企业编码、展位确认书

- 境外：护照、证件照、在职证明、企业编码


🛠️ 流程


1. 企业登录参展易捷通→新增/续办人员→提交交易团审核

2. 员工用广交会供应商APP完成人脸核验

3. 审核通过：老IC卡→现场激活；新办→现场领证；一人一证、一期一证


四、常见问题&避坑提醒


1. 境内采购商必须先找交易团拿邀请码，直接官网注册无效

2. 免费证全届通用；付费证按天、不连续需分开办

3. 所有证件实名、人脸核验、不可转借，违者拉黑

4. 邮寄截止4.10，之后只能现场领；建议提前办、避开开展前高峰

5. 补办仅1次、200元；付费后不退费
