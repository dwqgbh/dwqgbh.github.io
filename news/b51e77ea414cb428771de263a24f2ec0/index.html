elexcon2025亮点回顾|创龙科技:多款重磅新品,亮相“深圳国际电子展暨嵌入式展”!

elexcon电子展 2025-09-02

2025年8月26-28日，第22届“深圳国际电子展暨嵌入式展”，在深圳会展中心(福田)举行。作为国内年度重磅“嵌入式嘉年华”，本次展会以“ALL for AI, ALL for GREEN”为主题，为快速发展的AI与双碳提供全栈技术与供应链支持。



创龙科技携首款“工业单板机”重磅亮相，并展出50+款工业核心板，0.7s秒开、266帧高速图像处理、ARM + FPGA异构多核等方案，吸引了诸多开发者驻足咨询。

（1分钟云游深圳国际嵌入式展）


▍全新业务：工业单板机


创龙科技今年推出了全新业务：工业单板机，首款产品采用瑞芯微RK3562（4核Cortex-A53@2GHz + 1TOPS算力NPU）处理器为主控，并在本次大会强势亮相，产品特点如下：

（1）可通过扩展板实现多DI/DO、多串口、多网口等功能。
（2）扩展板原理图、PCB开源，并提供30天快速定制服务。
（3）提供4款现成整机，通过CE、FCC、RoHS等认证，产品上市快速。
（4）整机采用无风扇工业设计，支持导轨式、壁挂式、桌面式安装。



▍异构多核：赋能工业应用


ARM + FPGA异构多核产品是创龙科技多年来的独特差异化，广泛应用于工业自动化、能源电力、仪器仪表、医疗、通信、安防等行业。本次我们展出了如下ARM + FPGA异构多核产品：

（1）瑞芯微RK3588/RK3576/RK3568 + 紫光同创FPGA
（2）全志T3 + 紫光同创FPGA
（3）安路DR1、复旦微FMQL20SM
（4）TI Sitara + Xilinx FPGA
（5）Xilinx MPSoC、Zynq-7045、Zynq-7020


▍5款工业方案：助力产品开发


本次我们带来了5款工业方案动态演示，为工业用户提供了不同场景的应用解决方案，让客户产品开发更简单。

（1）ARM + FPGA多通道AD采集方案
基于瑞芯微RK3576 + 紫光同创Logos，实现8通道200KSPS同步并行采样、数据处理、波形显示功能。


（2）ARM + FPGA高速图像采集方案
基于瑞芯微RK3588 + 紫光同创Titan-2（可选贴Xilinx Kintex-7），实现266fps高速图像采集、处理与显示功能。


（3）低成本工业HMI秒开方案
基于全志T113-i双核ARM Cortex-A7 + RISC-V C906，实现0.7s显示开机动画、5.2s进入应用界面功能。创龙科技T113-i核心板工业级配置含税仅需79元，性价比超高。


（4）4K高清多屏异显方案
基于瑞芯微RK3568实现HDMI + MIPI +LVDS的高清多屏显示功能，分辨率可高达4K。


（5）EtherCAT多轴电机控制方案
基于瑞芯微RK3588 + 翼辉国产实时系统，在任务周期250us下实现4us抖动的电机控制功能，性能表现卓越。



▍媒体专访：深耕工业不动摇


本次大会现场，创龙科技业务总监蔡金杰先生接受媒体采访，他介绍道：创龙科技发展到现今近200人员规模，依靠的关键策略是长期“坚持深耕工业不动摇”，我们始终致力于打造高品质工业核心板，已成功帮助超过25000+家工业客户完成产品的快速开发与上市，不断为实时系统、现场总线、物联网、人工智能等技术提供应用平台。



▍大会获奖：开发者最喜爱奖


创龙科技很荣幸获得本次大会的“最受工程师喜爱的嵌入式领域大奖”。一直以来，创龙科技都以“提供高可靠性产品、快速响应，有效解决”作为服务宗旨，并且秉持“因我们的存在，让嵌入式应用更简单”为使命，持续为嵌入式开发赋能。
