通知:2026年全国优特钢棒线材精益智造与产品性能提升关键技术研讨会邀您相约淮钢特钢!

钢铁技术网 2026-07-02

关于召开“2026年全国优特钢棒线材精益智造与产品性能提升关键技术研讨会”的通知


各钢铁企业及有关单位：

    在钢铁行业高质量发展的关键时期，棒线材行业正加速向装备差异化、工艺差异化驱动产品差异化的方向演进。依托极致效率与极致能效，降低吨钢制造成本，进而实现产品品种丰富化、性能高端化与企业竞争力跃升，已成为行业共识。为更好地服务棒线材生产企业，搭建共商、共建、共享的交流平台，推动中国棒线材产业健康、可持续、高质量发展，工匠人冶金科技发展中心联合江苏沙钢集团淮钢特钢股份有限公司，定于2026年8月5—7日在江苏·淮安召开“2026年全国优特钢棒线材精益智造与产品性能提升关键技术研讨会”。会议主题为：“拓产品·控组织·提性能·铸精品·赋智能”。

    会议拟邀请北京科技大学、东北大学、上海大学、中冶赛迪、中冶华天、钢铁研究总院等行业专家，以及相关单位论文作者共同参与研讨。

    诚挚邀请贵司董事长、总经理；生产、技术副总；轧钢厂、总工办、生产制造部、技术质量部、技术中心、设备部等部门的工艺技术负责人；设计院校、装备等负责人及相关单位人员出席。

    现就会议有关事宜通知如下：

一、会议征文及报告研讨内容：

（一）高效、精益、智能轧钢与高端装备技术

1.数字孪生/大数据/AI在线智能检测在棒线材精益生产中的深度应用

2.轧线全流程自动判定/动态速度补偿与微张力/无张力闭环控制

3.粗轧高刚度轧机/‌精轧/减定径机组装备精度与稳定性提升

4. 精准控冷/高效穿水/气雾冷却系统与柔性冷却工艺控制

5. 连铸坯直接轧制/热装热送工艺及配套保温装备

6. 轧钢工序节能、余热回收与绿色低碳生产技术

（二）轧制工艺与组织性能调控关键技术

1. 压缩比设计对优特钢组织、性能与探伤的影响

2. 低温轧制与变形制度对组织细化的作用

3. 轧制工艺参数对棒线材通条性能均匀性控制

4. 大盘卷表面质量与内部组织协同控制技术

（三）控轧控冷与组织性能精准调控 

1. 优特钢控轧控冷组织演变规律与性能调控机制

2. 轧后冷却路径对强韧性匹配的影响

3. 冷床均匀冷却技术与组织均匀性提升

4. 小规格线材圈形稳定与组织性能协同控制

5. 在线组织性能精准调控与免调质工艺实现

（四）典型优特钢组织缺陷专项控制

1. 轴承钢网状碳化物、液析碳化物控制技术

2. 弹簧钢、齿轮钢表面脱碳全流程控制

3. 优特钢带状组织、中心疏松、夹杂控制技术

4. 冷镦钢、焊丝钢组织与拉拔性能匹配控制

5. 高端特殊钢疲劳寿命与组织稳定性控制

（五）高附加值优特钢棒线材新产品开发

1. 大盘卷特殊钢新品种开发与组织性能保障

2. 免退火/免调质/免球化退火优特钢产品开发

3. 高强度/高韧性/高疲劳寿命优特钢产品开发

4. 齿轮钢/轴承钢/冷镦钢/帘线钢等产品开发

5. 工程机械用高强韧合金棒线材产品开发

6.  高端装备用优特钢棒线材产品开发与组织设计

7. 稀土耐蚀普碳钢产品开发与应用

8. 高强钢筋、耐腐蚀钢筋开发进展

二、会议组织机构：

主办单位：工匠人冶金科技发展中心

协办单位：江苏沙钢集团淮钢特钢股份有限公司

媒体支持：中国冶金报、中国钢铁新闻网、轧钢之家、钢铁技术网
