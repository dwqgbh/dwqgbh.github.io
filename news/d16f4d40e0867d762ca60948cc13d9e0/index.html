阅会展 2026-03-30

中国国际工业博览会（CIIF）是国家级工业展会，更是全球工业风向标的核心平台。2026 年工博会以「工业新质 智造无界」为主题，汇聚全球顶尖工业资源，是企业展示实力、链接商机、提升品牌的黄金窗口。

📌 展会核心信息速览

  • 时间：2026.10.12-16

  • 地点：国家会展中心（上海）

  • 定位：国家级工业展会・全球工业风向标

  • 四大核心功能：展示、交易、评奖、论坛

  • 参展：Lily Ye18516756117


🛠️ 十大专业展，覆盖工业全产业链


2026 工博会精心打造十大专业展，精准覆盖工业自动化、机器人、数控机床、新能源、工业互联网等核心领域，为不同赛道企业提供专属展示舞台：

  • 工业自动化展（IAS）：运动控制、智能传感、工业软件、系统集成

  • 机器人展（RS）：工业 / 协作 / 人形 / 移动机器人、机器视觉、核心部件

  • 数控机床展（MWCS）：金属切削、激光加工、钣金冲压、机床功能部件

  • 智慧能源展（ES）：核电、风光储氢、智能电网、虚拟电厂、综合能源

  • 信息与通信展（ICTS）：工业 AI、工业互联网、5G/6G、云计算、信息安全

  • 新材料产业展（NMIS）：新能源 / 汽车 / 航空 / 电子 / 医用 / 生物基新材料

  • 芯工业未来展（NICE）：芯片设计、制造、封测、车规芯片、工业芯片

  • 智行未来展（FTIS）：未来出行、智能驾驶、汽车芯片、核心零部件

  • 绿色低碳展（GLCS）：零碳园区、节能装备、工业降碳、全生命周期管理

  • 科技创新展（STS）：科研成果、高校创新、高新技术、孵化项目
