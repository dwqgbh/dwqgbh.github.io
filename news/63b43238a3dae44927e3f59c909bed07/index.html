#1


广东佳的美智能科技有限公司

#

企业介绍

广东佳的美智能科技有限公司总部位于广东省清远市，另设广州分公司、深圳分公司。佳的美是一家集研发设计、生产制造、市场营销于一体的多元化电子产品品牌商、制造商，是首批获国家高新科技认证的企业。公司占地31万多平方米，内有五星级园林式厂区，环境优美，目前拥有员工2,000多人，约200项技术专利与多个专业技术授权。公司建有省级科技创新实验室，拥有强大的自主研发能力和完善的市场营销网络。

#

企业所在地

广东省清远市。

#

成立时间

1998年。

#

所属行业

科学研究和技术服务业。

#

主营业务

电子产品、集成电路；软件和信息技术服务业；科技推广和应用服务业；电子产品贸易；货物及技术进出口。

#

特色产品

充电器、热熔枪、移动电源、便携显示器等。

#

主要市场

日本、韩国、澳大利亚、美国、德国。


#2


中山市诚熙进出口有限公司

#

企业介绍

中山市诚熙进出口有限公司成立于2019年4月，是一家专注于进出口贸易的企业，涵盖多种产品，包括但不限于五金制品、塑料制品、电子产品及家居用品等。凭借专业的团队和广泛的市场渠道，公司致力于为客户提供高效、优质的国际贸易服务。近年来，公司年进出口代理量超3,000万美元。

#

企业所在地

广东省中山市。

#

成立时间

2019年。

#

所属行业

贸易代理业。

#

主营业务

代理出口包括但不限于五金制品、塑料制品、电子产品及家居用品等。

#

特色产品

广告牌、抽油烟机、办公桌椅等。

#

主要市场

东南亚、中东、澳大利亚等。

#9

广东省凯力鑫中药科技有限公司

#

企业介绍

广东省凯力鑫中药科技有限公司成立于2020年9月，由冬鹤堂（惠州）中药科技有限公司核心控股，主要生产“澳门三宝”相关产品，拥有著名保健品品牌“冬鹤堂”，其产品纯草本中药配方，药食同源，兼具食品级品质和药用功效，通过多项权威食品安全检测。产品已在300+港澳药房商超上架，在12个电商平台均有销售。

#

企业所在地

广东省惠州市。

#

2020年。

#

主营业务

药品销售、货物或技术进出口。

#

“澳门三宝”——心宝护脑舒络胶囊、胃保养胃护胃胶囊、祛湿排毒身轻胶囊。

主要市场

中国、新加坡、马来西亚、泰国、马拉维、英国、法国、刚果。

 建行全球撮合家

为促进全球经贸开放合作共赢，便利跨境贸易与投资，2019年末中国建设银行推出B2B跨境智能撮合平台“建行全球撮合家”。平台整合多方资源，以“科技+数据”搭建开放共享生态，旨在解决跨境交易场景下信息不对称和交易成本高等痛点难点，为境内外企业跨境贸易与投资提供商机发布、智能匹配以及综合服务解决方案。
