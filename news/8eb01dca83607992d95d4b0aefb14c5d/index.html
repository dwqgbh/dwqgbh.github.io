副会长谢竞青率珠海侨商赴郑州开展2026 “侨兴南粤” 经贸交流专场活动

广东省侨商会 2026-06-04



活动提要

TECHNICAL REPORT

为深入贯彻落实省“侨助经济高质量发展”工作部署，扎实推进“2026‘侨兴南粤’系列专场活动”走深走实，深挖豫珠两地侨资产业互补优势，搭建跨区域侨商资源对接平台，2026年5月28日至29日，珠海市委统战部侨务工作科科长黄毓飞，广东省侨商投资企业协会副会长、珠海市侨商企业协会执行会长谢竞青带队赴河南郑州、许昌等地，开展为期三天的企业走访与交流座谈活动，实地考察河南标杆侨商企业与特色产业项目，共叙侨商情谊，共拓合作新机。






PART 01

政企座谈叙侨情

TECHNICAL REPORT



5月28日，在河南省侨商联合会党支部书记罗建中、执行会长陈向辉陪同下，考察团重点走访了河南省侨商联合会执行会长单位——瑞美真发，实地参观智能化生产车间、品牌陈列展厅及党建中心，全方位了解发制品全产业链研发、海内外市场布局与企业发展历程。

谢竞青结合珠海会员企业产业特点表示，以本次“2026‘侨兴南粤’珠海市侨商赴郑州经贸交流专场活动”为契机，珠豫两地商会可建立常态化联络机制。协会会员深耕植发医疗器械产业，与瑞美真发产业链高度互补，合作空间广阔；同步邀约河南侨商组团到访珠海参观中国航展，借力粤港澳大湾区资源拓宽海内外市场。与会企业家围绕跨境渠道共建、海外侨胞互助、产品供需对接深入交换意见，达成多项初步合作意向。

座谈交流会上，许昌市委统战部侨务工作科科长段雍笛对珠海侨商考察团到访表示欢迎，围绕许昌区位禀赋、特色支柱产业、侨务营商环境及惠侨政策进行详细推介，热忱欢迎珠海侨商落地许昌投资兴业。


珠海市委统战部侨务工作科科长黄毓飞立足珠海区位与粤港澳大湾区发展优势，推介了珠海侨资产业集聚环境、高新产业扶持政策与城市发展机遇，诚挚邀约河南省侨商会组织企业家回访珠海，依托“2026‘侨兴南粤’系列专场活动”载体落地双向经贸合作。


河南省侨商联合会执行会长、瑞美真发董事长陈向辉介绍了河南省侨商会会务建设、会员服务体系与中原侨商产业版图，依托许昌全球发制品产业集群优势，提出与珠海侨商在医美植发、上下游配套、跨境贸易等领域深化联动。

最后，河南省侨商会党支部书记罗建中作总结发言。他代表商会欢迎大家来到许昌，并介绍了省侨商会自换届后展现出的蓬勃生机——现有1000余名会员，涵盖河南名优产品各领域，会员企业质量高、产业链完备。他强调，加入侨商会首先要有“侨的背书”，大家要诚信经营，坚守道德底线。他指出，河南企业相对保守，应向南方企业学习品牌建设和专利布局，打造自己的护城河，积极走出去。他祝愿珠海侨商会一行在河南考察企业的同时，多走走、多转转，感受河南的人文地理与深厚文化。









PART 02

走访中原优质侨企

TECHNICAL REPORT


这次“2026‘侨兴南粤’珠海市侨商赴郑州经贸交流专场活动”行程紧扣医疗健康、工程建设、医疗器械等重点产业，考察团先后走访河南省侨商会多家副会长单位，点对点开展产业对接。

华夏健康集团

在华夏健康集团，考察团参观中药材仓储中心、原生态百草园与品牌直营店，深度调研中医药全产业链布局、药食同源新品研发模式，对企业传统中医药现代化创新路径给予高度评价，探寻珠海康养市场与中原道地药材产业落地合作路径。

中机六院

在中机六院参访环节，郑州市政府副秘书长李峰、河南省侨商会副会长赵辉陪同参观企业展厅与技术研发中心。赵辉详细介绍工程设计、绿色建造、产业园区规划等七大核心业务。

谢竞青结合珠海城市更新、工业园区建设需求，洽谈工程技术落地合作，邀约企业赴珠实地踏勘，推进战略合作协议落地。




参观实况


仁惠医疗

在仁惠医疗，河南省侨商会秘书长柴莉、副会长管世昌陪同参观自研医疗器械展厅，考察团现场体验自研理疗医用设备，细致了解企业医疗器械研发、临床落地与产业化布局。双方约定依托“2026‘侨兴南粤’系列交流平台”，打通产品在珠海代理、华南市场拓展的合作通道。







PART 03

对标标杆民企范本

TECHNICAL REPORT


交流期间，考察团赴胖东来天使城、大咖国际茶饮梦工厂两家中原标杆企业实地研学。


在胖东来天使城，考察团深入学习企业人性化运营管理、服务体系建设与品牌运营思路，借鉴民营企业精细化管理先进经验。


在焦作温县大咖国际茶饮梦工厂，调研茶饮原料智能化生产、全链条供应链建设模式，学习食品加工智能制造与品牌市场化运营经验，为珠海本土消费类企业转型升级提供参考。







PART 04

总结

TECHNICAL REPORT

为期三日的“2026‘侨兴南粤’珠海市侨商赴郑州经贸交流专场活动”圆满落幕。活动以侨为纽带，打通粤港澳大湾区与中原地区侨商互联互通渠道，既深化了豫珠侨界情谊，又围绕医疗、健康、智能制造、轻工消费等多个领域敲定合作方向，切实落实了“2026‘侨兴南粤’系列专场活动”以侨促贸、以贸兴产的活动宗旨。下一步，珠海市将持续深化“2026‘侨兴南粤’系列主题经贸活动”，充分发挥市委统战部（市侨务局）的统筹协调作用，立足珠海、联动全国各省市侨商组织，持续为侨商企业搭建外出参访、招商对接、资源互通的优质平台，发挥侨商资源优势，助力珠海实体经济高质量发展。
