【国企招聘】覆盖硕本！济南二机床集团2026年招聘公告

一、单位简介

济南二机床始建于1937年，是大型国有企业，中国机床行业重点骨干企业，国家高新技术企业。主导产品包括锻压设备、金切机床、自动化设备等，服务于汽车、轨道交通等多个国家重点领域。其大型冲压设备国内市场占有率达85%，国际市场占有率达35%，被誉为“世界三大数控冲压装备制造商之一”。企业拥有国家企业技术中心、博士后科研工作站等国家级研发平台，坚持“事业留人、感情留人、环境留人、机制留人、待遇留人”的人才战略，为人才搭建广阔的成长平台。

二、招聘要求（条件）

（一）基本条件

  1. 拥护中国共产党的领导，政治立场坚定，遵纪守法。
  2. 品行端正，责任心强，具有良好的沟通协作能力和团队精神。
  3. 身心健康，具备适应岗位要求的身体条件和心理素质。
  4. 无违法犯罪记录，无不良诚信记录。

（二）资格条件

  1. 国内普通高等院校本科及以上学历毕业生；或经教育部认证学历学位的境外院校本科及以上学历毕业留学生。
  2. 毕业时须取得规定的学历、学位证书，国（境）外学历须获得教育部留学服务中心认证。
  3. 符合招聘岗位的专业要求，具有履行岗位职责所必需的专业知识和良好的职业素养。

（三）禁止报考情形

  1. 曾受过刑事处罚的，涉嫌违法犯罪被立案调查的，因违反《治安管理处罚法》受到行政机关处罚的。
  2. 被开除中国共产党党籍、开除公职的，曾参加非法组织、邪教组织或从事其他危害国家安全活动的。
  3. 被列为失信被执行人且尚未履行义务的。
  4. 在各级公务员、事业单位招考中被认定有舞弊等严重违反录用纪律行为的。
  5. 正在接受审计、纪律审查，或涉嫌犯罪、司法程序尚未终结的人员。
  6. 报名后构成回避关系的人员。
  7. 法律法规规定不得聘用的其他情形的人员。

三、招聘岗位

共招聘4类岗位：

  • 机械设计研发岗：专业要求机械、数学等相关，学历要求硕士及以上。
  • 电气设计研发岗：专业要求自动化、控制、电气等相关，学历要求硕士及以上。
  • 电气调试岗：专业要求自动化、控制、电气等相关，学历要求本科及以上。
  • 工艺技术岗：专业要求机械、材料成型、焊接等相关，学历要求本科及以上。

四、福利待遇

提供行业具有竞争力的薪酬及福利待遇，特别优秀者可“一人一议”。

五、报名时间和报名方式

  • 报名时间：自公告发布之日（2026年2月6日）起至 2026年5月31日。
  • 报名方式：统一通过公司招聘门户网站进行网申。
    • 报名网址：http://222.173.34.54:8443/recruit
    • 需上传材料：个人简历、成绩单、相关证书等。

六、其他

（一）招聘流程

  1. 资格审查：根据招聘条件进行审查和简历筛选。
  2. 综合测试：包括线上测评和面试。测评合格者进入面试，面试重点考察职业道德、专业知识和业务能力。
  3. 考察与体检：对通过人员进行综合考察，并统一组织体检。
  4. 录用：根据综合测试、考察和体检情况，择优录用并签订劳动合同。

（二）注意事项

  1. 资格审查贯穿招聘全过程。应聘者应对提交材料的真实性负责，弄虚作假者将被取消资格。
  2. 招聘各环节通知将通过短信、邮件或电话发出，请保持通讯畅通。
  3. 本次招聘不收取任何费用，不指定考试辅导用书，不举办也不委托任何机构举办辅导培训班。
  4. 公司有权根据情况调整、取消或终止个别岗位的招聘工作。
