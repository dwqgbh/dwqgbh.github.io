西门子Mobile Worker Lite 获 2024中国国际工业博览会 智能制造奖

Siemens过程工业软件 2024-09-24


在2024中国国际工业博览会开幕式暨颁奖与仪式期间，西门子Mobile Worker Lite数智工厂AR移动应用软件（以下简称西门子Mobile Worker Lite）从数百件参评展品中脱颖而出，斩获2024中国国际工业博览会智能制造奖（以下简称智能制造奖）。














什么是西门子Mobile Worker Lite？

西门子Mobile Worker Lite是一个移动终端应用软件，安装后，手持终端可以化身为操作控制器，通过软件即可获取工程设计数据、图纸、文档、三维模型和设备运行状态等关键信息。

西门子Mobile Worker Lite极大地简化了业务处理流程，将现场人员工作模式从人找数据变成数据找人，随时随地的获得企业运营所需操作指引、实时数据和工程文档。工程师们不再需要繁琐地查阅和携带大量文件，可以随时随地利用手机、平板、智能眼镜等移动应用设备完成各种业务场景的处理，大大提高工作效率，有效提升现有数据的价值，助力企业数字化转型与升级。

Mobile Worker Lite能做什么？


软件主要依靠数据集成与传输、数据检索与呈现、AR交互三个方面实现相关功能，是西门子工业智能数字化实践的又一生动产品。

数据集成与传输：

Mobile Worker Lite可以将设备资产数据与DCS/PLC数据进行集成，将中控室的实时数据与档案室的图文文档通过网络推送到生产现场，工程师们能够轻松浏览各种数据资料。

数据检索与呈现：

Mobile Worker Lite支持设备文档、图纸、视频与DCS/PLC动态数据一体化呈现，也可以通过移动终端扫描二维码、位置码、蓝牙、NFC等，定位设备并获取DCS/PLC数据及设备数据，监控设备实时运行状态，查阅设备相关资料。

AR交互：

Mobile Worker Lite采用了AR增强现实技术提升用户的交互性，混合现实世界与数字世界，能够通过移动终端直接在设备上显示出设备ID、名称规格、动态数据，一目了然。

在这三项技术的基础上，西门子Mobile Worker Lite支持智能巡检等业务应用，提供巡检计划、巡检项及标准、巡检路径规划、巡检点签到、巡检清单生成及结果上传、巡检过程记录、问题跟踪处置等功能。

西门子Mobile Worker Lite已经被广泛应用于多种工业场景，例如某大型制造企业利用Mobile Worker Lite显著减少了设备维护时间，提高了生产效率。Mobile Worker Lite能够提供详细的工作指导和实时数据反馈，帮助工程师在现场高效地完成复杂的维修和维护任务。

智能制造奖的授予肯定了西门子Mobile Worker Lite的技术创新，和推动工业数字化、智能化的成功实践。目前西门子Mobile Worker Lite数智工厂AR移动应用软件已上架西门子Xcelerator平台，西门子将持续致力于工业转型，为企业提供更多能够真正落地使用的数字化产品，推动工业的数字化转型和创新发展，积极投身中国智能制造浪潮，见证更多的灿烂与辉煌。
