龙华企业注意!参加进博会可领补贴啦

中融说企服 2026-03-04

龙华区商务局发布最新通知：第八届中国国际进口博览会深圳龙华交易分团（企业团）资助类 已开始受理！只要你是龙华区企业，参加了第八届进博会（含相关活动），就有机会领取4000元参展补贴！如果还签了大单，额外再奖1万元！

一、谁能申请？

  • 注册地要求：参展时企业在龙华区依法注册、具有独立法人资格。

  • 信用要求：未被列入严重失信名单，近三年无严重违法违规行为。

  • 参展要求：

    • 有到场参展记录（以深圳市商务局或展会主办方提供的入场记录为准）；

    • 或参加了深圳市在展会期间举办的相关活动（以市/区商务局确认的活动名单为准）。

二、补贴多少？

  • 基础补贴：符合条件的参展企业，每家 4000元。

  • 额外奖励：在展会期间签署合同，且 签约额超过1000万美元 的企业，额外再奖 1万元。

三、什么时候申报？

  • 申报时间：2026年3月4日 — 2026年3月18日

  • 时间仅有两周，请务必抓紧，逾期不再受理！

四、需要准备什么材料？

  1. 《第八届中国国际进口博览会深圳龙华交易分团参展企业补贴申请书》

  2. 营业执照复印件（多证合一新版）

  3. 法定代表人身份证复印件；经办人身份证复印件、法人授权委托书、经办人社保或劳动合同证明（非法人办理需提供）

  4. 申请1万元额外资助的：须提供展会期间签约额超1000万美元的有效合同复印件（涉及外文的需附中文翻译件）

所有材料先上传至 广东政务服务网 预审，通过后再提交纸质材料（胶装、盖章、一式两份）。

五、办理流程

  1. 线上申报：登录广东政务服务网 → 深圳市 → 龙华区 → 龙华区商务局 → 找到“中国国际进口博览会深圳龙华交易分团（企业团）资助类”在线填报。

  2. 预审通过后打印材料，提交至龙华区政务服务中心窗口。

  3. 区商务局审查 → 征求意见 → 公示5个工作日 → 资金拨付。

特别提醒

  • 基础补贴和额外奖励可叠加享受，最高可领 1.4万元！

  • 签约合同须真实有效，弄虚作假将依法追责。

  • 申报时间短，建议企业立即整理参展记录和合同材料，尽早申报！
