2024第三届江西·进贤(李渡)医疗器械经销商大会暨医疗器械展览会开幕

美年大健康贵州 2024-03-25


3月23—25日，2024贵州健康产业及医疗器械博览会(简称“医博会”)在贵阳国际会展中心隆重举行，贵州省人民政府原副省长，省政协原副主席孙国强出席并讲话，贵州省国际国内公共关系协会会长齐新潮代表主办单位致辞。


/ 贵阳美兆分院院长石雁冰（右六）与出席会议领导合影 /

贵阳美年大健康是唯一参展的体检机构代表，其旗下的体检中心——贵阳美兆健康医疗（简称：贵阳美兆），是一家将核磁共振作为标配的高品质体检中心。3月23日-24日，贵阳美兆展台吸引了众多参展的领导嘉宾和群众。此次医博会现场，贵阳美兆健康医疗主要介绍体检设备有：日本东芝3.0T冠脉核磁共振、西门子1.5T超导磁共振、飞利浦64排128层CT、机器人磁控胶囊胃镜、法国AIXPLORER系列E超。

/ 贵阳美年大健康展位号：2536、2537 /

日本东芝3.0T冠脉核磁共振

贵阳美兆引进原装进口冠脉核磁共振，该设备具有无创伤(不需要心脏插管)、风险小的特点。与冠脉CT和冠脉造影相比:冠脉核磁无放射辐射的伤害，耗时短，不需要注射任何造影剂，避免造影剂对肾脏伤害与过敏等反应。


西门子1.5T超导磁共振

西门子超导磁共振，大孔径设计，有效避免检查者的幽闭恐惧感;全数字高清，光纤超导，真正16接收通道，检查时间短，图像清晰,全景成像技术，超静音技术，国际听力协会认证。



飞利浦64排128层CT

飞利浦64排128层CT,连续扫描能力强，射线质量稳定，图像细节显示更丰富，图像更清晰。10秒完成全身扫描，可进行横断面、矢状面、冠状面等任意平面的图像重建及多平面成像。每周旋转缩短至0.4s，一次屏气18s，可以完成胸、腹、盆腔的连续扫描。根据客户胖瘦描绘出 mA 曲线图，受检者接触的射线剂量最多可降低 50%。

机器人胶囊胃镜

机器人磁控胶囊胃镜，只需随水吞服胶囊胃镜机器人，便能以无痛、无创、无不适、无麻醉、无交叉感染的原则，实现了“不插管，做胃镜”。这颗“神奇的胶囊”在医生的操控下可以在胃内进行运动，按照所需的角度对病灶拍摄，每次磁控胶囊胃镜检查会生成大量图像，胃里的情况一览无余。检查结束后随正常排泄排出体外，一次性使用不回收。


法国AIXPLORER系列E超

E成像超声拥有极速的超声平台，是传统超声系统采集图像速度的200倍之多。这种极速平台是可以同时处理两种波的超声平台:纵波形成高质量的灰阶图像，而横波可以使医生实时准确、无创重复地看到并分析组织的硬度。这项革新技术，又名E成像(SWE™)可以应用于乳腺、甲状腺、肝脏、前列腺等人体各组织器官筛查。


贵阳美兆先进的体检设备吸引了现场众多领导嘉宾。中共贵州省委办公厅“服务决策专家智库”专家、贵州省国际国内公共关系协会会长齐新潮携参会嘉宾一行巡视贵阳美兆分院展台。

/ 齐新潮会长在开幕式上致辞 /

/ 齐新潮会长一行巡视贵阳美年大健康展台 /
