宝鸡机床集团简介


宝鸡机床集团有限公司始建于1965年，目前是我国重要的中高档数控机床研发生产基地和智能制造基地，主营业务收入多年稳居中国机床行业前列。


公司坚守“匠心智造中国好机床”的企业使命，秉持“满足用户需求就是企业生存的底线”的市场理念，坚持科技创新与市场开拓“双轮驱动”，建立起集人才、技术、品牌于一体的完备研发制造生产体系，形成车削类机床、铣削类机床、自动化产线3大产品群，共14大类、200多个品种、400个规格，广泛应用于高端装备制造领域。承担国家重大专项30多项，在全球拥有五大营销服务中心。


公司拥有国家数控系统工程技术研究中心和国家级技能大师工作室，充分应用工业互联网、大数据、人工智能等新技术，不断推进生产、管理、服务数字化，加快实现单机智能化向产线智能化转型，致力于为用户提供完整的生产制造解决方案。公司汇聚了一大批职工创新团队、技术精英和能工巧匠，多次承办国家、省、市职工职业技能大赛。

02


入职福利

入职即享五险一金，按实际工资足额缴纳，公积金按 12% 顶格缴纳，骨干员工额外享有企业年金。同时配备企业补充医疗保险、高额意外险及女工大病险，医疗保障更全面。


严格执行国家法定带薪假期，包括年假、婚假、产假、育儿假等，工龄满一年即可享受工龄工资。


提供免费员工公寓，家电齐全可拎包入住，外地员工优先安排；设有员工食堂，餐补标准充足。


覆盖市区主要片区的免费通勤班车，出行便利；按规定发放高温补贴、取暖补贴、洗理费、住房补贴等各类津贴。


拥有完善培训体系，实行师傅带教，支持技能考证与学历提升，考证费用可报销。设置专业技术与管理双晋升通道，每年定期考工晋级，晋升即涨薪。


作为省属国企，工作稳定、制度规范，抗风险能力强，适合长期稳定发展。

03


薪资水平

一、研发 / 技术岗（本科及以上为主）

机械 / 电气 / 工艺工程师：8000-15000 元，应届生起薪约 8000 元，3-5 年经验可达 12000-15000 元，资深 / 骨干 15000-20000 元。
数控编程 / 工艺员：7000-10000 元。
质量 / 设备运维：6000-9000 元。
铸造技术员：4000-7000 元。

二、技能操作岗（中专 / 中技为主）
数控机床操作工：6000-8500 元，含技能津贴、加班费。
装配钳工 / 调试师傅：7000-13000 元，经验越丰富薪资越高。
检验员：6000-8000 元。
生产普工 / 辅助岗：4500-6000 元。

三、职能 / 管理 / 销售岗
行政 / 人事 / 财务：5000-7500 元。
销售 / 市场：底薪 5000-8000 元 + 提成，综合收入 8000-15000 元。
基层管理（班组长 / 主管）：8000-12000 元。

四、薪资结构与激励
结构：基本工资 + 岗位津贴 + 绩效奖金 + 年终奖金（通常 2.5-3 个月工资）。
补贴：持证补贴（高级技师 / 工程师 1000-1500 元 / 月）、高温补贴（6-9 月 300 元 / 月）、取暖补贴、餐补、交通补贴等。
晋升：技术 / 管理双轨，每年考工晋级，涨薪同步。
