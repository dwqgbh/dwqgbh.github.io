2025 亚太智慧城市奖项评选启动

中国国际高新技术成果交易会经国务院批准，自1999年以来高交会已成功举办二十六届，被誉为“中国科技第一展”。

为了更好地推动新型智慧城市建设发展，国家信息中心与亚洲数据集团共同承办的智慧城市发展论坛（简称：论坛）迄今已连续成功举办十一届，是高交会的重要组成部分之一，是高交会百余场活动中唯一聚焦于智慧城市领域的专业论坛活动，今年论坛将以“智启未来，筑梦城市”为主题，聚焦“AI+算力、政务、企业服务、信创”等核心领域为数字经济高质量发展提供底层支撑与创新引擎。

与论坛同期举办的“2025 亚太智慧城市奖项评选”作为智慧城市领域重要展示活动之一，以展示宣传亚太区当下最具代表性的优秀智慧城市建设项目案例、最具实力和影响力的智慧城市服务提供商，为城市提供优秀案例的设计和建设经验，为城市选择合适的服务商提供帮助和建议。


参评条件

2025亚太领军智慧城市、2025中国领军智慧城市

  1. 参评城市必须是以政府机构为申报单位；

  2. 参评城市必须有完整的智慧城市解决方案；

  3. 参评城市必须在2024年1月－2025年9月实施过至少1个智慧城市项目；

  4. 参评城市必须保证所提供资料和数据的真实性和完整性。


评选设置

2025中国领军智慧城市

  1. 中国领军智慧城市；

  2. 中国领军智慧县级城市；

  3. 中国领军智慧城区；

  4. 中国领军智慧园区。


参评条件

2025亚太领军智慧城市服务商、2025中国领军智慧城市服务商

  1. 参评企业必须是独立注册法人机构；

  2. 参评企业必须有自己的网址；

  3. 参评企业必须有自己的智慧城市解决方案或所推选的产品、解决方案、应用案例或项目已投入实际使用；或已动工、已通过产品评测、方案评估或项目验收且能在2025年上半年投产、投入使用或推出市场；

  4. 智慧城市类参评企业必须在2024年1月－2025年9月参与实施过至少1个智慧城市项目；或该产品、方案、应用实践在所属行业中起到一定的实验性、示范性作用；或该产品、方案、应用实践在有一定影响力，被行业广泛认可；

  5. 近两年获得过县、区级别或以上的，地市或以上级别社会组织、第三方权威机构颁发的相关行业荣誉资质并能提供资料佐证的；

  1. 参评企业必须保证所提供资料和数据的真实性和完整性。


评选设置

2025中国领军智慧城市服务商

  1. 综合类奖项：中国智慧城市领军解决方案提供商；

  2. 技术类奖项，包括：中国智慧城市领军云计算解决方案提供商、中国智慧城市领军人工智能解决方案提供商、中国智慧城市领军大数据解决方案提供商、中国智慧城市领军物联网解决方案提供商、中国智慧城市领军信息安全解决方案提供商、中国智慧城市领军智能制造解决方案提供商。

  3. 主要子市场奖项，包括：中国领军智慧政务解决方案提供商、中国领军智慧园区解决方案提供商、中国领军智慧统计解决方案提供商、中国领军智慧医疗解决方案提供商、中国领军智慧交通解决方案提供商、中国领军智慧水务解决方案提供商、中国领军智慧能源解决方案提供商、中国领军智慧环保解决方案提供商、中国领军智慧教育解决方案提供商、中国领军智慧社区解决方案提供商、中国领军智慧城管解决方案提供商、中国领军智慧楼宇解决方案提供商、中国领军智慧旅游解决方案提供商等；

  4. 数字化创新奖：旨在表彰在算力经济和数字经济背景下，以数据共享、数据转型、数据要素、元宇宙、AIGC、虚拟现实和增强现实等为中心的重点产业，布局建设全球领先的数字化发展高地，打造全国领先的新型数字化、智能化标杆，打造数字化应用场景试点示范工程，为数字化转型升级做出创新贡献的企业。

  1. 现场权益：获奖者将获得奖杯、证书等荣誉证明，并在全国范围内得到广泛宣传和推广。

  2. 通过媒体、网站、微信公众号、现场展示等多种宣传渠道对所有参选和获奖单位及方案进行持续性的展示与宣传；

  3. 优秀的单位简介资料、技术创新资料、产品或方案或应用示范案例将编撰入亚太智慧城市评选报告或数字化、智慧城市建设案例集；

  4. 获邀参加智慧城市项目对接会闭门会议，与国内多个数字经济、智慧城市建设单位的主管领导和产业链上下游专家、企业家、甲方开展深度交流；

  5. 主办方提供奖项专用标识，入选企业可在本公司网站及其他企业宣传资料上使用该标识。

  1. 将根据参评单位信息反馈和评选结果，编撰入“亚太智慧城市评选”报告；

  2. 将选择有代表性的项目或成果，进行案例分析和经验推广；

  3. 除了会后报告、线上（官方网站、微信公众号等）公开发布外，主办方将根据实际情况对高峰论坛、颁奖典礼等线下发布方式进行评估，并及时通知各参选单位。

  4. 所有参选单位所提供的信息将仅用于本次“2025亚太智慧城市评选” 活动，主办方对各参评单位的资料承担保密义务。
