特种设备相关安全法规知识全解-2025新版
文件编号：20471 

以下是中国特种设备相关的主要法律法规及标准体系，涵盖法律、行政法规、部门规章、安全技术规范和最新政策动态：

一、核心法律

1.《中华人民共和国特种设备安全法》

- 效力层级：法律，特种设备领域的根本大法。

- 主要内容：

- 定义特种设备范围（锅炉、压力容器、电梯、起重机械等八大类） 。

- 明确生产、经营、使用、检验、检测全链条的安全责任，要求建立分类监督管理制度 。

- 规定特种设备采用新材料、新技术、新工艺需通过技术评审 。

- 修订动态：2025年9月市场监管总局提出加快修订，重点完善风险防控和应急处置机制。

二、行政法规

1.《特种设备安全监察条例》

- 效力层级：行政法规，细化法律实施细则。

- 主要内容：

- 明确特种设备生产许可、使用登记、定期检验等具体流程 。

- 规定检验检测机构资质要求及监督管理 。

- 修订动态：2025年送审稿新增“特种设备检验属于公益事业”表述，强化检验机构公益性。

三、部门规章及规范性文件

1.《特种设备现场安全监督检查规则（2025新版）》

- 效力层级：部门规章，规范监管部门现场检查流程。

- 主要内容：

- 要求制定年度检查计划，突出风险导向。

- 明确对生产、经营、使用单位的检查内容（如资质、设备档案、应急演练等）。

- 对严重隐患设备可采取查封、扣押等强制措施。

2.《特种设备安全沙盒监管制度》（2025年试行）

- 政策背景：应对新材料、新技术、新工艺的潜在风险，允许企业在可控范围内试制试用 。

- 主要内容：

- 沙盒试验需在特定时间、场所进行，同步监测风险并消除隐患 。

- 申请单位需制定试制方案，购买产品责任保险 。

- 适用于安全技术规范未明确要求的“三新”特种设备 。

3.《缺陷特种设备召回管理规则》

- 政策背景：强化事后监管，消除同一性缺陷设备的安全隐患 。

- 主要内容：

- 生产单位为召回责任主体，需停止销售、发布公告并实施召回 。

- 召回流程包括技术评估、整改验证和效果公示 。

四、安全技术规范（TSG系列）

1.《特种设备使用管理规则》（TSG 08-2017）

- 主要内容：

- 规定使用单位需办理使用登记、建立安全管理制度 。

- 明确人员配备要求（如安全管理机构设置条件） 。

- 要求定期检验、维护保养并保存记录 。

- 修订动态：2025年计划修订，重点完善电梯、起重机械等设备的使用管理细则。

2.其他技术规范

- 《电梯安全技术规程》：2025年加快修订，强化电梯全生命周期安全管理。

- 《场（厂）内专用机动车辆使用安全治理行动方案》：2025年部署专项整治，打击无证作业、超期未检等行为 。

五、标准及地方规定

1.强制性国家标准

- 如《液化天然气罐车》《二甲醚钢瓶》等，2025年计划立项修订。

- 标准内容涵盖设备设计、制造、检验等具体技术指标。

2.地方条例

- 各省、自治区、直辖市可根据本地实际制定补充规定，例如：

- 北京市《电梯安全监督管理办法》。

- 广东省《特种设备安全条例》。

- 获取方式：通过地方市场监管部门官网查询。

六、执行与监督

1.监管部门：

- 国家市场监督管理总局统筹全国特种设备安全监督 。

- 县级以上地方市场监管部门负责现场检查和隐患治理。

2.法律责任：

- 对未登记、未检验、违规作业等行为，最高可处30万元罚款。

- 生产单位未召回缺陷设备的，责令改正并处罚款 。

七、最新政策动态（2025年）

1.强化检验公益属性：明确特种设备检验机构的公益性定位，推动检验机构改革。

2.动态调整目录：研究建立《特种设备目录》动态更新机制，适应技术发展。

3.数字化监管：推广“智慧电梯”“气瓶追溯系统”等信息化手段，提升监管效率 。

八、获取渠道

1、官方平台：

- 国家市场监督管理总局官网（www.samr.gov.cn）：发布部门规章、技术规范。

2、专业数据库：

- 特种设备安全技术规范网（www.tsg.org.cn）：提供TSG系列规范全文。

3、地方查询：通过各省、市市场监管局官网获取地方性法规。

4、点击文末阅读原文，在安全管理资料库里面下载

建议用户根据具体设备类型（如电梯、压力容器）和业务场景（如生产、使用），结合最新法规动态及地方要求执行。
