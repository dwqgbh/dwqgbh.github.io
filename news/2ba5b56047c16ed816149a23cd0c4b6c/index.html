高质量构建中老命运共同体的新机遇、新挑战及优化路径


尹君、储金、王元丰


摘要：中老命运共同体的构建是人类命运共同体理念在次区域合作中的先行探索与实践，自2019年中老两国领导人共同签署命运共同体行动方案以来，双方在政治互信、经贸合作和人文交流等多个维度深入合作，推动中老关系跃升到新阶段和新高度。2023年，中老命运共同体第二个五年行动计划的签署为更加全面、更高水平、更加深入的高质量合作关系构建提供了实践指南，中老命运共同体构建的整体态势保持稳定且发展向好。在区域合作方面，中老命运共同体的构建客观上推动了中柬、中泰、中越、中缅的双边合作与命运共同体的构建，助推了澜湄区域的多边合作进程，为构建中的澜湄命运共同体打下了坚实基础。然而，中老命运共同体在构建进程中仍面临来自政治稳定性、经济可持续发展、文化交流互鉴以及非传统安全威胁等多方面的挑战，这些挑战对中老关系高质量发展以及中老命运共同体的行稳致远产生一定的影响。在全球局势加速演进之下，中老命运共同体构建如何逆势而上向更高质量发展，不仅关乎中老两国核心利益，而且对澜湄命运共同体的构建以及全球政治经济秩序的调整具有更深层次的意义。

从人类命运共同体理念的提出到实践，是马克思主义“真正的共同体”“自由人联合体”理念的发展与实践。马克思主义共同体思想中将共同体的形态划分为自然共同体、虚幻共同体和真正共同体三种形式。(1)人类最终实现“真正的共同体”的过程十分漫长和艰难。在实现“真正的共同体”这一最终目标前，通过世界各国和各国人民的共同努力，推动构建“人类命运共同体”。从这一意义上讲，人类命运共同体的构建是未来实现“自由人的联合体”的必经阶段。(2)中老命运共同体的构建是中国“协和万邦”“义利并举、以义为先”优秀传统思想的当代实践。“中华文明始终在兼收并蓄中历久弥新。亲仁善邻、协和万邦是中华文明的处世之道。”(3)中老两国亲邻关系紧密而源远，双方弥合文化差异和发展差距，以义为先，倡导相互尊重、公平正义、弘义融利的国际合作观，为构建新型国际关系的理念开拓实践创新。

目前，学界对中老命运共同体主要从以下几个方面开展研究：一是聚焦于中老各领域合作成果，从政治、经济、文化等各个领域来剖析构建中老命运共同体的合作基础；二是从外部因素来分析构建中老命运共同体面临的挑战，指出部分西方国家对构建中老命运共同体持偏见态度；三是以“伙伴关系、安全格局、经济发展、文明交流、生态建设”五大路径为基本遵循来阐述构建中老命运共同体的路径；四是基于中老命运共同体构建的理念与实际行动，探讨中老铁路在助推中老命运共同体建设中的关键作用。这些研究成果为本文提供了重要参考，也指明了未来可进一步探索和拓展的研究方向。本文旨在阐述构建中老命运共同体已取得合作成就的基础上，分析在大变局加速演进背景下推动高质量构建中老命运共同体所面临的时代机遇与挑战，并尝试提出具体的优化路径。



一、义利并举：助推中老命运共同体构建跃升至新高度


自2013年首次提出人类命运共同体理念以来，中国在人类命运共同体的内涵和实践方面不断推进创新，逐步形成一个以推动建设“五个世界”为总目标，以打造全球伙伴关系为新起点，以构建新型国际关系为根本路径，以全人类共同价值为价值追求，以共建“一带一路”为实践平台，以全球发展倡议、全球安全倡议、全球文明倡议为重要依托的完整理论体系。(1)2 0 1 9年中老签署《构建中老命运共同体行动计划》以来，两国积极践行人类命运共同体理念，开拓创新、和衷共济、携手前行，在各领域都取得丰硕成果，构建中老命运共同体为构建人类命运共同体打下了坚实基础，正在向着更高水平、更高质量迈进。中老双方在构建命运共同体的过程中，强调平等相待，践行正确的义利观，义利兼顾，义重于利。双方在政治、经济、文化、安全、生态合作五方面推进两国深入合作。中老命运共同体的构建，弥合双方文明差异和发展差距，超越自身“小”利益，在政治上倡导公平正义、平等相待，在经济上坚持互利共赢，不断推动双边人文交流、安全、生态环保等合作，为双边和区域共同体的构建积累经验，成为先行实践的典范。


（一）弘义融利：中老共同构建命运共同体取得显著成效


在新型国际关系构建理念方面，人类命运共同体倡导国家之间要坚持对话协商，构建对话不对抗、结伴不结盟的伙伴关系，努力构建相互尊重、合作共赢的新型关系。中老从伙伴关系到中老命运共同体的构建，双方践行“五个世界”理念，成为人类命运共同体构建的一个双边实践案例。


1. 高度政治互信成为中老命运共同体稳定发展的坚固基石


中老双方通过共同推动和维护地区和平，增进政治互信，不断把双边关系推向更高层次。自2009年中老决定共建全面战略合作伙伴关系以来，中老高层通过互访、电话交流及信函往来等多种形式，深化党际、政府及立法机关间的互动，全面交流国家治理与政策制定的经验，持续夯实中老命运共同体的政治共识。中老双方相互坚定支持各自国家的社会主义建设进程及探索符合各自国情的现代化道路，加快对接国家发展战略，积极践行“三大倡议”，执行《中老两国外交部关于深化新时代合作的协议》，加强各层级的沟通与协作，增进双多边事务中的协调与配合，以共同维护双方的利益。据中国外交部数据，2009—2024年，中老两国领导人之间互访交流、会谈会见达一百多次(1)，两国领导人针对双方共同关切的事项展开及时而深入的磋商。签署《行动计划》之后，两国领导人就推动命运共同体构建的意义及实现途径多次展开交流和对话。另外，部长级会谈也频繁举行，双方在具体合作事项上达成诸多一致意见，有力地推动了两国关系朝着健康稳定的方向蓬勃发展。


2. 互利共赢的经贸合作是中老命运共同体的核心纽带


全球发展存在失衡、数字鸿沟和公平赤字等问题，中老双方在经贸、交通、能源、技术及合作平台机制建设等领域加强协调合作，持续践行共同繁荣发展理念。中老经贸产能合作成果显著，有力推动两国协同发展。在深化贸易投资层面，2022年中老双边贸易额为56.8亿美元，2023年增至71亿美元，2024年达82.3亿美元。(1)当前，中国是老挝第一大投资国、出口国及第二大贸易伙伴。两国共同实施老挝对华出口产品零关税政策，推进口岸及基础设施建设，提升通关效率，以促进双边经贸合作与人员流动，带动边境地区社会经济发展。在平台合作机制方面，双方不断优化协调。着力高质量维护中老铁路运营并开发沿线区域，有效运营赛色塔综合开发区等重点项目，稳步推进中老磨憨—磨丁跨境经济合作区建设，开展跨境产业合作，共同构建安全稳定的产业链与供应链体系。


中老两国在交通基础设施与能源建设合作领域同样取得了显著成效。一方面，交通基础设施建设成效显著，合作修建公路、铁路和机场为老挝经济发展提供有力支撑。万万高速公路、腊满高速公路的建成通车，有力改善了老挝的国内和国际公路运输条件。以中国为主投资建设的中老铁路自2021年12月开通运营以来，国际货运业务迅速发展，与中国长三角、珠三角和环渤海三大经济圈紧密连接，对外延伸到泰国、柬埔寨、越南等东南亚国家。同时，老挝不断完善现有机场设施并新建机场，增加了直飞中国的航线。公路、铁路和新机场的建设投用，改变了老挝国内国际交通和运输条件，尤其是提升了澜湄地区的运输效率，改善了老挝的营商环境，成为拉动老挝经济增长的重要动力。另一方面，能源领域的合作是中老命运共同体建设的重要领域，中老在能源建设领域合作成果显著。中老持续深化电力合作，助推老挝国内电网实现全覆盖和现代化，助力老挝将资源优势转变为经济发展优势(2)。2020年，中国企业投资开发的老挝南欧江梯级水电站项目二期首台机组发电，有效缓解了老挝电力资源紧张形势，进一步促进老挝成为“东南亚蓄电池”，南欧江梯级水电站等中国公司投资建设的水电站，已成为老挝大型清洁能源项目的代表。(3)截至2023年8月，中老两国累计实现双向电力互济1.56亿千瓦时。(4)2 0 2 0年，中国南方电网公司与老挝国家电力公司签署共同组建老挝国家输电网公司协议，2024年老挝国家输电网公司运营启动仪式顺利举行。


3. 文明交流互鉴搭建起中老命运共同体民心相通的桥梁


中老两国持续开放包容，推动双边关系在文明交流互鉴方面取得了重要进展。一方面，两国在图书翻译与出版领域的协作正不断深化拓展。从2017年开始，中国在东南亚国家持续规划并实施“东南亚中国图书巡回展”的学术与文化交流项目，在老挝获得广泛而积极的反响。中老双方计划在2026年之前，联合翻译出版50部各自国家的经典文学作品。另一方面，青年互动为文化交流增添更多活力。老挝国立大学中文系的学生通过诵读中国经典诗词，展现传统节日习俗，传播中国传统文化。2020年，老挝赛区成功举办第19届“汉语桥”世界大学生中文比赛。老挝国立大学的副校长宋占指出，老挝国内对于中文学习的热情持续高涨，中文教育已经被正式整合进老挝的国家教育体系之中(1)。2024年，第二届老挝国际中文教育合作发展论坛在琅勃拉邦成功举办(2)，此举密切了校际中文教育合作，提升了老挝中文教育水平，培养了更多懂中文、知中国的中老友好事业接班人。


4. 多领域安全合作筑牢中老命运共同体的安全防线

（二）交织与共生：从中老命运共同体到高质量中老命运共同体的跃升


中老命运共同体是区域命运共同体的一个先行实践典范。相较于其他双边和区域命运共同体，中老命运共同体率先被提出和付诸实践，为其他命运共同体的构建提供经验和借鉴。中老命运共同体的提出、建设和推进，总体上可分为全面合作共识达成、全面战略合作伙伴关系建设、构建中老命运共同体三个阶段。


第一，全面合作共识达成阶段。中老两国建交后一直保持友好合作的关系，进入21世纪后，中老两国增加了高层互访和战略对话。2 0 0 0年，中共中央总书记、国家主席江泽民对老挝进行国事访问，两国签署了双边合作的《联合声明》等6个文件。2006年，中共中央总书记、国家主席胡锦涛对老挝进行国事访问，双方签署多项合作文件，深化了两党两国传统友好与全面合作。此阶段两国在“长期稳定、睦邻友好、彼此信赖、全面合作”十六字方针的指引下，两国关系继续蓬勃向前发展，各领域交流与合作不断扩大。


第二，全面战略合作伙伴关系建设阶段。此阶段中老两国都有相互合作、促进发展的战略诉求，为构建中老命运共同体打下坚实基础。2 0 0 9年9月，中共中央总书记、国家主席胡锦涛与访华的老挝人民革命党中央委员会总书记、国家主席朱马里达成共识，决定把中老关系提升为全面战略合作伙伴关系。(1)2 0 1 3年9月，习近平主席同来华访问的朱马里会谈时首次提出，“中老关系不是一般意义的双边关系，而是具有广泛共同利益的命运共同体”。(2)2 0 1 6年5月，中老两国领导人达成共识，一致强调携手共建坚如磐石的中老命运共同体理念，双方均对此倡议表达了赞同与支持。(3)2 0 1 7年11月，习近平主席对老挝进行国事访问。其间，习近平主席与老挝人民革命党中央委员会总书记、国家主席本扬重申了中老两国作为“具备战略重要性的命运共同体”的共识，并一致决定共同致力于构建一个牢不可破且具有战略意义的命运共同体。(4)这一共识的达成，有力地推进了“中老命运共同体”理念向实际行动的转化进程。


第三，构建中老命运共同体阶段，此阶段彰显了双方命运的共生交织性。2019年4月，两国签署《行动计划》。中老两国遵循《行动计划》所规划的路径和时间表，在经济合作、政治互信、安全保障、文化交流以及生态保护这五大关键领域展开深入而具体的构建合作。2 0 2 3年1 0月，为了更好地规划未来两党两国关系的发展路径，并推动中老命运共同体建设高质量、高水平地持续稳定发展，双方经过协商，决定续签并执行新的为期5年的行动计划，即《中国共产党和老挝人民革命党关于构建中老命运共同体行动计划（2024—2028年）》。(1)2 0 2 4年1 0月，习近平主席在喀山会见老挝人民革命党中央总书记、国家主席通伦时指出，在中国周边外交布局中，中老关系占据着关键的战略地位，中老命运共同体的构建始终走在区域命运共同体建设的前沿。(2)


当前，中国正致力于全面推进中国式现代化建设，老挝也正处于国家发展的关键时期，双方都有增进政治互信、深化战略合作的意愿。在第一个五年《行动计划》的指引下，构建中老命运共同体已经取得诸多成效，构建中老命运共同体需要向着一种更高质量、更高水平、更加紧密、全面与深入的双边合作关系发展。中老双方领导人也强调“继续秉持长期稳定、睦邻友好、彼此信赖、全面合作方针和好邻居、好朋友、好同志、好伙伴精神，坚持政治上互尊互信、经济上互惠互利、安全上相守相助、人文上相知相亲、生态上共生共治，携手共建高标准、高质量、高水平的中老命运共同体，为构建人类命运共同体作出积极努力与示范”。(3)



二、势义相向：高质量构建中老命运共同体的新机遇


推动构建人类命运共同体是推动构建新型国际关系的最终目标，而推动构建新型国际关系则是推动构建人类命运共同体的基本路径。(1)构建人类命运共同体要靠不断推动构建新型国际关系来达成。目前，国际格局发生巨大变化，同时国际秩序进入到深刻调整阶段，全球面临战争、疾病、气候变暖等共同的威胁，在应对霸权、贸易保护主义、经济恢复与发展等方面，中老有共同的战略需求。中老两国在双边和国际事务中有诸多共同认知和利益，这为高质量构建中老命运共同体提供了有利条件。中国提出的相互尊重、公平正义、合作共赢的新型国际关系理念，为双方构建命运共同体的实践提供了路径支持。中国“一带一路”倡议和“三大文明”倡议为中老命运共同体建设提供价值引领、实践平台和重要依托。


（一）构建新型国际关系理念和“三大倡议”为中老命运共同体注入政治合作新动力


以构建新型国际关系和“三大倡议”为代表的中国理念和中国力量为构建中老命运共同体提供重要动力。长期友好交往的历史奠定了中老两国民心相通的基础，人类命运共同体理念在中老两国深入人心。


第一，构建新型国际关系是构建人类命运共同体的根本路径。新型国际关系理念以相互尊重、公平正义、合作共赢为基本内涵。相互尊重是首要前提、公平正义是核心准则、合作共赢是基本取向。这为中老命运共同体从政治、经济、文化、安全、生态合作领域构建新型双边关系提供了路径支持。


第二，“三大倡议”的提出为中老命运共同体建设提供新的价值引领。2023年中国“三大倡议”的提出，旨在缩小全球发展差距、应对国际安全难题、超越文明间的隔阂与冲突，同时为中国及世界各国合作推进人类命运共同体的构建明确了道路与方向。“三大全球倡议”从发展、安全和文明三个维度指明人类社会前进方向，彼此呼应、相辅相成，为推动构建人类命运共同体提供了坚实支撑。(2)三大倡议的提出，为中老命运共同体的构建提供了新的价值引领，也为双边关系在安全、发展、文明方面的合作与实践提供理论基础，增强合作动力。中老在应对全球安全局势恶化、经济合作环境恶化、文明冲突与对抗加剧等方面具有共同的需求，双方在安全、发展和文明方面的合作与交流，为双方对抗风险增强力量，也为地区注入更加稳定、和谐和发展的有利因素。


第三，中国发展所积累的力量为中老命运共同体构建提供动力支撑。当前，全球进入动荡与变革交织的新历史时期，各类全球性挑战呈现频发多变且难以预判的特征。中国在国际事务中的作用日益凸显，成为维护世界和平与促进共同发展的关键力量。在全球治理体系中，中国以更积极主动的姿态参与其中，成为国际公共产品的重要供给者与守护者。作为当今世界的重要行为体及人类命运共同体理念的积极倡导者，中国综合国力与国际影响力的持续提升，为深化国际合作、推动全球治理向更加公正合理的方向演进提供了有力支撑，也为中老两国深化政治、经济、文化等领域合作，以及构建命运共同体奠定了更坚实的基础。


（二）“一带一路”倡议、次区域合作机制及中老铁路助力中老经济合作

第一，深入共建“一带一路”为中老命运共同体构建提供新机遇。中国“一带一路”倡议与老挝当前推动经济社会发展的战略高度契合，这为构建高标准、高质量、高水平的中老命运共同体创造了有利条件。经济上，老挝作为中国的陆上接壤邻国，拥有得天独厚的地理优势。老挝发展战略与“一带一路”倡议有效对接，对老挝的经济社会发展具有重要推动作用。对外战略上，借助“一带一路”倡议所带来的互联互通便利，大力促进老挝与各国的互动合作，进一步拓宽其外交深度与广度。在社会与文化层面，中老加强人文交流，为经济合作提供基础，堪称“一带一路”倡议文化交流实践的典范。老挝积极参与“一带一路”建设，不仅致力于加强对外文化交流，还积极引进中国的先进管理经验，从而促使本国的发展与现代化、市场化等要素实现深度融合。

中老两国在构建中老命运共同体的过程中，秉持相互尊重、公平正义、开放包容的原则，超越传统理念中对“技术”的竞争与零和思维，让技术的创新为构建共同体服务，“技术”成为助推双方合作与推动生产力发展的重要工具，弥合发展差距，为双边合作开拓更新更广阔的空间。

中老两国在数字经济、人工智能等领域也存在合作的机遇。人工智能等技术的快速发展与应用，为两国在科技创新、产业升级等领域开辟了广阔合作空间。双方在数字经济领域的合作，推动了经济协同发展，拓展了贸易投资规模，进而深化了经济联系与相互依存度。以合作实践为例，中国与东盟曾将2020年确定为“中国—东盟数字经济合作深化年”。在同年举行的中国—老挝企业家交流研讨会上，老挝总理通伦·西苏里提出，“双方应在数字经济、区块链技术、电子产品制造等前沿领域开展深度合作，携手并进，共同推进老中命运共同体的构建与发展。”(3)2 0 2 4年5月，老中联手打造的老挝制造电子市集在老挝上线，而专为中国消费者定制的老挝农产品电商平台也将于近期上线。据第8届中国—南亚博览会东南亚馆老挝展区老挝科技有限公司工作人员介绍，上述实践不仅是老挝电商平台的推广，更是两国数字合作的生动写照。近年来，老挝政府一方面在国内出台支持数字产业发展的政策举措，另一方面积极参与国际数字产业相关论坛、展会等活动，以推动数字化转型合作。老挝国家工商会北京代表处首席代表王飞指出，“数字化正加速推动东南亚各国经济社会发展，老挝通过与中国，尤其是与云南开展数字化合作，为国内各行业数字化发展注入动力。”(1)数字经济与人工智能领域的深度合作，既为中老经济协同发展提供动力，也为中老命运共同体构建提供了新的路径与合作空间。



三、局势跌宕：高质量构建中老命运共同体面临新挑战


高质量构建中老命运共同体面临多方面的挑战。从国际局势变化层面，全球治理失衡，单边主义、贸易保护主义、地区热战、金融战、贸易战和科技战不断出现，国际局势变化速度较快，不可预判性增强。从大国博弈层面，美国遏制中国的手段和领域正在密集加强和拓展，中老关系的深入发展，使得美国在老挝乃至澜湄地区范围内对中国的竞争和对抗加剧，这对老挝的全球发展战略和政治心态具有重要影响，客观上使老挝在中美间选择的压力更大。在澜湄地区层面，中老命运共同体的构建改变了湄公河区域的传统地缘政治格局，牵动了湄公河区域越南、泰国和柬埔寨的神经，促使各国的地区和大国战略发生改变。在中老双边的合作层面，主要是发展的阶段性不同，经贸合作结构深入调整给双方带来挑战。


（一）政治与安全挑战：国际局势复杂变化、大国博弈、区域地缘政治格局演化


中老命运共同体的构建处于一个不稳定的国际环境中，在深入和持续合作方面带来新的挑战。


1. 国际秩序新旧交替带来的全球政治与安全合作急剧变化


当前，国际秩序正处于解体与重建的过渡期，旧有秩序崩溃而新的秩序尚未形成，全球局势变得更加不稳定，大国之间的紧张关系持续存在。一方面，随着新兴国家群体的迅速崛起、区域性组织力量的不断壮大以及跨国集团影响力的日益提升，冷战后所形成的国际权力结构已经丧失其原有的力量基础，美国及西方国家为维护原有权力地位，其政局及对外政策变化较快，对抗和冲突加剧。另一方面，世界百年未有之大变局正在加速演进，国际战略环境急剧恶化，特别是美国对中国的遏制与打压，以及以美国为主导的北约在全球范围内的无限扩张，这两大因素显著加剧了全球安全形势的紧张，将世界推向了大国对峙的险境。俄乌战争改变了二战以后形成的俄罗斯与欧洲的地缘政治格局，俄乌谈判加速了欧洲、美国和乌克兰三方的分裂，推动全球局势发生裂变。欧洲通过防务法案后，迅速推动建立独立的军事国防体系。中东巴以冲突不断打破国际法底线，战场不断拓展，形式变化脱离常规。美国在全球范围内尤其是亚太地区对中国的遏制和战略对抗，使得亚太地区国家在中美间难以抉择。美国特朗普政府上台以后，内外政策“失序”。同时，全球正面临前所未有的传统安全与非传统安全威胁，大国间的竞争焦点已扩展到经济、科技、文化、网络、数据、人工智能、生物安全、太空、极地和深海等多个安全领域。从历史的角度来看，这些挑战对国际关系格局的形成和瓦解紧密相连。在新旧国际秩序交替的情况下，构建中老命运共同体的外部国际安全形势复杂严峻，传统与非传统安全威胁交织，全球局势动荡不安，其所面临的挑战前所未有。


2. 澜湄地区地缘政治格局演化加剧


澜湄地区地缘政治环境中的复杂因素，如领土争端、民族矛盾等，都对中老命运共同体的建设产生潜在影响。老挝在中南半岛的地缘战略位置使它成为周边国家乃至区域外大国竞相争取地缘政治优势的关键点。周边国家在面对中国时，虽然总体上秉持合作与共同发展的理念，但仍不可避免地掺杂着一些消极情绪。例如，在湄公河流域水电开发领域，中老双边合作受到越南一定程度的影响。(1)越南方面对湄公河上游大坝建设可能给其至关重要的三角洲带来的粮食及水资源安全“威胁”表示担忧。面对越南对中老水电合作项目的谨慎态度，老挝认为自身作为一个资源匮乏的小型经济体，唯有通过实施“东南亚电池战略”方能更有效地促进经济发展。老挝采取了较为谦卑的姿态，旨在最大限度地降低中老合作可能对老越关系造成的负面影响。中老铁路开通以来，越南、泰国、柬埔寨等国心态发生不同程度的变化，与中国合作的意愿加强，地区地缘政治格局进一步演进。湄公河区域的地缘政治竞争加剧，这些国家间的主权争议及其他利益上的不一致，都为高质量构建中老命运共同体的进程增添不确定性。


3. 美国对中国的遏制加强，破坏中老合作成为美国的遏制手段


老挝被美国视作遏制中国发展的一个重要战略支点。虽然中老命运共同体本质上契合中国与老挝的共同利益，彰显中老利益纽带的加固，但美国基于对华战略竞争的考量，试图将中老命运共同体的概念地缘政治化，将中国推进命运共同体建设的努力解读为谋求地缘政治优势的行为。针对中国与周边国家在命运共同体框架下开展的经贸合作、基础设施项目等互利共赢的活动，美国将其“泛安全化”，在国际舆论上加以歪曲，将这些正常的贸易往来、投资活动以及基础设施建设合作等描绘成所谓的“安全威胁”，从而使其成为双多边合作中的敏感议题。对中老合作的大型开发项目，美国刻意放大其对自然环境的潜在影响，大肆宣扬中老在水电、采矿及其他基础设施建设项目合作中的负面效应，极力鼓吹项目进一步加重老挝的债务负担。这些宣传和舆论对高质量构建中老命运共同体产生不良影响。

（三）共同认知挑战：文化差异仍需弥合，文化交流平台有待完善


首先，中国和老挝两国在历史文化、共同认知、价值观念和生活方式上存在差异。在文化层面，老挝是一个多民族佛教国家，其人口中有90%信奉南传上座部佛教(1)，其佛教思想融入老挝人民生活中的方方面面。在高质量构建中老命运共同体时，必须与老挝佛教这一重要因素相协调。其次，老挝国内对于构建中老命运共同体这一构想的认知与接受度并不高，这给中老两国的合作项目推进带来一定阻碍。老挝民众对于这一源自外部概念的了解和接纳程度远低于中国民众，部分原因在于民众的历史观念以及教育水平差异。以中老铁路沿线民生项目为例，部分老挝民众因受历史记忆中外部介入模式的影响，对项目的长期效益持有谨慎态度。再次，民间交流平台的搭建与机制不够完善，文化交流的频率、规模和深度需要提升。比如，川圹高原的苗族村寨与中国云南跨境民族的传统贸易虽有百年历史，但尚未建立系统化的文化互鉴机制，导致手工艺传承、口述历史等非物质文化遗产未能成为增进理解的有效载体。


（四）安全领域的挑战：毒品犯罪新变化、新型跨国犯罪、公共卫生安全领域的新问题


中老面临毒品犯罪新变化、新型跨国犯罪、公共卫生安全等非传统安全威胁，(2)这些威胁不仅对老挝可持续发展造成困扰，也对高质量构建中老命运共同体形成挑战。


首先，毒品犯罪的新变化。毒品犯罪活动日益呈现出跨国性与复杂性的新特征。毒品犯罪的跨国性和复杂性使得两国在禁毒合作中面临诸多挑战。毒品经由走私、运输等手段跨越国界或地区界限，渗透至他国民众生活，严重损害一部分人的生活，同时对社会安全和经济发展构成威胁。老挝政府通过立法和国际合作加强了禁毒力度，例如，修改刑法、实施国家禁毒总体规划，并与东盟国家及中国等邻国合作打击毒品走私，取得了显著进展。但是，毒品走私活动依然活跃。(3)2 0 2 3年11月2 5日，老挝副总理兼和平与安全部部长维莱·拉坎蓬将军在第九届国民议会第六次常会上表示，通过实施2021—2023年国家戒毒议程，老挝累计侦破毒品案件10843起，逮捕涉案人员15966人（其中包括324名外籍人员），查获毒品种子及粉末32111.78千克、化学品260余吨，并排查出10900名吸毒人员。(1)数据显示，老挝虽持续保持对毒品犯罪的高压打击态势，但毒品问题尚未得到有效遏制。


其次，新型跨国犯罪威胁双方安全。老挝人口贩卖、电信诈骗、非法资源开采等跨国犯罪活动相对活跃。人口贩卖方面，一些不法分子利用老挝边境管理的薄弱环节，实施偷渡、拐卖妇女儿童等犯罪。洗钱、电信诈骗等新型犯罪在老挝也有所增加。这些犯罪活动呈现出集团化、网络化、新型化、远程操控等特点，使得打击难度大大增加。这些跨国犯罪活动严重威胁到老挝国内的安全与稳定，同时影响中老两国在经贸、文化、安全等领域的深入合作。


再次，面临公共安全卫生领域的新挑战，传染性疾病多发。长期以来，老挝一直饱受肺结核、登革热、疟疾、艾滋病和禽流感等多种传染病的侵扰，导致严重的生命和财产损失。目前，公共卫生领域问题的新变化，尤其是层出不穷的新病毒传染病问题，已经成为全球健康与发展的共同挑战。随着中老合作的深入推进，两国之间的跨境人员往来日益频繁，数量也不断增加，使得传染病的传播速度和数量的风险增加，危及中老双方公民的生命健康。公共卫生危机也将给老挝的国家卫生体系带来沉重的负担，进一步冲击其原本就脆弱的宏观经济，影响社会稳定，从而对高质量构建中老命运共同体产生不利影响。



四、何以破局：高质量构建中老命运共同体的优化路径


目前，中老命运共同体在2019年《行动计划》的指导下，已经探索出一条符合两国人民利益与需求的道路，但高质量构建中老命运共同体是一项长期事业，不可能一蹴而至，需要做好总体布局，加强统筹协调。在方法路径方面，遵循习近平总书记提倡的“建立稳固伙伴关系、构建安全合作框架、推动经济共同发展、促进文化交流互鉴、加强生态环境保护”(1)五条核心路径。在宏观层面上，坚持以“五个世界”，即建设持久和平、普遍安全、共同繁荣、开放包容、清洁美丽的世界为努力目标，以构建新型国际关系为基本支撑，以践行全人类共同价值为普遍遵循，以共建共享共赢为实现路径，以落实三大倡议为重要引领，(2)按照从易到难、逐步推进的方式，共同努力高质量构建中老命运共同体。


（一）夯实核心路径：推动中老各领域合作向高质量阶段发展


从方法路径层面来看，构建中老命运共同体的核心路径―“伙伴关系、安全格局、经济发展、文明交流、生态建设”(3)等方面在实践中不断丰富发展，以开创中老两国更加互尊互信、更加共赢共享、更加发展繁荣、更加健康安全的未来。


1. 加强高层战略沟通构建更高水平的政治互信


政治互信始终占据着两国关系的核心位置，中老两国要致力于密切保持高层次的战略对话，为中老关系的不断前行明确方向并稳舵领航。第一，通过加强治国理念与政策层面的深入交流，确保两国领导人在历次会晤中所达成的重要共识能够得以有效实施，同时推动双方的发展战略实现精准对接，以促进双方合作项目的顺利推进和长期稳定发展。双方应持续坚守友好合作、互利共赢的基本原则，通过加强高层交往、增进民间友谊、扩大经贸往来等多种方式，加速推动高质量构建中老命运共同体的进程，实现两国关系的全面深化和长远发展。第二，在多边合作层面，两国应进一步增强沟通与协作。在“一带一路”实践中共同推进基础设施互联互通建设、经贸往来要素的快速流动、文化与文明的交流互鉴等合作，为地区经济发展注入新动力。在澜湄合作机制下，双方可加强水资源、农业、旅游等领域的合作，共同推动中老命运共同体以及澜湄国家命运共同体的建设。双方可积极参与RCEP合作，在中国—东盟自由贸易区升级版框架下进一步探讨深化经贸合作的新路径，推动中国—东盟关系实现提质升级。


2. 推动经贸合作提质升级实现更高质量互利共赢


加强中老双边经贸合作，实现共同繁荣发展，推动高质量构建中老命运共同体的进程。老挝可利用中国经济发展的机遇和良好效应，中国积极对接老挝经济发展的需求，增进两国在双边及多边层面的合作与互动，推动两国合作的效应外溢到整个东南亚地区，协同推进区域经济发展。双方可以在中国“一带一路”倡议下，继续深化务实合作，有效执行两国共建“一带一路”的合作规划纲要及重点项目和任务清单。(1)持续推进中老两国在跨境电商、移动支付、智慧城市等新兴领域的合作，提升中老经济合作质量与速率。在多边合作机制层面，主动抓住“一带一路”倡议、澜湄合作、《区域全面经济伙伴关系协定》、中国—东盟数字经济合作等机遇，在实践中积极推进交通、能源、产能、农业和数字经济等多个领域的合作与协同，将中老合作推向新的阶段，并外溢到多边合作及周边区域中。


3. 加强人文交流促进更深层次文明交流互鉴


民心相通是构建中老命运共同体最为牢固的人文联系，提供着深厚而持久的民意基础。中老两国可在多领域及多样化的框架内积极开展人文交流活动。鉴于中老两国在经贸领域的紧密联系，可将人文交流融入经济合作之中，以此来提升中老人文交流的成效。从政党合作、政府各部门合作到民间机构多层面加强合作，从政治、经济、文化、生态、卫生等领域加强人文交流与合作，构建层次清晰、内容丰富的文化交流网络。同时，可通过双方的网络、融媒体、自媒体等平台和技术手段，搭建交流平台，缩短交流距离，为民心相通搭建“数字桥梁”。加强中老跨境民族交流与合作，使之成为中老民心相通的天然纽带，助力跨境经济带，丰富中老命运共同体在文化层面的内涵。


4. 深化安全合作构筑更普遍的安全环境


中老两国应更加重视安全领域的综合与务实合作，共同推动高质量构建中老命运共同体。第一，深化原有的安全合作机制，加强双方的协作与制度协调。在打击跨境犯罪、毒品问题、贩卖人口方面不断加强执法协作，完善案件协查、情报互通、联合行动等制度性安排。在公共卫生安全、生态环境保护等非传统安全领域拓展创新合作模式，例如，建立跨境疫情联防联控机制、澜沧江—湄公河流域生态监测联合实验室，以丰富中老安全共同体的内涵与外延。第二，水资源问题是澜湄区域的敏感问题。针对湄公河水资源管理这一敏感问题，中老两国应加强协作，共同推动湄公河上下游国家间在水资源方面的协调，关注区域内外多边层面和国家层面的各方利益，在水资源开发、防洪抗旱、生态保护等敏感议题上坚持协商与合作，避免因水资源问题带来的对抗和冲突。


5. 扩大生态保护合作力度促进更高标准可持续发展


在推进中老命运共同体构建进程中，生态命运共同体的建设具有重要地位。老挝正处于工业化和现代化的快速发展阶段，这一进程不能以损害环境为代价。第一，中老两国在具体项目合作开发中，应注重生态环境的保护，避免因环境问题带来的负面舆论和损失。在跨境基础设施建设和产能合作项目开发时，双方需预先开展环境影响评估。以中老铁路为例，其沿线配套工程采用生态护坡技术，并设置野生动物迁徙通道，以此减少项目对自然栖息地的影响，防范因环境问题引发社会风险和经济损失。第二，在环境监测和保护方面，中国应尽可能地与老挝加强合作，尤其是在防灾减灾方面，避免造成双方的损失。中国与老挝可共同建设湄公河流域水质监测网络，实现水文数据和污染预警信息的共享；双方还可以联合开展澜沧江—湄公河洪涝灾害演练，并开发跨境地质灾害监测预警系统。第三，在气候变化领域、可再生资源的技术转让和开放方面，中老应加强双边协作，共同应对挑战。中老双方可建立可再生能源技术转移的常态化机制，通过共建联合实验室、开展技术培训等途径，增强老挝的绿色发展能力，共同应对全球气候挑战。


（二）深化战略引领：推动中老命运共同体向更高质量阶段转变


从宏观层面来看，高质量构建中老命运共同体应从整体布局，坚持以构建新型国际关系为基本支撑，以全人类共同价值为普遍遵循，以共建共享共赢为实现路径，以落实三大倡议为重要引领，(1)共同推动构建更为紧密的中老命运共同体，使其成为区域命运共同体乃至人类命运共同体构建的典范。


1. 把构建新型国际关系作为高质量构建中老命运共同体的支撑


在复杂变幻的国际关系中，中老两国应该如何相处，应遵循何种准则，是中老两国在高质量构建中老命运共同体时面临的重要课题。习近平总书记在倡导推动构建人类命运共同体重要理念的同时，开创性地提出推动构建相互尊重、公平正义、合作共赢的新型国际关系，(2)这也为高质量构建中老命运共同体提供了基本支撑。第一，相互尊重是双边关系的政治基石。这种基于国家主权平等的尊重，既为两国关系稳定发展奠定基础，也为中老命运共同体构建提供政治保障。第二，公平正义是合作的价值准则。中老两国始终坚持公平、公正、透明的原则推动双方合作。两国还共同致力于维护地区和平稳定，推动建立更加公正合理的国际秩序，这种公平正义的合作理念为构建中老命运共同体提供价值引领。在共建“一带一路”框架下，中老双方建立跨境经济合作区利益共享机制，确保项目收益惠及沿线社群，为国际秩序民主化提供了实践的样本。第三，合作共赢是发展目标导向。中老两国以双方发展为目标，不仅在政治、经济、安全等领域的合作都取得了显著成果，而且两国在人文、教育、科技、卫生等领域的交流与合作，也推动着两国关系不断向更高水平发展，这种合作共赢的合作模式为构建中老命运共同体提供了源源不断的动力。


2. 在价值理念上普遍遵循全人类共同价值


全人类共同价值助力高质量构建中老命运共同体。2015年以来，习近平主席在多个国际场合提出坚守和弘扬和平、发展、公平、正义、民主、自由的全人类共同价值。第一，和平与发展的共同愿景是中老命运共同体构建的基础。中老两国都有着和平发展的夙愿，都一同致力于维护地区和平与稳定，通过加强双边合作，共同应对地区安全挑战。第二，公平与正义的价值追求。中老两国在合作中始终坚持公平、公正的原则。双方通过加强法律、司法等领域的交流与合作，推动构建更加公正合理的国际秩序和地区秩序，为两国人民提供更加公平、正义的司法保障。第三，民主与自由的共同理想。中老两国都尊重彼此的政治制度和发展道路，致力于推动民主化进程，加强政治体制改革，促进人民当家作主。在自由方面，中老两国都致力于保障人民的基本权利和自由，推动社会进步和发展。


3. 以共建共享共赢作为高质量构建中老命运共同体的实现路径


共建共享共赢作为构建中老命运共同体的实现路径，为双方关系的持续稳定发展提供有力保障。第一，遵循共建原则。共建强调双方共同参与、共同努力。要发挥多元主体的力量，坚持全体人员参与到共同体建设，通过共同制定合作规划、共同推进项目实施来实现中老两国优势互补、资源共享。第二，秉承共享原则。共享原则强调合作成果的共享性，即合作带来的利益应由双方共同享有。要坚持双方对合作成果的公平分配，无论是经贸合作、人文交流还是安全合作，中老两国都注重让合作成果惠及两国人民，实现共同发展、共同繁荣。第三，坚持共赢原则。共赢原则强调合作中的互利共赢，即双方都能在合作中获得利益。要坚持通过选择符合双方利益的项目进行合作，促进中老两国实现互利共赢、共同发展。


4. 以“三大倡议”为高质量构建中老命运共同体的价值引领


“三大倡议”不仅为全球发展、安全和文明进步提供中国方案，也为构建中老命运共同体注入新的动力，增加新的内涵。全球发展倡议的主张和内涵要求中老两国在经济领域加强合作，共同推动经济发展，实现共同繁荣，这也是高质量构建中老命运共同体的第一要务。在这一倡议的实践中，中老两国可加强政策沟通、设施联通和贸易畅通，推动两国经济深度融合。全球安全倡议的主张和内涵为中老两国在安全领域的合作提供指导。安全和稳定关乎中老两国核心利益，两国可以加强在打击跨国犯罪、维护边境安全等方面的合作，共同应对地区安全挑战，维护两国和地区的和平与稳定。全球文明倡议的倡导要求中老两国理解彼此历史文化的独特性，在人文领域加强交流与合作，增进两国人民的友谊。通过文化交流、教育合作等方式推动文化互鉴，促进民心相通，为构建中老命运共同体奠定坚实的民意基础。



五、结论


高质量构建中老命运共同体顺应时代趋势并符合中老双方战略需求，具有广阔的发展潜力。2019年签订中老命运共同体行动计划以来，双方根据行动计划不断推进，在政治、经济、文化、安全、生态合作等领域取得了极好的成效，为高质量构建中老命运共同体提供了坚实的基础。2023年中老续签新的行动计划―《中国共产党和老挝人民革命党关于构建中老命运共同体行动计划（2024—2028年）》，2 0 2 4年习近平主席对中老命运共同体示范性地位的肯定，推动中老命运共同体向更高质量发展。


当前，高质量构建中老命运共同体具有较好的时代新机遇。在政治层面，新型国际关系理念的提出为构建中老命运共同体注入新动力；在经济层面，“一带一路”倡议、次区域合作机制及中老铁路助力中老经济合作；在文化层面，“三大倡议”与人类文明新形态为中老命运共同体提供价值引领与目标导向；在技术层面，新科技运用、数字经济与人工智能为中老命运共同体构建提供新模式。但是，当前国际局势的风云变幻，也为中老命运共同体的持续构建带来了新的挑战。在政治上，大国竞争加剧，周边地缘政治局势不稳定，全球热战爆发，外部环境风险加剧；在经济上，全球经济下行趋势以及中老两国贸易结构不平衡给双边经贸合作带来新的考验；在文化上，双边认知差异仍然存在，民间交流平台有待进一步完善；在安全形势上，非传统安全问题的新变化、公共卫生安全问题的频繁爆发，对双方的安全合作提出新的要求。


面对以上的新机遇和挑战，可通过以下路径推动中老命运共同体向更高质量发展：在方法路径方面，遵循习近平总书记提倡的“建立稳固伙伴关系、构建安全合作框架、推动经济共同发展、促进文化交流互鉴、加强生态环境保护”五条核心路径。首先，要丰富核心路径，推动中老合作各领域迈向更高阶段。从政治安全互信、经贸合作升级、文化交流加强、安全和生态合作逐步向新领域推进方面入手。其次，优化总体路径，深化战略引领，推动中老命运共同体向更高质量发展。高质量构建中老命运共同体应从整体布局，坚持以构建新型国际关系为基本支撑，以全人类共同价值为普遍遵循，以共建共享共赢为实现路径，以落实“三大倡议”为重要引领。


从构建人类命运共同体意义上来看，高质量构建中老命运共同体为中国构建广泛且高质量的全球伙伴关系网络树立了典型示范作用，同时也能进一步推动老挝在澜湄区域的地缘政治格局变化、区域地位上升。构建中老命运共同体在澜湄区域的典范作用已经发挥，推动中老关系向着命运共同体构建的方向发展，为构建澜湄国家命运共同体打下坚实基础，并为构建双边和多边命运共同体提供实践经验和构建模式。
