智能制造学院

学院介绍

智能制造学院坚持以培养高素质复合型人才为目标，在以装备智能化技术为核心的智能工程技术系和以数字BIM技术为核心的智慧建造系的基础上，不断打造品牌专业和特色专业。学院高度重视人才培养工作，紧密围绕国家“制造强国”战略及新工科建设要求，注重学生实践能力、创新意识及数字素养的培养。2024年学科竞赛成绩再创辉煌，获得国家级奖项29项，省级奖项67项。学院毕业生去向落实率高达98. 41%，普通本科生考研升学率达14.66%，其中机械工程专业升学率高达23.8%。

专业介绍

（一）工业工程

 1. 概述

工业工程专业聚焦智能制造系统，面向智能制造国家战略和需求，助力制造业数字化、智能化转型升级，服务产业提质增效降本，满足社会对管理+工程技术人才的需求。工业工程已被列为国家紧缺专业。

2. 培养目标

工业工程专业培养掌握现代工业工程理论方法和管理科学知识，具备智能制造与服务系统的分析、规划、设计、决策、运营、评价、优化等能力，掌握智能制造运营管理能力和智慧物流解决方案的应用型管理人才。

3. 主干课程

生产运作管理、运筹学、质量管理与可靠性、人因工程、工业工程基础、设施规划与物流分析、Modeling and Simulation of System、工业大数据分析、Intelligent Logistics & SCM。

4. 就业前景

社会对工业工程专业人才需求广泛。毕业生可从事智能制造、企业管理、政府决策、公共服务、物流运输等领域的系统规划分析与设计、现场管理、生产计划、质量管理、成本管理、供应链管理、项目管理、精益管理、数据分析等工作。

（工业工程实验室）


（二）智能感知工程

1. 概述

智能感知工程是一门集电子信息技术、计算机科学、人工智能于一体的交叉学科，致力于研究用于环境感知、信息获取、数据处理和智能决策的相关技术。

2. 培养目标

智能感知工程专业培养具备智能传感器设计与开发、智能感知系统设计、工业机器视觉系统开发等基本能力，能够驾驭先进智能感知技术，深耕智能制造、人工智能等前沿领域的高素质工程技术与管理人才。

3. 主干课程

智能传感技术、智能感知电路设计、嵌入式系统设计及应用、无线传感网络及应用、机器视觉、大数据技术基础、云计算应用技术等。

4. 就业前景

毕业生可在智能制造、人工智能产品、智慧医疗、智能家居等相关企业、研究院所从事技术开发、测试、管理及技术服务等工作。

（智能感知工程实验室）


（三）智能车辆工程

1. 概述

智能车辆工程是一个结合车辆工程、人工智能、自动控制等多个学科的综合性专业，主要研究和应用智能车辆相关的核心技术，包括环境感知、决策规划、控制执行等，以实现车辆的高效、安全和舒适运行。本专业致力于培养具备创新能力和实践经验的高素质复合型人才，为智能车辆和智能交通领域的发展做出贡献。

2. 培养目标

智能车辆工程专业培养具有车辆智能化关键技术和汽车领域的专业知识，具备设计、性能分析、智能系统装调和测试技术以及分析与解决车辆系统复杂工程问题的能力，能胜任车辆及智能化研发、制造、测试、管理及技术服务等工作的高素质复合型人才。

3. 主干课程

汽车构造、汽车理论、汽车设计、新能源汽车技术、智能车辆环境感知技术、智能车辆线控技术、ADAS系统与自动驾驶、智能车辆仿真综合实践、智能网联汽车电子系统测试综合实践。

4. 就业前景

毕业生可在各大汽车制造企业、零部件企业、设计公司、自动驾驶研发公司、检测机构等从事车辆研发、软硬件开发、系统集成、控制算法开发、汽车测试、工艺管理等相关工作。

（智能车辆综合实践平台）


（四）智能制造工程

1. 概述

智能制造工程是将智能化贯穿于设计、生产、管理、服务等制造活动各个环节，以工业机器人、自动生产线等智能装备为载体，实现人机协同，让制造过程具有信息深度自感知、智慧优化自决策、精准控制自执行的一种新兴学科。

2. 培养目标

智能制造工程专业培养具备智能制造系统设计与应用、管理和创新能力，以及获取信息、分析解决工程技术问题的能力，在智能制造工程领域从事智能产品和装备设计、生产线的研发、智能系统运行管理的复合型工程技术人才。

3. 主干课程

数字化设计基础、3D打印装备与工艺、数字化制造与仿真技术、智能控制技术、智能机电传动控制、工业机器人技术及应用、智能制造系统集成、人工智能技术、物联网技术与应用、大数据系统分析、智能制造生产管理等。

4. 就业前景

毕业生可在智能制造技术相关的企业从事数字化设计、智能制造系统设计、智造产品开发、智能生产管理、智造技术应用研究等工作。

（智能制造工程实验室）


（五）智能建造

1. 概述

智能建造专业是面向国家战略需求和建筑业升级转型，在土木工程基础上融合智能制造、大数据、物联网、人工智能、机器人、工程管理等技术发展而成的新工科专业，为社会培养智能建造产业急需人才。

2. 培养目标

智能建造专业培养具备扎实的智能建造专业知识和工程实践能力，掌握智能建造基本理论、相关计算机开发语言、机械和控制原理、装配式设计与施工、BIM技术、数字建造等新技术，能胜任工程项目的智能设计、施工、运维和管理等工作的高素质复合型人才。

3. 主干课程

绿色建筑材料、智慧测绘、装配式结构设计与施工、5D项目管理、智能机械与机器人、智慧施工、BIM技术与应用、基础设施智慧运维、3D打印技术、工程结构基本原理、结构力学等。

4. 就业前景

毕业生就业范围广阔，既可选择传统建筑工程行业，也能胜任BIM全过程设计、智慧施工、智慧运维、工业化制造、智能装备、项目管理与咨询、建造管理软件开发等工作。

（智能建造实验室）


（六）交通工程

1. 概述

交通工程专业基于国家交通强国战略，以交通工程与智能感知、大数据、智慧城市交叉融合为特色，通过先进的交通管控方法、传感器和大数据分析，实现对交通流量、车辆运行状况及交通设施的智能监控和管理，致力提升交通系统的通行效率、安全性和可持续性。

2. 培养目标

交通工程专业培养学生具备核心专业基础与交通系统智慧升级能力，掌握交通工程设计、规划、管控基本原理与交通数据挖掘、智慧运维的关键技术，成为在交通系统工程、道路桥梁工程、智能交通等相关领域的高素质复合型人才。

3. 主干课程

交通工程学、交通工程系统分析、交通大数据技术分析与应用、智能交通系统、智能交通信息感知、智能交通仿真技术、道路勘测设计、桥梁工程、交通规划、BIM技术与应用等。

4. 专业方向

设置智慧交通方向。

5. 就业前景

毕业生可在交通行业管理部门、交通规划设计单位、智能交通系统开发与实施、交通设施维护管养单位从事相关规划、设计、管理、智慧升级工作。另外，每届都有较多比例的学生考入国内外优秀大学继续进行研究生深造。

（交通工程实验室）


（七）环境设计

1. 概述

环境设计专业是一门涉及艺术学、设计学、社会学、环境学等多个领域的综合性学科。该专业通过合理的平面和空间组合，利用专业设计手段，运用艺术表现形式，以现实或虚拟形态来体现特定的氛围和风格，满足人们的功能使用及视觉审美上的需求。

2. 培养目标

依托南京理工大学优质教学资源，环境设计专业培养具有良好的人文、科学素养，系统掌握艺术设计的基本知识和方法，了解各类艺术风格，熟练掌握各类设计创作技能，具有突出的艺术能力和工程能力，能在美丽乡村、文化公园、主题展会、数字媒体等方面从事设计创作、咨询管理、技术研究等工作的高素质复合型艺术人才。

3. 主干课程

空间透视与专业表现技法、展示空间设计、环境设施与公共艺术设计、版式设计、新媒体展示设计、VR虚拟现实技术、交互设计等。

4. 就业前景

毕业生可在企事业单位、专业机构和科研单位从事设计、管理和研究工作，主要岗位有景观设计、室内设计、网页设计、影视动漫制作、游戏场景设计、艺术和文化传播等。

专业特色

（一）工业工程

工业工程是工程、信息与管理的交叉学科，毕业生就业面广，市场需求量大，就业匹配岗位多、起薪高。本专业面向新兴工程领域，运用现代信息技术与工业工程方法，培养解决系统运行优化与生产过程改善相关问题的高素质复合型工程管理人才。

（二）智能感知工程

智能感知工程专业注重校企合作、学生创新活动与学科竞赛，专业以“人工智能+传感”为基础，以“工业机器视觉”和“工业感知互联”为特色，面向智能制造、人工智能等领域的需求，培养高素质复合型人才。

（三）智能车辆工程

作为江苏省内首批设立的智能车辆工程专业，本专业紧跟行业趋势，抓住发展先机，通过产学对接、交叉融合、内涵建设，以“车辆数字化”为基础，以“智能--网联”为特色方向，培养服务区域产业、经济、社会发展的紧缺工程技术型人才。

（四）智能制造工程

依托南京理工大学优质教学资源和办学优势，瞄准智能制造业数字化、智能化等产业高端，以服务长三角地区经济和社会发展的需求为导向，紧跟行业和市场需求，采用跨学科交叉融合培养新机制，培养高素质复合型人才。

（五）智能建造

智能建造依托校企合作资源及双师型教师队伍，融合智能感知、智能制造、人工智能等技术优势，建立以BIM、装配式、智慧施工与运维为基础的产学平台，强化学生“数据驱动工程”的能力，培养智能建造高素质复合型人才。

（六）交通工程

专业建设面向国家交通强国战略，以交通工程与智慧城市相融合为特色，培养学生具备扎实工程基础知识与智慧升级拓展能力，能够胜任交通运输、智能交通等各类岗位。

（七）环境设计

以艺术创作为引领，突出人文和创新设计，聚焦新媒体数字技术，注重锻炼学生自我学习和团队协作能力，将新艺科和新工科相融合，导师制培养“有艺术、能设计、懂项目、擅交流、勇创新”的高素质复合型艺术人才。
