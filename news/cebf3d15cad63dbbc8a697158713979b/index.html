1、数控机床与切削工具展区


高端数控机床（车铣复合、五轴联动、加工中心）、精密磨削设备、电加工机床及智能生产线；各类高性能刀具（硬质合金、超硬材料）、先进工装夹具、精密量具量仪及智能切削解决方案。

2、数控机床与切削工具展区


高功率光纤激光切割机、精密激光焊接 / 打标设备、激光清洗及增材制造系统；数控冲床、折弯机、剪板机、压力机（伺服 / 液压）、自动化钣金生产线及柔性制造单元（FMC）。

3、工业自动化与机器人展区


工业机器人（多关节、协作、SCARA、Delta）、机器视觉系统、运动控制与伺服驱动、PLC、HMI、传感器及工业通讯技术；自动化生产线集成、智能检测装备、工业物联网 (IIoT) 平台应用等。

4、数控机床与切削工具展区


智能立体仓库系统（AS/RS）、AGV/AMR、智能分拣输送线、穿梭车、堆垛机；集成 WMS/WCS 软件、智能拣选系统、自动化包装设备及厂内智能物流整体解决方案。

5、焊接 / 模具 / 五金机电 / 工业配件展区


智能化焊接机器人工作站、先进焊接电源与专机、激光 / 搅拌摩擦焊等特种工艺；数字化模具与设计制造（CAD/CAE/CAM）；五金工具（手 / 电 / 气 / 模具）、机电设备、紧固件 / 传动 / 密封件、工业配件、劳保安防。

6、表面处理与工业环保展区区


电镀、喷涂、阳极氧化、磷化、热喷涂等表面处理工艺及智能化设备；工业废气 / 废水 / 固废处理设备、粉尘控制技术、节能环保材料、清洁生产方案及环境监测系统。
