2025年广东省政府工作报告（笔记）

一、2024年工作回顾

（一）经济发展

  • • 经济指标

    • • 地区生产总值：14万亿元，连续36年全国第一

    • • 进出口总额：9万亿元，增长9.8%，连续39年全国第一

    • • 规模以上工业增加值：增长4.2%，企业约7.4万家

    • • 地方一般公共预算收入：1.35万亿元，连续34年全国第一

    • • 旅游总收入：突破万亿元

    • • 海洋生产总值：突破2万亿元

  • • 经营主体

    • • 净增：98万户

    • • 总量：突破1900万户，企业超830万户，均居全国第一

    • • 城镇新增就业：超143万人，超额完成国家任务

（二）产业科技

  • • 研发投入

    • • 研发经费支出：约5100亿元

    • • 投入强度：3.6%左右

    • • 高新技术企业：约7.7万家

    • • 区域创新能力：连续8年全国第一

  • • 产业发展

    • • 先进制造业、高技术制造业增加值占比：分别提高到56.7%、31.6%

    • • 新能源汽车产量：增长43%，占全国1/4

    • • 机器人产量：增长31.2%，占全国44%

    • • 智能手机产量：增长12.5%，占全国超4成

    • • 集成电路产量：增长21%，占全国18%

  • • 投资增长

    • • 工业投资：超1.5万亿元，增长6.7%

    • • 技改投资：增长11.5%，连续24个月两位数增长

（三）改革开放

  • • 改革举措

    • • 系统部署：500多项具体改革举措

    • • “百县千镇万村高质量发展工程”：集成式改革成效显现

    • • 科技体制改革：三年攻坚顺利完成

  • • 营商环境

    • • 市场化法治化国际化：成为广东鲜亮品牌

（四）基础设施与民生

  • • 基础设施

    • • 深中通道：通车运营，日均车流量超8万辆次

    • • 黄茅海跨海通道：建成通车

    • • 港珠澳大桥：车流量增长90%

  • • 民生保障

    • • 粮食生产：实现“三增”，单产创历史新高

    • • 城乡居民收入比：缩小至2.31:1

（五）区域协调

  • • 粤港澳大湾区建设

    • • 规则衔接、机制对接：新发布“湾区标准”54项

    • • 横琴合作区：全域封关运行，琴澳一体化关键一步

    • • 前海：14项全国“首创”、“首批”金融政策

    • • 南沙：开发建设全面展开

  • • “百县千镇万村高质量发展工程”

    • • 县域特色优势产业：15个主平台新承接项目约700个、总投资约3000亿元

    • • 新型城镇化建设试点：首批15个县城

    • • 乡村全面振兴：新建改造提升高标准农田112万亩

（六）科技创新

  • • 政策支持

    • • 全过程创新链“1+N”政策

    • • 科技创新条例

    • • 制造业高质量发展促进条例

  • • 科研投入与成果

    • • 省级科技创新投入：持续加大

    • • 散裂中子源二期、先进阿秒激光：动工建设

    • • 江门中微子实验站：建成

    • • 第12批省重点领域研发计划：布局

    • • 原生鸿蒙操作系统：正式商用

    • • 量子纠错新方案、天气预报人工智能大模型：入选年度中国科学十大进展

  • • 创新平台与企业

    • • 大湾区国家技术创新中心：推进

    • • 省中试平台：布局建设7个

    • • 国家制造业创新中心：新增2家

    • • 国家级制造业单项冠军：新增60家

    • • 专精特新“小巨人”企业：新增561家

（七）生态环境

  • • 生态建设

    • • 美丽广东建设实施意见和规划纲要：制定

    • • 绿美广东生态建设“六大行动”：推进

    • • 林分优化提升、森林抚育提升：超额完成“两个200万亩”任务

  • • 污染防治

    • • PM2.5平均浓度：降至20.6微克/立方米

    • • 空气质量优良天数比率：95.8%，创记录最好水平

    • • 地表水国考断面水质优良率：93.2%

    • • 劣V类国考省考断面：首次全面清零

（八）文化事业

  • • 文化活动与产业

    • • 中华人民共和国成立75周年：热烈庆祝

    • • “强国复兴有我”主题宣传教育：广泛开展

    • • 文化及相关产业增加值：稳居全国首位

    • • 龙舟赛、英歌舞：岭南传统文化活动火爆出圈

  • • 旅游发展

    • • 接待游客：超8亿人次

    • • 入境外国游客：增长87%

    • • 国家5A级景区：河源万绿湖入选

    • • 国家级旅游度假区：横琴长隆入选

（九）民生福祉

  • • 教育

    • • 新增基础教育公办学位：52万个

    • • 高等院校获批设置：7所

    • • 华南理工大学工程学：迈入ESI全球排名前万分之一学科

  • • 医疗

    • • 国家公立综合医院百强：12家医院位列

    • • 县域医共体人员下沉乡镇卫生院：全覆盖

  • • 社会保障

    • • 基本养老保险提质扩面行动：开展

    • • 养老金融产品：率先试点全品类

    • • 个人养老金开户数：近1800万

    • • 基本医保：放宽户籍限制，实现新生儿出生即参保，镇村医保服务全覆盖

（十）风险防范

  • • 社会稳定

    • • 法治广东、平安广东建设：深入推进

    • • “大平安”格局：加快构建

    • • 刑事治安警情、刑事立案数量：分别下降26.3%、32.6%

  • • 房地产市场

    • • 房地产政策：因城施策调整优化

    • • 房地产市场：止跌回稳

    • • 房地产企业风险化解：稳妥推进

    • • 保交房交付率：94.7%

二、2025年工作安排

（一）总体要求

  • • 以习近平新时代中国特色社会主义思想为指导

  • • 全面贯彻落实党的二十大和二十届二中、三中全会以及中央经济工作会议精神

  • • 围绕落实省委“1310”具体部署

  • • 坚持稳中求进工作总基调

  • • 完整准确全面贯彻新发展理念

  • • 服务和融入新发展格局

  • • 扎实推动高质量发展

  • • 进一步全面深化改革

  • • 扩大高水平对外开放

（二）主要预期目标

  • • 地区生产总值：增长5%左右

  • • 固定资产投资：提质提效

  • • 社会消费品零售总额：增长5%左右

  • • 进出口总额：稳量提质

  • • 规模以上工业增加值：增长6%左右

  • • 地方一般公共预算收入：增长3%左右

  • • 城镇调查失业率：5.5%左右

  • • 城镇新增就业：110万人以上

  • • 粮食产量：1285万吨
