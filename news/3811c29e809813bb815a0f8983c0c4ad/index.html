PLCopen中国携会员单位参加2026成都国际工业博览会

PLCopen 2026-03-27

PLCopen·成都



2026成都国际工业博览会于3月11-13日在中国西部国际博览城盛大举行。展会以“创链新工业，共碳新未来”为主题，PLCopen 中国携手会员单位共同展示PLC领域新技术，新规范，共同打造PLCopen生态圈。

CODESYS 软件是一款基于 .NET 架构和 IEC 61131- 3 国际标准的、面向工业4.0及工业物联网应用的软件开发平台，可支持多种主流 CPU 芯片和操作系统（包括国产芯片和国产操作系统）、主流总线协议栈，用户基于此平台可快速完成 PLC（控制器）、运动控制器及机器人控制器、HMI（人机界面）、冗余控制系统、Safety 功能安全控制器（SIL2、SIL3）的开发。CODESYS 软件已在工厂自动化、过程自动化、能源自动化、楼宇自动化、轨道交通、工程机械、工业机器人、特种车辆、数控机床等领域得到广泛应用。

Virtual PLCnext Control是菲尼克斯重磅推出的新一代开放式虚拟控制软件解决方案，它是基于菲尼克斯 PLCnext 开放自动化平台，结合了软件虚拟化技术，提供了一种灵活、可扩展且成本效益高的工业级控制解决方案。vPLCnext正将“软件定义自动化”理念转化为工程现实，赋予了工程师前所未有的灵活性：既可使用经典的IEC 61131-3语言，也可直接采用C++、Python等高级语言或AI框架进行开发，让复杂的算法模型与毫秒级实时控制在同一平台上无缝融合。由此，vPLCnext正成为驱动产线柔性化、智能化的核心引擎，赋能制造业快速响应市场变化。

杭州电子科技大学从2006年开始开展全自主可编程控制器的研究，攻克了实时操作系统、PLC运行时、图编译、开发语言组态等关键技术，开发了国内首个通过PLCopen国际认证的可编程控制器软件平台，实现了PLCopen Motion Control PART1（国内首个通过PLCopen运动控制PART1认证）、PART4运动算法功能块；提出了一种全新的控制器设计理念，将嵌入式系统升级到PLC系统（ePLC）。CASS相关技术充分利用了现有嵌入式处理器的强大处理功能，支持各种架构处理器芯片，可按需定制各类专用控制器，实现了从控制器到人机界面(HMI)的软件设计和代码自动生成。杭电团队通过多年持续地研发，在可编程技术、智能平台软件、运动控制等方面取得了大量的原创性成果，参与制定PLC国际标准2项，主持制定国家标准5项。

北京奥特思技术有限公司是国内国产化IEC61131-3可编程系统的供应商，技术能力对标全球先进水平，为广大用户提供自主可控且标准化程度高、易用性好的产品及服务。

其产品 AnyControl 是遵循 IEC61131-3 (GB/T 15969.3) 标准自主研发的，用于自动化控制的工业可编程软件系统。AnyControl作为自动化控制系统解决方案，可集成逻辑控制、运动控制、工业通讯协议、物联网通讯协议、人机界面、工业视觉以及行业应用算法等，并进行统一编程和调试。

四川零点自动化系统有限公司成立于2003年，拥有23年自动化行业经验，是一家专业从事工业通讯和自动化控制产品研发及相关技术服务的企业，为全球客户提供先进的自动化解决方案。

零点产品拥有CE 、 UL 、 RoHS、船级社以及100%国产化等多重认证，硬件与软件全国产自主可控，搭载的AIOSYS-跨界工业控制软件作为新一代工业控制软件，更是集成了分布式控制、事件驱动等功能，为各种工业场景提供高可靠、高灵活、易扩展的控制解决方案。

多年来，零点自动化凭借持续的研发投入，先后被认定为“国家科技型中小企业”、“国家高新技术企业”、“四川省专精特新企业”、“省级企业技术中心”、“绵阳市企业技术中心”，已成为中国制造业强国的重要组成部分!

零点自动化致力于为客户提供深度ODM的工业通信和控制解决方案，构建开放共赢的产业生态。在数字化转型浪潮中，公司持续推动技术创新，助力企业实现智能制造升级。

东莞固高自动化技术有限公司作为固高科技（股票代码：301510）旗下核心子公司，深耕工业自动化领域多年，拥有50余项专利技术，专注于高端智能装备系统集成与行业解决方案，在运动控制、机器人控制等领域形成核心技术优势，推出全国产化系列智能控制器，且产品集成AI技术，进一步强化核心竞争力，助力智能制造国产化突破。

东莞固高围绕 PLCopen 技术规范，集中展示了iDEABOX Pro 智能运动控制器、iDEABOX Plus 工业控制核心、iSpade 刀锋系列总线 I/O 模块、IO-Link 系列总线主从站模块等明星产品，全面覆盖工业控制、数据采集、设备互联等核心场景，依托集成的AI技术实现更精准的控制与数据分析，为高端装备提供精准高效的运动控制保障，助力企业高效完成数据采集与设备互联互通。

南京实点电子科技推出基于自研 X-bus 协议的背板 I/O 解决方案，直击工业控制器厂商自研 I/O 面临的高投入、长周期、技术壁垒高等核心痛点。可直接针对客户自有 PLC 硬件平台，完成 X-bus 主站协议深度适配，提供完整静态库实现无缝对接；无需客户从零研发，即可让自有 PLC 快速具备成熟稳定的背板I/O 扩展能力，仅需数周就能完成全系列I/O 集成落地。

实点 I/O 已广泛应用于 3C、锂电、光伏等核心工业制造场景，经海量复杂工况的长期验证，具备工业级超强抗扰与稳定运行能力，产品拥有 CE、UL 等齐全权威认证，一站式为客户落地成熟的背板 I/O 扩展能力，助力厂商高效抢占市场先机。

德国赫优讯（Hilscher）是全球领先的工业通讯专家，总部位于德国法兰克福。公司专注于为智能制造、工业4.0和工业物联网提供创新通讯解决方案，包括Profibus、CANopen、DeviceNet、CC-Link 等现场总线，PROFINET、 EtherCAT、EtherNet/IP、Powerlink、Modbus TCP、SERCOS III 等实时以太网和OPC UA、MQTT等云通讯技术的全系列“量身定做”解决方案。在本次PLCopen活动上，赫优讯重点展示了面向PLC控制系统的网络通讯方案，其中包括：PC板卡方案（通过PCI、PCIE等多种工业PC内部接口，实现各种网络通讯协议功能），以及基于最新技术的netX 900系列多协议主站芯片解决方案。netX 900全新芯片方案不仅内置网络CRA安全特性，同时提供千兆网络通讯平台，可以快速协助控制器厂商实现工业实时以太网和云通讯协议的高效集成，显著缩短产品开发和面世周期，实现多方共赢！

上海华元创信软件有限公司是中国电科旗下的基础软件企业，专业提供锐华嵌入式基础软件产品的深度开发定制以及应用解决方案设计服务，是锐华的市场化主体和产业化平台。公司聚焦工业自动化、能源电力、轨道交通、航空航天、国防电子等关键行业基础设施，基于锐华嵌入式实时操作系统和自主EtherCAT主站，打造的全栈国产化控制器解决方案已在众多关键领域实现规模化应用：覆盖数千套风力发电机、水电站控制主机及泵站控制系统，支撑数百万兆瓦级火电机组安全运行；同时，在全国超过10个城市、30余条城市轨道交通线路中，持续保障着日复一日的高效精准运行，推动自主可控系统产业化和规模化应用。

此次展会，PLCopen北京代表处携手会员单位给大家带来了多款国内外PLC软硬件产品及解决方案，给广大客户朋友普及和介绍了IEC 61131-3标准、PLCopen技术规范，吸引众多的新老朋友驻足参观交流。

        PLCopen北京代表处旨在为会员单位构建合作平台，实现更高的应用程序开发效率、更短的调试投产时间，以及更低的生命周期成本。欢迎更多朋友加入PLCopen。
