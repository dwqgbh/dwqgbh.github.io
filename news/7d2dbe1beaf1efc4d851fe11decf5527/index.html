【参展预告】启扬光学诚邀您参加CIOE 2025中国国际光电博览会

启扬光学 2025-09-04




启扬光学将参加2025年9月10-12日在深圳国际会展中心开展的第26届中国国际光电博览会（CIOE）。


 此次展会超3800家参展商，覆盖信息通信、光学、激光、红外、传感、新型显示、光电子创新、AR&VR等光电全产业链板块，是展示行业新产品、新技术的高效一站式平台。


展位号

4馆4B190




启扬光学专注于高端光学领域产品研发与制造，为客户提供高品质光学产品及定制化技术解决方案，产品广泛应用于激光、半导体、光通信、科研、医疗等前沿领域。


启扬光学此次将携棱镜耦合仪、微观缺陷检测系统、可变磁场单晶倍频测试系统、Z-Scan 三阶非线性测试系统等参展，期待您莅临指导。



OptoPrism系列 棱镜耦合仪

OptoPrism Coupler



利用棱镜耦合原理测量晶体、薄膜等材料的双折射率、膜厚度、热光系数、波导传输损耗等参数，折射率精度高达±0.0005，多种激光光源可选（标配405~1650 nm，可选配375 nm、266 nm），具备一体化、易操作等优势。


点击链接了解更多仪器信息

一文解析 | 棱镜耦合仪适配材料清单与测量范围（附实测案例）



微观缺陷检测系统

Micro Defect Inspection System



采用非接触式检测方案，实现高精度测量，可精准识别微米至亚微米级缺陷，并对各类关键缺陷实现自动检测与智能分类，涵盖划痕、裂纹、微气泡、镀膜不良、表面污染等。



可变磁场单晶倍频测试系统

Variable Field Monocrystal

Frequency Doubling Test System



在可变磁场下，测试单晶的二阶非线性倍频效应，定量表征材料在特定晶向下的二阶非线性系数。



Z-Scan 三阶非线性测试系统

Z-Scan Third-Order

Nonlinearity Test System



表征非线性光学材料的三阶非线性吸收系数、三阶非线性折射过程、三阶非线性极化率的虚部和实部。



启扬光学始终秉持“市场为导向、技术创新为驱动”的核心发展理念，依托深厚的精密光学技术积淀与对光学应用需求的深刻洞察，自主研发出一系列高精度检测与加工核心设备。凭借精准高效的AI智能技术赋能、全流程自动化智慧工厂支撑、全面严苛的质量管理体系保障及定制化开发的快速响应能力，全方位满足不同领域客户的多元化、个性化需求。


本届光博会期间，启扬光学不仅将集中展示最新科技成果与核心产品，更将为行业伙伴提供全方位的技术咨询及定制化解决方案，助力业界同仁携手探索行业新方向、共拓发展新空间。我们诚挚邀请各界朋友莅临启扬光学展台，一同见证光学技术的前沿突破与创新魅力！




参展小提示




地铁

搭乘地铁12/20号线，乘坐至国展站或国展北站可直接到达展馆门口！

①国展站C1/C2出口，抵达南登录大厅东侧（临近5-12号展厅）；②国展站D出口，有电瓶车接驳，快速2号馆10号门入口（临近1-4号展厅）。


公交车

您也可以选择乘坐公交车到达展馆，到达公交站点深圳国际会展中心、国展地铁站。


出租车/网约车

如果您想打车，请直接告诉师傅或输入：深圳国际会展中心。

①如您要到南登录大厅，请导航至深圳国际会展中心网约车南一区/南二区上下客区；②如果您要到南入口，请导航至深圳国际会展中心18号门；③如您要到北登录大厅，请导航至深圳国际会展中心网约车北区上下客区。


自驾

来馆请导航深圳国际会展中心P1/P2/P3/P4/P5/P6地下停车场入口或1、2、4、5、6号门。主办方设置了地面及地下停车场。


飞机

①从深圳宝安国际机场出发的小伙伴可在机场搭乘地铁11号线坐1站到达机场北站，转乘20号线在国展站下车到达登录大厅，具体参考上方地铁乘坐方式。

②从广州白云国际机场出发的小伙伴可以搭乘公共交通，如高铁等，到达深圳市后，再换乘公共交通抵达展馆。


高铁

①深圳北站（地铁乘坐）：乘坐5号线（赤湾方向）到达灵芝站——换乘地铁12号线（海上田园东方向）到达国展站；②福田站：乘地铁11号线（碧头方向）到达机场北——换乘20号线到达国展站；


城轨

①在福海西站下车，可乘坐地铁12号线抵达国展站或国展北站；②在机场北站下车，可乘坐地铁20号线抵达国展站或国展北站。


从香港中港城码头出发

乘坐客船到达深圳机场码头，再搭乘出租车到展馆。


从香港国际机场出发

展馆距香港机场仅75公里，您可搭乘公共交通，如跨境巴士、轮渡（海天客运码头至深圳蛇口码头）等到达深圳，再乘坐出租车或公共交通至展馆；

香港国际机场有4家跨境巴士前往深圳各跨境陆路口岸，分别是：永东巴士、中港巴士、中旅巴士、三通巴士，可在一号客运大楼抵港层（L5）接机大堂A出口方向M01-M05柜台购买。抵达深圳后，您可换乘其他交通工具前往展馆。


从香港西九龙站出发

乘坐高铁至福田站，搭乘地铁11号线（碧头方向）到达机场北再换乘20号线到达国展站或国展北站。


从香港各口岸出发

深圳湾、福田、皇岗及罗湖等口岸，在入关后均可直接换乘深圳地铁到达展馆。
