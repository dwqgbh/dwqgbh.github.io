2022粤港澳大湾区（广州）智慧交通产业博览会

近日，2022粤港澳大湾区（广州）智慧交通产业博览会在广州顺利闭幕，瑞数信息受邀参展，携瑞数动态安全系列产品精彩亮相博览会信息与网络安全展区。



本届博览会由广东省交通运输厅、工业和信息化部电子第五研究所指导，广东省交通运输协会，广东省港口协会、广东省静态交通协会、广州市信息技术应用创新行业协会、深圳市智能交通行业协会共同举办。


此次论坛以“信息技术赋能数字经济高质量发展，铸就行业数字化转型安全基石”为主题，探讨信息技术高质量发展技术路径，交流网络安全热点问题，为关键领域及重点行业数字化转型提供了新思路，为粤港澳大湾区数字经济高质量发展和高水平安全提供了重要平台。



瑞数动态安全Botgate

Botgate


瑞数动态安全 Botgate(机器人防火墙) 通过动态封装、动态验证、动态混淆、动态令牌等创新动态安全技术对服务器网页底层代码的持续动态变换，增加服务器行为的“不可预测性”;对企业内、外网的应用 提供主动防护，不仅可以防护传统针对应用的手动攻击，还可以有效的防御新兴的针对业务的自动化攻击。实现了从用户端到服务器端的全方位“主动防护”， 为各类 Web、HTML5 网站提供强大的安全保护。让攻击者 无从下手，从而大幅提升攻击的难度!


动态Web应用防火墙River Safeplus

River Safeplus


瑞数动态Web应用防火墙(River Safeplus)以创新的“动态安全”技术为核心，结合智能检测引擎和规则引擎，多重检测协同机制，覆盖传统 OWASP top 10 安全攻击及新兴自动化威胁，为用户提供更贴心的应用安全保障;每个引擎可以独立开启工作，实现纵深防御!


瑞数WAAP动态安全超融合解决方案

WAAP


瑞数动态安全超融合解决方案覆盖所有的业务接入渠道，包括Web、APP、API、微信、小程序等业务接入渠道，实现全业务渠道防护；通过全访问记录和多维度关联分析，将各业务接入渠道的数据进行超融合，实现用户访问数据追踪和透视。


结合行为分析技术，打造出“动态安全”+“AI 智能威胁检测”双引擎协同工作机制，提 供传统 Web 安全防御能力的同时，更能轻松应对新兴和快速变化的 Bots 攻击、API 攻击和应用 DDoS 攻击;结合可编程对抗和业务威胁感知功能，提供对企业 Web、APP、API 业务的威胁感知和高强度对抗能力，覆盖 Gartner WAAP 全功能，实现应用安全全功能超融合防护


瑞数 API 安全管控平台API BotDefender

API BotDefender


瑞数 API 安全管控平台(API BotDefender)由 API 感知、发现、监测和保护四大模块组成“ADMP 安全模型”，对来源环境和 API 资产进行感知，自动发现流量中“API”，并采用机器学习引擎对异常 API 请求行为、敏感数据进行识别与分析，调用动态响应机制对异常 API 请求进行拦阻、限速或脱敏等响应动作。


作为中国动态安全技术的创新者和Bots自动化攻击防护领域的专业厂商，瑞数信息持续致力于构建更全面、安全、可控的网络环境，提供涵盖Web、App和API的全渠道应用安全、业务安全、数据安全、云安全等在内的专业网络安全产品及服务。


未来，瑞数信息将继续以技术创新，助力交通行业数字化转型为己任，积极探索创新交通行业网络安全防护手段，为构建交通行业网络安全新生态作出更大贡献！
