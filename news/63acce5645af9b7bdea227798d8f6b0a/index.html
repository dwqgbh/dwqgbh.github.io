艺启2025 | 第五期:学涯引航,心灵守望——“2025级新生班主任来啦”

杭州师范大学美术学院 2025-09-05

2025级辅导员

     张涵婷

简介：

中共党员，美术学院2025级本科生辅导员。曾获杭州市教育系统优秀团员等荣誉。


教师寄语：

亲爱的25级新生们，欢迎加入美术学院！愿你们在这里勇敢追梦，不懈探索，期待与你们一同见证每一份努力绽放的光彩。


01

美术学（师范）

美教251

   吴静

简介：

杭州师范大学副教授，硕士生导师。近年来从事中国古代及近现代美术史、美育教育、艺术公共服务等领域研究。在《文艺研究》《文艺理论与批评》《美术》等核心期刊发表论文20余篇。主持教育部、省、市、校级课题多项。出版科普著作《走进博物馆》《文物里的地方志》。指导研究生设计并实施了《妞妞的唐装》《齐白石爷爷的菜市场》《我与丰阿宝》《观石造皴》等社会美育项目。


主讲课程：

《中外美术史》《艺术思想史》《艺术史研究问题与方法》


教师寄语：

祝贺大家正式开启“大学自由创作模式”。从现在开始，你真正成为了自己人生的主笔。别佛系，不躺平，把每一天都铺成画布，静心描线、大胆铺色，偶尔卡顿没关系，但别忘了按时“交稿”。 期待你们在这里，画出绝不雷同、闪闪发光的四年。


美教252

 钱江妍

简介：

杭州师范大学美术学院讲师。主要从事版画与美术教育理论与实践研究。现为浙江省版画家协会会员、杭州市美术家协会会员，国际教育学会（Insea）会员，杭州余杭区特聘文化导师。

发表学术论文及作品多篇，主持及参与杭州市哲学社会科学研究课题、杭州师范大学智慧教学课程建设、多项教学改革等科研及教学项目等纵向课题。

指导学生参加国家级、省市级学科竞赛、教学设计大赛、创新创业大赛数十项目，获得一类学科竞赛省级奖项1项，各级专业领域赛事10余项。作品多次入选全国青年美展、全国版画展、省美术作品展，多次获奖，并被浙江美术馆、挪威奥斯陆艺术大学收藏。


主讲课程：

版画表现、丝网版画、绘画基础（素描）、毕业创作


教师寄语：

“艺术教育，是心灵的呼吸与思想的栖息之地。”在这里，你们将用色彩与线条书写青春，用创意与激情点亮未来。愿你们扎根课堂，勇于探索，在艺术的世界中自由生长，成为用美育滋养心灵的教师，让灵魂在教育中绽放光芒。


02

绘画

绘画251

   金成

简介：

1976年出生于浙江杭州

1999年毕业于厦门大学美术系 获学士学位

2007年毕业于中国美术学院油画系 获硕士学位

现任职于杭州师范大学美术学院绘画系

作品展览：

2001 《走进新时代——浙江油画大展》

2002 《纪念毛泽东在文艺座谈会上的讲话发表60周年——浙江省美术作品展》

《浙江省油画家协会会员作品——第二回展》


教师寄语：

手执画笔时，专注勾勒每一处细节；心怀善意时，认真做好每一件小事。做眼里有光、心中有爱的画者，让作品与人品，皆成风景。


绘画252

 林素梅

简介：

美术学硕士，副教授，硕士生导师，主要研究领域为中国山水画创作与理论。在《文艺研究》等专业领域期刊发表论文、作品20余篇（幅）；主持并完成了人文社科纵向、横向课题5项；作品曾参加“国际书画大赏展”“中国山水画艺术双年展 ”“浙江省第十二届美术作品展”等，作品被杭州市政府、美国波士顿儿童博物馆等机构收藏；主持省一流课程建设项目1项，课程思政示范教学团队负责人，省高等教育课堂教学改革项目研究等5项教研教改项目；曾获“杭州市优秀教师”“教学创新大赛优秀奖”“十佳班主任”等荣誉。


主讲课程：

中国画（山水）临摹 中国画（山水）技法 中国画（山水）创作 艺术实践；


教师寄语：

亲爱的同学们，欢迎来到美术学院，成为这个充满可能与创造的新集体中的一员。

      我已经做好准备，要认真履行班主任的职责：我将尽我所能陪伴大家这段崭新的旅程——不仅关心你们的学业成长，更在意你们内心的声音与生活的状态。我期待与我们班每一位同学一起，共同营造一个彼此支持、自由表达、健康向上的班集体。

山水画讲究“外师造化，中得心源”。大学阶段，你们既是学习者，也是自我成长的塑造者。愿你们在笔触勾勒间发现自我，在墨色渲染中释放才华，以专注锤炼技艺，以真诚涵养心灵。

而我，会在这里，作为你们的老师、同行者，当然也可以是朋友。愿这四年，你们既能深耕专业、锤炼技艺，也能保持真诚、学会关怀，最终成长为明亮而笃定的杭美人。同学们，准备好了吗？我们学院见。


绘画253

 韩京雷

简介：

河北人，杭州师范大学美术学院副教授，硕士生导师，绘画专业负责人、教研室主任，中国美术学院美术学博士，浙江省人物画研究会理事，浙江省美协、浙江非遗书画院、杭州市美协、北京工笔重彩画会会员。主要研究方向：中国人物画创作与理论研究、综合材料绘画创作、美术教育。


主讲课程：

现代水墨人物画


教师寄语：

杭美院就像家门口的小超市，琳琅满目要啥有啥；杭美院也像一家小工厂，一如既往按部就班；杭美院更像一座花园，猫猫狗狗莺莺燕燕。既然来都来了，那就好好享受吧......


03

书法学

书法251

 王友贵

简介：

1976年生于浙江淳安。中国美术学院文学硕士艺术学博士。现为杭州师范大学美术学院副教授，硕士研究生导师。草书曾获中国书法家协会第九届全国书法篆刻作品展获奖提名、第三届全国青年书法篆刻作品展优秀作品、巨幅草书《祭皇帝陵文》获中国美术学院林风眠创作金奖。论文《〈张猛龙碑〉的接受史研究》入选全国第八届书学讨论会


主讲课程：

书法篆刻技法


教师寄语：

行而不辍，未来可期

04

公共艺术


公艺251

 周筱馨

简介：

杭州师范大学美术学院公共艺术专业教师。被评为杭州师范大学社会实践优秀指导教师，杭州师范大学优秀班主任、杭州师范大学优秀党务工作者、本科生毕业论文（设计）优秀指导教师等。


主讲课程：

展示与陈列、材料构成、观念造型、色彩设计


教师寄语：

欢迎开启设计学习之旅！设计是你对世界的提问，对生活的感知，对人格的展现，愿大学四年绽放出属于你自己的风采。


公艺252

   陈钢


简介：

陈钢，文学硕士，杭州师范大学美术学院设计系公共艺术专业教师。兼任浙江省美术家协会壁画艺术委员会委员、中国雕塑学会会员、浙江省雕塑家协会理事、浙江省雕塑学会理事、杭州市摄影家协会会员。


主讲课程：

公共空间设计 公共景观设计 公共设施设计 公共雕塑设计 材料与工艺（雕塑）立体造型专题设计研究（装饰艺术方向研究生）


教师寄语：

愿你们以画笔为舟，以灵感为帆，在艺术海洋中勇敢探索。保持热爱，专注创作，思辨求索，用作品书写属于你们的时代印记。


公艺253

   郑炜

简介：

杭州师范大学美术学院环境设计专业教研室主任、教授、硕士生导师。主持完成浙江文化工程重点项目1项，浙江省哲社规划课题4项以及市厅级项目3项。在各级各类期刊、报刊发表论文、作品20余篇；主持横向课题50余项，参与和主持了省市地区100余项建筑设计和景观设计项目；曾获得“杭州市优秀教师”“杭州师范大学硕士毕业论文（作品）优秀指导教师”“杭州师范大学暑期三下乡优秀指导教师”等荣誉。


主讲课程：

《制图与识图》《建筑初步》《公园景观设计》《乡村景观设计》《展览设计》


教师寄语：

设计是个很有意思的专业，希望大家在大学期间不仅能学会做出好的设计，更能理解设计背后的逻辑和人文关怀——好的设计永远是从人的真实需求出发的。期待能和大家一起探索设计的无限可能！


公艺254

  陈冀峻

简介：
       教授/硕士生导师，建筑学硕士/设计艺术学博士。主要研究领域：中外室内设计史比较研究、公共建筑室内设计及其理论、老旧住宅适老化设计以及既有建筑改造利用。主持或参与4项省级以上纵向科研和教研项目。


主讲课程：

建筑设计基础、室内设计基础、商业策划与时尚空间创意、场地设计、设计调研与分析等。


教师寄语：

行万里路，读万卷书，才不会成为AI的奴隶；深入社会，畅游书海，才能在纷繁世界中拒绝被谎言裹挟。


05

视觉传达









视传251

 毛勇梅

简介：

柏林艺术大学美术学硕士。杭州师范大学美术学院 视觉传达专业副教授、学/专硕导；设计学传达与媒体设计方向责任人，装帧实验室负责人。浙江钱塘江文化研究会理事、浙江/杭州工艺美术学/协会会员。主持及参与省、市各级各类纵向课题多项；在《文艺研究》《新美术》《光明日报》等核心期刊及各类报刊发表论文作品20余篇；带领团队进行亚运会标志设计、浙江省运会、西博会等设计项目。个人获得全国美展、中国美协、中国编辑学会等各级各类设计展、奖项多次。获得杭州市教育局系统优秀教师、杭师大杭师大师德标兵、杭师大教学十佳、“巾帼文明示范岗”先进个人等荣誉。指导学生获得国家级、省级各类展赛30余项。


主讲课程：

《版式设计》《信息设计》《书籍设计》《中西方版式研究》《装帧艺术语言》


教师寄语：

大学是精神上独自富养的开始，愿同学们能和四年后的自己对话！


视传252

   陆洲

简介：

视觉传达专业教师，从事品牌设计与包装设计课程教学。在教学过程中，始终强调“理论与实践并重”，注重培养学生的综合设计能力与创新思维。多年来指导学生屡次在全国性重要设计竞赛中斩获佳绩，并长期投身行业实践，积极参与品牌构建、文化创意等项目开发，推动设计教育与产业需求深度融合。始终坚持“教学相长”的理念，尊重学生个性，注重启发引导，致力于以有温度、有实效的教学方式，帮助每一位学生夯实专业基础、拓展视野潜力，陪伴他们从入门到卓越，成长为社会真正需要的创造型人才。


教师寄语：

大学是开放的起点，而非人生的框限。尤其在这个充满AI与变革的时代，愿你们保持好奇、跨界学习，重视思维锻造与自我规划。设计离不开洞察与热情，人生的答案也不止于专业。希望你们在这里真正学会学习、感受创造，拥抱属于自己的成长节奏——快乐、自由、踏实而坚定。
