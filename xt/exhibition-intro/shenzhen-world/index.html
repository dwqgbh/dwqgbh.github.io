深圳國際會展中心

深圳國際會展中心
地址：深圳市寶安區福海街道展城路1號
電話：0755-32937926
網址：www.shenzhen-world.com/
深圳國際會展中心位於深圳市寶安區展城路1號，緊鄰深圳機場，地處粵港澳大灣區灣頂、珠三角中心和廣東自貿區中心，廣深港核心發展走廊和東西向發展走廊的交會處，廣佛肇、深莞惠和珠中江三大城市圈交會處，地鐵12,20號線、穗深城際鐵路直達，區位優勢突出，具有良好的資源集聚效應和龐大的發展潛力。

深圳機場
距深圳機場T3航站樓7公里，距深圳機場T4樞紐3公里，距香港機場僅75公里。

深圳機場碼頭
緊鄰深圳機場碼頭（機場福永碼頭），搭乘高速客輪可往返中山、珠海、澳門和香港機場，時間約為45-70分鐘。查詢船班時刻表及購票可關注微信公眾號 “深圳機場碼頭”。
① 碼頭可搭乘出租車直達展館（全程約15公里）。
② 從碼頭搭乘M590巴士專線到達深圳機場T3航站樓（地鐵11號線機場站 → 機場北站 → 轉乘地鐵20號線至國展站/國展北站）。

高速公路
接駁2條高速公路（沿江高速和廣深高速）和1條快速路（海濱大道）。以國際會展中心收費站為主體的廣深沿江高速二期國際會展中心互通立交項目已通車，該立交還將銜接深中通道、機荷高速對接線，以及寶安國際機場互通立交。
廣州：S3廣深沿江高速
東莞：京港澳高速 ←→ 廣龍高速 ←→ 廣深沿江高速
惠洲：甬莞高速 ←→ 莞佛高速
中山：廣澳高速 ←→ 莞佛高速 ←→ 廣深沿江高速
香港：屯門高速 ←→ 廣深沿江高速
澳門：廣澳高速 ←→ 廣深沿江高速
註：從廣深沿江高速至展館，車輛不需要申請臨時通行證；車輛在限制期內從市區進入展館時，需要申請臨時通行證。深圳外地車限行時間：工作日( 7:00-9:00 ； 17:30-19:30 )。信息需要持續關注相關政策發布為準。
辦理貨車証 / 非深圳號牌小車入深臨時通行証
申請官網:http://ga.sz.gov.cn/
● 貨車通行證辦理流程
● 非深號牌小車入深臨時通行許可辦理流程

 

高鐵交通
廣州出發： 廣州南站 → 深圳北站 → 換乘地鐵5號線到前海灣 → 再換乘地鐵11號線到機場北站 → 再換地鐵20號線至國展站/國展北站。
（廣州南站至深圳北站車程32分鐘，車費74.5元）
香港出發： 香港西九龍站 → 深圳(高鐵)福田站 → 換乘地鐵11號線到機場北站 → 再換地鐵20號線至國展站/國展北站。
（香港西九龍站至福田站車程14分鐘，車費68元）
深圳出發： 深圳站（羅湖）搭乘地鐵1號線（機場東方向）到車公廟 → 換乘地鐵11號線到機場北站 → 再換地鐵20號線至國展站/國展北站。
福田口岸： 深圳(高鐵)福田站搭乘地鐵11號線到機場北站→再轉乘地鐵20號線至國展站/國展北站。

 

地鐵交通
深圳地鐵11號線站點：福田站、車公廟站、紅樹灣南站、后海站、南山站、前海灣站、寶安站、碧海灣站、機場站、機場北站、福永站、橋頭站、塘尾站、馬安山站、沙井站、後亭站、松崗站、碧頭站。
深圳地鐵12及20號線直達展館
深圳地鐵12號線途徑站點：福永站、福海西站、國展站、國展北站
深圳地鐵20號線途徑站點：機場北站、國展南站、國展站、國展北站、會展城站
*可與深圳地鐵11號線、穗深城際鐵路轉乘到達展館。
深圳地鐵官網：按此

 

穗深城際鐵路
穗深城際鐵路全線16個站點：廣州東站、新塘南站、中堂站、望牛墩站、東莞西站、洪梅站、東莞港站、厚街站、虎門北站、虎門東站、長安西站、長安站、沙井西站、福海西站、深圳機場北站、深圳機場站。
深圳機場北站 → 轉乘地鐵20號線至國展站/國展北站。
廣州東站到深圳機場站最快僅需1小時11分鐘，從新塘南站到東莞西站最快僅需11分鐘，從新塘南站到深圳機場站最快僅需53分鐘，大大壓縮了往返穗、莞、深三地的出行時間。

智慧公交車 ( 會展地鐵接駁專線 )
從11號綫塘尾地鐵站到深圳國際會展中心（直達）
服務時間：08:30 – 17:00
停靠站點：塘尾地鐵站、深圳國際會展中心①、國際會展中心場站
從國際會展中心場站到11號綫塘尾地鐵站（直達）
服務時間：09:30 – 19:00
停靠站點：國際會展中心場站、會展南站①、塘尾地鐵站、塘尾地鐵場站

出租車
由展覽館到深圳灣口岸，約40分鐘車程，車費約RMB120元。

 

停車場

地面停車場
深圳國際會展中心停車區域包括地面停車場和地下停車場，提供約1.5萬多個收費車位。
出租車落客區：兩個登錄大廳東側和南入口均設出租車落客區
VIP停車區：兩個登錄大廳西側和18號展廳外設VIP停車區，約555個車位；
貨車臨時停車區：展廳（除18號展廳）附近均設貨車臨時停車區，約895個車位；
貨車輪候區：展館最北端設貨車輪候區，提供1500個貨車位；
