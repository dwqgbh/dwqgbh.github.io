iQ-F系列兼容CC-Link IE TSN的运动模块上市

2021/11/4 11:43:58

随着智能制造成为主流，对运动控制系统也提出了多维度更高需求。三菱电机MELSEC iQ-F系列作为强力支援客户“制造业先锋产品”，满足不同客户从单机设备控制到系统控制的各种需求。最新推出基于CC-Link IE TSN前沿通信技术的运动模块，实现高精度实时控制，使IT系统和信息通信并存，构建工厂IIoT数字化系统。

一、新一代运动控制产品——iQ-F运动模块

01简单设定实现高精度运动控制

iQ-F运动模块新品是可以进行定位、同步、凸轮、速度、转矩等先进运动控制的控制器，实现运算周期500μs，能经由CC-Link IE TSN连接高性能伺服放大器，对设备的小型化和高性能化做出更高贡献。

模块型号：

优异性能：

◇ 具备兼容功能，可使用FX5-nSSC-S的程序；

◇ 最小运算周期500μs(4轴直线插补)实现高速运动控制；

◇ 通过同一网络且使用1台控制器，可实现包含远程I/O和PP模式的伺服放大器的系统构建。

◇ 经由伺服放大器、远程I/O，可与外部输入信号实现高精度同步。

02具备便捷升级的继承性

沿用原有的iQ-F简易运动模块工程，减少程序开发周期，实现顺畅迁移。在运用既有程序的同时，还可使用MR-J5的功能。

03快速建立多种设备的运动控制系统

利用CC-Link IE TSN的特点，支持多种设备连接1台控制器的系统配置，能够连接伺服放大器、远程I/O、支持TCP/IP通信设备等，实现构建高度灵活的IIoT系统。

04支持更最大规模的运动控制系统

iQ-F运动模块可以控制最多32轴的CSP模式伺服放大器+最多64站的PP模式伺服放大器。

二、与MELSERVO-J5共创全新价值


iQ-F运动模块与MELSERVO-J5形成整体驱动解决方案，具备先进性和升级便利性。通过检测部件的振动和摩擦变化，预测部件寿命，实现预测性维护，避免故障停机时间并减少维护时间，提升设备的运转率并保证高生产率与高品质生产。同时实现轻松编程，能够连接高性能的伺服放大器，还可以为用户提供更高效的伺服电机方案。

三、与MELSERVO-JET共建适用方案

作为支持CC-Link IE TSN通信的高性价比网络伺服MELSERVO-JET，能够轻松构建4轴~256轴伺服系统，通信周期最快达125μs，提供多种整定方式，支持同一放大器控制不同容量电机。MELSERVO-JET与iQ-F运动模块搭配，构建更高灵活性、可连接性、可维护性的极高性价比运动控制方案，并将大幅度缩短编程及工程实施周期，为用户提供更适用的解决方案。

四、MELSEC iQ-F系列运动模块使用方法

为方便大家了解，以对应CC-Link IE TSN的运动模块FX5-nSSC-G与伺服放大器MR-J5-G组合为例，介绍从构建到运行的设定步骤。

第一步：系统配置架构设计

第二步：系统配置设置


启动GX Works3，并通过选择[工程]菜单中的[新建]来创建项目，设置设备配置图/添加设备、网络配置、PDO映射、伺服参数等，在使用MR-J5-G时进行相应的伺服参数设置值、通信周期设置和模块参数(网络)的应用等具体内容。

第三步：简单运动控制模块设置

在GX Works3导航中双击[简单运动控制模块设置(模块扩展参数)]，启动简单运动控制模块设置画面，进行轴和电子齿轮的设置。

注：具体操作方法、注意事项等请联系区域销售负责人或者当地代理商。

五、融合生产现场和IT系统的开放式整合网络——CC-Link IE TSN

CC-Link IE TSN采用时间同步技术，保证了实时性控制，并实现了IT系统和信息通信在同一网络中并存，是数字化生产时代的开放式整合网络，也为智能制造提供了先进的IIoT网络基础。

定义运动控制新标准

三菱电机将CC-Link IE TSN导入运动模块新品，显著提升运动控制的精度、速度，保证时间同步的同时也实现了IT系统和信息通信的网络统一，辅助建立高效、灵活、低成本的控制系统，助力数字化管理和运维的实现。更多新品详情，欢迎来电垂询！
